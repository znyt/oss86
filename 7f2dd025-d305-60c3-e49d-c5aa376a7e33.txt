　　告别了林劲四人。
<br/>　　萧逸直朝外门宝库而去。
<br/>　　毫无意外，易老正坐在桌案前看书。
<br/>　　萧逸刚刚出现，还未开口。
<br/>　　易老便已经率先发现了他，并放下了书籍。
<br/>　　“来了。”易老淡淡道。
<br/>　　“嗯。”萧逸点了点头。
<br/>　　仍旧是毫无意外，两人的开场白，永远是这样的单调。
<br/>　　“师…”萧逸想叫一声师傅。
<br/>　　易老却率先打断道，“称我易老即可。”
<br/>　　“我说过，我们之间毫无关系。”
<br/>　　易老的语气，有些强硬，甚至是冷漠。
<br/>　　“额。”萧逸呆愣在原地，一时说不出话来。
<br/>　　气氛顿时沉默住。
<br/>　　半晌，萧逸率先说道，“我也说过，你便是我毫无关系的师傅。”
<br/>　　易老神色有些许波动，淡淡道，“随你怎么说。”
<br/>　　“但，你只需称我为易老。”
<br/>　　说这句话时，易老铁着脸。
<br/>　　语气，也变得无比强硬。
<br/>　　周围的空气，陡然间变得冰冷至极。
<br/>　　他的眼睛，锐利得宛如要划破空气。
<br/>　　让萧逸一阵难受。
<br/>　　“易老。”萧逸咬了咬牙，说道。
<br/>　　‘易老’二字一出，周围的空气，瞬间恢复平淡。
<br/>　　易老也收敛了脸色，点了点头。
<br/>　　“嗯，今日来此，是有修炼上的疑惑，要我解答吗？”
<br/>　　“并没有。”萧逸摇摇头，“只是刚刚回来剑派，想来看看您。”
<br/>　　“不过现在看来，易老似乎并不希望小子来此。”
<br/>　　“确实不愿意，我更喜欢清净。”易老的语气，毫无情感色彩，无比平淡。
<br/>　　“那小子告辞便是。”萧逸脸色难看地拱了拱手，转身离去。
<br/>　　易老没有说话，没有挽留。
<br/>　　他似乎，早就预料到萧逸的反应，嘴角略微咧过一丝微笑。
<br/>　　但他却没有料到，萧逸会忽然在一瞬间再次转过身。
<br/>　　并且清晰看到了他脸上的微笑。
<br/>　　“呵。”萧逸会心一笑，道，“易老，你在笑什么？”
<br/>　　易老的微笑瞬间僵硬，脸部抽了抽，没想到精明的自己，会着了萧逸的套。
<br/>　　“你又笑什么？”易老铁青着脸，问道。
<br/>　　萧逸笑道，“我笑，易老明明很乐意见我，却非要一副不近人情的样子。”
<br/>　　“我笑，易老你身上有故事。”
<br/>　　“我想听。”
<br/>　　‘我想听’三个字，让易老的手，微微抖了抖。
<br/>　　“没门。”易老淡淡地回了一句。
<br/>　　而后揶揄道，“我还是第一次看到后辈拜见前辈，会两手空空来的。”
<br/>　　“额。”萧逸脸色一阵尴尬。
<br/>　　“废话少说。”易老从桌案前的椅子上站了起来。
<br/>　　走到萧逸面前，问道，“你离开剑派后，我百思不得其解，你为何无法破丹化脉。”
<br/>　　“便找长老们探讨了一下。”
<br/>　　“无论怎么想，都想不明白。”
<br/>　　“唯一的可能，只是你要凝聚的气泉异常庞大，所以那些丹药不足以让你突破。”
<br/>　　“可是，气泉的大小，只与气丹、真脉和武魂的强弱有关。”
<br/>　　“气丹和真脉，每个武者都差不多。就算强，也强不了多少；就算弱，也弱不了多少。”
<br/>　　“唯一的因素，只剩下武魂。”
<br/>　　“你不过是最低级的控火兽武魂，不可能拥有庞大的气泉。”
<br/>　　边说着，易老摇了摇头，道，“我做出了无数的推论，几乎将整个剑派的修炼书籍都看了个遍。”
<br/>　　“始终没有结果。”
<br/>　　“哦？”萧逸笑道，“如此说来，我岂不是该非常自豪？”
<br/>　　“易老和长老们无论如何都想不通的问题，我却自己解决了。”
<br/>　　“如今，我已突破洞玄境。”
<br/>　　“我知道。”易老白了他一眼，道，“你已突破洞玄境，如今更是洞玄五重。”
<br/>　　“你当我瞎？看不出你的修为？”
<br/>　　“我在问你怎么回事。”
<br/>　　“当然。”易老摆了摆手，道，“如果事关你的秘密，便不要说，我也没兴趣。”
<br/>　　任何武者，总有一些事关自己底牌的重要秘密。
<br/>　　所谓底牌，自然是只能自己知道，别人不能知道。
<br/>　　若泄露了，便不算底牌了。
<br/>　　易老更加希望萧逸将自己的底牌捂实了。
<br/>　　他宁愿不知道。
<br/>　　“并非什么秘密。”萧逸笑道，“只是，如易老你猜测的那般。”
<br/>　　“我的气泉，很是庞大，所以当初没能破丹化脉成功。”
<br/>　　“出外历练近10个月，才成功突破。”
<br/>　　“哦？”易老疑惑道，“你的气泉有多大？”
<br/>　　萧逸想起了易老曾经的话，传闻中，有些绝世妖孽，能达到气泉五百丈的地步。
<br/>　　而，哪怕是他游历遍整个炎武王国，也不曾见过。
<br/>　　“五百丈。”萧逸笑笑，毫不犹豫地说道。
<br/>　　“五百丈？”易老瞪大了眼睛。
<br/>　　“易老请看。”萧逸大手一挥，一个巨大的真气漩涡凝聚在高空中。
<br/>　　漩涡的大小，在一百多丈以上。
<br/>　　萧逸如今是洞玄五重修为。
<br/>　　若他的气泉是五百丈大小，那么里头的真气量，就是250丈。
<br/>　　算上普通武者顶多一次性释放一半左右的真气。
<br/>　　便是一百多丈。
<br/>　　“还真的是。”易老眨了眨眼睛，惊讶地说道。
<br/>　　萧逸并没有如实告知自己三千三百三十三丈的气泉。
<br/>　　现在不过是五百丈，易老便惊讶成这个样子。
<br/>　　若如实告知，也不知道会不会被易老看成是怪物。
<br/>　　“控火兽武魂，怎么可能有这么庞大的气泉？”易老皱眉道。
<br/>　　“小子之前没有告诉易老，我曾经吞服过一些淬炼真脉的丹药。”
<br/>　　“以前在凡境时，也因异变，凝聚了比较强的气丹。”
<br/>　　萧逸回答道。
<br/>　　“什么丹药那么厉害？”易老更加疑惑了。
<br/>　　世界上的丹方，无数之多。
<br/>　　就是品阶再高的炼药师，也不敢说知晓所有丹方，听过所有丹药。
<br/>　　“五…”萧逸刚要回到。
<br/>　　“慢。”易老打断道。
<br/>　　而后，锐利的目光，直视萧逸，道，“我有感觉，你小子在说谎。”
<br/>　　“而且，话里头，真中有假，假中有真，甚是高明。”
<br/>　　“额。”萧逸尴尬地摸了摸鼻子。
<br/>　　易老不愧是人精，一眼就看出了萧逸在说谎。
<br/>　　不是萧逸不想如实告知。
<br/>　　只是，有些事，确实不能说。
<br/>　　“不用再说了。”易老沉声道，“我知道，你再说下去，就要涉及你自己的秘密了。”
<br/>　　“我没兴趣，也不许你说。”
<br/>　　“记住，底牌之所以是底牌，就是连父母都不能说，这样才能成为救命的东西。”
<br/>　　“好了，这件事到此为止。”易老说道，“既然你已经凝丹化脉成功了，我也不必再烦忧。”
<br/>　　“洞玄境之后的修炼，你可都清楚了？”
<br/>　　易老将话题牵移到修炼上来。
<br/>　　“嗯，都清楚了。”萧逸点点头。
<br/>　　“洞玄境修炼，以填满气泉为主。”
<br/>　　“气泉是定数，不会再变。”
<br/>　　“但，小世界会变。”
<br/>　　“洞玄境，武者每突破一重修为，小世界增幅一百丈。”
<br/>　　“不错。”易老满意地点点头。
<br/>　　“别以为小世界没用。”
<br/>　　“那是决定你日后操控天地灵气，以及其他特殊手段强弱的因素。”
<br/>　　“你虽天赋过人，但千万不能懈怠，修为绝不能落下。”
<br/>　　“修为不断提高，你的小世界才会不断变大，手段愈加厉害。”
<br/>　　萧逸点点头，认真道，“小子懂的。”
<br/>　　“好了，若没有其它疑问，你可离去了。”易老脸上尽是满意的表情。
<br/>　　“是。”萧逸点点头，拱拱手，准备离去。
<br/>　　“慢。”易老忽然道，“就这么走了？你真的没有疑问了？”
<br/>　　“没有。”萧逸摇摇头，奇怪道，“易老觉得，我该有什么疑问吗？”
<br/>　　“你回裂天剑派，就没有听说过顾长空修为大增的事？不知道他和叶铭打成了平手？”易老问道。
<br/>　　“听说过，也知道。”萧逸耸耸肩。
<br/>　　“就不想知道原因？”易老问道。
<br/>　　不等萧逸回答，他已经率先说道，“极界碑。”
<br/>　　“极界碑？”萧逸原本满不在乎的脸色，瞬间一惊。
<br/>　　“难道，裂天剑派曾经出过地极境武者？”萧逸瞬间色变。
