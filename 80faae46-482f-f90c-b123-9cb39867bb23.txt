　　第33章
<br/>　　见小白狼如此郑重，程大雷也来了兴趣：“什么样的大买卖？”
<br/>　　“一个月后是幽州王杨龙停的六十寿诞，届时会有一批寿礼从青牛山下过，如果程当家感兴趣，我们两家可以联手将这寿礼吞下。”小白狼。
<br/>　　程大雷陷入沉思中，帝国十三州一百零八城，青牛山就位于帝国最北的幽州。幽州王杨龙停是帝国罕见几个异姓王，他是幽州的主人。
<br/>　　“那么……这批寿礼是来自薛半川？”
<br/>　　小白狼点点头：“薛半川占了黑石城，现在正想法设法平息本地势力，如果他能得到幽州王的支持，城主的位置就坐的更稳了。所以，这批寿礼的数目一定了不得，而从黑石城到幽州城，青牛山是必经之地。现在青牛山的山寨都在打这寿礼的主意，我特意来找程当家，不知程当家感不感兴趣？”
<br/>　　“既然薛半川如此看重这件事，肯定不容有失，绝对会派重兵沿途护送。凭我们两家吃得下嘛，白当家为何不找别人，单独来找我？”对于性命攸关的事情，程大雷还是把美色搁在一边，更看重自己的寿命。
<br/>　　“杏花岭有五十人，都可以配合程当家行动，我之所以找程当家，是看重程当家的五雷法！”
<br/>　　“呃……”
<br/>　　程大雷无语，还真有信的啊。他见过薛半川，也见过薛半川练出来的兵，那绝非乌合之众。财富他自然也有兴趣，可在这场群雄争肉的游戏中，蛤蟆寨并没有竞争资格。
<br/>　　“五雷法乃是神术，多造杀孽，会惹来天谴的。”
<br/>　　“那……程当家的意思，是不插手这件事了？”小白狼。
<br/>　　“暂且没有拿定主意，容我考虑几日。”
<br/>　　“男子汉大丈夫，当断则断，如此优柔寡断，却是让我错看你了。”小白狼。
<br/>　　程大雷呵呵一笑，没将小白狼的激将放在心里。
<br/>　　“白当家，不必心急，我另外有件买卖和你商量？”程大雷。
<br/>　　“什么？”
<br/>　　程大雷引着小白狼来到培育辣椒的暖房，他冲小白狼道：“我们山寨正在种辣椒，如果白当家感兴趣，可以种在杏花岭，这辣椒的产量要远远高于其他作物，等辣椒成熟时，我可以用一斤粮食换一斤辣椒，小麦，大豆，粟子都可以。白当家，这笔交易如何？”
<br/>　　小白狼看着暖房内绿油油的辣椒苗，眉头皱了起来，最后，她一摆手道：“程当家，你还是自己留着种吧，这件事程当家不愿意合作，自然有人愿意合作。”
<br/>　　说罢，小白狼便一甩手扬长而去，看着她离去的背影，程大雷轻叹一声：我也想冲冠一怒，谁来灭谁，可蛤蟆寨这点家底，经不起折腾啊。
<br/>　　得，自己还是安心种自己的辣椒吧，先发展农业也未必不行。不过，想起小白狼那鼓鼓的胸脯，程大雷还是怔了怔……都说胸大无脑，看来这句话果然有道理。
<br/>　　他晃悠在暖房，照顾着自己的辣椒，山寨发展的第一桶金，可就全靠它们了。
<br/>　　等程大雷忙完从暖房出来，就看到徐神机正和一个年轻人交谈，徐神机满脸不耐烦，而那年轻人虽然满面胀得通红，却还是陪着小心。
<br/>　　“什么事？”程大雷问了一声。
<br/>　　“程大当家！”那年轻人赶忙喊了一声：“我是朱家寨的。”
<br/>　　徐神机走过来，低声冲程大雷道：“朱家寨的朱小朋，想耕地没有粮种，想要管我们借一些。”
<br/>　　程大雷对朱家寨有一定了解，朱家寨只有三十几个人，当然，作为一个只有六个人的山寨，程大雷也没有资格嘲笑对方。不过这朱家寨是特别穷，山寨有老人有小孩，种地的人手都不够。今年是穷得连种子都吃完了，所以朱小朋才厚着脸皮来朱家寨借粮。
<br/>　　“大当家，多少借给我们点，等到收成了，一斗借一斗二还。”
<br/>　　程大雷眼珠一转：“你想种辣椒吗？”
<br/>　　“辣椒？”朱小朋第一次听到这个词。
<br/>　　程大雷又把对小白狼说的话，对朱小朋说了一遍。朱小朋听得半信半疑：“这是真的么？”
<br/>　　小白狼可以干脆的拒绝程大雷，朱小朋可没有这个底气，最后虽然还是不太相信程大雷，却还是拿了一些辣椒苗走了。
<br/>　　程大雷推销出第一笔，心情愉悦。只靠蛤蟆寨一家种，能种多少辣椒，自己这是学前世那些农业公司，将种植外包给农户。
<br/>　　过了几日，林少羽和黄三元从落叶城回来。将苏樱送回去后，他们这次回来又买了两只小猪，几十只鸡仔，还有一些山寨急需的布匹，盐，纸笔。
<br/>　　根据他们带回来的消息，据说这次寿礼的事情，连落叶城都知道了。许多青牛山之外的山贼，也已经在磨刀霍霍，假若青牛山没有人动手，他们绝不会放过这件事。
<br/>　　而这几天，程大雷已经感受到青牛山的气氛，所有人都没打算让这批寿礼离开青牛山。除了小白狼，另外还有几家小山寨来找程大雷商量合作的事情。
<br/>　　程大雷都客客气气的婉拒了，他不下手，不是不敢，而是没这个实力。不说别的，就是山寨现在养的鸡和猪，已经让蛤蟆寨的人手捉襟见肘。
<br/>　　所以这一次，程大雷不想参与这件事。他依旧种他的辣椒，养他的小猪，小鸡，最重要的，是刷抢劫任务。
<br/>　　程大雷，秦蛮，林少羽轮流值班，每天在山下收着，十几天过去，也不过将任务进度推到5|10 ，所抢的也不过是一些普通的小商小贩，无非是一些微不足道的东西，有时候甚至往对方身上摘一颗扣子，就算抢劫了。不仅如此，有时还得把对方平安送出青牛山，真让程大雷对无辜之人下手，他是做不到的。
<br/>　　就像林冲林教头上山之前，要纳一份投名状，假若事情发生在程大雷身上，这份投名状，他是绝对给不了的。
<br/>　　不知不觉，已是夏末，不过天气依旧炎热。这天下午，程大雷在山下值班，远远的，就看到两个人沿着山道过来。
<br/>
