　　这玉符之中记载的，正是一个丹方，这丹方的一部分，已经是被楚风眠抹除，这一部分需要的圣药，都是现在楚风眠所拥有的。
<br/>　　这玉符记载中的丹方，还是楚风眠当初是一本古籍之中所见到的，名为玉露破命丹，专门是用来冲击生死台阶的武者，所用的。
<br/>　　一枚玉露破命丹，便是足以让一名生死境巅峰的武者，轻而易举的超越生死，冲击到碎命之境。
<br/>　　只是这只是对寻常武者而言的，对楚风眠来说，以他自己的估计，炼制了这玉露破命丹，大约只能够是为他增添三到四成的突破机会。
<br/>　　不过这三到四成，对楚风眠来说也是弥足珍贵了，以楚风眠的资质，有着三到四成的机会，楚风眠便是有冲击碎命之境的希望。
<br/>　　这玉露破命丹的丹方，极为珍贵，现在整个九域，只怕是也就唯有楚风眠一个人才能拥有的。
<br/>　　就算是在九域之中号称炼丹为王的药王谷，都不一定会有着如此珍贵的丹方，这样完整的丹方自然不能给古家家主看。
<br/>　　不过这一半丹方，也足以是证明楚风眠的所言不虚了。
<br/>　　“炼丹而来？”
<br/>　　本来听到楚风眠的话，古家家主脸色还有些怀疑，心中半信半疑，毕竟是炼丹而来，也不用专门跑到他们宁州古城来。
<br/>　　可是当古家家主，看到这丹方的一刻，他的目光都被死死的吸引住了。
<br/>　　脸色时而变化这，就以他的阅历，似乎都微微震惊，这其中记载的内容，其中的种种圣药，甚至是以他的见识，都没有见到过。
<br/>　　古家，可不是武道家主，而是不折不扣的商业家主，身为宁州古城的城主，古家家主跟各大商会之间都是关系极好，阅历极广。
<br/>　　以他的阅历，看到这丹方的一刻，都是震惊无比。
<br/>　　“这，这是炼制圣品灵丹的丹方！”
<br/>　　古家家主脸色大变，震惊的开口道。
<br/>　　“如此之多的圣药，虽然只有一半，但是能够炼制出来的灵丹，绝对不是天级灵丹，而是圣品灵丹！”
<br/>　　“你，居然是要来炼制圣品灵丹！”
<br/>　　古家家主惊讶的目光看向楚风眠。
<br/>　　先不要说这圣品灵丹丹药的珍贵，而是这圣品灵丹，就算是让无数炼丹大师前来，只怕是给他们足够的材料，都不一定真正炼制的出来。
<br/>　　他还从未听说过楚风眠关于炼丹一道上还有造诣，没想到现在楚风眠居然是要炼制圣品灵丹。
<br/>　　“炼制圣品灵丹，并非是楚某之事，楚某只是负责来寻找丹方所需要的圣药的。”
<br/>　　楚风眠平静的开口道，他的话中意思倒是极为模糊，也没有说明，到底是不是他要炼制这圣品灵丹。
<br/>　　能够炼制圣品灵丹的炼药大师，在整个九域都是难寻，凤毛麟角一般，就算是在药王谷中，有能力炼制圣品灵丹的炼药大师，也不会超过三位。
<br/>　　楚风眠也不想引得这古家家主胡乱猜测，便是随意找了个理由蒙混了过去。
<br/>　　“此人，居然有着如此珍贵的丹方，莫非此人的背后，还有着一位炼药大师的存在？”
<br/>　　古家家主目光闪烁，似乎是在思考着。
<br/>　　楚风眠的名字，从九域之中也是突然冒出来的，在几日之前，古家家主都不知道九域之中还出现了这样一位人物。
<br/>　　楚风眠虽然之前已经是拿到过了北域百绝榜的第一位，但是北域百绝榜，从来都不会中域的强者注意到，毕竟北域可是荒凉之地。
<br/>　　在无数中域强者的心中，北域这种地方，几乎是无法诞生什么像样的天才的，也不会在意。
<br/>　　一个天才的崛起，单单凭借一个人的力量，肯定不足，楚风眠有着如此可怕的实力，莫非是楚风眠的背后还有着其他势力？
<br/>　　想到楚风眠居然是还敢跟七大宗门为敌，这更加是加剧了古家家主心中的猜测。
<br/>　　若是真的这样，一个前途无量的楚风眠，就算是跟七大宗门为敌，还真的值得拉拢一番。
<br/>　　“古家主，不知道你看完这丹方没？”
<br/>　　楚风眠出言开口道。
<br/>　　“楚某此番前来宁州古城，也只是为了寻找这丹方之中的圣药而来，至于价钱好说，不管是什么价格，只要是能够找到这上面的圣药，楚某都愿意出钱购买。”
<br/>　　“这丹方上面记载的圣药，我们古家所有的，只有一小部分，还有一部分，我们宁州古城之中的确是有，不过是在一些大商会的手中，楚公子愿意购买的话，老夫也可以去牵线搭桥，只要是楚公子能出合适的价格，我想也可以买到。”
<br/>　　古家家主放下丹方，摇了摇头道。
<br/>　　“不过还有一些圣药，就算是老夫都闻所未闻，前所未见，只怕是老夫都无能为力了。”
<br/>　　“无妨，这上面的圣药，只要是古家主能够帮楚某寻找一二，就已经是大恩了，这人情，将来楚某必报。”
<br/>　　楚风眠听到古家家主的话，心中也是极为满意。
<br/>　　看来这宁州不愧是中域中庞大的商业城市之一。
<br/>　　这记载炼制玉露破命丹所需要的圣药，其中大部分都是极为珍贵的，宁州古城中能够拥有一部分，已经是超出了楚风眠的意料了。
<br/>　　毕竟楚风眠也没有想过，可以一下子将这炼制玉露破命丹的圣药，全部收集齐。
<br/>　　圣品灵丹既然号称为圣品，自然不单单是想要炼制极难，炼制圣品灵丹的材料，都是极为珍贵的世间难寻，甚至是为了收集一些圣品灵丹的圣药。
<br/>　　一些炼药大师，花费数十年，数百年的都有，这结果已经是足以让楚风眠满意了。
<br/>　　“人情就不用了，楚公子既然救了小女，这点事，就算是我们古家报恩了。”
<br/>　　古家家主笑着开口道。
<br/>　　“楚公子稍等，老夫去去就回。”
<br/>　　古家家主的身躯，突然消失在这密室之中，似乎是去了其他地方。
<br/>　　楚风眠坐在椅子上，安静的休息这。
<br/>　　几分钟后，古家家主的身影，便是再次来到了这密室之中。
<br/>　　从古家家主的手中，如今还是突然多了数个玉盒。
