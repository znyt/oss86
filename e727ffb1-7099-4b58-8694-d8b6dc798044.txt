“这段时间在夏家是谁照顾爷爷的，我上一次给爷爷扎针的时候爷爷还没有中毒，这毒是最近才下的。”
<br/>
<br/>        “这段时间都是…妍妍照顾老爷子的，”说着夏振国就站起了身，震惊的看着夏夕绾，“你是怀疑妍妍给老爷子下毒的？”
<br/>
<br/>        夏夕绾冷笑了一声，“不是怀疑，而是笃定，爷爷是被李玉兰夏妍妍母女给谋害的！”
<br/>
<br/>        “不可能！”夏振国迅速否定，“李玉兰和妍妍不会给老爷子下毒的，你还说十多年前是她么母女推老爷子下楼的，这些我都不信。”
<br/>
<br/>        叶翎站在一边出声道，“这对母女还真是歹毒，她们将老爷子给谋害了，现在人证物证都没有了，她们只要抵死不认就可以了。”
<br/>
<br/>        夏夕绾拽了一下拳，夏妍妍应该看出爷爷有苏醒的迹象了，为了不让以前的事情败露，她们选择了斩草除根，直接给爷爷下毒。
<br/>
<br/>        只要爷爷死了，她们就可以高枕无忧了。
<br/>
<br/>        夏振国又是一个眼瞎的，被猪油蒙了心，根本就不信她们是凶手。
<br/>
<br/>        “我不相信这些事情，除非拿出证据，现在不要纠结这些了，老爷子死了，我们先办葬礼吧。”说完夏振国就转身去办丧事了。
<br/>
<br/>        这时身后传来了夏夕绾清冽的嗓音，“站住！”
<br/>
<br/>        夏振国回身，“又怎么了？”
<br/>
<br/>        夏夕绾站起了身，她哭到红肿的澄眸里溢出了几分凛冽的冷光，“爷爷的死讯暂时保密，秘不发丧。”
<br/>
<br/>        “什么？”夏振国一僵，“老爷子都死了，你不让他入土为安这是想要干什么？”
<br/>
<br/>        夏夕绾看着夏振国，“李玉兰夏妍妍母女以为将爷爷毒害了就可以高枕无忧，永守秘密了？这世上法网恢恢疏而不漏，我要将她们揪出来，让你亲眼看一看这些年盘在你身边的究竟是怎样的毒蛇！”
<br/>
<br/>        “你…你想怎么做，老爷子都死了。”
<br/>
<br/>        “明天就是你的生日了吧，我会过去给你庆祝生日，你只管开开心心的过，然后生日上透个口风给她们，就说爷爷苏醒了！”
<br/>
<br/>        夏振国震惊的看着夏夕绾，“你这简直是荒唐，你要我配合你演戏，我不干…”
<br/>
<br/>        他的话还没有说完，夏夕绾眸光倏冷，厉声呵斥住了他，“夏振国！”
<br/>
<br/>        夏振国从来没有见过这么凶的夏夕绾，女孩气场全开宛如掌控全场的女王，他腿一抖差点给跪了。
<br/>
<br/>        “夏振国，你是不是忘记了爷爷的遗训，你要听命于我，我现在不是跟你商量，而是通知你，这一场戏你必须拿出奥斯卡影帝的演技给我演好了！”
<br/>
<br/>        夏振国额头全是冷汗，“知…知道了，你别这么凶嘛，我就是影帝本人，保证给你演好了。”
<br/>
<br/>        “回去了，别让她们看出破绽，我会留在这里守着爷爷的！”夏夕绾手一挥，懒得再跟他废话了。
<br/>
<br/>        夏振国灰溜溜的就跑了。
<br/>
<br/>        ……
<br/>
<br/>        叶翎看着夏振国落荒而逃的身影，“绾绾，没想到夏振国不是你的父亲，现在你有什么打算？”
<br/>
<br/>        “翎翎，等处理完李玉兰夏妍妍母女，我要去帝都，我总觉得帝都像是有什么东西在召唤我。”
<br/>
<br/>        “绾绾，你去帝都，那陆先生呢，跟你一起？你们结婚还没有多久，不会要两地分居吧？”
<br/>
<br/>        提到陆寒霆，夏夕绾的眸里迅速覆上了晶莹的水雾，他已经不要她了。
