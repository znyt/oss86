　　高空之上，苏莫急速而行，他已经飞行了一年多。
<br/>　　这一年多的时间，包括大夏界域在内，他已经穿过了三个界域。
<br/>　　“这样下去不是办法！”
<br/>　　苏莫飞行之中，苏莫微微叹了口气，以他现在的速度，飞到终极剑域，估计要五六年以上。
<br/>　　这实在是太浪费时间了！
<br/>　　苏莫抬头看了一眼天穹之上的九个太阳，辨认了一番方向，便停了下来。
<br/>　　好像往西去，再穿过一个界域，就是时空界域。
<br/>　　上次殿主夏千渊曾言，时空界域可以让他在奥义之道上有所进展。
<br/>　　略一沉吟，苏莫便改变了方向，向时空界域而去。
<br/>　　他准备先去时空界域看看，然后，一边修炼，一边赶去终极剑域。
<br/>　　修为不能拉下，等到达终极剑域之前，他要成神。
<br/>　　随后，苏莫继续不停的赶路。
<br/>　　时间飞逝，一晃眼，便有又过去了三个多月。
<br/>　　苏莫的身形，停在了一片荒林之前。
<br/>　　望着眼前苍茫的荒林，他深深的吸了口气，只要穿过这片荒林，便是时空界域了。
<br/>　　经过这段时间的赶路，苏莫也在中途，了解了一番时空界域的情况。
<br/>　　据闻，时空界域乃是无数岁月以前，一位叫时空至尊的人物所建。
<br/>　　这位时空至尊，据闻拥有时空武魂，当年乃是能与轮回天尊、亘古魔祖等太古至强者争锋的人物。
<br/>　　只不过，不知道什么原因，时空至尊消失了。
<br/>　　千万年岁月了，时空至尊再也没有出现过，唯留下了时空界域的永恒神宫这个势力。
<br/>　　“时空武魂！”苏莫眸中闪过夺目的溢彩，这可是太古武魂榜排名第四的武魂，排名还在轮回之门之上。
<br/>　　据闻，时空界域有一处神奇的宝地。
<br/>　　这处宝地对于普通武者而言，充满了危险，但对于天才而言却是充满了机遇。
<br/>　　这处宝地名为时空之城，传闻乃是时空至尊当年的一件宝器所化，蕴含超强的时空法则。
<br/>　　此时空之城内，有扭转时间与空间的能力，甚至是时间倒流，改天换地。
<br/>　　总之，传言无比的神奇，让人神往。
<br/>　　时空之城，本该是时空至尊留给永恒神宫的宝物，不过，失去了时空至尊的永恒神宫，根本不能独占时空之城。
<br/>　　所以，时空之城，便成为了神域的开放之地。
<br/>　　深深的吸了口气，苏莫身形一展，继续向时空界域而去。
<br/>　　飞跃了荒林，飞跃了一座座雄伟的古城，苏莫来到了一座浩大无比的古城。
<br/>　　这座古城，占地足有千渊城数十倍，城高数百丈，巍峨壮阔。
<br/>　　不过，这并不是时空之城，而是一座名为‘极空’城的城池。
<br/>　　苏莫身形缓缓的降落了下来，落在了城门之前，大步进城。
<br/>　　时空之城，便在这座极空城之中。
<br/>　　极空城极为繁华，远不是千渊城可比，城内武者成群结队，人头涌动。
<br/>　　“这些年，时空之城陨落了不少人！”
<br/>　　“是啊，亘古魔国的魔烨，这么天才的人物，都没有从时门之中出来，还是空门安全。”
<br/>　　“空门虽然安全一些，但是，还是时门更加吸引人！”
<br/>　　走在极空城之中，苏莫的耳中，能听到各种各样的议论之声。
<br/>　　这些议论之声，多是与时空之城有关，而成中的武者，也多是来之神域各地。
<br/>　　“时门？空门？”苏莫心中沉吟，时空时空，那就是与时间和空间有关。
<br/>　　那么，所谓的时门和空门，便说明，时空之城内分为两部分。
<br/>　　“逆转时空，若是真的这么厉害，岂不是能救活芊寻月了？”苏莫稍稍有些振奋。
<br/>　　不过，时空之城具体是什么情况，他并不清楚。
<br/>　　苏莫快步而行，一步数丈，向极空城中央而去。
<br/>　　片刻之后，他来到了城中央。
<br/>　　这里，是一片开阔地带，方圆数里没有任何一物，只有平坦的大地。
<br/>　　大地之上，黑压压的一片，聚集了不下数万人。
<br/>　　有人在闲聊，有人在打坐，还有人在谈武论道。
<br/>　　只见，在高空之上，有两个虚幻的门户。
<br/>　　这两个门户，均是有虚幻的空间之力凝聚而成，高约三丈，宽约两丈，彼此相距百丈。
<br/>　　只不过，这两个门户，此刻却是大门紧闭。
<br/>　　苏莫来到了人群之中，感知力弥漫，静静的听着周围的谈论之声。
<br/>　　“还有六天！”
<br/>　　片刻之后，苏莫便清楚了大概的情况，这时空之城的时门和空门，均是九天自信开启一次。
<br/>　　而距离下一次开启，还有六天的时间。
<br/>　　随即，苏莫便找了一个空地，盘膝坐了下来。
<br/>　　既然还有六天的时间，那他只能再等六天了，到时候进入看看，一探究竟。
<br/>　　苏莫闭上了眼眸，静静的打坐，在离开千渊城之后，他再次做了隐藏。
<br/>　　比如，体内的混沌之力，分解出很大一部分，化为五行玄力，这让他的气息大大的改变。
<br/>　　其次，他的吞噬元神，隐藏的极深，以后不到万不得已，不会动用。
<br/>　　不过，安静打坐了半个时辰，苏莫有睁开了眼眸，转头看向不远处。
<br/>　　在他的视线之中，不远处，站立着四道年轻的身影。
<br/>　　这四人一男两女，看起来都是双十年华，很是年轻。
<br/>　　那名男子身穿黄袍，体型高大，面容刚毅。
<br/>　　而那两名女子，均是天姿国色，其中一女身穿绿色长裙，个子娇小，容颜秀美，眼睛极大，充满了灵气。
<br/>　　另外一名女子，气质和姿色更甚绿裙女子，其穿着一袭白裙，五官精致，肌白如雪，面色淡雅而华贵。
<br/>　　此时此刻，那名娇小的绿裙，眼眸中充满了喜色，正叽叽喳喳的和两位同伴说个不停。
<br/>　　苏莫之所以看向这三人，倒并不是因为两位女子美丽，而是他听到了三人的谈话。
<br/>　　这三人，是从轮回神朝而来，是轮回神朝的人。
<br/>　　苏莫打量了三人一番，这三人修为都不是很高。
<br/>　　除了那位气质华贵的白裙女子，貌似修为不止一品真神之外，另外两人均是一品真神。
<br/>　　苏莫面带沉吟之色，他在思考，自己要不要去认识一番。
<br/>　　帝一魂如今的修为，肯定已经非常可怕，对方肯定会在轮回神朝。
<br/>　　那么，从这三人的口中，或许能打听出帝一魂的情况。
<br/>　　比如，帝一魂如今的修为，再比如，帝一魂如今在轮回神朝的地位等等。
<br/>　　想到此处，苏莫长身而起，向那一男两女走了过去。
