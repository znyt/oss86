        她想留下这个孩子，却让大哥走上了一条不归之路。
<br/>
<br/>        那个时候，她不懂，她只知道肚子里的孩子是条活生生的小生命。
<br/>
<br/>        “我从来没想过，星儿是我的负担。我和你一样，都很爱这个小丫头。”傅夜沉搂住童筱颖，面色不再严肃，“我一点都不后悔自己的决定。她是我们一起想要留下来的小丫头。所以，筱颖，你们是我现在唯一的精神支柱。”
<br/>
<br/>        连华生当年劝过他，要他去劝童筱颖拿掉肚子里的女宝宝，但是他拒绝了。
<br/>
<br/>        另一边，粉色的公主房间里。
<br/>
<br/>        连星儿坐在露台上的摇椅上，一边吃着妈咪刚送过来的水果沙拉，一边给明司寒发短信。
<br/>
<br/>        我们分手吧！
<br/>
<br/>        打完这几个字，连星儿又删除了重新输入。
<br/>
<br/>        反正，她感觉自己说什么都不是。
<br/>
<br/>        她不能阻碍明司寒去追逐他的梦想，但是她自己本身又不喜欢异地恋。
<br/>
<br/>        连星儿想了许久，内心也挣扎了很久，才鼓足勇气给明司寒打了通电话。
<br/>
<br/>        “我的公主殿下，是不是想我了？”电话一接通，明司寒那好听的声音，从听筒里传出来。
<br/>
<br/>        连星儿顿了一下，才应道：“你在哪？我们出来见个面。”
<br/>
<br/>        “我在滨江城自己的家中，要不，我去接你？”明司寒微笑道。
<br/>
<br/>        连星儿直接拒绝了：“你直接去中心公园吧！那里有个儿童沙场。我们就在那儿见面。”
<br/>
<br/>        “好！”明司寒应道。
<br/>
<br/>        连星儿随后没再说什么了，直接将电话给挂了。
<br/>
<br/>        但她如约赶到中心公园的儿童沙场边时，那边的香樟树下，明司寒已经在石凳上坐下。
<br/>
<br/>        连星儿朝他走了过去，在他身边坐下后，看着沙场里那些正在堆沙堡的小朋友，心里很是感慨地说道：“小时候，就是在这里，遇见了那个让我一直念念不忘的小哥哥。”
<br/>
<br/>        “睹物思情？”明司寒的脸色瞬间不好看了。
<br/>
<br/>        好歹他这个正牌男友兼未婚夫在这里，她却来这里为了怀旧。
<br/>
<br/>        “明司寒，我们分手吧！”连星儿语气平淡地说出这句话。
<br/>
<br/>        明司寒身心一滞，刚刚就不大好看的脸色，此刻又变得更加惨白。
<br/>
<br/>        “我发现，自己只是被你帅气的外表所吸引，心里好像并不是真正的那么喜欢你。”连星儿微微偏头，看着明司寒，波澜不惊地说。
<br/>
<br/>        明司寒也侧过头来，惊怔地看着连星儿。
<br/>
<br/>        良久后，明司寒苦涩地问道：“这是你的真心话？”
<br/>
<br/>        “对，真心话。你给我的感觉，其实跟洛羽给我的感觉一样。因为你们不仅长的帅，对我又好，所以我才会喜欢。但这种，应该不是爱吧！”连星儿淡淡地回应。
<br/>
<br/>        明司寒却突然嗤笑了一声：“原来，这段时间，一直都是我在自作多情。”
<br/>
<br/>        “对不起啊！这么好的你，遇上我这种不懂爱的女孩子……”连星儿瘪了瘪嘴，声音也越来越弱。
<br/>
<br/>        明司寒突然从石凳上起身，居高临下地看着连星儿：“连星儿，我再次问你一遍，你到底喜不喜欢我？”
<br/>
<br/>        “喜欢啊！”连星儿仰头对上明司寒的目光，嘴角微扬，“但应该不是爱。”
<br/>
<br/>        她脸上始终是浅浅的笑意，没有一丝的忧伤与哀愁，就好像她是在跟他陈述一件无关紧要的事情一样。
<br/>
<br/>        明司寒心痛地单手握拳，指骨“咯吱”作响，眼神里有愤怒，也有悲恸。
<br/>
<br/>        连星儿也跟着起了身，面对着他，一脸抱歉地接着说道：“对不起。”
<br/>
<br/>        慢慢地，明司寒的手放松后，从裤兜里掏出手机，当着连星儿的面，删除了她所有的联系方式。
<br/>
<br/>        “连大小姐，我这个人，最不喜欢的是吃回头草。今天，你甩了我。从今以后，我会跟你断绝一切来往。我不是那种，会与前任藕断丝连，或者对前任恋恋不忘的男人。”明司寒删完连星儿所有的联系方式后，把话也一并给说完了。
<br/>
<br/>        他不再留恋地转身，走得干脆利落。
<br/>
<br/>        连星儿愣在原地，看着明司寒那决绝的背影，在风中渐行渐远。
<br/>
<br/>        到底什么是爱啊？
<br/>
<br/>        心有点痛了，这算不算是爱？
<br/>
<br/>        而这个时候，裴萱妍从香樟树后走了出来，一边啧啧嘴，一边来到连星儿的跟前，感叹道：“连大小姐，你甩男生的本事都是一流的啊！”
<br/>
<br/>        “要你管。”连星儿一脸不悦地皱起了眉头。
<br/>
<br/>        “哎——”裴萱妍不禁摇着头，感慨道，“我终于算是解放了！”
<br/>
<br/>        只要明司寒出国留学了，她就可以不用这么时时刻刻地跟在明司寒身边了。
<br/>
<br/>        连星儿此刻偏过头看着裴萱妍：“怎么哪儿都有你啊？”
