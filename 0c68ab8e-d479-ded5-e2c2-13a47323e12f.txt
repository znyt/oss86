　　“哈哈！不自量力！”
<br/>　　“把他们打下去！”
<br/>　　三名鬼族的妖孽，见苏莫等人上来，顿时不屑一笑，纷纷出手了。
<br/>　　三人均是大手探出，如同鬼爪一般的手掌，幻化出漫天漆黑的爪影，向众人疯狂攻击而来。
<br/>　　三位鬼族妖孽，均是二品高级的修为，又是妖孽人物，战力之强，无与伦比。
<br/>　　漫天爪影，裹挟无匹的威势，如同地狱的恶魔，向一众人王宫的天才攻击而来。

<br/>　　人王宫的天才，包括苏莫在内，共有十二人。
<br/>　　众人各施绝学，各种剑气飚射、掌印飞舞，疯狂的还击。
<br/>　　苏莫也出手了，轻描淡写的斩出了一剑，剑气如虹，和众人的攻击一起，迎向漫天爪影。
<br/>　　轰轰轰！！
<br/>　　爆响声惊天，双方的攻击碰撞在一起，不断的爆炸。
<br/>　　与此同时，其它几根天尊的手指之上，同样爆响连连，争夺战彻底的爆发了。
<br/>　　轰隆隆！
<br/>　　恐怖的爆炸，响起滔天气浪，三位鬼族的妖孽，实力果然非同小可，远胜于十余名人王宫的天才。

<br/>　　这五名天才，有三名是一品高级的修为，两名是二品低级的修为，实力实在太弱了。
<br/>　　而骆妤等人，虽然没有被击出去，但也不好受，个个面色有些苍白。
<br/>　　“你们还是自行退出吧！免得受到不必要的伤害！”
<br/>　　一名身材高大的鬼族妖孽，环视众人王宫天才，朗声喊道。
<br/>　　对于这群人王宫的弟子，他完全不在意，他在意的是，接下来和两位同门的争斗。
<br/>　　毕竟，一根手指之上，只有一个名额，他还需要击败两名同门。
<br/>　　骆妤、沁儿、褚彦等六名人王宫的天才，面现无奈之色。
<br/>　　她们本以为，能争一争，没想到现实居然如此残酷。
<br/>　　只是一轮攻击，便有一小半的人，被淘汰出局。
<br/>　　他们几人，根本不可能翻盘了。
<br/>　　这三位鬼王宫的妖孽，俱是名声响亮的天才，每一个都是战力超强之辈。
<br/>　　不过，她们也并没有多大的失望，因为本来就不报希望。
<br/>　　骆妤转头看了一眼其它几根天尊的手指，各大王宫的队伍，基本上是节节败退。
<br/>　　修为的差距，不是人多就能弥补的，唯有林无霜所在的天尊手指之上，呈现势均力敌之态。
<br/>　　那根天尊手指之上，林无霜和那名地王宫的妖孽，面对十八名魔王宫天才的围攻，稍稍有些吃力。
<br/>　　主要是，十八名魔王宫的妖孽之中，足有一半都是二品中级的修为。
<br/>　　还有就是，天王宫的队伍倒是颇为强大，面对一名魔王宫和两名兽王宫的二品高级的妖孽，居然能勉强抗衡。
<br/>　　主要是，天王宫的队伍之中有一名高瘦青年，修为虽然只有一品高级，但是战力非常强大。
<br/>　　此人，和一众天王宫弟子联手之下，面对三名二品高级的妖孽，居然短时间没有落入下风。
<br/>　　“你们怎么看？”骆妤转头，看向沁儿和褚彦等人，她没有立刻认输，要征得其他人的意见。
<br/>　　“算了吧，还是不争了！”沁儿摇了摇头，争下去也是自取其辱而已。
<br/>　　她才二品低级的修为，哪有实力去争！
<br/>　　“我同意！”褚彦点了点头，他从未想过，能得到前十的名额，这根本就不可能的。
<br/>　　就算他们侥幸打败了这三名鬼族妖孽，他也是要败给骆妤，不可能得到名额。
<br/>　　“我也同意！”
<br/>　　“没希望了！”
<br/>　　另外两名人王宫的天才，纷纷赞同沁儿的意见，他们也是不想做无谓的争斗了。
<br/>　　苏莫闻言无语，这几个家伙，也太怂了吧！
<br/>　　居然这样就认输了，让他颇为无奈。
<br/>　　“那就算了，我们下去吧！”骆妤没有询问苏莫的意见，招呼一声，便闪身飞了下去。
<br/>　　褚彦和另外两名人王宫的天才，跟着骆妤，齐齐飞了下去。
<br/>　　“段惊天，你估计没有机会表现了！”沁儿没有急着离开，转头看向苏莫，面带无奈的说道。
<br/>　　她知晓，苏莫是要靠这次比斗，表现自己的潜力，好让他父亲看重。
<br/>　　但是，现在的情况，却是没有机会了。
<br/>　　“有的是机会！”
<br/>　　苏莫轻轻一笑，手掌推了一把沁儿，道：“沁儿姑娘，你先下去吧！”
<br/>　　苏莫用的力道不小，沁儿猝不及防之下，顿时被苏莫推出了天尊手指。
<br/>　　“你这是？”沁儿心中一惊，身形飞落的同时，惊疑的看向苏莫。
<br/>　　她不知苏莫此举是何意，难道还要拼命一搏？有可能吗？
<br/>　　“他在干什么？”骆妤站在下方，看到这一幕，顿时眉头一皱。
<br/>　　这个段惊天，居然不愿意下来，这是还要战斗？
<br/>　　“哈哈，段惊天，你居然还不下来，小心被虐成狗！”褚彦见此，顿时就乐了。
<br/>　　这段惊天也太有意思了，修为低的可怜，居然还痴心妄想，想要获得名额？
<br/>　　所有人王宫的弟子，均是有些无语，这段惊天怎么想的？
<br/>　　“呵呵，小子，就你还想要争？”
<br/>　　“真是搞笑！”
<br/>　　三名鬼族的妖孽，见人王宫的人都下去了，唯有苏莫留了下来，顿时笑了起来。
<br/>　　“给你们一个机会，也自己下去吧！”
<br/>　　苏莫面色淡漠，手中长剑闪烁出森冷的寒光，接下来，就是他表现的时间了。
<br/>　　“你说什么？”
<br/>　　三位鬼族妖孽，闻听苏莫之言，顿时脸色一沉。
<br/>　　他们差点以为，自己听错了，这个人王宫的弟子，不过才一品高级的修为，居然要让他们认输？
<br/>　　啊？
<br/>　　骆妤、褚彦、沁儿等人王宫的人，听闻苏莫之言，均是面色一呆。

<br/>　　言罢，褚彦看向刚刚落在身边的沁儿，道：“沁儿，你看到了吗？这段惊天简直就是个奇葩，这也太丢人了！”

<br/>　　若是被鬼族妖孽轻松击败，到时候，不仅没有证明天赋，反而弄巧成拙了。
