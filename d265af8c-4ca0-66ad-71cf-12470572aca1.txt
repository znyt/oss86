　　朝阳出声，晨雾弥漫，道宗之内又逐渐热闹了起来。
<br/>　　苏莫来到了山腰处的广场上，白玉铺就的广场上，已经聚集了三、四十名弟子，个个都是武帝境九重的修为，气息浑厚，实力不凡。
<br/>　　苏莫的到来，引得广场上所有弟子频频侧目，这倒不是他外表多么不凡，而是他与众人不同。
<br/>　　聚集在此地的人，均是要去太阴族参加会武的人，只有武帝境九重修为的核心弟子才有资格。
<br/>　　而苏莫修为不符，和众人站在一起，倒显得是鹤立鸡群了。
<br/>　　“这家伙是谁啊？”
<br/>　　“他也是要去参加会武吗？”
<br/>　　“应该不是吧！武帝境八重去当看客吗？”
<br/>　　众人打量着苏莫，纷纷的议论了起来，他们均是不认识苏莫，所以有些疑惑。
<br/>　　苏莫扫视了众人一番，便在广场的边缘伫立，他没有熟人，也不认识任何人，并没有与这些人打招呼。
<br/>　　广场上的众人，也没有过多的关注苏莫，随便议论了几句之后，便各自和相熟的人闲聊了起来。
<br/>　　“这次会武，我们道宗压力巨大啊！”
<br/>　　“是啊！七大势力之中，我们道宗的实力本就垫底，门下天才比之其它六大势力，也是有些不如！”
<br/>　　“尤其是啸天族和闪灵族，太难对付了！”
<br/>　　“这次会武，我们道宗也就林皇、吕罡他们几个能与其它六大势力的妖孽天才争锋了！”
<br/>　　数十名参加会武的弟子，三三两两的围拢在一起，谈论这次会武之时。
<br/>　　苏莫耳中听着众人的议论，心中有些讶然，那个吕罡的实力，看来在道宗核心弟子之中，还是最顶尖的存在。
<br/>　　对于太阴族下辖的七大势力，苏莫也有所耳闻，这七大势力只有三个是普通的人族势力，分别是道宗、天虚宗、御剑谷，然后还有两大异族势力，分别是闪灵族和黑炎族，最后是两大妖族势力，分别是啸天族和雷云禽族。
<br/>　　这七大势力之中，轮总体实力而言，道宗的确是属于垫底的存在，其中最强大的莫过于啸天族和闪灵族。
<br/>　　其它几大势力，均是相差不多，即便稍弱一些，差距也并没有多大。
<br/>　　苏莫对于这几大势力的强弱，并不关心，几大势力之中都有那些妖孽天才，他也不关心，他想的只是要见到青璇。
<br/>　　时间流逝，日上三竿，阳光倾洒大地，广场上的人越聚越多，很快便超过了两百人。
<br/>　　苏莫心中颇为惊讶，道宗弟子的实力，比之天冥星巫族支脉，都要强大不少啊！
<br/>　　那天冥星巫族支脉，好像都没有如此多的武帝境九重的后辈天才。
<br/>　　踏！踏！踏！
<br/>　　就在此时，沉重的脚步声响起，身材高大，面容粗犷的吕罡来了。
<br/>　　“吕师兄！”
<br/>　　“吕师兄早！
<br/>　　“吕师兄气息浑厚异常，实力好像又有提升啊！”
<br/>　　不少人向吕罡抱拳，当然，众人也没有太过恭敬，毕竟大家都是武帝境九重的修为，吕罡也只是实力比他们强而已。
<br/>　　“嗯！”吕罡随意的点了点头，目光扫视一番，顿时神情一怔，因为他看到了一袭蓝衣的苏莫。
<br/>　　“呵呵！”
<br/>　　吕罡冷笑一声，他可没有忘记苏莫上次在宗务殿的狂妄，居然藐视他。
<br/>　　“苏莫，你来干什么？难道是去观看会武？”吕罡面带戏虐的问道，他不认为苏莫是去参加会武，因为苏莫的修为只是武帝境八重境界。
<br/>　　“你能去，我去不得？”苏莫面色淡淡的反问道。
<br/>　　“呵呵，你去涨涨见识也好！”吕罡冷笑道，他心中猜测，苏莫可能是认识这次带队的某位长老，所以才能跟着去观看会武。
<br/>　　苏莫面色淡淡，懒得理会吕罡，对方若是不挑衅他，他也不想惹事端，但若是对方不识抬举，那就别怪他不客气了。
<br/>　　“林师兄！”
<br/>　　“唐师兄！
<br/>　　“两位师兄早！”
<br/>　　就在此时，人群中再次响起一阵嘈杂的声音，只见又有两名青年联袂而来。
<br/>　　这两名青年，一名身穿明黄.色长衫，身材欣长，面容俊逸。
<br/>　　另一名青年，则是身形壮硕，虽没有吕罡那般高大，但气息丝毫不弱于吕罡。
<br/>　　这两人，那名身材欣长的青年，名为林皇，身形壮硕的青年，名为唐古。
<br/>　　两人均是道宗核心弟子之中的顶尖天才，实力不比吕罡弱。
<br/>　　林皇和唐古两人，亦是向众人打了声招呼。
<br/>　　旋即，林皇望向吕罡，沉声道：“吕罡，这一次七大势力会武，也是我们表现的机会，你可不要让人失望！”
<br/>　　“哼！这一次会武，我倒要看看其它六大势力的天才，有何逆天之处！”吕罡面上带着自信，显然对于自己的实力，有着很大的把握。
<br/>　　林皇点了点头，这一次会武，道宗也就只有他们三人能扛起大旗了。
<br/>　　众人继续等待，又过了小半个时辰之后，广场上的人数达到了两百六十余人，便再也没有人来了。
<br/>　　其实，道宗武帝境九重修为的弟子，并不是只有这些，只是有些人并不想去参加会武，或者是在闭关，亦或是在外历练未归。
<br/>　　片刻之后，长老们来了，只见从上峰的上方，十数名长老飞身而来，落在了广场上，个个气息滔天，如同大浪涛涛。
<br/>　　这十几名长老，有男有女，有中年人也有老者，为首的是一名头发半白的老者，身形消瘦，弓腰驼背。
<br/>　　“见过诸位长老！见过真枯半圣！”
<br/>　　众人见此，立刻纷纷抱拳行礼，苏莫亦是微微抱拳，原来这位驼背的老者，居然是一名半圣。
<br/>　　半圣，意味着半只脚已经踏入了武圣之境，堪称半步武圣，武圣境之下的最强者。
<br/>　　“嗯，大家免礼！”驼背老者，也就是真枯半圣挥了挥手。
<br/>　　“他居然也来了！”苏莫打量着十几名长老，微微有些意外，因为其中居然有品行极差的冯光。
<br/>　　冯光来到广场上，一眼便看到了苏莫，肥胖的脸庞上露出意外之色，难道苏莫真的突破到了武帝境九重境界。
<br/>　　不过，他稍一打量，便心中恍然，原来并没有突破，他看出了苏莫依旧是武帝境八重境界，虽然已达武帝境八重的巅峰，但比武帝境九重差距还是很大的。
<br/>　　那苏莫为何来此？
<br/>　　冯光心中有些疑惑，难道是找真贤道圣帮忙了？破例去参加会武，或者是去观战，增长见识？
<br/>　　这时，真枯半圣目光如电，扫视众人，朗声道：“这一次会武，背后的意义重大，希望你们拿出全部的实力，为道宗争光，也为你们自己争光！”
<br/>　　“是！”众人朗声应和，声浪震天。
<br/>　　“嗯！”
<br/>　　真枯半圣微微颌首，随即目光落在了吕罡、林皇和唐古的身上，沉声道：“你们三人，是本宗参加会武的弟子之中，实力最强的三人，不要让老朽失望！”
<br/>　　“真枯半圣放心，我绝不会让您老失望！”吕罡率先开口，目光中战意凛然，更留露出绝强的自信。
<br/>　　“我们不会让真枯长老失望的！”林皇和唐古亦是齐齐开口，面色坚毅。
<br/>　　“很好，老朽很期待你们的表现！”
<br/>　　真枯半圣见此，老脸上露出了一丝笑容，其实，在道宗的核心弟子之中，还有两人的实力，不弱于吕罡、林皇和唐古三人，但那两人外出历练已久，都没有回来。
<br/>　　所以，这一次的会武，重担全部落在了吕罡三人的身上。
