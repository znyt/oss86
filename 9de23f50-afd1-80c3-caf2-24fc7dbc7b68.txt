　　一颗小型星辰之上，高空之中，苏莫御剑而行。
<br/>　　这颗星辰也是生命星辰，荒山密布，偶尔能发现不少人类武者，只不过修为均是很低，基本上全部都是武皇境以下的武者。
<br/>　　远远的，苏莫再次发现了一名武者，闪身追了上去。
<br/>　　这是一名老者，花甲之年，穿着一身灰色长袍，修为达到了武皇境三重。
<br/>　　嗖！
<br/>　　瞬息之间，苏莫追上了老者，拦住了对方的去路。
<br/>　　“什么人？”
<br/>　　老者被苏莫拦住了去路，顿时停了下来，口中发出一声厉喝，满脸警惕的打量着苏莫。
<br/>　　因为他发现，他完全看不透苏莫的修为，只能感觉到眼前的青年气息深邃无比，修为深不可测。
<br/>　　“前辈，在下路过此星，有个问题特来询问一番！”苏莫向对方微微抱拳，客气至极，并没有恃强凌弱。
<br/>　　“什么问题？”老者疑惑的问道，同时心中惊疑，路过此星，难道是其它星辰的人？
<br/>　　“我想问前辈，此星上是否有星空传送大阵？”苏莫沉声问道。
<br/>　　“星空传送大阵？”
<br/>　　老者闻言双眉一挑，略一沉吟，点了点头道：“的确有！”
<br/>　　“哦！在哪里？还请前辈告知！”苏莫闻言一喜，急忙问道。
<br/>　　“星空传送大阵在玄羽宗内。”老者道。
<br/>　　“玄羽宗在何地？”苏莫问道。
<br/>　　“从此地向北飞行，大约十日的路程！”老者道。
<br/>　　“多谢前辈相告！”苏莫向对方抱拳道谢，便准备前往玄羽宗。
<br/>　　老者见此，急忙提醒道：“这位小友，那玄羽宗是这玄巍星上的霸主势力，他们的星空传送大阵并不给外人使用！”
<br/>　　“是吗？”苏莫闻言一笑，沉声道：“多谢前辈提醒！”
<br/>　　言罢，苏莫身形如电，划破长空，急速远去。
<br/>　　无论玄羽宗的星空传送大阵给不给外人用，他都要去，因为他必须乘星空传送大阵离开。
<br/>　　老者望着苏莫离去的背影，暗暗摇了摇头，想使用玄羽宗的星空传送大阵，可不是容易的事。
<br/>　　若不是玄羽宗的人，外人想要使用的话，要付出难以想象的巨额代价，根本不是一般的武者能承受的。
<br/>　　苏莫告别了老者之后，便一路向北疾驰，他的速度很快，老者说十日的路程，想必要不了这么久。
<br/>　　时间如白驹过隙，转瞬即逝，苏莫路过了一座座城池，一片片山脉，仅仅用了三天的时间，便来到了所谓的玄羽宗。
<br/>　　只见前方一座高达万丈的通天山峰上，建筑连绵，各种精美的宫殿、阁楼，数之不尽。
<br/>　　无数精美的建筑，流光溢彩，恢弘而又美丽，造就了一副如画般的美景。
<br/>　　“想必这就是玄羽宗吧！”苏莫暗道，因为他在前方的这座山峰上，感受到了不少强大的气息，其中甚至还有一位武尊境的强者。
<br/>　　虽然好像只是武尊境一重的修为，但对于这颗小型星辰来说，应该已经是至强者了。
<br/>　　他一路飞来，路过了不少地方，可是从来没有发现有武尊境的强者。
<br/>　　这颗星辰上的武者，修为普遍较低，连武皇境和武帝境的武者都极为稀少。
<br/>　　这个宗门有武尊境强者，还有不少的武帝境武者，在此星上应该是最强的势力，想必就是那玄羽宗。
<br/>　　嗖！
<br/>　　随即，苏莫身形一闪，划破长空，直接向玄羽宗的山门处飞了过去。
<br/>　　少倾之后，他身形降落而下，落在了山门之前的石质平台上。
<br/>　　山门前，屹立着巨大的白玉石门，高大十丈，宽约五丈有余，恢弘大气。
<br/>　　抬头看去，石门上‘玄羽宗’三个大字龙飞凤舞，铁画银钩。
<br/>　　此地，的确是玄羽宗。
<br/>　　苏莫并没有直接闯进去，既然要用别人的传送阵，自然不能失了礼数。
<br/>　　“什么人？”两声厉喝声突然响起。
<br/>　　只见山门之前，两名守山弟子看到突然从天而降的苏莫，顿时飞身拦在了苏莫的身前，满脸警惕之色。
<br/>　　“在下上官昊，想求见贵宗的宗主，劳烦两位通传一声！”苏莫抱拳说道，言语颇为客气。
<br/>　　使用星空传送大阵，这并不是小事，显然不是两名弟子可以做主的，所以他直接求见此宗的宗主。
<br/>　　至于上官昊之名，苏莫只是冒用而已，因为现在在巫族以及其他大势力的眼中，他已经死了。
<br/>　　所以，以后很长一段时间之内，苏莫之名都不能再用了，而且他要一直保持改变容貌的状态。
<br/>　　从今以后，他就叫上官昊了。
<br/>　　“想见宗主？”
<br/>　　两名守山弟子闻言一怔，随即仔细打量了一番苏莫，其中一名高大的弟子顿时冷笑道：“简直是笑话，我们宗主岂是你说见就能见的？”
<br/>　　这名守山弟子直接拒绝了苏莫，虽然他只有真玄境的修为，看不出苏莫是何修为境界，也不会为苏莫通传。
<br/>　　他们玄羽宗的宗主是什么人物？
<br/>　　乃是玄巍星第一强者，俯瞰天下，至高无上，身份何等尊贵，岂是什么阿猫阿狗都能见的？
<br/>　　上官昊？
<br/>　　这个名字可是从来没有听闻过，显然不是什么大人物！
<br/>　　一个无名之人，想要求见他们宗主，更是不可能了。
<br/>　　呃~~
<br/>　　苏莫闻听此言，顿时眉头一皱，这守山弟子居然如此嚣张，难道还要自己用强吗？
<br/>　　“好吧！告辞了！”苏莫略一沉吟，便点了点头，随即闪身离去。
<br/>　　两名守山弟子见此，并没有什么意外，继续驻守上门，这种要来求见宗主的人，他们见的多了，最后都是被他们轰走了。
<br/>　　他们宗主作为玄巍星第一强者，受天下武者敬仰，很多武者都想要得到他们宗主的指点，这很正常。
<br/>　　苏莫当然没有离开，飞出两名守山弟子的视线之后，他便动用了隐身战魂，随后飞了回来，大摇大摆的走进了玄羽宗之内。
<br/>　　他还要指望用玄羽宗的星空传送大阵，返回古周星呢！怎么可能就这样离开？
<br/>　　走进玄羽宗之后，苏莫便顺着山上的大道，直接向山峰上方走去，因为山峰上方的一处广场上，有大量玄羽宗的人，好像那位武尊强者也在其中。
<br/>(本章完)
