　　苏莫否定了夺舍紫袍青年的肉身，混进太阴族的想法，此法不可取。
<br/>　　旋即，他琢磨这是否能在寻找一名太阴族之人夺舍，但考虑了良久，觉得还是不太可行。
<br/>　　首先，太阴族太强大，远不是天冥星巫族一脉可比，靠夺舍肉身混进太阴族，有被识破身份的风险。
<br/>　　其次，他想要去追求青璇，肯定要和大量的妖孽天才竞争，一旦夺舍别人，实力将会大幅度下降，到时候怕是竞争不过别人。
<br/>　　毕竟，这终究是一个讲究实力的世界，若是实力不够，被人击败甚至杀死，将失去任何竞争的资格。
<br/>　　不过，他若是不夺舍太阴族之人，他又没有办法进入太阴族，这倒是一个两难的局面。
<br/>　　苏莫皱眉思索，查看搜魂得来的记忆，看看有什么好的办法！
<br/>　　“道宗！”
<br/>　　良久之后，苏莫眸中精光一闪，终于想到了一个办法。
<br/>　　真贤道圣是明虚星河‘道宗’之人，而根据紫袍青年的记忆，道宗就是太阴族下辖的七个超级势力之一。
<br/>　　这七个超级势力，归属于太阴族统领，七大势力的天才，经常都能出入太阴族，而且七大超级势力中有特别强大的天才，还能被吸纳进太阴族，成为太阴族的一份子。
<br/>　　“就去道宗吧！”苏莫低声自喃，这是目前最好的办法了，进入道宗，成为道宗弟子，日后就能进入太阴族。
<br/>　　而且，有真贤道圣之助，他也更好行事。
<br/>　　旋即，苏莫离开了山洞，向东方飞去。
<br/>　　七大超级势力，全部都在太阴星之上，道宗自然也不例外。
<br/>　　道宗的山门，坐落在问道山之上，就在东方，居此地大概五天的路程。
<br/>　　苏莫一路疾行，五日之后，他便来到了问道山，
<br/>　　问道山，是一座孤零零的巨啊山峰，高达万丈不止，直.插云霄，大小比之万界山都不差太多了。
<br/>　　当然，问道山只是一座普通的山峰，完全无法和神奇的万界山相提并论。
<br/>　　整个问道山，山腰以上的位置，全部隐匿的云层之上，仿佛欲与天比高。
<br/>　　云雾缭绕在山峰之上，显得缥缈而梦幻，如同人间仙山。
<br/>　　无数的精美建筑，从山峰下方连绵而上，通向云雾缭绕的九霄之上。
<br/>　　嗖！
<br/>　　苏莫速度不停，直接飞到了问道山下，落在了山门之前。
<br/>　　几名驻守山门的弟子，见到苏莫到来，凌厉的目光顿时落在了他的身上。
<br/>　　“在下苏莫，有事求见真贤道圣！”苏莫向几名守山弟子抱拳说道。
<br/>　　“你要见真贤长老？”
<br/>　　几名守山弟子闻言，均是眉头微挑，随即仔细的打量了苏莫一番，一名为首的弟子道：“真贤长老不在宗内，你请回吧！”
<br/>　　这几名弟子倒还算客气，没有狗眼看人低，直接轰走苏莫。
<br/>　　“不在宗内？”苏莫闻言皱眉，沉声问道：“那他什么时候回来？”
<br/>　　“我们怎么可能知道！”
<br/>　　那名弟子摇了摇头，继续道：“就算他在宗内，也不一定会见你，除非真贤长老认识你！”
<br/>　　苏莫闻言叹了口气，对方没有必要骗他，既然真贤道圣不在道宗，那他只能等对方回来了。
<br/>　　随即，苏莫走到一边，盘膝而坐，静心等待了起来。
<br/>　　几名驻守山门的弟子见此，均是暗暗摇了摇头，既然苏莫愿意等，只要不绕路宗门，他们也不会多管。
<br/>　　苏莫闭上了眼眸，境界的打坐，体内玄力运转，一个大周天又一个大周天，周而复始。
<br/>　　他突破到武帝境八重境界，已经过去了很久了，这段时间以来，一直没有安生的修炼。
<br/>　　境界的沉淀，还要一些时间，随着修为的不断提高，想要提升修为变得越发缓慢。
<br/>　　时间流逝，一日又一日，苏莫盘坐在道宗的山门之前，仿若磐石，亘古不变。
<br/>　　“这家伙是谁啊！好像在这里坐了半个月了！”
<br/>　　“谁知道呢！”
<br/>　　“估计是想加入我们道宗吧！”
<br/>　　道宗山门处，不时的有大量道宗弟子进进出出，不少人对苏莫评头论足。
<br/>　　苏莫不管不问，不理会任何人，只是静心的修炼，巩固修为根基。
<br/>　　又过去了约莫半个月左右，问道山的上空，虚空突然裂开，一声道袍的真贤道圣，从空间裂缝中走了出来。
<br/>　　“咦？”
<br/>　　刚刚走出空间裂缝，真贤道圣便发现了山门处盘膝而坐的苏莫，顿时轻咦一声。
<br/>　　“这小子，怎么来了？”真贤道圣疑惑，随即身形一动，瞬间消散不见，当再次出现之时，已经站在了苏莫的面前。
<br/>　　“见过真贤长老！”
<br/>　　“见过真贤长老！”
<br/>　　几名驻守山门的弟子，见到突然出现的真贤道圣，立刻躬身行礼。
<br/>　　真贤道圣可是道宗六大核心长老之一，武圣境强者，地位崇高无比。
<br/>　　“嗯！”
<br/>　　真贤道圣随意的挥了挥手，道：“免礼吧！”
<br/>　　这时，苏莫已经睁开了眼眸，看到身前的真贤道圣，顿时心中一喜，他立刻站了起来，抱拳道：“真贤师……！”
<br/>　　最后一个‘兄’字还没说完，真贤道圣便立刻打断了苏莫的话语：“苏莫，随本道进宗内一叙！”
<br/>　　“是！”苏莫点了点头，他知道对方可能不想让人知道彼此乃是同门的关系。
<br/>　　随即，真贤道圣道袍一摆，直接向问道山上方的山巅部位飞去。
<br/>　　苏莫见此，身形一展，立刻跟了上去。
<br/>　　这一次，有真贤道圣带路，没有人再敢阻拦他。
<br/>　　“这个苏莫是何需人也？”
<br/>　　“不清楚啊！不知道和真贤长老是何关系？”
<br/>　　“肯定是来投奔真贤长老的！”
<br/>　　几名守山弟子，望着苏莫离去的背影，低声议论，个个目中充满了羡慕之色。
<br/>　　能和真贤长老攀上关系，那以后前途无量。
<br/>　　苏莫跟着真贤道圣，飞到了问道山的上方部位，身形穿过飘荡的云层，落在了一座宫殿之前。
<br/>　　目光扫视四方，这问道山的上方部位，建筑极少，非常的宁静，周围没有半个人影。
<br/>　　身前的宫殿建造的极为雅致，附近白云飘飘，还有不少灵鸟飞舞，将宫殿衬托的仿若是天上宫阙。
<br/>　　随后，真贤道圣带着苏莫，走进了宫殿之中。
