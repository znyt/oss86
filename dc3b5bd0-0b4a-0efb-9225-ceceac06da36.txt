　　吴嘉伟静静地看着他装逼，等他说完了，说道：“你这么能耐，你怎么不去像那些穿越剧里演的那样，去改变历史啊。”
<br/>　　四宝道：“神经病啊，我改变历史做什么。”
<br/>　　吴嘉伟道：“高官厚禄什么的，还不是举手就来了。”
<br/>　　“没兴趣。我现在只想回去，我的小雯……”四宝眉头一皱，猛然想到了什么，表情顿时僵住了，过了好半天，他深吸了口气，说道：“大伟，你说……假如我真的去这么做，对后世会造成什么样的影响？”
<br/>　　吴嘉伟道：“那要看你做的事情大小吧，我不清楚。”
<br/>　　四宝道：“像我刚说的，如果我有机会告诉张司令，直系部队会绕到卢沟桥后面偷袭他，让他提前防备，甚至故布疑阵……前提是人家听了我的话，你说，结果会怎么样？”
<br/>　　吴嘉伟怔了半晌，道：“东北军有可能反败为胜？然后……”
<br/>　　“然后历史可能会被改写。”四宝嘴角抽搐了一下，感慨道：“卧槽，想到我有可能改变历史，这感觉真的是……说不上来啊。”
<br/>　　吴嘉伟想了一会，说道：“这只是最理想的结果，你一个普通人，根本就见不到军阀，哪怕机缘巧合，让你见到了，战争这么大的事，人家不可能听你一句话就改变战略计划。”
<br/>　　四宝点头说道：“我也就是这么幻想一下，就算真给我这样的机会，我也不会这么做，改变历史这种事虽然刺激，但想起来也太可怕了。”
<br/>　　吴嘉伟在草甸上坐下来，望着大殿里破旧的佛像，说道：“我只是法师，我对这些跟法术界无关的事情，一点兴趣都没有。”
<br/>　　“我也是法师，所以，我们只做自己擅长的就好，想要用我的计划寻找少阳，我们得有一定的身份地位。”
<br/>　　吴嘉伟抬头看着他：“要怎么做？”
<br/>　　四宝在他对面坐下，揉着自己的肚子，说道：“等时局太平了再说吧，现在乱世，咱们什么都做不了，先把肚子填饱了再说。那个啥，你再去煮几个红薯……”
<br/>　　吴嘉伟听到“红薯”两个字，面瘫的脸上出现了恶心的表情，嘴角抽搐了一下，“宝爷，咱俩能吃点别的吗？”
<br/>　　四宝揉着咕咕叫的肚子，怒道：“不吃红薯，你还想吃什么呀。肯德基麦当劳？黄焖鸡米饭？”
<br/>　　吴嘉伟的肚子也咕的叫了一声，擦了一把口水，吸着气说道：“你别放毒了，咱们还是吃红薯吧，我去煮……”
<br/>　　“今天换个口味，我们吃烤红薯。”
<br/>　　吴嘉伟已经站起来往外走，听见这句话又站住了，说道：“烤红薯不也吃过好几次了。”
<br/>　　四宝两眼一翻，“那你告诉我，什么口味的咱们没吃过？”
<br/>　　吴嘉伟语塞，竟然无言以对。
<br/>　　桃花山是杭州城外的一座小山，名字虽然好，但并不知名。山上有一座道观，恰逢乱世，就算是富庶的杭州城，也受到了波动，基本上没什么人上山烧香，桃花山的前厅也很破败，只有两个小道士在前山做接引童子，不过一天到晚也没几个香客，只是偶尔有来借宿的，两人奉命驱赶走。
<br/>　　两个童子在大厅里，一个拿鸡毛掸子，一个拿拂尘，在清理大殿里的神像，看上去百无聊赖，也只是给自己找点事干。
<br/>　　“师兄，你说，我们什么时候能像那些师兄一样，开始修炼法术啊。”一个年纪小点的抱怨起来。
<br/>　　“着急什么，那些师兄，不也都是我们这样子坚持过去的，总是有机会的。”
<br/>　　年纪小的道童看了师兄一眼，说道：“师兄你可真看得开。”
<br/>　　“看得开要等，看不开，也是要等，都是一样的，干嘛自己找不自在。”
<br/>　　小道童笑起来，说道：“师兄啊，你这觉悟，如果我是师父，一定让你入外院开始修道。”
<br/>　　两人一边干着活，一边说笑起来，小道童找来抹布，将香案擦拭了一遍，出去倒水的时候，见到一个人走上山来，忍不住看了一眼。
<br/>　　来的是一个穿黄色长衫的人，一头长发束在脑后，面色蜡黄，两只手拢在一起，顺着山道缓缓走上来。
<br/>　　小道童盯着来人的脸庞看去，感觉相貌极为普通，但却透着一种说不出的感觉……于是站在一边讷讷地等着，直到这人上了山。
<br/>　　小道童凑上去，说道：“施主……”
<br/>　　刚说了这两个字，来人抬起一只手，捏了一个七星诀。这是道门的手法，两人一见，知道是道门中人，只是不知道对方的辈分，但看捏了个子午诀，一人说道：“前辈何来？”
<br/>　　黄脸男子看也没看他们，走进了大殿。道观的大殿有前后之分，前殿一般供奉的都是自家宗门的祖师爷，后殿有三进，供奉各路神仙，最后一进是三清正殿，不用说，供奉的是道教三清。
<br/>　　黄脸男子走进前厅，从几尊神像前面一个个走过去，最终停留在最中间的一座神像前面，抬头看着那尊神像，嘴角露出一抹笑意。
<br/>　　两个道童相互看了一眼，都是一脸迷茫，其中年纪大的那个走了过去，行礼说道：“前辈，您这是……”
<br/>　　“把你们桃花山掌教叫来。”黄脸男子望着神像，目不斜视，细声慢语地说道。
<br/>　　两人一下子怔住了。
<br/>　　虽然对方语气平淡，但是不回答自己的问题，甚至看都不看自己，一开口就要找掌教，简直有点冒犯了。两人登时都有点火气，一个道童说道：“你有什么事要找我们掌教！”
<br/>　　“我有几句话要跟他说。”黄脸男子的语气仍然平静的很。
<br/>　　那道童一怒，哼了一声说道：“阁下想见我们掌教，至少得通个姓名吧？”
<br/>　　黄脸男子似乎想了一下，说道：“我叫李浩然。”
<br/>　　两人对视了一眼，皱起了眉头。
<br/>　　“恕我直言，我没有听说过阁下的大名，不知道阁下从哪里来，找我们掌教有什么事？我们才好去通报，不然我们掌教一向是不见外客的。”
<br/>　　“我要见他，他就得来。”李浩然轻声说道。
