　　之后的排名，也一一被宣布。
<br/>　　第四，第五，分别是另外的两个剑主。
<br/>　　第六、第七…一直到三十六名。
<br/>　　第一轮比赛，将剔除一半参赛者。
<br/>　　72位炼药天才，只有一半，也就是36位，能进入第二轮。
<br/>　　这时，木长老站了出来，高声道，“第一轮，控火之争，结束。”
<br/>　　“第二轮比赛，名为随机炼药。”
<br/>　　“地点在药楼之下。”
<br/>　　“前36名，随我来。”
<br/>　　“后36名，可随同各长辈一起观战。”
<br/>　　“亦可现在离开药王谷。”
<br/>　　说罢，木长老转身带路。
<br/>　　萧逸、戒墨等前36名者，跟随而去。
<br/>　　至于后36名，虽然心有不甘，但还是跟随前往。
<br/>　　虽无缘第二轮，但前往见识一番也是好的。
<br/>　　当然，他们来药王谷一趟，其实也算不虚此行。
<br/>　　在药田内得到的天材地宝，是一笔价值不菲的财富。
<br/>　　最重要的是，此番操控铁巨人。
<br/>　　锻炼控火能力。
<br/>　　此次之后，他们的控火能力，必会有所增长。
<br/>　　所以，这后36名，虽然脸色不甘，但很快便褪去了。
<br/>　　只有一人，脸色难看到极点。
<br/>　　正是万山剑主。
<br/>　　以他的本事，本来进入第二轮绰绰有余。
<br/>　　进入第三轮也是轻而易举。
<br/>　　此番，却止步第一轮。
<br/>　　日后回到万山郡，必会成为全郡笑柄。
<br/>　　这无疑让他极为愤怒，甚至是怨恨。
<br/>　　而他的怨恨对象，并非得到他所有天材地宝的叶铭。
<br/>　　而是易霄与戒墨。
<br/>　　“紫炎易霄，戒墨。”万山剑主心头愤怒。
<br/>　　“此番大仇，我若不报，誓不为人。”
<br/>　　“待我进了剑宗，修为有成。”
<br/>　　“日后再慢慢与你们算账。”
<br/>　　万山剑主心头想罢，咬了咬牙，还是跟上了队伍。
<br/>　　在他看来。
<br/>　　裂天剑宗，作为全炎武王国武道圣地。
<br/>　　他以剑主身份进入，必能得到重视和栽培。
<br/>　　而易霄不过是个猎妖师，戒墨也不过是郡王之子。
<br/>　　假以时日，他必能远远超越两人。
<br/>　　殊不知，他的想法，只是一个笑话。
<br/>　　……
<br/>　　第二轮比赛，在药楼之下。
<br/>　　第三轮比赛，才是药楼之内。
<br/>　　此刻，药楼下。
<br/>　　一众炼药天才，成名炼药师，等待着。
<br/>　　木长老，则吩咐手下执事，将一份份材料运到此处。
<br/>　　不多时，一切准备好。
<br/>　　木长老高声说道，“第二轮，随机炼药。”
<br/>　　“36位参赛者，请前来抽取丹方。”
<br/>　　“每人抽取三张。”
<br/>　　说罢。
<br/>　　前36名，纷纷前往抽取丹方。
<br/>　　药王谷执事，早已准备好抽签箱。
<br/>　　箱内，拥有三百张丹方。
<br/>　　品阶，从五品中级丹方，到六品高级。
<br/>　　另有10张特殊丹方，位列六品巅峰。
<br/>　　每人抽取三张丹方。
<br/>　　这一轮的胜败。
<br/>　　取决于三张丹方的成功率。
<br/>　　若成功率一样，则以耗费药材的多少计算。
<br/>　　若耗费的材料也一样。
<br/>　　最终则以丹药的成色来决出排名。
<br/>　　劣质、合格、良好、完美，是丹药的四种成色。
<br/>　　不过，这一轮多少讲些运气。
<br/>　　若是三张都抽到五品丹方，炼制起来容易。
<br/>　　若抽到六品丹药，炼制起来麻烦。
<br/>　　若抽到特殊丹方，几乎没办法炼制。
<br/>　　……
<br/>　　当然，这一轮主要考验的。
<br/>　　是炼药师的丹方涉猎程度，以及炼药知识的深浅。
<br/>　　箱内的丹方，有许多都是偏门丹药。
<br/>　　起码，对于普通的年轻炼药天才来说。
<br/>　　这些偏门丹药，大多没有听说过。
<br/>　　若是炼药师的丹方涉猎程度大，听说过这个丹方。
<br/>　　甚至是亲身炼制过，那自然是好的。
<br/>　　若没有听过、炼制过，那只能自己慢慢研究丹方。
<br/>　　这时，便考验炼药知识的深浅了。
<br/>　　炼药知识越深，研究丹方自然也越简单。
<br/>　　这一轮比赛，实力、天赋、运气，缺一不可。
<br/>　　……
<br/>　　待得36人抽取丹方完毕。
<br/>　　每人的三张丹方。
<br/>　　会被执事悬挂于旁边的一块巨大木牌上。
<br/>　　用于记录。
<br/>　　萧逸看了眼自己的三张丹方。
<br/>　　顿时眉头紧皱。
<br/>　　就连另一边观赛的炼药师，也是倒吸一口凉气。
<br/>　　“那紫炎易霄，运气够背的。”
<br/>　　其中一个中年炼药师说道。
<br/>　　“第一张丹方，六品初级，地元丹。”
<br/>　　“这个还算好。”
<br/>　　“听闻他在自己郡参加炼药决赛时，就曾经成功炼制过。”
<br/>　　“第二张，六品高级，无杂质地元丹。”
<br/>　　“他有麻烦了。”
<br/>　　任何品阶的基础丹药，若成为无杂质丹药。
<br/>　　都会在该品阶，上升为高级丹药。
<br/>　　例如，洞玄丹，只是四品初阶。
<br/>　　但，无杂质洞玄丹，就会成为四品高级。
<br/>　　周围一些颇有名气的炼药师，开始摇头。
<br/>　　“六品的无杂质丹药，太难炼制了。”
<br/>　　“五行属性，排列得异常复杂。”
<br/>　　“而且，每次炼制，若妖兽内丹的属性不同，还要每次重新专研。”
<br/>　　“还要考虑各种不同的变化。”
<br/>　　“无杂质地元丹，起码要六品高级以上的炼药师。”
<br/>　　“才有一定把握炼制成功。”
<br/>　　“别说这些年轻人。”
<br/>　　“就是我们这些成名已久的炼药师。”
<br/>　　“恐怕在场的，也就只有包括莫老前辈在内的寥寥几人有把握炼制成功了。”
<br/>　　众人的目光移动。
<br/>　　当看到第三张丹方时。
<br/>　　看向萧逸的目光，已经变了。
<br/>　　似乎带着一些可惜和遗憾。
<br/>　　“第三张丹方，天意丹。”
<br/>　　莫老前辈的脸色，立刻变得肃然。
<br/>　　而后，摇了摇头。
<br/>　　“以紫炎的炼药术，本应进入第三轮。”
<br/>　　“但现在看来，他的运气太差了。”
<br/>　　“如无意外，他会在这里止步。”
<br/>　　天意丹，正是那10张特殊丹方之一。
<br/>　　位列六品巅峰。
<br/>　　最重要的是，这种丹药，极其稀罕，又效果极大。
<br/>　　天意丹，顾名思义。
<br/>　　武者服之，可上感天道，揣摩天意。
<br/>　　半天之内，进入绝对的天人合一状态。
<br/>　　对于武者突破瓶颈，大有裨益。
<br/>　　最恐怖的是，这种丹药的效果，能够叠加。
<br/>　　比如，服用10粒天意丹，效果便会增幅10倍。
<br/>　　传闻，许多年前，炎武王国曾有一位大势力的继承人。
<br/>　　武道修为达到地元三重时，遇到瓶颈，连续10年不得突破。
<br/>　　但，在吞服了30粒天意丹后。
<br/>　　竟瓶颈瞬间破除。
<br/>　　而且，地元九重以下，再无瓶颈。
<br/>　　之后，这位继承人吞服下大量无杂质丹药。
<br/>　　在毫无瓶颈下，一个星期内，连跳六重修为。
<br/>　　直达地元九重。
<br/>　　那一次，天意丹的恐怖效果，让得世人惊叹。
<br/>　　当然了，这种丹药，需要的数十种天材地宝，皆为珍稀行列。
<br/>　　加之，他的炼制难度极大。
<br/>　　就连莫老前辈，这个被誉为七品之下的第一炼药师。
<br/>　　也毫无把握炼制。
<br/>　　只有七品炼药师，而且还是手段高深那种。
<br/>　　才有可能炼制出来。
<br/>　　也难怪，现在所有人都对紫炎易霄失去了信心。
<br/>　　这时，药王谷执事，送来一份份材料到萧逸面前。
<br/>　　正常丹方，每张给予五份材料。
<br/>　　若全部失败，这张丹方上的丹药，将再无机会炼制。
<br/>　　视为彻底失败。
<br/>　　特殊丹方，天意丹，只给了两份材料。
<br/>　　实在是，天意丹的材料，太过珍贵。
<br/>　　药王谷也不会给予太多。
<br/>　　也就是说，别人的材料，都有15份。
<br/>　　萧逸的材料，则只有12份。
<br/>　　萧逸眉头紧皱，心头暗道，“地元丹和无杂质地元丹都没问题。“
<br/>　　“倒是这天意丹，有些麻烦。”
<br/>　　忽然，观赛者，再次一声惊呼。
<br/>　　“怎么回事？”
<br/>　　“戒墨，还有那两位剑主的第三张丹方，竟然都抽到了天意丹。”
<br/>　　“难道，上一轮的四强，就因为运气差，要止步于第二轮？”
<br/>　　莫老前辈等人，纷纷说着。
<br/>　　一旁的万山剑主，心头冷笑一声。
<br/>　　“易霄，戒墨，看来不用我对付你们，你们率先就栽在这里了。”
<br/>　　孰料，木长老忽然站了出来。
<br/>　　高声道，“抽到五品丹方者，限时半天内炼制出来。”
<br/>　　“抽到六品高级丹方者，限时两天，自行研悟。”
<br/>　　“抽到特殊丹方天意丹者。”
<br/>　　“将由我药王谷派出四位长老，悉心指导三天。”
<br/>　　“三天内，可随意请教，我们知无不言。”
<br/>　　“三天内，可到我药王谷宝库。”
<br/>　　“随意查阅长老权限以内的药王谷炼药典籍、心得。”
<br/>　　此话一出，在场所有人尽皆色变。
<br/>　　看向萧逸、戒墨和两位剑主的目光，顷刻间变得羡慕无比。
