　　岁月悠悠，转瞬万年即逝。
<br/>　　洪荒还是大战不发，小战不止，天地之间的劫煞之气却是越来越重了，仿佛在酝酿这一场前所未有的大战。
<br/>　　永恒岛上，玉虚宫中，元始却是从悟道之中醒了过来。万年闭关，对于他来说，只是眨眼间的时间，修为并没有什么明显的进步，他此次闭关只是消化了三清论道所得，老子的无为阴阳之道，通天的截天剑道、阵道都有独特之处。
<br/>　　毕竟每个人的道都有不同，就算同一种道每人的领悟也会有所不同，没人敢说自己全部都会。
<br/>　　最重要的是元始所走元始大道注定要取百家所长，不能闭目造车，所以，与道友论道，注定是无法避免的。
<br/>　　此次闭关还有一个目的，就是增强无极印。无极印初创，还没有完善，元始此次闭关就是为了完善无极印。结果并不是十分理想，无极印虽然完善了，但还是初级阶段，并不能发挥出巨大威力，比不上开天印、灭世印、诛神印等道印。
<br/>　　不过，元会并没有气馁，自己以后有的是机会继续完善它，它既然都已经被创造出来了，那么开始都有了，还怕过程的艰难吗？
<br/>　　同时，元始也发现了自己的肉体真身还是不够强大，《玄黄不灭》一直停在第三重，是时候进入第四重了，不知道敖玉完没有完成他布置的任务。要是完成了，那更好，要是没有，那只好自己动手了。不过，不到最后一步，他还是不想自己动手，因为这是在量劫之中，一不小心，就会被扯入量劫之中。
<br/>　　要是牵涉入量劫之中，要么就是入劫，等量劫结束，要么就是死在量劫之中，除此之外，没有其他可能。更别想躲过量劫，能躲过，它就不叫量劫了。
<br/>　　“嗯？”走出玉虚宫，元始发现自己院中多了几个小孩。
<br/>　　小孩，不错，就是小孩。
<br/>　　院前多了四个小孩，两男两女，看上去只有十一二岁左右，两个男的，一个身白衣，脸色苍白，眼神清纯。另一个一身黑衣，脸色坚毅，不苟言笑，眼神之中充满坚定之色。
<br/>　　两个女孩，一个身穿墨绿色衣裙，满头墨绿色长发，一双墨绿色眼瞳，脸色红润，不言不语。另一个女孩身穿白色衣裙，头戴白色莲花，肌肤雪白，小脸洋溢着雪莲般的笑容。
<br/>　　让元始想到一句话，最是那一低头的温柔，像一朵水莲花不胜凉风的娇羞。
<br/>　　“见过老师！”四个小孩见到元始出来，顿时走过来向他躬身行礼。
<br/>　　“嗯？老师？”元始错愕，老师，自己何时收过徒弟了，怎么他自己不知道。
<br/>　　不过，心念一动，元始顿时就知道这是怎么回事儿了。这让元始哭笑不得。
<br/>　　要说这四个孩子的来历，还真与元始有关。
<br/>　　万年之前，元始三人讲道，并没有控制各自的声音，所以，讲道之时的大道之音传遍整个永恒岛，永恒岛上的生灵都尽皆收益，有的得以开启灵智，有的得到修炼之法，而有的却是修为上涨，很快化形。特别是靠近玉虚宫的几个生灵。
<br/>　　而眼前这四个孩子，就是靠近玉虚宫的几个生灵，他们得到机缘，并以此化形而出。两个男孩，白衣男孩是玉虚宫上的白鹤化形，黑衣男孩是宫前的玄松化形。而两个女孩中，墨绿色衣裙的为宫殿之前墨竹所化，白色衣裙的女孩为神泉池塘中一朵白莲所化。
<br/>　　看着眼前这四个孩童，元始顿时不知如何是好，他们的资质只是上等，不算太好，也不算太差，收了他们，是不是有些对不起我元始天尊的名号啊。
<br/>　　但，要是不收他们，那又要如何？放他们自由，自己毫不理会？那样不太好吧。
<br/>　　思索良久，元始下定决心，道：
<br/>　　“罢了，罢了，既然你们与本座有缘，那本座就收你们做本座的记名弟子。以后你们就住在这玉虚宫了。”
<br/>　　这一刻，元始想通了，去你妈的资质，资质一般又怎么了，难道我元始天尊还会在意这区区资质问题，我可不是另一个洪荒之中的元始天尊啊。就算资质不好又如何？我一样能够让他修炼。
<br/>　　“多谢老师收留！”
<br/>　　四人连声感谢道，小脸之上充满喜悦之色，就好像离家的孩子一下子找到了自己的家。
<br/>　　元始看到四个孩子脸上的纯真笑容，连带着自己都仿佛被感染了，脸上不由露出一丝微笑。自己好久没有见到过这样的笑容，久到自己都快忘了自己是从哪里来的。
<br/>　　这样的笑容，是自己前世的前世才能看到过，自从自己来到混沌，做了因果魔神，来到洪荒，成了元始天尊，一天到晚不是修炼，就是算计，这样的日子，一直持续了不知多少元会，久到自己都懒得去算了。
<br/>　　自己都快忘了自己骨子里还是一个人呐。
<br/>　　是啊，既然是人，就会有七情六欲。自己走的又不是太上忘情一道，人的情感，不是逃避，忘情就可以避免的，正所谓压抑越久，爆发的就越厉害。
<br/>　　自己走元始一道，从来不忌讳自己的感情，整天板着一个脸，都忘记了笑容是什么了。看来，在洪荒呆久了，脑子都出现了问题。
<br/>　　我走的是大逍遥，大自在之道！
<br/>　　我要的是不受别人约束，自己的命运只能掌控自己！
<br/>　　我要的是踏天地之上，超大道之前，永恒不朽！
<br/>　　而不是现在这样，整天除了修炼，还是修炼，都快不知道人生的乐趣了。
<br/>　　所以，从现在起，我要做真正的我，而不是一个只会修炼的我。
<br/>　　元始心中闪过这些念头，脑海不由一轻，好像挣脱了什么束缚，心境更上一层，就连元神都增加不少，《神元九转》第五重飞速增加着，转眼间就到了第五重后期。
<br/>　　“哈哈！
<br/>　　我有道心一颗，久被尘牢封锁，
<br/>　　今日枷锁尽去，照破山河万朵！”
<br/>　　人身本没有枷锁，是我们自己偏要给自己以各种这样那样的理由套上无尽枷锁！何日能真正认清自我，才能断去这些枷锁！
<br/>　　修为增加，心境增强，更重要的是自己开始正视自己，内心舒畅，元始再来看眼前四个孩童时，顿时觉得他们还是非常不错的。
<br/>　　他决定好好奖励这四个童子一番。
<br/>(本章完)
