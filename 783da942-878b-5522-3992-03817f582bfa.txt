<br/>　　陈少爷的神之狙击能力，至始至终都没有任何扩展，从一阶到十阶的区别只是射击命中的范围不同而已。
<br/>　　其余的，则就是在射击范围内的绝对命中！
<br/>　　这看似单一的能力，却在专攻其一的这项上形成了强大的后期支援，陈少爷可以只凭借自身就做到其余狙击手都做不到的事，甚至可以与高科技媲美。

<br/>　　它森白空洞的瞳孔露出了一抹惊愕，但在中弹的这一霎那除了向前倾倒，丧尸头领没有其他反应，这出乎意料的一枪打乱了它的节奏，也让它迟疑了片刻。
<br/>　　此时已经从前方二十米处回身奔跑而来的楚涵，则是枪声响起的一瞬间就前进了五米，与那丧尸头领的距离只剩下十五米。
<br/>　　这不是一个良好的进攻距离，哪怕中弹的丧尸头领，也完全有机会和时间躲避亦或是再次反攻。
<br/>　　这只丧尸头领的等阶在与楚涵对招的时候已经显示而出，超过了所有狼牙战团战斗过的丧尸等阶，这是一只九阶的超级大丧尸！
<br/>　　很长一段时间内他们所遇到的丧尸最高等阶都只有八阶巅峰，九阶似乎是一道新的门槛，持续了大半年都没有出现。
<br/>　　而当这只丧尸头领出现后，也让很多的狼牙战团成员明白过来，九阶的丧尸究竟有多恐怖，尤其是这还是一头九阶的超级丧尸，是这一千五百万尸潮的丧尸头领！
<br/>　　无论是速度、战力还是防御，这只丧尸都远超这片地域的所有丧尸和人类，从它比楚涵先一步改变方位的速度和灵敏度就看得出来，它的各项数据都可以将楚涵碾压。
<br/>　　眼下的楚涵想要战胜它，根本是无稽之谈。

<br/>　　明显是被动的劣势，让蒋天庆一颗心再次一沉：“果然二十米太远了吗？”
<br/>　　不退二十米陈少爷无法开枪，退了二十米却对劣势的情况毫无改变！
<br/>　　那丧尸头领自然也明白这一点，所以虽然眼前的楚涵冲来的速度快到惊人，它依然不慌不忙，相对眼前这个已经被它视为蠢人的楚涵，它对身后那个能开枪两次命中它的狙击手，更感兴趣！
<br/>　　第一枪是在它刚跳进基地的时候，那是一次出其不意。
<br/>　　而这第二枪，丧尸头领认为这是这名狙击手的极限能力，值得被它稍微佩服一下。

<br/>　　猝不及防响起！

<br/>　　第二发绝命狙击枪的子弹，再次射·入了丧尸头领的后背！
