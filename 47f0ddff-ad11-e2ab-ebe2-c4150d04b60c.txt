　　　　只是这条龙蛇涂小安看了几眼之后，察觉到有丝不对劲，它盘踞在水缸中，居然毫无生机。
<br/>　　旁边的李清照看的兴奋之余，也微微蹙眉起来，好奇的问：“小安哥，你有没有觉得这条龙蛇好像病怏怏的”
<br/>　　不但李清照有这个质疑，就连旁边很多观看的客人都议论纷纷起来。
<br/>　　“大家看，这条蛇的眼睛一直都没睁开过，不知道的，还以为是一条死蛇呢”
<br/>　　“怎么可能是死蛇，一定是在睡觉”
<br/>　　“不可能，蛇是非常敏感的动物，我们那么多人围着它看，它怎么可能视若无睹”
<br/>　　一个个围观的人七嘴八舌，更为一位客人肆无忌惮的走上去，用手敲了敲水缸，“咚咚”...
<br/>　　只见水缸中的龙蛇在如此大的动静下，既然一点反应都没有，蛇躯没有扭动一下，蛇眸没有睁开看一眼。
<br/>　　这更是引起了客人们的疑惑，不少人纷纷猜测，这蛇是不是真的已经死了。
<br/>　　这里有很多人都是冲着龙蛇的名头来的，都知道龙蛇富贵逼人，气运加人，大家都想过来沾沾运气。
<br/>　　可这要是条死蛇，那么就成了晦气了。
<br/>　　不少人愤愤不平起来，什么意思呀，摆一条死蛇来忽悠人吗。
<br/>　　“服务员，服务员”
<br/>　　顿时间，很多客人纷纷的叫嚣了起来。
<br/>　　一位迎宾急急忙忙的小跑了过来，感觉要出事情，连忙道：“各位客人，出什么事情了”
<br/>　　一位西装革履的男子站出来，质问：“你们店太过分了，居然给我们看一条死蛇，当我们眼瞎吗”
<br/>　　“死蛇？”
<br/>　　迎宾女子一听，大吃一惊，直接否认：“不可能，这条龙蛇可是我们老板花了重金觅得的，怎么可能是死蛇”
<br/>　　“什么不可能，这蛇从始至终动都没有动一下，眼睛也没睁开过，不是死蛇是什么，不信，你自己看看”
<br/>　　“这...”迎宾朝着水缸看去，看了接近一分钟，确实里面的龙蛇没有动过，顿时冷哼涔涔起来。
<br/>　　这蛇要是真的死了，那么就糟糕了，老板曾经说过，这个店都没这条蛇值钱。
<br/>　　奇怪，昨天龙蛇还好好的，今天怎么成死蛇了。
<br/>　　她瞬间意识到这件事情不是自己可以处理的，甚至任何员工都处理不了，必须老板亲自来，而且老板对这条龙蛇非常的看重，它可不能出一点事情。
<br/>　　“请各位客人稍等”
<br/>　　迎面火急火燎的转身离开，只留下一个个七嘴八舌的客人。
<br/>　　甚至有不少人暗中抱着看戏的态度，嘴角挂着笑容。
<br/>　　一般有身份的人，都会选择来味鲜阁就餐，照成很多餐饮界的人眼红。
<br/>　　这条蛇要是死了，看你拿什么吸引客人眼球。
<br/>　　“谁说我的龙蛇死了”
<br/>　　一会儿后，突兀的，一位青年从楼梯上缓缓的出现。
<br/>　　青年一身范思哲西装，面带微笑，神情自若的走上来，脸上噙着沉稳跟自信。
<br/>　　他的身旁还跟着一个四十多岁的黑衣中年人，步履沉稳，矫健，好像是他的贴身保镖，如影随形。
<br/>　　“大家好，我是味鲜阁的老板，苏羽”他走上来，看着众人自我介绍。
<br/>　　苏羽？？？
<br/>　　青年报出自己的名字的那一刹那，大厅内，几乎所有人都倒吸了一口凉气。
<br/>　　苏羽？温城有名的大家族苏家二公子，做生意非常的有一套。
<br/>　　这古街建成还没几年，他的味鲜阁就成了远近驰名最有名的餐馆。
<br/>　　而且这还不过是他随便小试牛刀的副业，温城最大的五星级大酒店，博尔曼就是他手下的产业。
<br/>　　比起五星级大酒店，这家餐馆到不算什么了。
<br/>　　不过能成为一条古街最好的餐馆，也说明了这个人的厉害。
<br/>　　可众人没想到，这位年少多金，拥有不少大产业的老板，居然会因为一条蛇，就立刻出现了。
<br/>　　可见，他对这条蛇的重视程度。
<br/>　　“苏老板，都说你的店里有一条镇店之宝龙蛇，我等仰慕过来看一看，可随知发现居然是条死蛇，这样欺骗客人，是不是有点说不过去了”
<br/>　　苏羽来的时候已经听说，眼睛烙印在水缸上，微微蹙眉。
<br/>　　这条蛇是不是真的死了，必须拿出来检查一番才知道，但就算没死，他也看出来这条蛇的萎靡跟病怏怏。
<br/>　　“老板，这条蛇还活着，没死”
<br/>　　他身边的贴身保镖凑到苏羽的耳边说了一句话。
<br/>　　苏羽点了点头，道：“打开，将龙蛇拿出来检查一下”
<br/>　　“各位稍安勿躁，我苏羽在商界立足，向来最讲诚信，绝对不会拿一条死蛇来忽悠大家”
<br/>　　贴身保镖闻言，直接大步迈过去，干脆利落，将水缸盖打开，伸出粗粗的手臂直接放进去，将一动不动的龙蛇给抓了出来。
<br/>　　“还说不是死蛇吗，被人如此抓着，也软绵绵的，眼睛也没睁开”
<br/>　　这句话刚落下，龙蛇的蛇眸却慢慢的睁开了，无精打采了看着众人，也没什么反应。
<br/>　　“老板，这蛇应该是病了”贴身保镖淡淡的开口。
<br/>　　“好端端的怎么会病了”苏羽蹙眉的问。
<br/>　　这条龙蛇他得来不易，那可是在顶级的拍卖会上花了整整五百万给拍下来的，这才拿到店里多久，就病怏怏的。
<br/>　　龙蛇富贵逼人，当初他请来的风水师就说过，只要有这条龙蛇在，他的这个餐馆的生意就会一直红火下去。
<br/>　　但要是这条龙蛇死在店中，那么味鲜阁的风水也会骤变，变的非常的晦气。
<br/>　　而且甚至还会影响苏羽自身的气运。
<br/>　　凡是有利就有弊，这种关于气运的动物，一旦死了，等于玉石俱焚，会被反噬。
<br/>　　贴身保镖迟疑了一下：“老板，对于蛇这些动物，我也不是很了解”
<br/>　　“那就去请个兽医过来看看”
<br/>　　苏羽蹙眉起来，看着萎靡而软绵绵的龙蛇，一脸的担心。
<br/>　　“你这个问题，我到是可以回答你”
<br/>　　大厅内，一道清冷的声音响了起来，众人顺着声音放眼望去，只见一位清秀的少年神情自若的开口。
<br/>　　苏羽一见说话的人，如此年轻，到也没太轻视，问：“你知道我的龙蛇为什么变成这样？”
