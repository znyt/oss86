　　四个人在工地上随意的四处走动，所谓的鼠灾，自然说明老鼠的数量是非常的恐怖。
<br/>　　但迄今为止，他们连一只老鼠的踪影都还没看到。
<br/>　　在人类的世界中，老鼠这种小动物应该是除了狗跟猫，大家最常见的，不同狗跟猫有看家镇宅的美誉，老鼠就是一个无恶不作的害人精，可偏偏又是一只具有无比灵性，聪慧神秘的小生灵。
<br/>　　民间早在几千年前就流传着所谓“四大家”，“五大门”的动物原始崇拜，五大门就是五大仙，蛇被唤为白仙，而老鼠在五大仙中称之为灰仙，可见它的地位，同时也是人们对狐狸、黄鼠狼、刺猥、老鼠、蛇的敬畏心理的反映。
<br/>　　老鼠的听觉，嗅觉，味觉，都很敏锐，能攀登，能游水，能打洞，能破坏，食量还很大，几乎人们吃的东西它都吃，酸、甜、苦、辣，它都不怕，完全一副你吃啥，我就吃啥的无赖样子。
<br/>　　真正让人们痛恶老鼠的并不是以上说的这些，而是老鼠会带来鼠疫，流行性出血热，钩端螺旋体病等病源，对人类危害极大。
<br/>　　你白吃白住，不断的搞破坏，人类都可以忍，你还传播的鼠疫那就太不讲道理了。
<br/>　　此时工地的鼠灾透着诡谲，因为有了余子童这位捉妖师，涂小安退居二线，充当一位看客，当然他也没有闲着。
<br/>　　涂小安意外的发现工地已建成的石柱跟水泥地面，有非常多的坑坑洼洼，像是一个个小洞。
<br/>　　“这些小洞都是被那些老鼠咬成这样的”苏羽看着涂小安解答。
<br/>　　老鼠打洞？
<br/>　　涂小安脸上透着一丝古怪，这可不是什么泥土跟木头，可是水泥而成的坚硬墙壁跟水泥地，就算是拿把手抢开火，也顶多钻出一个窟窿眼罢了。
<br/>　　这些老鼠的牙齿真的厉害到了如此恐怖，能将水泥墙都咬出一个个洞。
<br/>　　看着这些小洞，难不成里面成了老鼠窝？
<br/>　　坑坑洼洼中，有不少是大洞，足足能塞下一个成年人的拳头。
<br/>　　恍然间，涂小安想起了自己刚开始重生为蛇，那时候自家后院冒出来一只鼠王，那只鼠王体型达到一只小兔子的大小，牙齿锋利如刀，出场就抱着一块石头来吃，就跟吃面包一样的轻松随意。
<br/>　　如果是那种级别的老鼠，在墙壁上打洞，到也有可能。
<br/>　　这时，涂小安站到一面墙壁面前，整面石墙有很多的小洞，他将一只手试着伸了进去。
<br/>　　“小安兄弟，这洞中既有可能藏着老鼠，小心...”
<br/>　　涂小安笑了笑，没在意，一只手在里面捣鼓了下，发现里面好像形成了一条小型的通道。
<br/>　　不得不说老鼠打洞的本事天下一流，但是能在水泥墙壁中打洞，也可谓恐怖如斯了。
<br/>　　忽然，涂小安一阵蹙眉，自己的食指被什么东西咬了一口，他伸出手来，错愕的发现自己的食指直接就咬没了。
<br/>　　成了断指。
<br/>　　涂小安连忙用身子一侧，挡住了苏羽的视线，不让他看到自己的断指。
<br/>　　眼睛微微一眯，这只断指再次神奇的长了出来，恢复原状。
<br/>　　好厉害的老鼠呀。
<br/>　　一口将自己的手指咬断了，也好在是他，这是虚假的身体，要是换了别人，恐怕就成了九指了。
<br/>　　涂小安面不改色的再次将手伸进去，突然，他猛然的一抽，一只肥大的老鼠被他抓着长长的尾巴给揪出来了，直接扔到了地面。
<br/>　　“吱吱...”
<br/>　　肥大的老鼠出现在空旷的工地上，顿时冲着众人一副凶相的尖叫了起来。
<br/>　　“好大的老鼠呀”
<br/>　　苏羽跟余子童都微微的吃惊。
<br/>　　这只肥大的老鼠被揪出来暴露，既然也不四下逃窜，而是后肢站立，前肢挺立，跟人一样的站了起来，小小眼睛散发异样的光，凶性很大。
<br/>　　涂小安发现这只肥大的老鼠，跟自己后院当初的那只鼠王很像，都是成长大了一定的变态级别。
<br/>　　这种老鼠，在一般的人家成窝，都是王的级别。
<br/>　　嗷...
<br/>　　肥大的老鼠出现，余子童怀中的黄金瞳虎啸一声，震耳欲聋，骤然出现在了大老鼠的面前。
<br/>　　只见黄金瞳的阴阳眼光芒大亮，兴奋的盯着这只大老鼠，浑身金毛炸起，尽显老虎之势。
<br/>　　大老鼠见人都不怕，但是看到这只威风凛凛的猫，一下子没了脾气，贼眉鼠眼中流露出了害怕情绪。
<br/>　　这一害怕，黄金瞳动如狡兔的扑了过去，直接将直立的老鼠扑倒，凶煞残忍的冲着一顿撕咬，画面充满了血腥。
<br/>　　这只大老鼠根本没的反抗，被咬了没几口后，瞬间气绝身亡，而黄金瞳依然不放过它的尸体，大快朵颐的吃了起来。
<br/>　　“这老鼠体大如兔，通人性，厉害的很，不过并没有觉醒，怪不得我感觉不到妖气”
<br/>　　余子童脸色微微流出笑容：“不过在厉害的老鼠，碰到黄金瞳，都是被吃的命运，苏老板，我看就是这只鼠王在你的工地为非作歹，现在它死了，那些老鼠小弟，想必也不敢出现在你的工地了”
<br/>　　“这只就是鼠王？”苏羽虽然吃惊这只老鼠的体型，但是说它是鼠王就有点太过了。
<br/>　　这不过是涂小安随意在小洞中揪出来的。
<br/>　　一揪就把霍乱的鼠王给揪出来的？
<br/>　　“如此大的老鼠，不是鼠王还能是什么，而且它能直立行走，非常的聪明，若不是碰到我的黄金瞳，常人可对付不了它”
<br/>　　余子童笑着解释，看似对付的轻松，但主要是他的猫牛逼。
<br/>　　涂小安开口：“余大师，你太乐观了，这不过是一只藏匿在工地上的普通老鼠”
<br/>　　“这老鼠还普通？”
<br/>　　虽然不是觉醒生物，但成长到这种级别，也是非同小可。
<br/>　　“很普通，不信你看”涂小安冲着一个方向指过去，大家顺势看去，顿时呆若木鸡。
<br/>　　只见坑坑洼洼的小洞中，钻出一只又一只的大老鼠，每一只都无比的肥大，通体灰毛，一双眼睛冒着摄人的光。
<br/>　　一只，两只，三只，五只，十只...
<br/>　　短短几秒，就有十只刚才那么大的老鼠钻出来。
<br/>　　霎时间，只见工地上出现了密集的吱吱声，不绝于耳，好像是从四面八方传来的。
<br/>　　一个转身看去，原来不单单是坑坑洼洼的小洞中冒出老鼠，工地上附近阴暗的角落也冒出了几十只老鼠。
<br/>　　而且这种数量在迅速的增加，没半晌，就多达百来只老鼠出现。
<br/>　　旋即，这些大老鼠朝着四个人包围了过来，虎视眈眈，凶煞霍霍，小小眼睛中，冒出贪婪的光，好像人类才是它们的猎物似的。
