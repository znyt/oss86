　　今日楚风眠既然是要大开杀戒，当然不会放过任何一个人。
<br/>　　这陈少主的身上，有着吞天兽的血脉，杀了此人，吞噬了他的血脉，对楚风眠也有着巨大无比的好处。
<br/>　　更何况这陈少主背后的势力，也无比惊人，要知道陈家如今来到九域的古帝了，都有着十几位，八劫古帝都有着三位。
<br/>　　这样的势力已经比起大罗天星域的许多二流势力还要强大的多了。
<br/>　　要是放走这陈少主，无异于是放虎归山！
<br/>　　今日楚风眠跟着陈家之间的仇怨已经是达到了一种不死不休的局面了，也没有什么足以让楚风眠顾忌的了。
<br/>　　楚风眠身形一闪，立刻追逐上了陈少主，大手一挥，整个宫殿之中的空间都被楚风眠所封锁。
<br/>　　楚风眠拥有始祖天龙血脉，在封锁空间之上，没有人比起楚风眠施展出的还要更加玄妙。
<br/>　　一旦是被封锁，纵然是陈少主全力之下都难以逃出生天，更何况他现在失去了肉身，已经是化为了一团精血。
<br/>　　“给我镇压！”
<br/>　　楚风眠看着陈少主，没有丝毫的犹豫，运转起灵力向着此人镇压而去。
<br/>　　纵然现在陈少主已经是强弩之末，失去了肉身，更是被天龙变的力量所封锁，无法逃出生天。
<br/>　　但是陈少主毕竟可是来自于阎罗星域，也许会有着楚风眠所不知道的一些秘术，楚风眠也不敢掉以轻心。
<br/>　　一股股的灵力，在楚风眠的掌心之中，凝聚成为了一座七层宝塔，这宝塔之上，盘旋着九条远古战龙。
<br/>　　这一招也是始祖九式之中所记载的一招无上封印之术，九龙封天决！
<br/>　　七层宝塔凝聚而出，从天而降，死死的镇压在陈少主的身躯之上。
<br/>　　只见陈少主那一团精血，在被不断的压缩，一点点的被蚕食镇压。
<br/>　　“不，我是绝世天才！我可是要征服九域！成为圣域之主！怎么会被镇压在这里！”
<br/>　　陈少主疯狂的咆哮道，他的眼神之中露出几分狰狞之色。
<br/>　　“阎罗大祭！”
<br/>　　陈少主疯狂怒喝这，一大口精血突然从他的口中喷了出来，这一口精血喷出来的一刻，在空中突然凝聚起来。
<br/>　　一团团精血，在空中凝聚成为了一个个古来的文字，这文字的古老，就连楚风眠都没有见到过。
<br/>　　但是从上面，却是散发出一种无比神圣，古老而又神圣的力量。
<br/>　　“少主！不！”
<br/>　　看到陈少主施展出的这一招来，其他的陈家古帝一个个皆是脸色大变，跟寒月漓，洛汐缠斗的那两名八劫古帝，更是脸色大变。
<br/>　　丝毫不顾被寒月漓跟洛汐力量轰击，掉头就跑，相要逃出这宫殿之中。
<br/>　　“祭祀？这是祭祀之道？还是血祭！”
<br/>　　楚风眠死死的盯着陈少主，突然之间明白了过来，这陈少主施展的是什么手段。
<br/>　　祭祀。
<br/>　　这是最为古老的武道之一，祭祀之道，可以祭祀天下万物，来获取力量。
<br/>　　而祭祀之中，最为野蛮，也是最为古老的，便是血祭。
<br/>　　以鲜活的生灵祭祀。
<br/>　　在蛮荒的时代之中，许多武者都是擒下其他武者，妖神，进行血祭，来获取力量的。
<br/>　　这血祭之道，实在是太过野蛮，以至于在远古时代，便是彻底消失了。
<br/>　　现在这陈少主，竟然是懂得血祭之道？
<br/>　　“你们不用逃了！你们得到陈家的恩惠，就注定有着这一日！小子，你的手段的确不错，足以逼迫我施展血祭之道！但是今日你死定了！我要不惜一切的杀了你！才能弥补我这一次的损失！”
<br/>　　陈少主看着逃跑的一群陈家古帝，大笑一声道。
<br/>　　“血祭！”
<br/>　　血祭二字一出。
<br/>　　那些陈家古帝的身躯，在空中全部都停滞住了，下一刻，更是突然间的爆炸开来。
<br/>　　漫天血雾，都向着陈少主面前的那一堆古老的文字，飞了出来，融为了一条血河，化为了极为精粹的灵力，融入到了陈少主的身躯之中去。
<br/>　　这陈少主本来是被楚风眠毁灭的肉身，在这一股股的力量之下，重塑归来。
<br/>　　甚至是身上的气息，要比起之前还要强大的多。
<br/>　　十几名古帝，包括两名八劫古帝，都被这陈少主血祭，化为了他的力量。
<br/>　　楚风眠看的出来，这些陈家古帝的身上，应该是都被刻下了一种烙印，一种在血脉之中的烙印。
<br/>　　所以这陈少主已经催动，就足以将他们血祭。
<br/>　　“这手段好狠！”
<br/>　　楚风眠眼神无比冰冷，这一手，显然是早有准备，这烙印，甚至可能是这些陈家弟子，在刚刚出生的时候，就被刻入到血脉之中的
<br/>　　为了就是这一刻，来血祭，这些人，甚至都称不上是真正的陈家人，只是奴隶，被这陈少主血祭的奴隶。
<br/>　　在血祭的效果之下，陈少主身上的灵力节节攀升，他缓缓的睁开双眼，看向楚风眠的目光，尽是杀意。
<br/>　　“小畜生，今日你竟然敢逼我到了这一步，这是你自己找死！”
<br/>　　陈少主的心中都有些心疼。
<br/>　　施展这血祭之术，足足是令他们陈家十几名古帝，全部陨落。

<br/>　　这种损失千万年都难以恢复的过来，但是今日在楚风眠的逼迫之下，他已经是没有选择了。
<br/>　　陈少主看向楚风眠的表情，无比狰狞，他现在只想要杀了楚风眠，得到了楚风眠的一身奇遇，甚至是得到了楚风眠的仙术，才足以弥补他的损失。

<br/>　　轰隆！
<br/>　　陈少主一出手，一拳直指楚风眠轰击而来。
<br/>　　在血祭了十几名陈家古帝之后，现在陈少主身上的力量，比起之前要恐怖了数倍。
<br/>　　一招之下，就足以惊天动地，这样的实力，已经是接近于小绝帝那个级别了，甚至是还要在小绝帝之上。
<br/>　　“一起出手！”
<br/>　　楚风眠长啸一声。
<br/>　　不用楚风眠提醒，寒月漓跟洛汐，也已经出手了。
<br/>　　一道剑光，一股混沌，顿时向着陈少主轰击而去。
<br/>　　这血祭之术虽然强大，但是那两名陈家八劫古帝的死，也让寒月漓跟洛汐可以来助楚风眠一臂之力。
