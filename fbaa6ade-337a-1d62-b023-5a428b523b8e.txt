　　　　不过总归来说，这些出现的蛇，不管是无毒蛇，还是有毒蛇，都是很一般品种的蛇。
<br/>　　为什么这些蛇会一起跟商量好的出现在了校园。
<br/>　　难道它们也想来上学？
<br/>　　涂小安看了一眼满是兴奋心情的李清照：“你们学校眼前也出现过蛇吗”
<br/>　　李清照就跟进入了动物园，这看看，那瞧瞧，一地的蛇让她没有一点惧怕，还让她觉得非常的有意思。
<br/>　　“没有，要是以前也出现过蛇，我至于那么激动吗”
<br/>　　出生在白镇的孩子，可是将蛇为朋友，根本不会怕，但这丫头要不要那么的幸灾乐祸呀。
<br/>　　她考虑过别的学子们的心情吗。
<br/>　　人都有惧蛇症，也就她看到满地的蛇，乐的合不拢嘴。
<br/>　　跟一个疯丫头一样。
<br/>　　“嘻嘻，我又找到了一个直播的好噱头”李清照眼眸闪过狡黠，拿着手机出来就准备开直播，嘴巴还呢喃自语着：“要起一个什么标题呢，不然就叫校园百蛇出没”
<br/>　　涂小安一个劲的翻白眼，好奇的问：“丫头，你就不奇怪为什么这个学校一下子出现了那么多蛇吗”
<br/>　　“好奇啊，但现在也没人给我答案，而且这不是我考虑的事情”
<br/>　　李清照忽闪忽闪眼眸，没心没肺的说。
<br/>　　正准备开直播，李清照被前方一道靓丽的身影所吸引。
<br/>　　“苏老师，哇，你好厉害呀”
<br/>　　只见一位身材高挑的美女抓着几条蛇走来，惹的李清照叫了起来。
<br/>　　此女人扎了一个利落的马尾，简单干练，但五官长的极美，眉宇间有英气逼人，整个人落落大方，破有点女中豪杰的味道。
<br/>　　这女中豪杰主要是她手中抓了四五条的蛇，印象有点另类。
<br/>　　“晴照，怎么是你呀，这学校不知道怎么回事，出现了大量的蛇，有不少新生学子都被蛇给咬了”
<br/>　　苏菲蹙眉而说，然后看了涂小安一眼。
<br/>　　李清照连忙在旁边介绍：“这是我们学校的生物学老师苏菲，这是我的朋友涂小安”
<br/>　　“你好”
<br/>　　“你好”
<br/>　　两人微微一点头，涂小安诧异，一个女的教生物学呀，怪不得不怕蛇。
<br/>　　“晴照，你不是白镇人吗，快帮老师一起抓蛇”苏菲继续道：“你朋友怕蛇吗，不怕就一起来”
<br/>　　“好在现在还没真正开学，不然可就麻烦了，这些蛇都要尽量的给捕抓了，不能会影响新学期的上课”
<br/>　　李清照闻言，开心道：“能跟苏老师一起抓蛇，是我的荣幸，我这位朋友更不怕蛇，他可是捕蛇人”
<br/>　　“捕蛇人？”苏菲格外看了涂小安一眼，见他长的眉清目秀，说难听点，有点小白脸。
<br/>　　还真的不像捕蛇人，或许还真是人不可貌相。
<br/>　　笑的道：“那太好了，同学，帮帮忙，一起抓蛇，校方已经报警”
<br/>　　涂小安点点头，自然不会拒绝，而且这位苏老师也将他当成了这个学校的学子。
<br/>　　很快苏菲找了一个大桶过来，专门用来放蛇，别看学校学子很多，但敢真正徒手抓蛇的却不多。
<br/>　　女孩子都更是屈指可数了。
<br/>　　谁能想到，温城大学开学招生，招来的却是百蛇出没。
<br/>　　涂小安并没有动用自己的特殊能力，而是跟普通人一样的捕蛇，一条一条的抓。
<br/>　　脑海中不禁的想，这种事情要是发生在白镇，那么白镇人民该多高兴呀。
<br/>　　山都不用上了，直接有蛇可抓。
<br/>　　别看白镇已蛇出名，可蛇都是乖乖待在山上，人们生活的地方，基本上不会出现蛇。
<br/>　　蛇出现在白镇，找抓吗。
<br/>　　涂小安早就意识力笼罩这个学校，发现蛇的数量还真的不少，绝对不少于一千条，不过都是很普通的蛇，连罕见蛇种都看不到。
<br/>　　蛇虽然不少，但是这些普通的蛇一般人能对付，也是涂小安没有动用自己实力的主要原因。
<br/>　　今天他就是来参观大学面貌的，苏菲也说了已经报警，到时候事情自会得到解决。
<br/>　　此时，大学各个地方散开了捕蛇行动，如今真是那些男生他们展现男性荷尔蒙的时候，捕蛇的样子，在女生的眼中是很帅的。
<br/>　　涂小安，李清照，苏菲三人组成捕蛇小组，效果出奇的高，看到蛇，二话不说，伸出手抓起来，扔桶里面，一气呵成，绝不拖泥带水。
<br/>　　唯有李清照抓到蛇，还逗皮皮蛇玩一下，才扔桶里。
<br/>　　嘴巴还嘟嘟的说：“还是我的小蛇蛇最可爱了”
<br/>　　太阴蛇也没有出场，乖乖趴在李清照的肩膀上，无论她如何的大动作，这蛇就是掉不下来，跟沾到她肩膀上似的。
<br/>　　“苏老师，你说我们学校为什么会突然出现那么多的蛇呀”
<br/>　　李清照发问。
<br/>　　苏菲双眸似水，十指纤纤，一举一动都像是在舞蹈，一条条蛇被她轻松的扔进了桶里，边做事，边回答：“这件事情我也说不好，处处透着诡谲”
<br/>　　“最近城中无雨水，天气炎热，应该不会把蛇给引出来，而且温城大学在市中心，后背无山，都是城市楼宇，就算有蛇隐匿，也不会有多少”
<br/>　　“这一下子冒出那么多条蛇，实在让人觉得匪夷所思”
<br/>　　苏菲就算是生物学的老师，也二丈和尚摸不着头脑。
<br/>　　她这说法等于是普通人的正常判断，现代人，尤其是这些生物学老师，讲的是科学依据，既无雨水多，又不是后背靠山林，如今这些蛇没头没尾的出来吓人，实在找不到问题所在，就像凭空出现。
<br/>　　也只有先将它们都抓起来在说。
<br/>　　别说苏菲找不到问题关键所在，就连涂小安这条蛇王，也有点莫名其妙。
<br/>　　大规模的蛇出动，有两种现象，一是蛇繁衍，会汇聚在一起，这是最正常最科学的说法。
<br/>　　二便是有可能是蛇报复，某某人身上沾上了蛇仇恨，从而吸引了大量的蛇出现攻击。
<br/>　　这种说法有点诡谲，但在白镇人心中却是真实有可能发生的。
<br/>　　但涂小安发现不管是有毒蛇，还是无毒蛇，其实它们的攻击性并不强，好像只有人类不小心踩到它的身上，它们才会凶性大发的攻击你。
<br/>　　然而，你抓它，它也不跑，任由让你抓。
<br/>　　这或许也是不幸中的大幸，这些都是没什么危险的蛇，要是出现的是佘山的毒蛇，那么这些新生的学子恐怕咬到就要死翘翘。
