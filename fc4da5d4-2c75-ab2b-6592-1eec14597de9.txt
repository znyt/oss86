　　“轰！”
<br/>　　盖世大劫，这是名副其实的灭世，乱天动地，雷光席卷一切，苍宇四裂，天地万物俱灭。
<br/>　　强大如万劫魔尊，当冲入雷劫之中时，也差点被当场劈成碎块，仅仅只是一个照面，他便血肉横飞，血雨飘散，白骨外露，可怕无比。
<br/>　　万劫魔尊的躯体差点被劈开，有的地方露出森森白骨，这是一种可怕的景象，古今不知道有多少人死在雷劫之中。
<br/>　　世上最不缺的便是人杰！
<br/>　　万劫魔尊如今的大劫，更是远胜其他，雷光之强，世所罕见，此劫已经堪比古今往来最强一列之内。
<br/>　　“斯拉，咔擦！”
<br/>　　一大片的暴雷劈了下来，五光十色，各种灭世之威，穿过层层异象，劈在万劫魔尊身上，尽管他受伤很重，好几次都差点被劈了只剩下碎渣，但他依旧复活过来，继续牵引雷劫，运转《万劫魔典》，吸收这无尽的雷劫之气。
<br/>　　这样的机会可不多，万劫魔尊修炼的是万劫之气，而天地间的劫气，哪有此刻雷劫之中的多，在此地修炼一年，抵得上平时修炼千年不止，若不好好利用此次的机会，尽管提升实力，他恐怕都不会原谅自己。
<br/>　　此刻，尽管他被暴雷劈得险死还生，但他心中依旧希望，此次的大劫，能够延长更多时间，好让他完成蜕变。
<br/>　　洪荒中的其他人见到这一幕，心中只会认为是天罚圣主不想让万劫魔尊成功突破，想让他死在雷劫中，可谁又会知道，天罚圣主这是在利用自己的权力为万劫魔尊提升实力？
<br/>　　《万劫魔典》是一本可以驾驭天地万劫的功法，乃魔祖所创，自是十分不凡，若是有一天，它更是能够执掌天地万劫。
<br/>　　天地万劫可不是天罚，而是比天罚更厉害。或者说，天罚也只是天地万劫之中的一部分。掌管天地万劫，那时便可以随时控制天地量劫，或者无量劫，那才是真正的大恐怖！
<br/>　　“嗤嗤嗤嗤……”
<br/>　　黑色的，金色的，白色的，银色的，紫色的……各种颜色的神雷，带着毁天灭地的气息，劈在万劫魔尊身上，使他的身体一震溃烂，这些雷霆之中，不仅带有强烈的腐蚀能力，更是像瘟疫一般，向着他的全身扩散，到最后，甚至能够腐蚀元神。
<br/>　　各种神雷全部都往他身上劈来，万劫魔尊的身体一震焦黑，像是被劈成了黑炭，腐蚀、灼烧、毁灭、破灭等各种力量交织在一起，破坏他的身体，各种神雷之中，仿佛是存在某种神秘联系，能够爆发出恐怖的威力，宇宙在颤栗，苍生在敬畏。
<br/>　　“吼！”
<br/>　　万劫魔尊突然大吼一声，他撤去全身所有防御，让自己暴露在雷海之中，任由那些劫雷劈在自己身上。
<br/>　　“轰隆隆！”
<br/>　　一瞬间，雷劫在没有遮拦的情况下，成千上万道倾泻下来，将万劫魔尊淹没，他的躯体被劈成碎片，只剩下一颗头颅，神光一阵涌动，一瞬间，一具全新的身体再次出现在雷海之中。
<br/>　　毁灭，新生，毁灭，新生……
<br/>　　万劫魔尊沐浴在五光十色的雷光之中，运转功法，仿佛化身千万混洞，在吞噬着周边的劫雷，。各种神秘的劫雷在他身上肆掠，交织着灭世的力量，到最后，万劫魔尊的整个身体燃烧起来，骨骼与血肉都燃烧起来，五彩的火焰绽放，像是大道之火，焚烧万物，焚寂苍穹。
<br/>　　万劫魔尊身上的血肉一块块脱落下来，不过这个时候，也有强大的生机绽放，有新的血肉在生成，就这样，在不断毁灭与新生之中，以雷劫炼体，淬炼无上万劫魔体，万劫魔尊的躯体强大到了极致，滴血重生实在只是小意思，仅凭肉体，如今的他便已经强过先前的十倍不止。
<br/>　　九年时间过去了。
<br/>　　万劫魔尊犹如一尊亘古长存的黑色雕像，屹立在无尽雷海之中，任由狂暴无比的雷电打在他身上，就是不能毁灭他分毫。
<br/>　　“咔擦！”
<br/>　　一声轻响，像是蛋壳破碎，可以看见，万劫魔尊左脸之上，一块漆黑的外壳脱落了，像是鸡蛋壳脱落。
<br/>　　露出里面的皮肉，很是晶莹，银白剔透，泛着九彩的霞光。
<br/>　　“咔咔咔……咔咔……咔……”
<br/>　　接着，像是起了连锁反应一般，响声不断，伴随着一声声轻响，万劫魔尊身上漆黑的焦壳都露出了裂缝，然后轰然炸开。
<br/>　　晶莹、雪白、霞光灿灿，熠熠生辉，这是一次新生，在雷劫之中，褪去了原先的血肉，衍生了全新的躯体。
<br/>　　“成功了？”
<br/>　　洪荒之中，关注此处的大神通者们，瞳孔有些收缩，看着雷海之中的万劫魔尊，新的躯体衍生出来，每一寸血肉都变得晶莹，九彩的光芒流转，淡淡的九彩光华流转，血肉新生，铸造出新的魔体，然后，在他们的视线中，万劫魔尊的头发以肉眼可见的速度长了出来。
<br/>　　很明显，万劫魔尊在雷劫之下，得到了巨大的好处，铸成了强大无匹的魔体，在如此灭世般的雷劫之下，他完成了最后的蜕变。
<br/>　　“撕啦……噼啪……”
<br/>　　大片大片的雷劫降临下来，犹如瀑布般冲击在万劫魔尊身上，万劫魔尊全然不去理会，他好似忘了所有，呆呆的看着自己的魔体，任由那恐怖的雷劫落在他身上，留不下半点伤痕，可见，雷劫的威力，已经不足以伤害到他了。
<br/>　　“万劫不灭邪圣仙，万道森罗魔至尊！”
<br/>　　“本尊乃万劫魔尊，掌万劫，御万魔，区区雷劫，能奈我何，给本尊散！”
<br/>　　万劫魔尊抬头，眼睛睁开，爆射出魔性的光芒，可以看见，他的瞳孔是苍茫色的，带有浓浓的魔性力量，宛若两道巨大的混洞，带着吞天噬地的神秘力量，更是有一种魔性、威严、淡漠的气息。
<br/>　　一股恐怖的气势从他身上爆发，那淡漠威严的双眼之中，两道魔光爆射而出，射入雷海之中，瞬间便将那狂暴的雷电世界吞噬一空。
<br/>　　不过瞬息时间，刚刚还遮天蔽日，雷海爆闪，天地俱颤的雷电世界，眨眼间便被吞噬一空，两道神光吞噬完漫天劫雷之后，再次返回到万劫魔尊眼中，让万劫魔尊显得更加威严神秘了。
<br/>　　雷劫消散，大日再次横空，晴朗的天空，一贫如洗，若不是周围的虚空变成一片虚无，星空也变成一片废墟，谁也想不到，此处刚刚发生一场毁天灭地的灾劫。
<br/>　　原本雷劫之后还有心魔劫，但万劫魔尊自己就是天底下最大的魔头，天地间还没有谁能够给他下心魔。心魔乃是魔祖当年以魔道补天，补全天道，之后才出现的，如今万劫魔尊可以说是魔祖第二了，从来只有他给别人下心魔，还没有人能够给他下心魔的。
<br/>　　混元之劫，整整十五年的时间，三年风劫，三年火劫，九年雷劫，无一不是惊天动地，九死一生。
<br/>　　大劫过后，看着静立在虚无之中的万劫魔尊，众人直觉得那是一尊亘古长存的魔神，身上带着无上的伟力，仿佛主宰天地的神魔一般。
<br/>　　洪荒之中，在此刻全部安静了。他们实在是没有想到，如此逆天的天劫，万劫魔尊都能渡过，还有什么他不能做到的吗？
<br/>　　一阵安静之后，洪荒彻底沸腾了！
