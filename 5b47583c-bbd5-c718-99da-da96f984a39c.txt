　　副校长将眼前悬浮一排的蛇莓收了十颗回去，一脸平静的说：“收了你十颗蛇莓圣果，测量宝石的事情就不跟你计较了”
<br/>　　涂小安无所谓的耸了耸肩膀，剩下来的十颗蛇莓飘了回来，他直接在老头子的面前一颗一颗的吃了起来。
<br/>　　此一幕出现，让副校长猛然的从椅子上站了起来，瞪大双眼，如此蕴含能量的圣果，他当零食吃？
<br/>　　“你这是做什么”
<br/>　　涂小安笑着说：“反正你不要，我就给吃了呗”
<br/>　　“你这样吃，简直是在暴殄天物”
<br/>　　副校长心中在流血，早知道这样，他就全给收了，跟这条蛇精客气什么。
<br/>　　这蛇莓蕴含纯粹的能量虽然能单吃，也要是作为药引来炼丹，肯定效果更佳。
<br/>　　就他这样吃，简直不可理喻。
<br/>　　涂小安邪笑一下，他就是故意的，谁让这老头还跟自己端着，旋即开口道：“超等天才班的事情你安排下”
<br/>　　他要进去虐虐那个白君然。
<br/>　　“老夫知道，明天让你去挑战里面的第五十名学员，胜了，你就代替他进入超等天才班”副校长看着他道：“不过你要答应老夫，下手轻点”
<br/>　　“下手轻点？”涂小安莞尔一笑：“不是说超等天才班都是一等一的天才修武者，全是高手吗，你怎么长他人的志气，灭自己学员的威风”
<br/>　　“哼...老夫虽然年老，但没有老眼昏花”他忽然忌惮的看着涂小安。
<br/>　　这条还没证道的蛇精，却给副校长莫大的危险感，可谓深不可测，否则他也不用那么的客气。
<br/>　　就是不想惹恼了对方，节外生枝，毕竟学院全是学员，要是这条蛇精发狂了，可是会死很多人的。
<br/>　　最关键是什么。
<br/>　　副校长进来就观察他，既然发现看不出对方的虚实，他甚至隐隐觉得，就算自己出手，都不一定拿的住这条化形的蛇精。
<br/>　　所以，很多事情就退一步，大家和谐相处。
<br/>　　况且白镇蛇王，算是正蛇，他才敢这样做。
<br/>　　要是邪蛇，岂能容他留在古学院。
<br/>　　“你先下去吧，我会派人给你安排住处，别的事情明天在说”
<br/>　　“好，听你的”
<br/>　　涂小安朝着书房外走去，身后传来苍白的声音：“你记得，不可在学院造杀戮，凡事三思而后行”
<br/>　　毕竟是妖，副校长不得不在提醒一句。
<br/>　　“人不犯我不我不犯人，人若犯我，我必吃人”
<br/>　　淡然冷漠的声音响起，副校长看着涂小安清瘦的背影，却有一条巨大的蛇影在地面显化，狰狞可怖。
<br/>　　“吃人？”副校长阴沉的呢喃自语：“老夫这样安排，到底是对是错”
<br/>　　这蛇精是一个极大的变数，或许能带来一些好处，但也可以带来遭难。
<br/>　　涂小安离开副校长的办公室，微微笑了起来，既然跟学院的副校长都通了气，那么他就不用担心自己的身份会暴露。
<br/>　　因为已经暴露了。
<br/>　　重新回到小广场，在一片新生的崇拜目光下，涂小安找到了东方袁，还有林凡，他也一直没离开。
<br/>　　涂小安看的出来，他想跟自己处点交情出来。
<br/>　　他也不拒人于千里之外。
<br/>　　“大神，你现在可是出大名了，新生报道就要成为整个学院的风云人物，还将测量宝石震爆了，副校长打算怎么处理你”
<br/>　　“处理？”涂小安微微一笑：“那老头叫我进去喝了杯茶而已”
<br/>　　旁边有不少耳尖的学院听到涂小安大言不惭的话，目瞪口呆？
<br/>　　将测量宝石震爆了，副校长不但没处罚，还请他喝茶？
<br/>　　这很难令人相信，但看涂小安风轻云淡的走出来，不见一点事的样子，不由信了几分。
<br/>　　最强新生果然就是最强新生，连副校长都要给他面子，非常有牌面。
<br/>　　在广场聊了一会儿，有一人来，将涂小安等人带到了一个分配的住所。
<br/>　　被带到住所前，竟是一栋独立的两层小楼。
<br/>　　涂小安有点诧异，看着林凡道：“古学院给新生的福利那么好？每人配备一栋楼房”
<br/>　　林凡问了问那人，是不是带错了，那人说，是副校长吩咐的。
<br/>　　他大吃一惊，看着涂小安：“大神就是大神，这可是高级教官才有的待遇，我们学员都是住集体宿舍的”
<br/>　　“哦，是吗”
<br/>　　涂小安看了看周围，这算是一处比较安静的楼房，他微微一笑，看来那老头是担心自己跟一些学员住在一起，学员有危险。
<br/>　　难道还真怕自己吃人？
<br/>　　这小楼是由木结构搭建而成的楼房，屋内的家具应有尽有。
<br/>　　什么电脑，电视机，空调，洗衣机之类的既然一览无余，简直就是一键式的拎包入住。
<br/>　　这绝对算是一栋个人别墅了。
<br/>　　“大舅哥，你太有牌面了，我现在终于知道陈伯伯为什么让你来当我的保镖”
<br/>　　涂小安斜睨了东方袁一眼，淡淡的道：“小胖，你的话说的不对，不是我当你的保镖，而是你当我的小跟班”
<br/>　　“什么，明明就是你当我保镖，怎么我还成了你的跟班”
<br/>　　涂小安伸出手，不轻不重的拍了拍他的肥脸，邪魅一笑：“你忘记了吗，蛇博士让你听我的，所以我怎么说，你就怎么听，懂吗”
<br/>　　“不然，你就去下等班的宿舍住去吧”
<br/>　　东方袁见识过涂小安的实力，现在对他是不敢得罪的，连忙服服贴贴，嬉皮笑颜道：“大舅哥说什么就是什么吧”
<br/>　　来古学院，还有别墅住，简直太棒了，傻子才去住下等班级的宿舍。
<br/>　　他是谁，温城第一号富二代，住宿舍开什么玩笑。
<br/>　　“林凡，这里大，房间多，你要是想来，也可以一起搬进来住”涂小安看着他说。
<br/>　　林凡一惊，顿时心喜过往：“以后你就是我的老大了”
<br/>　　此时的他，毫无傲气，跟工地上判若两人。
<br/>　　这个世界，实力代表一切。
<br/>　　显然，他这种生物掌控者，在古学院也吃不太开，找涂小安这等强者抱大腿，绝对不会吃亏。
<br/>　　这一天，是古学院的新生报道，不算正式上课，也没什么事情，涂小安无事可做的出去熟悉熟悉环境。
<br/>　　涂小安也跟林凡了解了不少古学院的秘闻，还有一些学员的风云人物。
<br/>　　既然来到了古学院，那么自然都要打听打听清楚，之后的日子，想必也会很有趣。
<br/>　　他虽然算新生，但是恐怕没人管的了他，想待在古学院就待着，想出去也没人拦得住他。
