　　不仅文奇胜等人被镇住，鲁初雪等段氏基地的高层们，在见到这三万人队伍好好狼狼而来之时，也全体不可置信的楞在当场。
<br/>　　他们提前了一周到达段氏基地，三万五千人的队伍衣着并不整齐，甚至很多人的衣服都看起来有些破烂，但是他们的队形却整齐到不可思议，而每一人的状态更是充满肃杀气势。
<br/>　　“我仿佛看到了当初第一次见到狼牙战团时的场景，正是倾巢大战前的出征。”一名段氏基地的高层出声带着感叹：“那时候的狼牙战团人数只有几千，数量比不上任何一个大基地的军队，但是他们集体出行的那震撼场面，让我永生难忘。”
<br/>　　“但是现在的这批队伍，人数已经翻倍了。”鲁初雪在一旁提醒道。
<br/>　　“是啊，短短几个月而已，狼牙战团竟然随手就可将一批流民，训练成这等气场的战团！”
<br/>　　“太不可思议！”
<br/>　　“看看这些人的眼神，那是真正从死亡边缘走出来的。”
<br/>　　“一路所过之地的村庄城镇丧尸都被屠尽，怕是一刻不停保持着战斗状态过来的。”
<br/>　　段氏基地的众人不停感叹，对这三万新兵的感观对比两个月前，早已有了翻天覆地的变化。
<br/>　　分界线大战时这些新兵还是流民，他们居无定所，只能在战区边缘徘徊，不知道往哪去更没有能力独自生存下去，哪怕遇上数量少于他们很多的北部联盟军，这些人也毫无招架之力。
<br/>　　但在看看现在，这三万人要是再次出现在当时的分界线大战上，恐怕不用其余几个基地联合，光是这三万人就足以将整个北部联盟军一次性解决。
<br/>　　三万人进入段氏基地后只休息了一夜，这一夜他们将身上的脏衣服换下，里里外外收拾了一翻，吃着基地后勤送来的热腾腾食物，住在早已空出来的基地居民楼中，全体放下了一路而来紧绷的神经，享受这仅有一夜的短暂放松。
<br/>　　狼牙五千老兵同样如此，只不过他们经验丰富，早已习惯了这种不断征战的节奏，对比新兵他们的状态要好上许多。
<br/>　　所有人都有条不紊，哪怕这三万五千人一次性入住到这段氏基地之内，也没能让这个基地热闹起来，反而依旧好似没多少人般冷冷清清。
<br/>　　尤其是那些负责后勤的人感触最深，当他们走进狼牙战团的休息区时，顿时就好似走进了荒无人烟的地带，明明每个房屋中安排了人休息，明明身旁还时不时的有狼牙战团的人走动。
<br/>　　但就是安静的可怕，或者说是肃静到了极致！
<br/>　　何峰等一干军官则是没有太多时间休息，刚一到达段氏基地，就直奔基地中心。
<br/>　　在那里狼牙的各部门成员已经等候多时，包括鲁初雪等段氏基地的高层们，也聚集在了这里。
<br/>　　“物资都已经准备妥当，明早出发的时候统一分配。”鲁初雪望着何峰交代道，紧接着又有些歉意的说道：“只是条件不怎么好，物资称不上丰富，我们一时半会儿也拿不出那么多抵抗丧尸的防护服。”
<br/>　　何峰点点头表示理解：“我们这两个月一路杀过来也没用防护服，衣服都是每到一个城市就现场扫荡，食物问题更是如此，一无所有都能走过来，更何况现在还有这么多准备好的物资？”
<br/>　　鲁初雪心中不禁对这些士兵钦佩不已，对楚涵的才能也又一次肯定，而想到楚涵，鲁初雪也自然而然的皱眉问道：“对了，楚涵上将怎么还没来？”
<br/>　　何峰同样不解，只能看向了刘玉定：“你知道怎么回事么？”
<br/>　　“老大说他战前会过来，现在在哪里我也不清楚。”刘玉定虽面上不显，但同样心中焦急万分。
<br/>　　大尸潮眼看就要来临，所有参战者都已到达最南方，提前出发此时已经在南署基地的那五万人，则是已经准备就绪，就连苏行的工程队都展开了最后冲刺，所有人每天只睡四个小时，用最快速度将南署基地尽可能的改造。
<br/>　　两个月的时限是楚涵定的，但他也说过大尸潮具体什么时候出现没有定数，也就是说可能会延后也可能会提前。
<br/>　　而在一切不确定的情况下，时间就变成了最珍贵的东西，这也是苏行的工程队如此拼命的原因。
<br/>　　听着刘玉定的话众人都心中担忧，谁都清楚眼下这已经扩张到九万人的队伍，影响最大的还是楚涵。
<br/>　　那些从外基地调来的基地老兵，多少人冲着楚涵而来的，为的不仅是对抗丧尸，更是想要亲眼见识一下狼牙战团的战斗，想要亲自感受一次被楚涵指挥的大战。
<br/>　　而那些留在基地不走的三万新人，以及狼牙战团老兵不远千里从分界线带来的三万人，之所以能到现在这个地步，更是与楚涵的影响力息息相关。
<br/>　　他们知道楚涵要对抗大尸潮，知道楚涵会指挥这次空前盛大的战役，狼牙一次又一次的奇迹和楚涵不断呈现而出的统帅能力，让这些人选择了当一次勇者。
<br/>　　因为热血，所以赴敌！
<br/>　　但楚涵若是不在……
<br/>　　虽然他们的战斗依然可以进行，他们加急紧赶的提前完成任务也是为了预防不测，但狼牙战团却不是最强的狼牙战团。
<br/>　　何峰看着沉默的众人，目光望向了卢宏盛打破沉寂：“武器准备的如何？”
<br/>　　卢宏盛连忙回答：“新赶制了五千冷兵器，我都添加了特殊金属，相信这批武器要比上一批好用很多。”
<br/>　　一旁的和上也开口说道：“科技部与锻造部一起研发制造了五百新武器，眼下已经送到南署基地。”
<br/>　　两人说话的口吻都很平静，当然也只是在称述一个事实，但听着这些话的段氏基地高层们，可就一个个内心波涛汹涌了起来。
<br/>　　其中一人更是见了鬼般的看着和上和卢宏盛：“你们开挂了吧？！”
<br/>　　短短两个月不到的时间，还是在特殊金属需要每天等待送达的急缺状态下，这两个狼牙部门竟然还能制造出这么多的武器！
<br/>　　怎么做到的？
<br/>　　狼牙是运来了很多东西，但也没搬个工厂过来啊！
