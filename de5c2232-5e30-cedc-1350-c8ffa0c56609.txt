        &lt;h3&gt;第三卷第一百八十六章  美好的新年&lt;/h3&gt;
<br/>
<br/>        寒家是大家族，新年都是聚在一起的，这也算是一个感情的纽带，但是大家族有大家族的快乐和忧愁，从古至今家族的事情也是不曾改变的套路。
<br/>
<br/>        寒伯安的父亲是大房，而寒伯安亦是嫡长子，父亲对他寄予厚望，寒伯安也不负众望，接手寒家之后不但将寒家的事业风生水起，自己还另辟捷径，踏着时代的鼓点，跟上时代的步伐开辟了新的篇章。
<br/>
<br/>        在寒伯安雷厉风行，坚持原则强硬的带领下，寒家的生意也一日比一日强盛。
<br/>
<br/>        俗话说：树人先树己，只有自己坚持原则，以身作则才能身正为刚，做事情也是如此，爷辈如此，父辈也如此，不过父亲做人要圆润很多。
<br/>
<br/>        寒伯安的婚姻父亲并没有过多干涉，因为安男是书香世家，安林曦是泰山北斗式的人物，真可谓桃李满天下，他手中的资源可不是哪一个家族能比得上的。
<br/>
<br/>        寒伯安在他的婚姻之事上可以说得偿所愿，一切都是他自己说了算，无非就是知会一下自己的父母，什么时候要结婚。
<br/>
<br/>        安男是很少到寒家的，算起来这次新年也是第二次到寒家，之前俩个人一直都没有公开，在前两天寒伯安在年会上向安男求婚，引起了巨大的轰动，几天都是寒伯安和安男的消息，而寒家很多人也才开始接触安男。
<br/>
<br/>        “男男，我们的婚姻关系公开了，你和我以后都是公众人物，我们的婚姻不能出差错，当然也不会出差错。作为公众人物会有很多烦恼，且一定要以身作则，积极向上。”寒伯安轻抚安男的发稍。
<br/>
<br/>        “我知道的，我以后一定会当好寒太太，不托阿琪的后腿。”作为安男，和寒伯安相恋很多年，且已经住在一起了，如果一直让她默默的站在寒伯安的身后，她也是愿意的，只是作为女人，谁不想光明正大的牵着自己老公的手出现在众人的面前？
<br/>
<br/>        寒伯安每次出席重要的宴会以前都是别人陪着，安男知道寒伯安不会对那样的人动心，但是心里多少还是有点不舒服，总感觉自己家的东西被别人占用着，却又不能多说什么，她并没有想到寒伯安会带她出席年会，并在年会上公开向她求婚，虽然只是一个仪式，但是还是给她很大的惊喜。
<br/>
<br/>        “不，是我们携手前进，一生如共！今天你会见到寒家的各种亲戚，以后你也会要面临着和他们打交道，必要的应酬还是要有的，我怕你会受委屈，你今天就一直和我在一起，吃完晚饭我们就去杭州陪爸爸妈妈。”
<br/>
<br/>        “可以吗？”安男惊喜的问，她和寒伯安在一起，家里就是父亲和母亲了。
<br/>
<br/>        “当然可以，我们回去杭州也就两个多小时。”寒伯安轻点了一下安男的樱唇。
<br/>
<br/>        “冰玉过年没有回来吗？”安男是知道寒家的兄弟的个性。
<br/>
<br/>        “和落雪在欧洲，说是过二人世界，还让我带他向你问好，祝你永远十八岁。”
<br/>
<br/>        “落家和我们寒家也算是打断骨头连着筋的吧？”安男问寒伯安。
<br/>
<br/>        “落家的存在是一个奇迹，而落雪的撅起也是一个奇迹，虽然有我弟弟辅佐，但是落雪能在短短的时间内让大家认可，而且神不知鬼不觉的做了那么多事，不得不让人佩服她的心智和谋略。对了，她还是陈子昂的恩人。”寒伯安评论落雪的时候看着安那微微皱着的眉头。
<br/>
<br/>        “她和陈子昂怎么会有联系？”安男不解。
<br/>
<br/>        “陈子昂初来北京，身无分文，流落街头的时候被落雪看到了，落雪将陈子昂带回家，这一带就成就了一个商界黑马，而这个黑马却和你我又扯上了关系。”寒伯安拉着安男的手走向大厅，大厅里人来人往。
<br/>
<br/>        “缘分还真是奇妙。”安男也是觉得不可思议。
<br/>
<br/>        “只能说这个世界上的事情一切皆有可能。不过我才发现陈子昂为什么会如此优秀。”
<br/>
<br/>        “为什么？”
<br/>
<br/>        “因为你们留着同样的血液，你们有着一样优秀的基因。”
<br/>
<br/>        “你这夸人的功夫日夜渐长，俩个人都被你夸了。”俩个人说说笑笑的走到宴会大厅，大家族的仪式也是很繁琐的。
<br/>
<br/>        寒伯安带着安男一圈下来，安男穿着高跟鞋的脚已经累得不行了。
<br/>
<br/>        寒伯安带着安男走到他的父亲面前：“爸，我和安男要去杭州，我们先走了。”
<br/>
<br/>        “这么晚去杭州，大家都还没有走呢。”爸爸倒是没有说什么，旁边的小阿姨开口了。
<br/>
<br/>        “阿姨，我和我爸爸妈妈说好了，晚上回去陪他们吃饭，要不就让伯安留下来，我先回去。”安男对着小阿姨说。
<br/>
<br/>        “这个......”
<br/>
<br/>        “伯安，那你和安男去吧，路上开车小心点。”寒老爷子发话了。
<br/>
<br/>        “谢谢叔叔。”安男礼貌的笑了一下。
<br/>
<br/>        “带我向你爸爸妈妈问好。过几天我去拜访他们。”寒老爷子对于安男的这门亲事是非常满意的。
<br/>
<br/>        不说安男的家庭条件，单说安男的容貌和气度及机敏还有对寒伯安的那份爱，就足够让他满意了。
<br/>
<br/>        为了不让寒伯安和他小阿姨起冲突，主动将矛盾过度到她的身上，化解了一场尴尬。
<br/>
<br/>        能有这份心怀的女子怕也不多，寒老爷子不是糊涂人，只是对于自己的家事只是睁一只眼闭一只眼而已。
<br/>
<br/>        寒伯安拉着安男的小手和父亲打了声招呼转身走了。
<br/>
<br/>        到了自己的家里，帮着安男将鞋子脱下来：“以后还是少穿高跟鞋应酬，太累了。”
<br/>
<br/>        “我的习惯呢，我可不希望以后有别人挽着你的胳膊。”安男笑着说。
<br/>
<br/>        “那我以后尽量不去应酬了，我们下班就在家里。”寒伯安替安男揉了揉脚，套上了一双低跟的靴子。
<br/>
<br/>        俩个人开着车赶到安家时，安林曦和代凉还没有开始吃年夜饭。
<br/>
<br/>        看着俩个孩子回来心里一喜，原来安男都是在家里，今天安男去了寒家，虽说女儿总归是要嫁人的，真的到了有一天不回家过年的时候，俩个人心里也还是有那么一丝失落的。
<br/>
<br/>        寒伯安和安男看着满桌子香气扑鼻的饭菜，使劲吸了吸鼻子：“好香啊，我们饿了。”
<br/>
<br/>        代凉喜滋滋的摆上了两套餐具，一家四口其乐融融的吃起了年夜饭。
