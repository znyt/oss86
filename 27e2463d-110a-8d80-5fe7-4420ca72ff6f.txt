　　“这……！”
<br/>　　“这是怎么回事？”
<br/>　　“不会吧？”
<br/>　　一众巫族的天才，看清了玉盘上的情况，顿时一片哗然，纷纷面面相觑了起来。
<br/>　　因为此刻，玉盘上大量的红点，有一半都是集中在一起，并且是在快速的移动。
<br/>　　其移动方向，正是他们所在的方向。
<br/>　　这让一众巫族天才惊讶不已。
<br/>　　这种玉盘，乃是他们巫族特制的测魂盘，专门探查武魂和战魂。
<br/>　　而且只有武魂等级达到天级的人，并且修为较高之人，测魂盘上才会显示。
<br/>　　也就是说，这测魂盘上的红点，正是苍穹世界的巅峰强者。
<br/>　　其中一大半的红点，零散的分部在整个测魂盘的各处区域，而有一小半的红点，居然向他们而来。
<br/>　　而且，这一小半的红点，明亮异常，远超其他零散的红点，说明这些人最为强大。
<br/>　　这是要干什么？
<br/>　　难道这些苍穹世界的土著，居然如此自不量力？还想主动来和他们硬碰硬？
<br/>　　这让他们都惊讶了，因为按照族中之人所说，以前的试炼，苍穹世界的土著，可不敢反抗，只会一味的逃命。
<br/>　　现在居然有近百人，直接向他们杀了过来，这可真是稀奇！
<br/>　　“有趣!”
<br/>　　巫天御的嘴角勾起了一丝笑容，如此的话，倒是给他们提供了方便，不用再去一个个寻找了。
<br/>　　“娴妹，看来这些土著知道无法活命，想要拼死一搏了！”巫天御向身边的皮裙女子笑道。
<br/>　　皮裙女子名为巫娴，不仅是大祭司的曾孙女，也是巫族天冥支脉中，武帝境之下的后辈中，仅次于巫天御的天才。
<br/>　　“土鸡瓦狗而已！”巫娴不屑的道，完全没将苍穹世界的强者放在心上。
<br/>　　“那就先解决他们吧！”巫天御点头道。
<br/>　　随即，巫天御吩咐三百余人，分散而开，前往苍穹世界各地，前去诛杀那些分散的苍穹世界强者。
<br/>　　而他和巫娴两人，则是率领剩下的六、七百人，迎着那些快速到来的红点而去。
<br/>　　……
<br/>　　广遨无边的长空之中，苏莫率领一众苍穹神宫的强者，一路急行，向南洲的方向疾驰。
<br/>　　众人都是强者，不到两日的时间，他们便飞出了中洲的地界。
<br/>　　到达了与南洲交汇之地，死亡之海。
<br/>　　死亡之海方圆数百万里，是苍穹大陆的一处内海，同样也是一处禁地，传闻是远古大战之时所造成。
<br/>　　海中有很多怨气所化的凶灵，对于一般的武者来说，无比的危险，但对于苏莫等人来说，没有丝毫的威胁。
<br/>　　“等等！”
<br/>　　飞驰之中，苏莫突然一摆手，吩咐众人停止飞行。
<br/>　　“宫主，怎么了？”
<br/>　　“苏莫哥哥，怎么回事？”
<br/>　　众人停了下来，见苏莫面色凝重，均是心中有些疑惑。
<br/>　　“他们发现我们了！”苏莫眸中爆射出璀璨的绿光，直射前方，他已经将幽冥魔瞳武魂催动到了极致，看到了数十万里之外，有数百名年轻人向他们飞了过来。
<br/>　　如今的苏莫，实力暴涨了不知道多少倍，幽冥魔瞳武魂全力催动之下，足可以查探到数十万里之外的情况。
<br/>　　在他的视线之中，这数百名年轻人，个个气息强大无比，如同惊涛骇浪，直冲苍穹，没有丝毫的掩饰。
<br/>　　而在这些人眉心之间，都有着和巫森同样的三叉戟印记。
<br/>　　显然，这些人便是巫族的试炼者。
<br/>　　“发现了？”众人闻言心中一惊，他们本来还想伺机而动，个个击破，擒拿和击杀那些分散的试炼者，现在居然被发现了！
<br/>　　如此的话，此法便行不通了！
<br/>　　“宫主，我们怎么办？”海明江急声问道，其他人眸中亦是露出焦急之色。
<br/>　　“只能正面一战了！”苏莫声音低沉的说道，他们现在想躲藏已经来不及了。
<br/>　　而且，这些试炼者显然有办法查到他们的踪迹，躲藏也没有什么用！
<br/>　　“正面一战？”众人闻言心中一震，随即纷纷深吸了口气，既然苏莫要正面一战，他们自然不会退缩。
<br/>　　他们已经抱着不成功便成仁的打算了，即便是死，也要尽力一博。
<br/>　　苏莫面色郑重，眸中绿光幽幽，他之所以要正面一战，也是有些底气的。
<br/>　　因为，他发现这些试炼者之中，没有武帝强者，基本上全部是准帝和武皇境九重武者。
<br/>　　所以，他才敢留下来正面一战，他相信，这些巫族的试炼者就算战力再强，不到武帝之境，也不可能打败他。
<br/>　　他对于自己的实力，有着绝强的自信。
<br/>　　随即，苏莫便伫立在虚空中，一众强者站在他的身后，静等巫族试炼者的到来。
<br/>　　死亡之海，要么是他们的死亡之地，要么是巫族试炼者的死亡之地！
<br/>　　嗖！嗖！嗖！！
<br/>　　少倾之后，光芒闪烁，破空声不断的响起，数十万里的距离，对于一众巫族天才而言，不过是片刻之间的事情。
<br/>　　六、七百名巫族天才，转瞬即到，而后身形停在了苏莫等人前方千里之处。
<br/>　　“不错，居然有十几名准帝！”巫天御打量了一番苏莫等人，微笑着点了点头。
<br/>　　他本来还猜测，孱弱的苍穹世界，能有几名准帝就不错了，没想到居然一次性就出现了十几名。
<br/>　　他神情淡然，显然没将苏莫等人放在眼中，虽然同样是准帝，但苍穹世界的准帝，对于他而言，不过是蝼蚁般的存在。
<br/>　　“一群蝼蚁！”巫娴面带讥讽之色，就眼前这近百人，还想对抗她们？真是自寻死路！
<br/>　　“哈哈哈！真是一群白痴！”
<br/>　　“看他们的模样，是要与我们拼命了！有意思！”
<br/>　　“土著就是土著，真是不知死活啊！”
<br/>　　“……！”
<br/>　　一众巫族天才，见到苏莫等人之后，顿时哈哈大笑了起来，个个面带轻蔑之色。
<br/>　　没有人将苏莫等人放在眼中，他们一路到达此地，路上也顺便击杀了几名苍穹世界的强者，实在是太弱了，弱到他们提不起兴趣。
<br/>　　苏莫眼眸微眯，静静的打量着为首的巫天御和巫娴，眸中显出了一丝凝重之色。
<br/>　　这两人极度强大，尤其是巫天御，给他的感觉深不可测，比之当初武帝境的巫森，都强大了不知道多少倍！
