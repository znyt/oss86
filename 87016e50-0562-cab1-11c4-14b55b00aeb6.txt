　　第343章
<br/>　　目前程大雷手下大概分为这样几波人。士兵训练方面由关鱼负责，程大雷偶尔管管；后勤方面由刘悲负责，程大雷偶尔管管；军事战略方面由徐神机负责，主要是吴用管理，程大雷得经常盯着。
<br/>　　落玉寨是由樊梨花盯着，蛤蟆寨轮值，每十五天换一支队伍驻扎。
<br/>　　后勤又分为商业组、医疗组、工匠组。工匠又包括铁匠、木匠、泥瓦匠、裁缝等等。
<br/>　　程大雷把工匠组的负责人叫了过来，这人名叫郭莱，是程大雷召唤而来的生活职业，原先是个铁匠，忠诚是绝对不成问题的。
<br/>　　程大雷站在院子中，双手持弩，二十步外一棵杨树，程大雷瞄准射击。
<br/>　　连发十弩，准头略差些，只有四箭射中，箭头完全镶入大树。
<br/>　　威力还算可以，只是对自己的准确度程大雷有些不太满意。
<br/>　　“瞧瞧看看，这东西工匠坊能不能拿出来。”程大雷冲郭莱道。
<br/>　　郭莱接过连击弩，仔仔细细检查一遍，道：“大当家，这武器威力这么大，你从哪里搞到的。”
<br/>　　见识过前生枪炮的威力，这连击弩在他眼里实在算不上有威力。只不过郭莱这个问题，却是让程大雷不好回答。
<br/>　　“这个……那个……”
<br/>　　“啊，我知道了。一定是神仙梦中传授，大当家放心，我不会说出去的。”
<br/>　　看来黑牛献宝的事情，郭莱也知道了。程大雷点点头：“嗯，要注意保密。”
<br/>　　“一定，一定。”郭莱道：“这东西我要拿回去让所有人一起研究，看上去构造并不复杂，应该可以仿制。”
<br/>　　“嗯，好好做，真拿得出仿制品，你们都有赏。”程大雷挥挥手。
<br/>　　都说系统出品，必属精品，但以山贼之心的尿性来看，还真不能这么说。
<br/>　　只希望自己手下的工匠组可以给力些，这样自己也能省点恐惧值。系统周扒皮的性格，程大雷有些难以承受。
<br/>　　总得来说，这次占领琴川关后，程大雷的收获还是颇丰的。
<br/>　　自己搞到了坐骑和全套橙装，实力升级到绝世，又开出了连击弩这样的杀器。
<br/>　　这要让程大雷自己建造，还不知道要花多少力气，多少时间才能将山寨生到四级。
<br/>　　看来对一个山贼而言，抢劫果然比按部就班发展快呐。
<br/>　　想到这里，程大雷忽然心中一动。
<br/>　　琴川关其实不算大，这里重要在战略位置。和那些大城相比，琴川关只能算个中型城镇。
<br/>　　但占领琴川关就能给自己带来这样的收获，那么如果攻打一座大城，会不会直接越级提升。
<br/>　　程大雷现在手上有一万余人，十八家头领的人加起来，超过两万。如果按照程大雷计划的发展，只需要半年时间就能成为一股精兵。
<br/>　　两万精兵想打一座城池，未必打不下来。
<br/>　　当然，目前还是安稳发展重要，不适宜出头，太出头的话，容易遭雷劈。六字真言铭记在心：猥琐发育别浪。
<br/>　　就算现在的发展，程大雷已觉得有些力不从心，重点还是人手不够。山寨还没有建立健全的组织，也缺乏能总揽全局的管理人才。有很多人都是一人身兼数职，就连程大雷也得经常照看着。
<br/>　　三军未动，粮草先行。这里的粮草不单指人吃马嚼，主要是指后勤。
<br/>　　古往今来，能领兵打仗的猛将不缺，缺的恰恰是心细如发，管理照应一切的人才。
<br/>　　系统还奖励一次抽奖机会，程大雷今日连番收获，感觉手颇顺，在把事情照应开后，直接选择抽奖。
<br/>　　扭蛋机再次在脑海中出现，还未开始转动，程大雷先收到一条系统提示。
<br/>　　嘟，正义的少年喔，你已拥有四级山寨，可以选择抽奖范围。
<br/>　　“抽奖可以选择范围了？”程大雷怔了怔
<br/>　　“可选范围有：装备，技能，人才，其他？”
<br/>　　程大雷只是随口道了一声，却没想到，竟得到了系统回应。程大雷发觉，自从自己用隐藏属性向系统交易后，系统好像比以前有了些灵性。
<br/>　　“选择抽取人才，抽奖。”
<br/>　　扭蛋机开始选择，随之程大雷便收到一条系统提示。
<br/>　　嘟，获得绝世文官——和珅。
<br/>　　咚！
<br/>　　程大雷差点倒在地上，心脏砰砰直跳，险些跃出胸腔。我就知道这个召唤人才不能随便用，看吧看吧，果然出事了。把这古今第一贪官召唤出来了。
<br/>　　话说刚出现一个没多大用的吴用，有出现一个古今第一大贪，都说物以类聚人以群分，最近自己的人品出现了什么问题。
<br/>　　会不会是自己的隐藏属性『一个好人』丢了，所以召唤到的都是这些反面角色。
<br/>　　程大雷隐隐觉得自己的推测有些接近真相，本来刘关张这些人怎么说也是忠臣良将，可吴用与和珅这种人就很难让人有什么好观感了。
<br/>　　和珅的能力程大雷是毫不怀疑的，没有能力能成为满清第一巨贪么。可他的存在也实在令程大雷不寒而栗。
<br/>　　程大雷已经决定，这货敢来投奔自己，直接一刀杀了，就算浪费一次抽奖机会也在所不惜。
<br/>　　因为程大雷自量凭自己这点心眼，可真玩不过对方。
<br/>　　咚咚咚！
<br/>　　外面响起敲门声，徐神机爽朗的笑声响起，程大雷走出书房看到徐神机领着一个人站在大厅。
<br/>　　“大当家，你看谁来啦，这可是我的忘年交……”
<br/>　　程大雷见徐神机身边这人，三十岁左右年纪，面白无须，看上去平平无奇。
<br/>　　“你和他早就认识？”程大雷。
<br/>　　“是呐，小时候我还抱过他呢。大当家，莫看他年纪小，可是个大大有用的人才，人老实，还仗义。”
<br/>　　估计也就徐神机这样的憨货会用老实仗义形容对方。得，这人设就来了，还和徐神机扯着关系。
<br/>　　据徐神机所说，此人是他的同乡，古道热肠，有贤才之名。这次是特意拖家带口来投奔徐神机的。
<br/>　　呵，还附赠老婆孩子。
<br/>　　程大雷若是不知道，还真被他唬了，实在对方长了一副值得信任的相貌。可程大雷在系统加持下，早就一眼看穿。
<br/>　　和珅，字致斋，倒不是戎族。
<br/>
