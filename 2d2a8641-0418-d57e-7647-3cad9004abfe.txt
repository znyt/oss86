　　百里范围中，有些修士察觉到有些不对劲，甚至体内的灵气都不受自己控制，面面相觑，不由得心头大骇。

<br/>　　“难道天要塌了不成，如此下去，我们体内的灵海都会枯竭！”
<br/>　　有修士绝望的破口大骂，那些本就弱小的修士，因为灵气的流逝，身影在空中不断挣扎，一副摇摇欲坠的模样。
<br/>　　天空中的灵气组成灵气狂潮，连同着修士们体内的灵气，霸道的掠夺了过去。

<br/>　　如此浩大的声势，未免太霸道了，直接将天地间的灵气都抽取一空，甚至他们福地的灵气也都全部被吸走。

<br/>　　尤其是前面合道期的宗主，他当初就是有过奇遇才有今日的成就，对自己的猜测已经有了一定的自信。
<br/>　　此时，心中惊骇尽是化作惊喜，浑厚的声音传遍宗门。
<br/>　　“方圆百里内尽是霸刀门的范围，如此异象，定是有至宝出世，在百里范围内，至宝就是霸刀门的机缘，门派长老都随本座来。”
<br/>　　霸刀门数十位长老闻言，顿时面露狂喜，难道真的是至宝出世？
<br/>　　当其中一名练虚期长老认真感应了一番，发现天地间的能量确实是朝一个方向汇聚过去的，霎时间眼神一亮。
<br/>　　他们虽是长老，但年纪已经不小，练虚期已经是他们的极限，若是有天大的机缘，可能就是他们一飞冲天的机会啊。
<br/>　　在场长老各怀心思，都是心头火热，脑海中闪过无数的念头。
<br/>　　酆都城后方，空间结界中，无尽的能量不断的涌入慕清霄体内，经过《御女心经》的运转，纷纷化作阴阳灵气。
<br/>　　同时，他的实力也在一步升华，气息节节攀升，更加的出尘浩瀚。
<br/>　　大乘四重初期，中期，巅峰……
<br/>　　此时，慕清霄沉浸在一个奇妙的状态中，丝毫不知晓，自己突破带来的声势，已经将方圆百里的修士给吸引了过来。
<br/>　　“呼呼……”
<br/>　　顷刻间，慕清霄气息骤然提升，体内响起一阵闷沉如雷的声音，血肉，筋骨，细胞都泛起一抹金色光泽。
<br/>　　旋即，一团诡异的黑色火焰自他体内席卷而出，四周空间在瞬息就化作一片虚无……
<br/>　　相对于先前，慕清霄明显感觉到，他体内的黑凤血脉，似乎完全与自己的肉身融合，各方面都有所提升，特别是肉身。
<br/>　　“好强的感觉，大乘期七重初期，哪怕渡劫期修士也丝毫不惧。”
<br/>　　眼眸微闭，慕清霄嘴角勾起一抹满意的弧度，将火灵珠炼化，顺利的提升了三个小境界，从大乘四重提升到了大乘期七重。

<br/>　　当初在古藤林吸收土灵珠后，慕清霄就觉得体内的阴阳灵气似乎更精纯了些，而且在向某个方向转变。
<br/>　　在雷州吸收雷灵珠后，那种感觉就变得更加清晰。
<br/>　　直到刚才，将火灵珠完全炼化后，慕清霄觉得这根本就不是错觉，比起以往，不管是肉身还是灵气，都更加精纯了。

<br/>　　就在神火收回体内后，慕清霄剑眉一挑，目光一凝，脸色一瞬间就变得阴沉了下来。
