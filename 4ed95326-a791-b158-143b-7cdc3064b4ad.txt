　　望着两人快步离开的背影，屋内的三名妇人和少年都呆在原地，半响没能反应过来。
<br/>　　无论是一名少校的恭敬态度，还是两人对话间包含的信息量，都让在场四位安罗城最普通的居民无所适从。
<br/>　　这个他们每天一起打麻将的袁阿姨，怎么就被一名少校请走了呢？
<br/>　　“这小袁，是谁啊？”少年的母亲呐呐出声：“我好像听到那军官喊她袁熙叶院长啊？”
<br/>　　“袁熙叶？”少年猛地惊醒过来，而后面色大变：“那是楚涵上将的母亲！”
<br/>　　哗——
<br/>　　屋内的几名妇人都感觉眼前一黑，而后便涌·出了一种颇为不真实的感觉，楚涵上将的母亲，那岂不是，岂不是这狼牙地界最尊贵的人吗？
<br/>　　这种人物怎么会在这里，跟他们一起打麻将？
<br/>　　此时的少年又有些想不通，皱眉道：“但是据说楚涵上将的母亲一直在狼牙研究部担任院长，每天研究项目很繁忙，怎么会在安罗城还有空闲玩乐？”
<br/>　　“人家或许是休闲一下呢？”一名妇人随口猜测道：“眼下这有事，不就立即被请走了吗？”
<br/>　　“大人物就是大人物啊！处理的事情都是紧急事件，还是一名少校亲自来找的。”
<br/>　　随着一声声的感叹，少年却想起了袁熙叶之前的那番话，如果只是出自一名普通妇人之口，仅仅是能打动人心而已，可如果说出这番话的人是楚涵上将的母亲袁熙叶，那么意义就将大不一样。
<br/>　　来到安罗城主府的袁熙叶马不停蹄，立即进入了安静的屋内查看信件，年轻的少校知道这座城市不需要他多此一举的守卫，便抓紧时间休息，毕竟第二天他还需要前往战区一线。
<br/>　　尚九娣听闻了消息后也火速赶来，当她进入屋内的时候，却看到袁熙叶正坐在那里沉思。
<br/>　　“发生了什么事？”尚九娣微微喘着气，显然是一路狂奔而来。
<br/>　　袁熙叶的面上带着一丝严肃：“南方边境的尸潮出现了一些异样。”
<br/>　　尚九娣顿时紧张起来：“第二批尸潮已经开始进攻了吗？战争开始了吗？”
<br/>　　“这倒不是。”袁熙叶摇着头，道：“从信息上来看，第二批尸潮已经抵达了边境，但很奇怪的是它们没有发起进攻，而是就此散开徘徊在段氏基地附近。”
<br/>　　“啊？”尚九娣一愣，下意识道：“这不可能啊，尸潮一旦形成，就必然有进攻的目标，那是丧尸的本能，怎么可能放着段氏基地那么强烈的热能量无视，改为在附近徘徊呢？”
<br/>　　“所以才说有异样。”袁熙叶说着，紧接着道：“有关尸潮的形成及常规，可有具体资料？”
<br/>　　“有。”尚九娣连忙从一堆文件中翻找，同时皱眉不解道：“但这些对于您来说不都是早已知道的常识吗？那些资料都是早期观察的客观描述，没有定论的，还不如您的推理呢！”
<br/>　　“我要的就是客观描述。”袁熙叶笑道：“有关尸潮的情况，狼牙战团每一次对抗丧尸后的战报，都有详细描述，我需要的就是这些所有战报的数据。”
<br/>　　尚九娣虽然不解，但还是依照吩咐，将所有档案都找了出来。
<br/>　　这一夜，安罗城主府内的某件屋内彻夜灯火通明，袁熙叶和尚九娣就在这里将末世爆发至今的所有尸潮资料，都翻阅了一个遍。
<br/>　　当清晨的阳光刚照在窗户上的时候，袁熙叶这才开始动笔，在此过程中她没有说任何话，更没有与尚九娣讨论。
<br/>　　尚九娣静静的待在一边看着，她陪同袁熙叶将所有资料翻阅过后，并没有查到一星半点的有用信息，所以她也没能理解袁熙叶要这些资料有什么用。
<br/>　　袁熙叶很快写好，将一封信递到尚九娣手中道：“去交给那个来自上京基地的少校。”
<br/>　　——————
<br/>　　上午时分，段氏基地照例每天的繁忙工作，围聚在这处基地的猎者们也照常出动。
<br/>　　此时来自狼牙基地的一架直升机，却冲入了众人视野，年轻的少校快速的走下机舱，直奔段江伟所在的方向而去。
<br/>　　段江伟收到那封信的时候，表情相当古怪：“袁熙叶的亲笔信件？”
<br/>　　“是。”少校神情严肃：“院长昨夜查了所有资料，今天清晨才写完让我送来的。”
<br/>　　“进来。”段江伟转身带着人进入最为封闭的办公室，确保屋内都是自己人后，他才打开那封信极其认真的阅读起来。
<br/>　　屋内除了那名跟进来的少校，还有鲁初雪等一干在段氏基地颇为重要的高层，此时他们都极其安静，等待段江伟的开口。
<br/>　　“信上让我们小心有诈。”段江伟用一句话概括了这封信的内容。
<br/>　　屋内的人都一愣，显然对这没头没脑的话表示不能理解。
<br/>　　段江伟将信拿起，神色认真的继续道：“这封信提供了这些年来狼牙战团遇到每一次尸潮或是丧尸群的战斗数据概括，并且得出了一个概率惊人的结论，那就是已经聚集的丧尸群除非有丧尸头领的命令，否则根本不可能凭白四散，更不可能在一个区域内不停的游荡。”
<br/>　　鲁初雪愣愣出声：“可以段氏基地周围的那些丧尸，就是在游荡徘徊啊？”
<br/>　　“所以附近这些丧尸中没有丧尸首领是不可能的。”段江伟的眼神犀利：“不仅有，恐怕还是一只拥有智慧的超级丧尸，并且是一个极为恐怖的存在。”
<br/>　　众人纷纷警惕起来，很多问题也浮现在明面上。
<br/>　　“可是这只丧尸为什么要这么做？明明已经带着丧尸群来到了附近，却偏偏没有行动，让丧尸们都散开，当下被猎者围杀的丧尸数量可不少啊，这么下去的话，这尸潮的数量不就越来越少了么？”
<br/>　　“既然它有智慧，还是很高的智慧，就不可能不清楚会有这种结果才对。”
<br/>　　“这些丧尸群聚集在一起才是最可怕的力量，几百万头齐动，眼下的段氏基地岌岌可危，为什么偏偏没有进攻？”
<br/>　　段江伟眯起眼睛：“如果说丧尸并不知道眼下的段氏基地很好攻打，只是因为前一场大尸潮战役的战败而变得警惕起来，那么……”
<br/>　　“是在等同伴！”鲁初雪语出惊人，更是猛然大喊出声。
