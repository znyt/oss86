　　翌日一早，苏莫再次的来到了人德宫主的宫殿。
<br/>　　宫殿之中，人德宫主面带威严，已经在等他了。
<br/>　　“拜见宫主！”苏莫向对方见礼。
<br/>　　“苏莫，这是给你的五行源石，共十五亿中品源石，五种属性每种各三亿！”人德宫主坐在主座之上，手掌一抛，一枚戒指便向苏莫飞了过来。
<br/>　　这枚戒指，通体雪白，其上刻画着无数的纹路。
<br/>　　苏莫接住戒指，意念一探，不由得心中暗喜。
<br/>　　这是一枚空间戒指，品质很高，其内是一片广阔的大地，大地之上耸立着五座大山。
<br/>　　五座山峰，赫然是由五种源石堆积而成，闪烁着五色的光芒，照耀八方。
<br/>　　“宫主，大恩不言谢，日后有机会必报！”苏莫满脸郑重的说道。
<br/>　　对方给予他的帮助，可谓是雪中送炭了。
<br/>　　“不求你报答什么，但愿老夫没有白白付出！”人德宫主摇了摇头道，他不指望苏莫能给予什么报答。
<br/>　　对于他来说，帮助苏莫，也是在帮助轮回天尊，又是在帮助整个荒界。
<br/>　　无论苏莫将来是死是活，只要能成长到更高的地步，都是有益的。
<br/>　　毕竟，就算苏莫将来，被帝一魂融合了，苏莫越是强大，对于帝一魂的帮助，也就更多一分。
<br/>　　当然，若是苏莫真的能创造奇迹，后来居上，超越帝一魂和拓跋青云，那自然更好。
<br/>　　只不过，这个可能性，基本上可以忽略不计。
<br/>　　言罢，人德宫主又道：“今日，老夫的孙女姬月沁，会外出历练，你随他一同出去！”
<br/>　　“和沁儿姑娘一起出去？”苏莫闻言疑惑，这好像不太合适吧！
<br/>　　和沁儿一起出去，还不如他自己动用天幻神丹，然后混出人王宫。
<br/>　　“当然，并不是让你光明正大的出去！”
<br/>　　人德宫主知晓苏莫的心思，他手掌一翻，掌中出现了一个小玉瓶，与上次给予苏莫丹药的玉瓶类似。
<br/>　　随即，人德宫主微笑道：“这玉瓶是一间简单的空间宝器，其内有一些资源，是给沁儿之物，你便藏身在其中，随沁儿一起出宫吧！”
<br/>　　对于安排苏莫出去，人德神王也是煞费苦心了。
<br/>　　毕竟，人王宫乃至人王城，都在监视之下，不得不小心谨慎。
<br/>　　就算苏莫有他炼制的天幻神丹，也不保险，所以还是小心谨慎为妙。
<br/>　　那玉瓶之中，不仅有资源，还有掩盖气息，阻挡神念探查的阵法，就算是一般的神王，都不会发现苏莫在其中。
<br/>　　“这……到时候方便离开吗？”苏莫有些迟疑的问道，他倒是没想到，宫主会让他用这种方法离开。
<br/>　　“玉瓶只是最简单的空间宝器，以你的修为，可以直接出来，不过，你不能向沁儿和其他人透漏身份！”人德宫主叮嘱道。
<br/>　　“明白！”
<br/>　　苏莫点了点头，随即又道：“那沁儿姑娘不会发现我藏身在其中吧？”
<br/>　　“不会！”人德宫主摇了摇头，沁儿资源多的用不完，根本不会检查给予了她多少资源。
<br/>　　再说了，玉瓶之中还有遮掩气息的阵法，沁儿不会发现的。
<br/>　　“那就好！”苏莫点了点头。
<br/>　　“那你进来吧，沁儿片刻之后就到！”人德宫主吩咐道。
<br/>　　“嗯！”苏莫点头，随即神念一动，包裹住玉瓶，轻易的便进入了没有多少防护的玉瓶之中。
<br/>　　玉瓶内的空间不大，只有方圆数十里，苏莫进来之后，立刻动用了天幻神丹，并且隐藏了起来。
<br/>　　毕竟，为了更好的隐藏，他连以前的隐身战魂都动用了。
<br/>　　虽然只是战魂，并不是元神，作用不大，但总归有些作用。
<br/>　　……
<br/>　　待苏莫进入玉瓶之后，人德宫主在殿中等待。
<br/>　　约莫半刻钟之后，沁儿来了，款款的走进了宫殿之中。
<br/>　　“沁儿见过爷爷！”沁儿向人德宫主行礼，并没有像面对玄天长老那般随意。
<br/>　　“沁儿，听说你过几日准备外出历练，爷爷给你备了一些资源，你拿去吧！”人德宫主面带微笑，拿出了玉瓶。
<br/>　　“爷爷，我资源多的是，不需要了！”沁儿笑道，他的身上，资源根本就用不完，哪里还需要资源啊！
<br/>　　“你这丫头，谁还会嫌资源多呢？”人的宫主莞尔一笑，随即手掌一抛，玉瓶便向沁儿飞了过去。
<br/>　　“那好吧！”沁儿接过玉瓶，也没有多想，便受了起来。
<br/>　　至于检查玉瓶，她压根就没有想过，玉瓶内资源的多少，她根本就不在乎。
<br/>　　“嗯，你回去吧，切记在外注意安全！”人德宫主叮嘱道。
<br/>　　“爷爷放心，有月松哥和我一起，不会有事的！”
<br/>　　沁儿笑着说道，言罢，她略一沉吟，目光紧紧的望着人德宫主，道：“爷爷，段惊天真的不会回来了吗？”
<br/>　　关于段惊天的消息，她询问过玄天长老数次，得到的答案均是段惊天可能不会再回来了。
<br/>　　这让她有些无奈，这么厉害的万古天骄，居然就这样走了。
<br/>　　“或许吧！谁又能知晓呢！”人德宫主摇了摇头，没有过多解释。
<br/>　　“好吧！”沁儿无奈的叹了口气，随即没有在多言，便告辞离去。
<br/>　　待沁儿离去之后，人德宫主立刻召玄天长老来见，他还要做些安排，以掩人耳目。
<br/>　　苏莫在玉瓶空间之中，可以监察到外界的一切。
<br/>　　他看到，沁儿回到了月沁园之中，便坐在院中的凉亭中发呆。
<br/>　　凉亭之前，是一个不大的水塘，水塘中有成群的鱼儿游荡嬉戏，荡起微波连连。
<br/>　　“段惊天，你会去哪里呢？”沁儿低声自喃，一对柳眉微微翘起。
<br/>　　“还会不会去时空界域？”沁儿想到了这个可能，但是也只是有这个可能而已。
<br/>　　玉瓶空间之中，苏莫听到了沁儿的自语，不由得心中暴汗。
<br/>　　这个姑娘，居然在猜测他去了哪里？
<br/>　　叹了口气，苏莫便不再关注沁儿，在玉瓶空间之中，修炼了起来。
<br/>　　一日无话，第二日，褚彦、骆妤和姬月松三人，来到了月沁园。
<br/>　　而后，四人一起，离开了人王宫，向天池神朝的方向而去。
<br/>　　今日，人王宫之中，出宫的弟子比平常多不少。
<br/>　　几乎是成群结队的弟子，外出历练，仅仅是半日的功夫，便有不下五千名弟子外出。
<br/>　　不仅如此，玄天长老也离开了人王宫，向天星神朝的方向而去。
<br/>　　“奇怪！”
<br/>　　人王宫之中、人王宫之外，人王城各处，有数十双眼睛，在时刻注意着人王宫的动向。
<br/>　　见到这这么多弟子外出，这些人不由得心生疑惑。
<br/>　　随即，这些人立刻将消息传到了天王宫。
<br/>　　天王宫之中，天尘宇端坐在一间修炼室之中，面带沉吟之色。
<br/>　　他收到了一众探子的消息，自然也觉得事有蹊跷，苏莫很可能蒙混出去。
<br/>　　但是，面对此情此景，他却是感觉无从下手。
<br/>　　难道将每个外出的人王宫弟子拦截，检查一番？
<br/>　　这显然不现实，不仅人手不够，也不能这么做。
<br/>　　“玄天长老！”天尘宇眸中神光一闪，此人离开了人王宫，是最有可能带着苏莫离开的。
<br/>　　看来，他要亲自出去一趟了，会一会名震神域的玄天神王了。
<br/>　　同时，他也立刻传讯，命令那些探子，去跟踪那些可疑的人王宫弟子。
<br/>　　比如，身为玄天神王之女的姬月沁，此女和苏莫有些关系，也有可能帮助苏莫离开人王宫。
