　　美国，马里兰州米德堡。
<br/>　　从NSA总部离开之后，影天使没有返回自己的住处，而是直接赶往机场。
<br/>　　影天使拿出一部藏在汽车座椅夹缝中的手机，开机，然后拨通了一个电话号码。
<br/>　　电话很快就被接通，影天使戴上耳机，问道：“小丑，什么情况？”
<br/>　　“影天使，你的潜伏任务结束了。”小丑说道：“当初利用西伦科特的帮助，让你成功进入NSA的EG，就是为了获取病毒实验室的网络代码，现在东西拿到了，你也不需要继续再那里待着了。”
<br/>　　“我现在要去哪儿？”影天使问道。
<br/>　　“坐飞机去冰岛，那里会有人接应你，然后从那里转机，前往列颠国，到了那里会有下一步行动指示。”小丑说道。
<br/>　　“明确。”
<br/>　　影天使挂断电话，甩手把手机从汽车中扔了出去，摔落在马路上的手机，瞬间就摔裂开来，紧接着被一辆跟在影天使汽车后面的汽车，压得粉碎。
<br/>　　把汽车停放在机场停车场后，影天使打开汽车后备箱，从里面拿出一个行李箱，带着背包进入了停车场的厕所。等到影天使从停车场厕所出来的时候，那只行李箱已经不见了，而影天使也是改变了一副装扮，而他原本藏在背包中的手枪，也被丢弃在了厕所隔间的废纸篓中。
<br/>　　影天使利用假身份很快就购买到了一张前往冰岛的机票，并且通过了安检，进入候机区等待登机。
<br/>　　……
<br/>　　与此同时，中东。
<br/>　　两颗BGM-109D型“战斧”导弹以0.72马赫的速度，朝着意大利“眼镜蛇”特种部队和龙隐部队所在的方向飞去。两架F/A-18E“超级大黄蜂”也正朝着西伦科特藏匿的别墅飞去，战斗机驾驶舱中，飞行员打开了武器保险，准备投放JDAM炸弹对别墅进行毁灭性摧毁。
<br/>　　联合行动司令部中，终于有技术人员发现“战斧”导弹巡航轨迹遭到了改变，再次确认情况后，技术人员脸色紧张的站起身来，对汉密尔顿说道：“将军，十五分钟前，有人侵入了我们的GPS系统对两颗导弹的攻击坐标做出了修改。”
<br/>　　“你说什么！？”
<br/>　　原本脸上挂着胜利笑容的汉密尔顿从椅子上站起身来，不仅是汉密尔顿，其他国家的军事代表脸上的笑容也是瞬间凝固起来。
<br/>　　李然从座椅上站起来，快步走到技术员身边，问道：“导弹现在的攻击坐标是哪里？”
<br/>　　“是‘眼镜蛇’特种部队和‘东方龙’特种部队！”技术员脸色难看的回答道。
<br/>　　“立刻修改坐标！”李然命令道。
<br/>　　“来、来不及了！”技术员说道：“导弹已经抵达攻击区域上空，两分钟后，布撒型战斗部中的小口径炸弹就会对攻击坐标进行火力覆盖！”
<br/>　　“草！”
<br/>　　李然一脚推翻椅子，把耳麦切换到龙隐部队所使用的频道，用喊得方式对罗昊说道：“狂龙，立刻带人从你们现在的位置撤离，两分钟后，一颗携带了166枚BLU-97B小口径炸弹的‘战斧’导弹会对你们所在的位置进行火力覆盖，有人侵入了GPS系统，对‘战斧’导弹的攻击坐标进行了修改！”
<br/>　　而同样听到技术员话的Benito却是彻底慌了神，甚至忘记了通知自己的特种部队撤离，最后还是汉密尔顿对“眼镜蛇”特种部队下达了紧急撤离命令。
<br/>　　“黄蜂1-1确认目标，实施攻击！”
<br/>　　两架F/A-18E“超级大黄蜂”飞抵海豹六队行动小组的上空，确认坐标后，投放精确制导钻地炸弹。四枚炸弹呼啸着，向海豹六队行动小组潜伏的树林坠落下去。
<br/>　　听到李然从耳麦中传来的喊声，罗昊脸色顿时为之一变，抬头朝着天空中看去，他已经能看到蓝色的天空中，一颗“火流星”正朝着自己所在的方向坠落而来。
<br/>　　“撤离！所有人立刻撤离！”
<br/>　　罗昊大声叫喊着让所有人从树林中撤离出去。
<br/>　　而就在龙隐部队战术中队撤离的时候，BGM-109D“战斧”导弹的布撒型战斗部从导弹主体分离出来，在重力的作用下朝着地面坠落下去，在坠落的过程中，战斗部打开，藏在战斗部中的166枚BLU-97B小口径炸弹如天女撒花般朝着地面散落而去。
<br/>　　联合行动司令部中，李然眼睛紧盯着正前方巨型屏幕中某一刻方块，那个方块中的画面因为罗昊剧烈奔跑的关系，在不断地晃动着，李然双手紧紧的握成了拳头，他的指甲盖因为用力的关系而泛白。
<br/>　　突然画面发生了变化，镜头突然对准了天空，爆炸声从李然的耳麦中传来，令得李然心脏狠狠地抽搐了一下，耳朵嗡嗡作响。
<br/>　　隔了很长一段时间，画面也没有发生改变，画面中依旧是树林上方的天空。
<br/>　　李然想要向罗昊询问情况，张嘴，却发现自己喉咙口就像是被卡了一块骨头一样，发不出任何的声音来。
<br/>　　努力的咽下一口口水后，李然采用干涩的声音对罗昊问道：“狂龙？狂龙？狂龙！？”
<br/>　　最后一个“狂龙”，李然几乎是在嘶吼了，可是回答他的除了沉默还是沉默。
<br/>　　在龙隐部队遭到BGM-109D“战斧”导弹攻击的时候，意大利“眼镜蛇”特种部队和海豹六队行动小组也是遭到了攻击，瞬间，汉密尔顿和Benito也跟自己手下的行动部队失去了联系。
<br/>　　至于其他几个国家的特种部队，情况也好不到哪里去。在西伦科特从别墅撤离的时候，就命令自己的“影子部队”全部出动向各个国家特种部队所在的位置包围过去，所以那些没有遭到“战斧”导弹和精确制导炸弹攻击的特种部队，遭到了“影子部队”的袭击，在完全被动的情况下，遭到了袭击。
<br/>　　联合行动司令部中已经是乱成了一锅粥，各国的军事代表都在拼命呼叫着自己国家的行动部队，想要弄清楚情况。
<br/>　　李然扯掉塞在耳朵中的耳麦，转身朝着房间外走去。
<br/>　　“妖瞳！”汉密尔顿从背后叫住李然，问道：“你要去什么地方？”
