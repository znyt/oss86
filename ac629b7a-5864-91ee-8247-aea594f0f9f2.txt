　　　　不管这个蛇王是不是真的神灵，但现在对舒瑶来说，都没什么不能接受的，她已经无路可走了。
<br/>　　“说吧，你是怎么怀上蛇胎的”
<br/>　　涂小安对于这点特别的好奇，貌似怀孕，都是有一个过程，一个人与人之间互动的过程。
<br/>　　但她这个怀的是蛇胎，中间到底是个什么过程，就让人不敢往深处想了，想多了浑身都鸡皮疙瘩掉一地。
<br/>　　“我不知道，我真的不知道自己为什么会怀上蛇胎，我是跟我男朋友同居的，我们谈了三年的恋爱了，都已经见过双方的父母，本来这次的怀孕，就准备结婚，可没想到会变成这样”
<br/>　　舒瑶捂着头，痛苦的说着，全身搐动，一声声压抑的，痛苦的唏嘘，仿佛是从她灵魂的深处艰难地一丝丝地抽出来，散布在这庙堂之内。
<br/>　　她的样子顿时吸引了不少人的注意，渐渐的，大家都看向了她。
<br/>　　涂小安知道想要了解整件事情，不是三言二语说的清，在这个人来人往的地方显然不合适，不过无妨，他现在有一个忠诚的信徒可以使唤。
<br/>　　只见一身道袍的李顺像是突然得到了什么指示，立刻走到了舒瑶的面前，面带笑容道：“这位小姐，请跟我来”
<br/>　　“跟他去”
<br/>　　诡异的声音出现在舒瑶的脑海中。
<br/>　　舒瑶擦了擦眼泪，浑然不怕的跟了上去，最后来到了寺庙一间安静的房内。
<br/>　　李顺关上门，退了出去，守在了外面，不让任何人进去，同时他的脸上满是激动，双手攥拳：我居然能为蛇王做事，祖坟冒青烟了，真的是祖坟冒青烟了。
<br/>　　来自李顺的信仰之力增加200点...
<br/>　　房内，舒瑶坐立不安，一双眼睛四处的打量着，现在，越来越多稀奇古怪的事情都好像发生在了她的身上。
<br/>　　“不要紧张，我是来帮你的”
<br/>　　一条灿然生辉的金银蛇不知何时的悬浮在了舒瑶的面前，她娇躯颤栗，吃惊的捂住了即将尖叫出来的声音。
<br/>　　她并不是本地人，也自然不信仰什么蛇王，平日听归听，但也只是微微一笑。
<br/>　　她甚至不明白白镇的人为什么那么崇尚蛇这种动物。
<br/>　　要知道人一般都是有惧蛇症的，看到就怕了，还崇拜？信仰？
<br/>　　对于那传说中的蛇王更是觉得可笑，一条蛇还能是神灵，既然还有寺庙供奉。
<br/>　　可最近发生的事情，好像都让她跟蛇纠缠到了一起去。
<br/>　　“你就是蛇王？”
<br/>　　半响，舒瑶回神，被整个白镇人民供奉的蛇王原来是那么小的一条蛇，但却好好看。
<br/>　　她只能用好看来形容，这条宛如漫画中游出来的蛇，从上到下都与众不同。
<br/>　　而且，它还会飞，飞在空中，一双发光发亮的蛇眸望着自己。
<br/>　　希望，她好像看到了一丝希望跟曙光，猛然的下跪，可在她快要跪在地上的同时，无形中好像有一股力量托着她，硬生生不让她跪下来。
<br/>　　“不用跪，也不用紧张，把你所经历的，都告诉我”
<br/>　　说实话，涂小安还真是第一次那么肆无忌惮的在一个陌生人面前现身。
<br/>　　不过他要帮助这个可怜的女人，完成系统任务，就不能束手束脚，只能打开天窗说亮话。
<br/>　　对方的反应到也还算冷静，或许她哀莫大于心死，没什么对她能特别冲击的。
<br/>　　还有什么比一个女人怀上蛇胎更能毁天灭地的。
<br/>　　舒瑶深深吸了一口气，对方真的是神灵，是拥有法力的。
<br/>　　他肯定是老天爷派下来拯救自己的。
<br/>　　舒瑶决定放松，也不拿他当什么蛇王，只当是一位聆听的观众，听自己的秘密。
<br/>　　因为她最近太压抑了，她太害怕别人知道她怀的是蛇胎，拿她当怪物一般看待。
<br/>　　很快，她陷入了一场回忆之中，也是噩梦的开始。
<br/>　　那是在半个月前的一次早晨，她起床发现自己的例假将近有一个星期没来了，似有所感之下，舒雅拿了一根验孕棒，验了一下，是两条扛。
<br/>　　刹那间，她的整个世界都亮了，春光明媚，本来她就跟自己的男朋友准备要结婚的，只是一直找不到一个很好的契机。
<br/>　　现在好了，她怀孕了，契机也来了，那么接下来，就是水到渠成的事情了。
<br/>　　她欢天喜地的把这个好消息告诉了自己的男朋友，他也高兴极了，抱着舒瑶在原地转了好几圈。
<br/>　　赫然，这个房间内，充满了欢声笑语。
<br/>　　幸福是什么，它不是长生不老，不是大鱼大肉，不是权倾朝野，幸福是每一个微小的生活愿望达成，当你想吃的时候有得吃，想被爱的时候有人来爱你。

<br/>　　舒瑶觉得自己的新人生要开始了，她对未来充满了无限的憧憬跟美好。
<br/>　　可谁又能想到，这不过是噩梦的开始。
<br/>　　这个噩梦是从她跟他男朋友去一趟医院拉开的序幕。
<br/>　　到此时此刻，舒瑶还清晰的记得，当他男朋友知道她肚子里怀的不是宝宝，而是一肚子蛇，是怎样惊恐的表情，看她瞬间跟看一个怪物一样。
<br/>　　他甚至觉得自己碰到了传说中的白素贞？是人怎么可能怀上蛇。
<br/>　　真的如晴天霹雳一般，将这对幸福的情侣一下子打入了十八层地狱。
<br/>　　他男朋友无法接受事实的跑了，不知道是跑到了天边，还是海角，反正她找不到了。
<br/>　　男朋友将舒瑶当成了一个怪物，医院要将她留下来，好像要拿她当什么实验品似的，也用看待怪物一样的目光审视她。

<br/>　　悬浮在空中的金银蛇对着舒瑶开口。
<br/>　　舒瑶摇了摇头：“没有，我们连小猫小狗都不养，更别说蛇了”
<br/>　　“那你身上发生过什么奇怪的事情吗，跟蛇有关的？”
<br/>　　人不可能平白无故的怀上蛇胎，除非她是蛇，那么就可以解释了，偏偏她不是。
<br/>　　舒雅努力的回想，痛苦挣扎的回想，眉头紧蹙，忽然，她好像想起了什么，怔怔的望着悬浮眼前的金银蛇。
