<br/>        就这样，两人一起出了门。

<br/>        话说，都快十二点了，他俩才去上班，合适吗？

<br/>        诺筱颖表示不理解，她只知道哥哥在某婚介网站当编辑，至于那个唐倩兮，她也不知道她是做什么，不过，月薪应该不低吧！

<br/>        梁、以、沫……

<br/>        傅昼景掏出之前诺筱颖写给自己的那张借条看了看，随后将这张借条撕碎，扔进了垃圾桶内。

<br/>        他此刻最想做的一件事情，就是，立即马上去洗澡。

<br/>        客厅杂志报纸扔得到处都是，茶几上饮料罐、泡面盒、零食垃圾一大堆，那边壁挂的电视上都落满了厚厚的一层灰。

<br/>        实在是受不了这家中的脏乱不堪，诺筱颖主动带伤打扫卫生。

<br/>        中途，手机短信提醒，她的银行卡里还真收到了那个好心的大帅哥汇过来的一万块钱。
<br/>
<br/>        这边，傅昼景搞完自己的个人卫生，让助理赵斌给诺筱颖汇完款后，便在家吃过午饭睡了一个午觉。

<br/>        傅昼景午睡醒来时，见时间差不多了，于是拿起车钥匙出了门。

<br/>        下午三点钟，他准时到了滨江城机场。

<br/>        出口那陆陆续续有旅客出来，傅昼景掏出手机，给苏漫雪打电话。

<br/>        起初打第一个未打通，连续打了好几个后，苏漫雪的电话终于通了。

<br/>        “喂，你好！你是昼景吗？”
<br/>
<br/>        电话接通后，未等傅昼景先开口说话，苏漫雪那嗲声嗲气的声音便从听筒里传了出来。
<br/>
<br/>        傅昼景听着，不由地起了一身鸡皮疙瘩：“是的，大嫂。”

<br/>        看来，大哥傅夜沉事先将他的手机号码告知了苏漫雪。

<br/>        “你是不是穿着一件白色衬衫和银色的西裤？”苏漫雪又问道。

<br/>        傅昼景怔愣地应了声：“嗯。”

<br/>        难道，她看到他呢？

<br/>        就在傅昼景疑惑的时候，出口的人群里，有个女人举着手机一脸笑容地向他挥手。

<br/>        只见她浓妆艳抹，身材高挑，一袭墨紫色连衣裙，贵气十足。
