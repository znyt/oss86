
<br/>　　<br/>　　    王惊蛰皱了皱眉，又接着往下擦拭了一下，祭坛圆鼎上的那幅图案的全图，渐渐的全都显露了出来。<br/>　　<br/>　　    他的手指顺着线条慢慢的移动着，从头开始接着往下观望，这是一幅祭祀图，主要讲述的是古人在祭坛上祭天的活动。<br/>　　<br/>　　    “巫师，祭祀，这个是……血祭？”<br/>　　<br/>　　    古代祭天，多数都是以牲畜为祭，杀猪宰羊什么的，稍微离谱些的，也有用活人来祭的，在变态点的话，用童男童女，幼男幼女也没准，基本上都是一个路子，以鲜血为祭，就拿面前圆鼎上讲述的祭天来讲，应该就是个巫师一类的角色，用手中的法杖或者兵器刺穿了一个人的胸膛，让他的鲜血流到了祭坛上。<br/>　　<br/>　　    这时，王惊蛰几乎已经看完了全幅图，他身后就是文峰和李菲儿，看见他在圆鼎前驻足了很久，就也想着游过来看看，但就在这个时候，王惊蛰的心里突然一颤，脑袋中一根弦顿时绷紧了。<br/>　　<br/>　　    几分钟之前，抚仙湖的湖面上，雨下的不是特别大，淅沥沥的，但下的比较密，有点水天连一线的意思，不过水面没有风浪，船上的黄詹和黄九郎等人也没太担心，这种不算太恶劣的天气下，水下深潜的危险性还不算太大，如果要是起风掀起大浪了，那潜水肯定就得终止了。<br/>　　<br/>　　    于此同时，抚仙的北岸，诗远沉尸的地方。<br/>　　<br/>　　    陈三岁潜在水中一直观望着尸体，他抬手看了下手腕上的潜水表，自己下来已经有半个多小时的时间了，王惊蛰那边估计差不多是在一个小时左右，再有一会工夫，他们这次下水就得该往回返了。<br/>　　<br/>　　    抚仙湖上，云层忽然之间，压得低了下来。<br/>　　<br/>　　    又等了片刻，陈三岁估计是到点了，尸体也没有啥变故，他就伸开手划着水面准备上去了，但就在陈三岁转身的瞬间，诗远的脸上一双眼睛突然就睁了开来，毫无感情色彩的目光直视着他的后背。<br/>　　<br/>　　    陈三岁刚刚游了几米远，他就木然感觉后脖颈子上有种如芒在背的感觉，两手往前拨了下顿住身子，他狐疑的转过身，然后拿起手电筒晃了晃，当灯光落在诗远的脸上时，他顿时看见诗远的两只眼睛居然睁开了，正直勾勾的看着他。<br/>　　<br/>　　    “唰”陈三岁手一松，手电筒就掉了下去，他惊愕的张着嘴巴，咬着的呼吸管都脱落了。<br/>　　<br/>　　    “咕嘟，咕嘟……”一连呛了几口水，懵逼了的陈三岁顿时回过神来，他惊恐的拼命的用手划着水，人直挺挺的就朝水面升去。<br/>　　<br/>　　    “哗啦”当他从水面钻出来后，急剧的喘了几口粗气，平复下了内心的惊恐。<br/>　　<br/>　　    陈三岁眼珠子转了几圈后想起王惊蛰给他的那道符纸。<br/>　　<br/>　　    “草，要出事！”陈三岁骂了一句，然后解开潜水服从衣服里拿出那张符纸，直接撕开了两半。<br/>　　<br/>　　    陈三岁撕开这张符纸的时候，王惊蛰正在祭坛前看完最后一幅图，他木然就感觉心里一紧，知道是陈安岁把符纸给撕了，肯定是诗远的尸体有异样了。<br/>　　<br/>　　    王惊蛰“唰”的一下就转过身，正看见文峰和李菲儿朝自己这边游了过来。<br/>　　<br/>　　    王惊蛰双腿略微一弯，脚下蹬着祭坛的地面，身子就升了起来，同时他快速的朝着两人摆了摆手，然后朝着头顶指了指，那意思是赶紧往上面游，离开古城。<br/>　　<br/>　　    文峰儿和李菲都是错愕的愣住了，因为按照他们的计划时间，还得要二十分钟以后才能离开古城返回到水面上，不光他俩没反应过来，几个蛙人和另外两个考古队员也呆了下，只有这几天一直跟王惊蛰戴在一起的耿明知道是怎么回事。<br/>　　<br/>　　    当王惊蛰身子已经游上来的时候，见下面人都没动，他就在心里骂了一句，不得不朝下又潜了回来，随即一个劲的摆着手朝上指。<br/>　　<br/>　　    船上的好几副视频画面中都显示着这一幕，黄詹诧异的问道：“时间还没到，他什么意思？”<br/>　　<br/>　　    黄九郎的脸色顿时一变，他不知道发生了什么事，但知道王惊蛰让人往回返肯定是大有深意的，可惜的是下面的人都没领悟他的意思。<br/>　　<br/>　　    黄九郎阴着脸说道：“这帮人脑子都他么被水泡的太长时间烂掉了么？之前怎么说的，水下古城中的一切活动都听王惊蛰的，一个个的都傻了，愣着干嘛呢？”<br/>　　<br/>　　    张生利皱眉说道：“黄主任他们可能也不明白王惊蛰是什么意思，毕竟离约定的时间，还差二十分钟左右呢”<br/>　　<br/>　　    黄九郎当即愤愤的咬牙说道：“服从！连最基本的听命行事都不知道么……”<br/>　　<br/>　　    王惊蛰向水下游去的时候，一连摆了好几次的手，同时耿明游上去的时候，也在示意人上来，几个蛙人就知道是啥意思了。<br/>　　<br/>　　    当过兵的人反映确实快，执行力也强，几个蛙人分别一拉身边考古队人，直接就把他们给提了起来。<br/>　　<br/>　　    但就在这个时候，祭坛四周围拢一圈的士兵，忽然间全都“唰”的一下睁开了眼睛，随即又再次同时仰头，望向了正从自己头顶游过的几道身影。<br/>　　<br/>　　    湖面的床上，盯着视频画面的黄詹顿时就懵了，被吓的身子一仰直挺挺的就朝后面摔了过去，张生利不可置信的指着电脑说道：“活，活了……”<br/>　　<br/>　　    水下，王惊蛰直接顿住身形，脑袋“嗡”的一声就响了。<br/>　　<br/>　　    “噗”当一个考古队员，从祭坛旁士兵的头顶上游过去的时候，尸体手里的画戟突然就抬了起来，锋利的刃尖直接就从这人的胸腹间穿了过去，然后尸体猛地一拉手里的画戟，将对方就从水中给硬生生的撤了下来，顺势往下一压，就将此人给钉在了地面上。<br/>　　<br/>　　    鲜血顿时浑浊了一片水域，这一幕当即把所有的人都给惊住了，就连知道会有变故的王惊蛰都没想到会发生这种状况。<br/>　　<br/>　　    “咕嘟，咕嘟……”好几个人的嘴里都冒出了水泡，这种惊悚的变故，让人都忘记他们此时是深在水中了。<br/>　　<br/>　　    王惊蛰顿住身子后，眼看着那人已经被牢牢的钉在了地上，身体还在痛苦的扭动了几下，但明显已经没救了，他果断的脚下一蹬，人重新朝上游去，随即示意考古队的人赶紧跟上。<br/>　　<br/>　　    “唰，唰”祭坛周围的士兵，全都抬起了手中的画戟，遥遥的指着水里的几道人影，这时候变故已经发生了片刻，人全都回过神来了，开始拼了命的朝着水面游去。
