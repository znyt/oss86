“孙少，你有没有见过你的未婚妻，那夏家的小女儿如何，是不是跟夏家人那样彪悍啊，该不会长得也跟如花一样吧，哈哈哈。”
<br/>
<br/>        大家都在笑话孙进，笑成一团。
<br/>
<br/>        提到这个话题，孙进就想骂街，这孙夏有婚约，夏家的小女儿就是他的未婚妻。
<br/>
<br/>        帝都城有四大豪门，厉陆徐夜，这四大家里虽然没有夏家，但是帝都城还有三绝，其中一绝就是夏家。
<br/>
<br/>        夏家是道上混的，江湖三十六帮派，道上所有的码头都听命于夏家，夏家就是江湖老大哥，没人不怕的，整个夏家人一身武艺，出手不凡，夏家现在的大当家性格又极度暴躁护短，没人敢娶夏家女儿的。
<br/>
<br/>        以前夏家的大当家曾经救过孙老爷子，所以孙老爷子就订了这门婚事。
<br/>
<br/>        孙进用手指了指那些看笑话的人，发狠道，“我是不会娶夏家那个小丑八怪的，你们等着，我一定会退了这门婚事的。”
<br/>
<br/>        大家哈哈大笑，
<br/>
<br/>        孙少，你敢退婚，小心夏家的大当家把你的脑袋拧下来当球踢！
<br/>
<br/>        不但如此，他们还会把你的身体大卸八块！
<br/>
<br/>        孙进气死了，他没有见过夏家那个小女儿，不过那种莽汉武夫的家庭里肯定生不出什么娇滴滴的美人，说不定那个小丑八怪真是如花，他才不要娶，要不然每天夜里做噩梦。
<br/>
<br/>        “不要聊我的事情了，嫣然表妹，听说今天陆少会来party上给你庆生？”孙进八卦的看向厉嫣然。
<br/>
<br/>        这个问题成功的吸引了所有人的注意，这场豪华party几乎集聚了整个帝都城的上流贵族，陆寒霆是从不跟他们一起玩的，在他们在大院里念书的时候，陆寒霆十几岁就去玩转商界了，在他们看来这个商界鬼才变态到令人发指。
<br/>
<br/>        后来陆寒霆陆陆续续的出事，陆家封锁了一切消息，外界知道的并不多，总之他们对陆寒霆这个人又敬又怕，他是传说里存在的人。
<br/>
<br/>        现在陆寒霆在三个月前回归，他正式接手了陆司爵手上的陆氏帝国，上任这三个月雷厉风行的血洗了各个部门，让人望而生畏。
<br/>
<br/>        但这样铁血而高高在上的男人注定是万众瞩目的，他像是一块磁铁石，让人畏惧时又忍不住屏息去仰望他。
<br/>
<br/>        几个闺蜜拉住了厉嫣然，
<br/>
<br/>        嫣然，你真的跟陆少在一起了？你不怕他吗？
<br/>
<br/>        嫣然，这陆厉联姻是不是真的啊，我可听说你和陆少快要订婚了。
<br/>
<br/>        厉嫣然漂亮夺目的小脸变得红扑扑的，陆寒霆去了海城整整七年，她等了七年，现在她终于修成正果了。
<br/>
<br/>        厉嫣然用力的点头，“是啊，我跟寒霆哥哥在一起了。”
<br/>
<br/>        这时不知道是谁激动的喊了一句，“你们快看，陆少来了！”
<br/>
<br/>        众人纷纷抬头，只见大门被推开，外面的寒风迅速侵袭了进来，一道高大英挺的身躯也跃入了众人的视线。
<br/>
<br/>        陆寒霆来了。
