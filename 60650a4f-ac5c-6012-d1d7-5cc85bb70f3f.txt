
姜红芍一双大眼那是相当炯炯有神的盯着自己，她迟疑后开口，“你怎么知道这些的？”眼眸子里仍然有着错误估计的疑惑。<br/>　　<br/>　　他们整个团队研究这个方向的时候，其实是久经战阵的化学指导教师想的题目，这个题目涉及现在的前沿科学，其实放此时蓉城的高中生领域，十中团队若是能够在创新指向性上面有所突破，就绝对妥妥能拿第一了。<br/>　　<br/>　　科技大赛是由省科协、省教育厅、省科技厅等十个政府机构共同主办，组委会主席是中科院专家，同时汇集六十名来自各高校各学科领域的专家组成的评审团评审，若是他们的课题需要通过论证，还要交给相关大学的实验室验证。<br/>　　<br/>　　整个科技大赛有包括创新新苗奖，创意之星奖，十佳科技辅导员，青少年创意学科一等奖等等诸多奖项，他们十中这次瞄准的是“工程，技术”类学科一等奖的位置。一般而言这种学科类一等奖含金量最高。他们的辅导员是连续夺得十佳辅导员的人物，精于此道。<br/>　　<br/>　　这次瞄准的课题也很是刁钻，也可以说是挑战，在这个过程中的任何一点前景的结论和预测，都能让他们脱颖而出。<br/>　　<br/>　　在这次课题上面，他们这个汇集当前年段这方面最好学生的团队突击集训一个月，虽然目前集训时间只是过半，但每个人这方面的知识增长，绝对超出了普通学生所能接触得到的范畴。有的内容，就是大学本科生，恐怕都没有听说过。<br/>　　<br/>　　所以姜红芍有理由疑惑。<br/>　　<br/>　　程燃在短暂的一怔过后，道，“我家的伏龙不仅仅是做交换机，也会生产电信便携设备，市面上购买不到的电池，外购定制的成本高了，长远考虑就只能自己制作研究，所以我知道啊。”<br/>　　<br/>　　姜红芍笑道，“你作弊。”<br/>　　<br/>　　“那你们查资料算不算作弊？这只是解决了你们当前的基本难题，最后课题结论还是要你们自己做的……”<br/>　　<br/>　　姜红芍点点头，“算你的功劳。”<br/>　　<br/>　　尽管程燃这么说，但她很清楚，迫于他们研究的局限性，程燃给出的路径，其实帮了大忙，让本应该直接进行不下去的课题，重新推动了下去，因为接下来他们只需要理论上探讨出可能性就够了，就像是一后面的那个零，如果没有前面的一，零的意义就不大了。<br/>　　<br/>　　“怎么报答？”程燃笑道。<br/>　　<br/>　　“报答……”姜红芍愣了一下，似乎觉得他很得寸进尺。但想了一下，她就打开书包，从里面取出一根巧克力威化饼干，放桌上，“我的能量棒，最后存货了……给你。”<br/>　　<br/>　　程燃顿时觉得自己像是剥削长工的资本家恶霸，怎么突然觉得老姜有点可怜。<br/>　　<br/>　　她包里装着这些小零食，大概有时候就是她做题饿了补充能量之用，甚至有时候，可能这就是中午饭了。所以不要看她成绩出类拔萃，这背后其实都有常人很难想象的毅力和努力。<br/>　　<br/>　　程燃拿着那支威化饼干，笑着道，“你这真是把我当猪了？”<br/>　　<br/>　　“不要的话……还来。”姜红芍伸手要夺。<br/>　　<br/>　　程燃撕开包装，咬了一口，在嘴里嗑嗤嗑嗤，顺手把剩下一半递给她，“给你啊，拿着不要客气。”<br/>　　<br/>　　姜红芍接过去，反手剥开饼干后半部分的包装，从最后他没吃过的地方咬了一口，只留中间一截，递还在了程燃手上。<br/>　　<br/>　　程燃一脸“还能这样！？”的嫌弃表情。<br/>　　<br/>　　老姜嘴里示威性的嗑嗤嗑嗤，眉梢间有那么一抹得意。<br/>　　<br/>　　然后她漂亮双瞳随后陡然睁圆，想下意识伸手去夺，程燃已经反手把剩下三分之一塞嘴里了。<br/>　　<br/>　　姜红芍顿时面如火烧，和程燃大眼瞪小眼。<br/>　　<br/>　　片刻之后，她低下头去，捋了捋鬓发，然后两耳不闻窗外事的埋头翻书做笔记，像是耽搁了太多时间，要赶快补回来。<br/>　　<br/>　　红木窗窗外覆盖的爬山虎，滤过的微光穿透进来，见证过无数人无数段光阴的这个不大且简陋的图书馆，再度将时光中的这个片段……永恒的封印了。<br/>　　<br/>　　……<br/>　　<br/>　　事情在下午的时候就变得有些不同寻常，姜红芍中午和叫做程燃的转学生一起去了图书馆，这件事就足以在一些圈子里引起关注，并且耐人寻味了。<br/>　　<br/>　　当然人们首先第一个念头，还是这个转学生到底是何方神圣？<br/>　　<br/>　　下午到学校教室，老姜和他是一起从图书馆回来的，但路上又被遇上的人叫走了，程燃独自到教室的时候，看到张平的座位旁边是姜红芍的同桌魏舒，也就是最看不顺眼他的那位。<br/>　　<br/>　　他们旁边里三层外三层的围了不少人，有的人程燃比较熟，属于平时大家中午过后一起吃饭的，这个时候听着张平和魏舒之间的说话，那表情满是不可思议。<br/>　　<br/>　　等正主儿回来，前面的一群人就撤开来，程燃发现围在一起的除了平时吃饭和张平关系不错的普通学生之外，还有几个是班上成绩不错的，平时挺特立独行的人。这些人有文学好的，也有当科代表的，平时即便早回教室了，除了和关系很亲近的会聊聊天自成一个小圈子之外，基本上都是自己做着手头上的事情，不会参与到其他人的闲聊中去，但此时都像是上课前没事干，跻身进入了张平魏舒的话题中。<br/>　　<br/>　　人群分开，各式各样古怪的，打量的目光望着程燃，魏舒瞪大眼看程燃，“程燃，你说，你和姜红芍什么关系？”<br/>　　<br/>　　张平干咳了两声，觉得这女生语气太硬，委婉道，“程燃啊……现在该说说了吧，你和姜哥怎么回事啊……他们科技大赛团队遇到难题了，结果却拉你一起去了图书馆，跟你商量……跟你商量什么啊？你会啊那些参赛内容？”<br/>　　<br/>　　魏舒也是一脸狐疑。<br/>　　<br/>　　省青少年科技创新大赛在十中普通学生之间也算是高大上的内容，参赛团队的那些成员，都是各个班的优秀学生，很多人不愿意承认，但委实有时候看着他们，就像是精英一样的感觉。<br/>　　<br/>　　偶尔也会有人关注一下大赛的课题团队的情况，一打听也是高川仰止，总之是寻常学生听着就是一脸懵的内容。<br/>　　<br/>　　所以今天中午姜红芍拉着他的这一幕，才越发有一种半路杀出程咬金的感觉。<br/>　　<br/>　　在参赛队员所属的一些尖子生圈子里，这件事倒是率先传开了。<br/>　　<br/>　　“大概是，以前的习惯吧……”程燃道。<br/>　　<br/>　　魏舒露出一个“这才合理”的表情，问道，“他们说你和姜哥以前都是山海一个学校的，真的？”<br/>　　<br/>　　程燃点了点头。<br/>　　<br/>　　一群人恍然了。<br/>　　<br/>　　郝迪和张平对视了一眼，张平才有一种恨不得钻地里面的感觉，人家程燃第一天和姜红芍见面的时候，就跟张平说过了两人其实是认识的事情，结果张平自己在那里自以为是，以为姜红芍是抓壮丁，要程燃当个什么小组长之类的，不过这也怪不得张平，因为那第一个星期的班会课，程燃委实就成了他们那一个组的小组长，负责每天收作业，给自己组分发卷子。<br/>　　<br/>　　张平尴尬道，“嗨……难怪，我说呢……他们团队遇到难题了，姜哥找你陪她去图书馆，就是想在以前你们一个地方的熟人面前放松一些吧，能理解能理解，毕竟他们参赛集训，还担当着为十中争光的任务，压力还是很大的啊……”<br/>　　<br/>　　魏舒撅着嘴巴斜眼看程燃，难怪当天他直接走到姜红芍旁边座位，原来有这层关系。<br/>　　<br/>　　可还是不爽啊……姜红芍只能和自己去图书馆看书，他们一个团队的，两人研究起来事半功倍啊，这程燃何德何能？<br/>　　<br/>　　……<br/>　　<br/>　　十中的科技大赛参赛圈子中，对这件事有的人倒是耿耿于怀。<br/>　　<br/>　　有的人对姜红芍，倒是略有些不满起来，私底下说着，“我们是一个团队，团队有争论，有分歧是正常的事情，可谁也别觉得谁大牌，这临时找个外人来商量算是个什么事？噢，就算是空余时间……真觉得一群人的智慧，比不上你们单独开小灶？还有那个程燃是谁啊，哪来的乱七八糟的人？”<br/>　　<br/>　　有这样不客气的声音传出来，也有年级上一些消息很灵通的人，听说了他们参赛团队的事情的，有的人会自告奋勇，想要出面帮忙的，可打听了一下，据说辅导老师都向大学的教授求助了，表面上的骄傲就有些打退堂鼓。<br/>　　<br/>　　下午体育课的时候，秦芊和苏洋的班级正好有课相重合，两个人又碰面了。<br/>　　<br/>　　秦芊沿着跑道走，苏洋在旁陪同，说着中午后发酵的事件，“据说红芍以前在山海读书，和那个叫程燃的转学生是一个学校的，两个人在那个时候就认识了……中午的事情，其实我理解她……”<br/>　　<br/>　　秦芊停住脚。<br/>　　<br/>　　苏洋微笑道，“当时一个团队争得很厉害了，虽然没有到吵的地步，但大家各执一词，确实需要一段时间冷静……还有，我们现在觉得，凭我们本身的力量，是没有办法继续把课题深入下去了，现在就看辅导老师那边能不能找到突破，但这次他是打算让自己的导师给建议的，课题能不能研究下去，如果能，怎么突破。如果不能，他们现在怎么转，能不浪费之前的集训知识……同时还要那位省内著名教授，帮忙在大赛组委会上面，修改一下已申报的课题内容……”<br/>　　<br/>　　“红芍是个不服输的女生……这样的情况，想必她也感觉烦闷吧。一时想要躲避一下，这是符合常理的。”<br/>　　<br/>　　秦芊“啊！”得轻轻点头，“原来是，这样啊……你好聪明。”<br/>　　<br/>　　苏洋面色郝然，片刻又正色道，“秦芊，那个程燃，你不会认真的吧？”<br/>　　<br/>　　秦芊“吖吖”的招财猫般摆摆手，露出个俏皮的笑容，“……看他表现吧！”<br/>　　<br/>　　嘿嘿，破译奥飘~
