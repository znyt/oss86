　　“宫主，据我的调查，数十万年之前，绫裳宫主加入终极神王殿之前，曾经是一个小势力的弟子，这个小势力在时空界域，名为玉虚宗。”楚天墟的眸中精光熠熠，声音低沉而有力。
<br/>　　“玉虚宗？”
<br/>　　紫霄闻言，陷入了回忆之中，由于时间太过久远，他都快忘记了此事。
<br/>　　少倾，紫霄点了点头，道：“不错，当年绫裳的确是玉虚宗弟子，不过，他在玉虚宗只是匆匆过客，待到时间好像很短，便加入了终极神王殿！”
<br/>　　“宫主，我的意思是，从玉虚宗下手！”楚天墟眸中闪过睿智的光芒，沉声说道。
<br/>　　紫霄闻言，眉头一皱，道：“此法并没有什么意义，绫裳和玉虚宗早已没有了任何的联系，而且，玉虚宗是时空界域霸主永恒神宫的下属势力，并不是能随意拿捏的宗门。”
<br/>　　“宫主，你有所不知，这些年我一直在查探绫裳宫主的过去，据我调查，那玉虚宗的宗主玉虚子，在数十万年之前，曾经救过绫裳宫主的命，对绫裳宫主有大恩！”
<br/>　　楚天墟声音低沉，继续道：“而绫裳宫主，一向是重情重义之人，很难抛弃这段恩怨！”
<br/>　　“哦？”紫霄闻言，顿时眸光微闪，若是这样的话，倒不失为一个机会，以玉虚子来引出绫裳。
<br/>　　“呵呵，天墟，没想到你如此有手段？”紫霄微微一笑，赞赏道，这些年他命令对方追查苏莫的下落，没想到对方一直在调查绫裳，倒是有些手段。
<br/>　　“为宫主效力，天墟不敢怠慢！”楚天墟立刻向紫霄抱拳，继续道：“宫主，我们现在，要制定详细的计划，怎样利用玉虚子，来引出绫裳，只要找到了绫裳，就能找到秦陨。”
<br/>　　紫霄闻言，沉默不语，此事的确要详细的计划一番。
<br/>　　毕竟，玉虚宗是永恒神宫的下属势力，不能得罪永恒神宫。
<br/>　　再者，他们的行动，也要背着终极神王殿，不然的话，就算找到了秦陨，终极剑道和终极神剑，他也得不到。
<br/>　　“天墟，此事只能够由你出面了，你是为师的密门弟子，若是为师将来得到了终极剑道，必然和你共享！”紫霄语重心长的说道。
<br/>　　“师尊，弟子不为终极剑道，只为您能完成心愿！”楚天墟立刻向紫霄抱拳，面色恭敬的说道。
<br/>　　“天墟，只要为师得到终极神剑和终极剑道，不敢说神域无敌，能击败为师的人，将屈指可数，到时候为师不会亏待你的！”紫霄郑重的说道。
<br/>　　“多谢师尊！”楚天墟躬身。
<br/>　　“去吧，此事就看你的了！”紫霄挥了挥手。
<br/>　　“师尊，等弟子的好消息！”楚天墟再次抱拳，随即转身离去。
<br/>　　望着楚天墟离去的背影，紫霄微微的叹了口气，现如今他在终极神王殿的日子，很不好过。
<br/>　　无论是终极神王、还是风雪神王和毁灭剑王，都不再信任他。
<br/>　　一些宗门的大事，甚至终极神王都不再通知他。
<br/>　　但是，他却是不能脱离终极神王殿，只能继续留下来。
<br/>　　“秦陨啊秦陨，不杀你我如何能立足？”紫霄低声自喃，苏莫的存在让他如鲠在喉，咽不下去也吐不出来。
<br/>　　……
<br/>　　轮回神朝，地王宫。
<br/>　　大殿之中，主座之上，端坐着一位矮壮老者，老者面容刚毅，不怒自威。
<br/>　　此人，不是别人，正是地王宫宫主，地渊神王。
<br/>　　下首，除了地渊神王之子地藏神王之外，还有一位高大的神王级别的强者在坐。
<br/>　　除此之外，还有一位黑袍青年，青年身形不高，但是体格比地渊神王还要雄壮。
<br/>　　青年的面色，略显黝黑，下巴上长着钢针一般的胡茬，显得很是霸气。
<br/>　　“藏儿，具体是有何事？如此郑重？”
<br/>　　地渊神王看着地藏神王，疑惑的问道，今日，对方召集他们在此，显然是有要事。
<br/>　　其他几人，也是目光凝视着地藏神王，面带疑惑之色。
<br/>　　“父亲，最近我发现了一件怪事！”地藏神王沉声说道。
<br/>　　“什么怪事？”地渊神王问道。
<br/>　　“这些年，因为那个秦陨之事，我们和天王宫以及终极神王殿向都向人王宫施压，此事诡异！”地藏神王面色郑重的道。
<br/>　　“大哥，有何诡异之处？”黑袍青年疑惑的问道。
<br/>　　“青云，你有所不知，我发现天王宫有些诡异！”地藏神王望着黑袍青年，对方乃是他的弟弟，拓跋青云。
<br/>　　众人没有再询问，静等地藏神王讲述。
<br/>　　“明面上，天王宫好像是为了终极剑道和终极神剑，才向人王宫施压，但实则不然！”
<br/>　　地藏神王眸中闪过一道精光，继续道：“据我暗中派人调查，天王宫一直在追查秦陨的下落，天尘宇这些年，一直游荡在天池神朝，目的就是找到秦陨！”
<br/>　　“这秦陨还有什么秘密？”地渊神王沉声问道。
<br/>　　“秦陨还有什么秘密，不得而知，不过，天尘宇此人和帝一魂关系不浅，所以，此事很可能和帝一魂有关！”地藏神王说道。
<br/>　　“和帝一魂有关？”众人闻言疑惑，秦陨会和帝一魂有什么联系呢？
<br/>　　“难道秦陨是分魂？”拓跋青云声音略显沙哑的说道。
<br/>　　“什么？”众人闻言，顿时心中一震，纷纷惊讶的看着拓跋青云。
<br/>　　拓跋青云，面色不变，淡淡的说道：“目前，能让帝一魂非常在意的事情，少之又少；而这秦陨能得到终极神剑和终极剑道，又能杀了流明，这种天赋万古罕见，所以，这秦陨有可能是分魂，才会让帝一魂如此的重视！”
<br/>　　“这……不太可能吧？”地藏神王惊疑不定的说道。
<br/>　　“六位分魂，不是只剩下你们两人了吗？”地渊神王面色凝重的问道。
<br/>　　“六位分魂，我曾经融合了一位，当年帝一魂曾言他已经融合了三位，我们两人将会一决胜负，但是父亲、大哥，你们谁又曾亲眼见到，那另外三位分魂被帝一魂融合了？”拓跋青云轻笑一声，不置可否的说道。

<br/>　　“父亲，搞清楚此事，怕是不简单，若是秦陨真的是分魂，人王宫和天王宫之中，估计知道的人都不会多，我们的探子也无法调查！”地藏神王皱眉道。

<br/>　　“该来的迟早会来，父亲，这次我亲自出手镇压天尘宇！”拓跋青云面色郑重的说道。
<br/>　　地渊神王闻言沉默，面带沉思之色，他知道，拓跋青云对于此事的重视，不然不会亲自出手。
<br/>　　毕竟，牵扯的分魂，此事甚为重大，必须弄明白。
<br/>　　“好吧！”
<br/>　　少倾之后，地渊神王重重的点了点头，随即对地藏神王道：“藏儿，查出天尘宇的行踪，然后在告知青云。”
<br/>　　“嗯！”地藏神王点头，随即二话没说，便起身离开了大殿。
