　　昆仑山，又称昆仑虚、昆仑丘，或玉山，有着“万祖之山”、“中国第一神山”的称号，在中华民族文化史上有着显赫的地位，古人称昆仑山为中华“龙脉之祖”。
<br/>　　作为“万山之祖”的昆仑山一向有着不少不为人知的神秘色彩，而今天，这里却汇聚了不少神秘人，在他们之中，有修炼大家族的长老，有魔门魔头，有武宗武者，也有西方的吸血鬼……
<br/>　　若是寻常，这些三教九流聚集在一起，早就喊打喊杀了，又怎会如此安静？但今天，他们却都没有行动，而是把目光汇聚在前方不远处的一个山头上。
<br/>　　在那里，一个身穿白袍，头发及腰，腰挂长剑的男子静静地站立在那儿，看面相，男子不过三十来岁模样，他闭着眼睛，长发随风飘扬，白袍在风中略显凌乱。他仿佛在神游天外，一副冯虚御风，遗世独立的样子，仿佛下一刻就要羽化登仙而去。他就这样孤傲的站在那里，仿佛世间再没有什么值得他留恋的地方。
<br/>　　“鬼老怪，你也来看元始梦魔渡劫？”旁边，一个白发老头对着另一个黑发老怪感慨道。
<br/>　　“桀桀桀……！”老头一阵怪笑，听得旁人有种毛骨悚然的感觉，但白发老头却毫不在意。
<br/>　　“元始魔头乃八百年难道一见的鬼才，他的成仙劫当然值得一看，桀桀桀……！”
<br/>　　白发老头皱了一下眉头，似乎是对他的回答有些不满，但也没多想，却又对着老鬼头说道：
<br/>　　“是啊！八百年难得一见的天才啊！可惜啊！可惜！”
<br/>　　“桀桀桀……！”这一次，鬼老怪却没有多说，只是一阵怪笑，就沉默了，显然是认同了白发老头的话。
<br/>　　他们在可惜什么，当然是在可惜这世道了。
<br/>　　他们现在所处的地球正处在末法时代。
<br/>　　末法时代，道法不显，灵气不现，天地道则寂灭，简直就是修道者的末路。
<br/>　　距今已有八百多年没有听过有人飞升了，要不是这世界上一直流传着八百年之前武当创始人张三丰张真人拳破虚空，飞升仙界的传说，大家都不相信这世界上有着仙界。
<br/>　　自张真人之后，就一直不见有人飞升，而八百年之后的今天，终于又有人感应到天劫，要渡劫飞升，大家自然都想来看看了。
<br/>　　亦善亦邪魔武道，唯心唯意元始梦！
<br/>　　元始梦魔，在修真界就是一个传奇。
<br/>　　元始梦魔，真名元释，纵横世间三百载。在这灵气枯竭，万法不显的时代，仅仅三百多年就修炼到渡劫成仙的地步，他的天资不可谓不强。就算在灵气充裕的上古时代，这样的人也不多。
<br/>　　元始梦魔，道武双修。道修魔道，武修剑道。道武双绝，因其修炼魔功，与魔头一般，又因其名字元释与元始同音，被称为“元始梦魔”。
<br/>　　众人只知道元始魔头天资出众，功法诡异，却并不知道，元释一开始并非如此，他只是一个普通小道士，因为一次意外得到了一块寸许大小的圆印，这一切才开始改变。
<br/>　　圆印上刻着元始两个字符，元释自从得到这块元始印后，他就发现自己的头脑特别清晰。开始并不怎么样，他只当自己得到宝贝，隐隐觉得它十分不凡。后来他渐渐开始更多关注这块圆印，于是，他发现了自己有了一种奇怪的神通——他心通！
<br/>　　他能够借助元始印看透别人的想法，当然，这也是有限制的。开始时他只能看透普通人的想法，而且只能在别人做梦的时候。
<br/>　　元释称此神通为盗梦。
<br/>　　可是，渐渐地他能够看透更多人的想法，他也开始接触到修真者。于是，就这样他借助自己的神通开始走进修真的道路。
<br/>　　可是，这个世界正处于末法时代，修道何其艰难，随着接触的更多，他也知道的更多，他也得到了更多功法秘术。
<br/>　　但是，没有灵气，一切都是白搭。他不甘心就这样放弃。后来，他借着元始印和自己的神通，自创了一种诡异的功法——《大自在梦心魔功》。
<br/>　　天地之间的能量除了灵气就没有别的了吗？不！有，而且还有很多。如：星辰之力，大地之力，太阳之力，……不过，这些都需要特殊功法才能吸取修炼，否则就是作死。
<br/>　　元释在一个魔头梦里盗到一种魔功，名《大自在魔功》，这是一种特殊的魔功，他能借助心魔修炼，欲成仙，先入魔，最后再超脱出去。这是一本疯狂的功法，一个不小心，就会玩火自焚。
<br/>　　元释得到这本功法后，再借助元始印，多次修改创出了《大自在梦心魔功》。这是一本能够盗取别人做梦时的负能量修炼的魔功。
<br/>　　在这个世上，灵气有限，但人的欲望却是无穷的，每个人心中都有一个堕落天使和光明天使。元释的魔功就是借助这些人的黑暗邪恶的一切念头来修炼，这是一本专修元神的功法。
<br/>　　只是，修炼这种魔功存在着很大的弊端，负能量多了，自己就容易被心魔驱使从而变成魔头。后来，元释通过多做善事来消除这种负能量。
<br/>　　就这样，凭借着他诡异的神通及魔功，他在修道者中闯出赫赫威名，当然，这只是魔名。
<br/>　　只修元神不行，因灵气枯竭，之后他又创出来依靠吃东西就能修炼肉体的功法《饕鬄战体》，依靠此功法修炼出强大战体。
<br/>　　一代强大的梦魔就这样出现在世间，独领风骚数百年。
<br/>　　而今，终于到了他渡劫的时候了，只是不知他能否成功？
<br/>　　“来了！”
<br/>　　元释猛地睁开双眼，眼中闪过一丝黑光，令人心悸，他抬起头来，看向天空，在那里，一阵乌云正在会聚，雷霆欲起。
<br/>　　“来了，终于来了！”
<br/>　　“是啊，多久了，没有看到劫雷了，不知他能否度过？”
<br/>　　“嘿嘿！”
<br/>　　远处，见到这一刻，有人激动，有人期待，有人担忧、更有人幸灾乐祸……
<br/>　　元释却没有关注别人的想法，他此刻目光紧紧地盯着高空的雷劫，既有期待，也有凝重，却不见丝毫后悔，到了他这一步，修道如修心，没有后悔一说。
<br/>　　“哗啦——”
<br/>　　酝酿已久的劫雷化作一条白色匹练，划破长空，直直向元释劈了下来。
<br/>　　“来得好！”
<br/>　　元释没有丝毫胆怯，反而迎难而上，飞到上空，一拳轰出，雷霆一轰而散。这时，第二波雷霆化作两道一同向元释劈来。
<br/>　　元释没有丝毫畏惧，修炼之途，不进则退。迎难而上，一掌劈空，虚空顿时出现一个巨大手印，向着两道劫雷劈去。
<br/>　　劈空掌！
<br/>　　“咔擦——！”
<br/>　　雷霆破碎，天空上的劫雷好像被触怒了，又是一波闪着白、绿、蓝、红、黄五色的雷霆狠狠的劈了下来。
<br/>　　五行劫雷！
<br/>　　看到这一幕，元释也变得凝重起来，这才是第三波，就是五行劫雷了，后面还有六波，之后又怎么办？
<br/>　　“斩——！”
<br/>　　剑光乍现，迎着五行劫雷狠狠劈去，“哗啦！”劫雷被皮城两半，剩下的依旧朝着元释狠狠轰下来。
<br/>　　“咳咳！”
<br/>　　元释咳嗽几声，从地上站立起来，此时，第四波劫雷再次降临……
<br/>　　第五波……
<br/>　　第六波……
<br/>　　……
<br/>　　一直到第九波，只是这最后一波，还在酝酿着，一旦降下，就是雷霆之势。而此时元释的身体早已龟裂，身上血迹斑斑，头发焦糊，气息早已若有若无了。
<br/>　　“还是不行吗？”
<br/>　　“嘿嘿，还是不行啊！”
<br/>　　众人有绝望，有幸灾乐祸的，什么样的都有。
<br/>　　“我——绝不——放弃！啊——！”
<br/>　　元释大吼一声，提剑冲向最后一波，最后一波整整九九八十一道紫雷，毫不留情地轰向元释。
<br/>　　“咔擦——！”
<br/>　　狂暴的雷霆轰在元释的身上，这一刻仿佛静止了，昔日的种种，快速在眼前浮现，有回忆，有甜蜜，也有苦涩，却没有后悔。这一刻，他仿佛明白了什么。
<br/>　　“精气神，三宝合一么？只是，没有灵气，又哪来的气？”
<br/>　　雷霆之中，元释的身体在快速消失，正在此时，一阵狂笑从雷霆中传出，声声震耳，响彻九霄：
<br/>　　“哈哈哈！无悔今生走一遭，有缘来世傲九宵！”
<br/>　　……
