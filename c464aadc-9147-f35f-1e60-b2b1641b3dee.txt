已经在他心中徘徊了好久了！就想好好的喝点酒，哪怕不能喝的尽兴，就是喝上两三杯也是满足的。<br/>　　<br/>　　    但是护士和医生是坚决不容许，而且陈默的医嘱中也是没有让其喝酒，所以就没有办法了！<br/>　　<br/>　　    现在看到陈默了，还不赶紧问问，要不然自己又要等一天的时间。<br/>　　<br/>　　    陈默听见胡国忠这样说，也不禁莞尔！<br/>　　<br/>　　    低头想了想，对身边的护士说道：“从今天开始，每天晚餐的时候，可以让病人喝一两的药酒！记住，是我配置的药酒，你将医嘱拿过来我签字，然后通知配餐部！”<br/>　　<br/>　　    “好的！”护士和医生立刻答应！<br/>　　<br/>　　    现在在这里，陈默的权威还是非常的足的。即使医生以前在医院的时候，是什么科室的主要负责人，还是什么其他的专家等等，在陈默这里，是龙也要盘着，是虎也要踞着！<br/>　　<br/>　　    反正就是陈默说什么，他们就做什么。这也是他让这些护士和医生能够进来，并且跟随自己看着治病的要求，谁要是不遵守或者有疑问，那么就不要待了，只能出去！<br/>　　<br/>　　    这里是陈默自己的地盘，他想怎么做就怎么做！<br/>　　<br/>　　    所以值班医生，听到陈默要让胡国忠在晚饭的时候，可以喝一两药酒，嘴唇动了动，但是最后却没有反对，只是将医嘱递给陈默，然后在一边看着。<br/>　　<br/>　　    三十好几的年龄了，在陈默这位只有二十几岁的年轻人面前，丝毫不敢逾越！<br/>　　<br/>　　    这也是陈默治疗好几个特殊病历以后，所形成的权威，所以他们只能看不能说。<br/>　　<br/>　　    陈默自然也知道这些人心里想的是什么，即使反对也不好说出来。但是他们哪里知道，这些酒都是乾坤珠里的溪水兑过的，而且这些药酒里的药材，都是从乾坤珠内培育的。<br/>　　<br/>　　    所以药酒的药效非常的好，即使胡国忠有肝病，只要不多喝，也是没有问题的。所以陈默才容许他喝上一两！<br/>　　<br/>　　    “哈哈！谢谢陈医生，谢谢陈医生！”胡国忠笑着说道。<br/>　　<br/>　　    自从他的身体越来越好，他对陈默也就越佩服，所以陈默说什么他都会听的。这次要不是酒瘾发作，还真的不好意思说出口。不过听到能喝一两酒，还真的是喜出望外。<br/>　　<br/>　　    至于说酒的价格，六千也好九千也好，对于他来说，已经不算是什么，都是小钱！<br/>　　<br/>　　    “胡老，虽然准许你喝一两酒，但是要记住，只能在晚餐的时候喝，另外就是不能多，希望你能够严格遵守医嘱！”陈默为了以防万一，所以叮嘱了一下，然后转头又对胡老的专属护士交代了一句！<br/>　　<br/>　　    胡国忠和护士都是同时答应！<br/>　　<br/>　　    这帮老头，要是不叮嘱一下，兴致上来以后，就可能不管不顾的喝起来，要是出个什么三长两短的事情，不是砸自己的招牌么？所以陈默这才一再的叮嘱。<br/>　　<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
