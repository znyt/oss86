　　“这是小镇对交易者的等级划分。”
<br/>　　子午向其余五个人解释道：“之前说过，进入这个小镇进行源代码交易的人来自各行各业，有自有黑客、有地下军火市场的军火商、也有像刚才洛菲克那样代表着洛克希德公司这种庞然大物前来的武器公司代表。”
<br/>　　“而这个小镇根据交易者不同的背景，划分成三个等级，利用红色、紫色、黑色三种卡片来进行区分。三种卡片在小镇滞留时间上没有区别，如果是在平时，这三种卡片基本没有差别，唯一可能有的差别就是在进入旅馆、酒吧这种地方的时候，接待态度上的区别，以及在你交易完成后，其他人要不要对你下手的顾虑。”
<br/>　　“可是到了‘源代码拍卖会’举行的时候，这种卡片颜色的等级划分就显得重要起来，因为它直接决定了，卡片持有者在拍卖场中的座位位置。”
<br/>　　周柏拿出自己这张红色卡片，翻看了一下，说道：“这么说，我们这种卡片是级别最低的一类咯，难怪刚才进来的时候，门口那家伙一脸轻视，而对那个洛菲克就跟见了自己亲爹似得。”
<br/>　　“不过，如果只是拍卖场座位位置关系的话，那其实也没有多大关系。”李毅峰想了想回答道：“毕竟拍卖看的是自身的实力，跟座位没有关系。”
<br/>　　“的确，如果仅仅只是拍卖场座位区别的话，那么这种等级划分没有任何意义。这里举行的‘源代码’拍卖会跟我们平时所见到的拍卖会并不一样，在拍卖开始前，我们不会得到一本小册子，然后从那本小册子上知道这次拍卖会上将会有哪些东西进行拍卖。”
<br/>　　子午向其他五个人解释道：“未知性，是‘源代码拍卖会’最大的特色，参加‘源代码拍卖会’就是一场赌博，因为你永远不知道下一件拍卖品是什么，也许你会在不断的等待中，错过最有价值的东西；当然你也有可能能够等到这次拍卖会上最具价值的拍卖品。黑色卡片与其他另外两种颜色卡片最大的区别，不仅仅是在拍卖场上可以坐在最前面的位置，而且他们能在拍卖开始之前就得到整个拍卖会将要进行拍卖的东西。”
<br/>　　“也就是说，黑色卡片的持有者，可以比我们率先一步得知拍卖内容，然后根据自己的需要制定拍卖策略。”李然说道：“这样的话，持有黑色卡片的人就是有准备的进行拍卖，而不会在一些鸡肋拍卖品上浪费自己的资金。”
<br/>　　“没错。”子午点头说道：“‘源代码拍卖会’还有一个特色就是你可以用先前拍到的东西，作为第二次竞拍的资金。就是说，如果能够提前得知拍卖品的内容，那么提前就可以用一个相对比较低的价格买到一样有价值的东西，然后再把这样东西作为资金投入到第二次竞拍中，这就是黑色卡片所有的特权。不过能来参加‘源代码拍卖会’的，没有一个是简单货色，所以想要捡漏几乎是不可能的。”
<br/>　　小镇上的旅馆有很多，价格自然也是不一样，罗昊他们选择了比较普通的一家旅馆，对于他们这种即便是在危机四伏的热带雨林中都能和衣而眠的人来说，只要有张床就能休息，根本不在乎星级酒店和路边宾馆的差别。
<br/>　　旅馆前台，六个人向服务员出示了自己的红色卡片，服务员将六张卡片逐一在机器上照了一下，随后连同三张房卡交还给罗昊等人，说道：“现在房源比较紧张，你们只能两个人住一间房，退房的时候把你们身份卡片和房卡一起交给我。”
<br/>　　子午自己留下一张房卡，将剩下两张房卡分别交给罗昊和李毅峰，便是朝着楼梯的方向走去，这家旅店一共就只有三层，所以并没有安装电梯，只能经由左右两侧楼梯上下楼。
<br/>　　就在罗昊接过房卡，准备上楼的时候，洛菲克和那个负责保护他安全的白人男子从旅店对面的酒店中走了出来。看到罗昊，白人男子目光立刻变得有些凝重起来，但很快，他就又撇开了目光，跟着洛菲克往小镇上的一个酒吧走去。
<br/>　　走在路上，洛菲克对跟在自己身边的白人男子问道：“David，跟我们遇到的这些人有什么问题吗？”
<br/>　　洛菲克能代表洛克希德公司前来参加今年的“源代码拍卖会”，自然不会是简单的人物，他也是注意到了自己这个来自黑水安保部队的雇佣兵队长David的神情变化。
<br/>　　“洛菲克先生，难道您不觉得奇怪吗？”David向洛菲克问道：“他们拿的是级别最低的红色卡片，持有这种卡片的，一般都是没有身份背景的自由者，可是很显然，这六个人都是军人，他们的步伐很好的说明了他们的身份，而且他们都是亚洲人，这是不是能说明一些问题？”
<br/>　　“你是说，他们是……”
<br/>　　洛菲克立刻就明白了David想要表达的意思，但是最后他还是没有把后面四个字说出来，眼睛中露出思索之色，说道：“他们怎么会来这里？难道是为了SHADOW而来？！”
<br/>　　没有不透风的墙，EG组长沙耶?本?华莱士就是消失已久的传说级黑客SHADOW的事情，虽然被NSA极力隐瞒，但还是被一些人得知了消息，洛克希德公司这种每年都会承接美国军方大笔订单的公司，在美国政府内自然是有着他们的人。
<br/>　　“当年GHOST和SHADOW最后一战之后，两个人就全都消失了，直到近年来，才偶尔会有GHOST活动的痕迹。”洛菲克思索着对David说道：“一直传闻GHOST参了军，为中方培养电子信息对抗的人才，如果他们的身份真的如我们猜测的那样，并且也是因为这次勒索病毒而为了SHADOW而来的话，那么……”
<br/>　　“GHOST很有可能就藏在他们中间！”David立刻明白了洛菲克话中的意思，随即对洛菲克问道：“GHOST和SHADOW同一级别的黑客高手，洛菲克先生，我们要怎么办？”
