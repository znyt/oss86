　　三日的时间，苏莫已经将功法完全转化完毕，一身玄力彻底转化了魔力，挥手间魔气滔天，魔威滚滚，实力再次暴增。
<br/>　　这时，他才算是真正的进入了混天魔功的第一层境界。
<br/>　　当然，仅仅是第一层的入门而已，距离第一层大圆满的地步，还相差甚远。
<br/>　　混天魔功之中，有很多独特的手段，非常强大，只不过苏莫现在没有时间修炼，准备等过几日再加以练习。
<br/>　　若是不考虑战魂等因素，单轮修为实力的话，现在这具肉身，比之苏莫的本体都丝毫不弱了，甚至更强三分。
<br/>　　当然，苏莫的本体，才武帝境二重修为而已。
<br/>　　苏莫离开了修炼室，走出了地灵塔，再次向武斗场走去。
<br/>　　不多时，他便来到了武斗场。
<br/>　　今日的武斗场，格外的热闹，人山人海，座无虚席，喧嚣震天。
<br/>　　四方的看台上，甚至有很多的巫族长老人物。
<br/>　　三日前，巫尘以武帝境三重修为，轻易斩杀巫烈和巫玉儿，然后又放下狂言，接下所有挑战之人。
<br/>　　此事，在黑耀城虽然不算多么轰动，但也在全城传开了，让得很多人得知了此事。
<br/>　　所以，今日才会有这么多人前来，甚至连不少族中长老都被吸引了。
<br/>　　战台上空无一人，并没有其它人或兽比斗，显然武斗场为苏莫留好了场地。
<br/>　　苏莫刚一走进武斗场，原本喧嚣震天的武斗场，立刻变得安静了下来。
<br/>　　唰唰唰！！
<br/>　　无数道目光，瞬间便落在了他的身上，让他成为了全城的焦点。
<br/>　　苏莫目光环视全场，看到了不少熟悉的面孔，巫天御、巫长远、巫源等人全部都来了。
<br/>　　巫源的目光中，带着冷笑之色，杀机毫不掩饰。
<br/>　　苏莫面无表情，身形一闪，瞬间便飞落到了战台之上。
<br/>　　“今日，凡是与我同等修为境界的人，想要挑战我，我巫尘来者不拒！”苏莫再次环视全场，朗声高喝。
<br/>　　他这是在告诉所有人，他只接受同阶之战，修为高于他的人，一律不战。
<br/>　　苏莫虽然自信现在的实力，但也不敢在巫族中肆意越级挑战，因为巫族中超级天才非常的多，有的人越级战斗实力超强。
<br/>　　以他这具肉身的实力，同阶一战他丝毫不惧，但越级挑战他就没有把握了。
<br/>　　不过，苏莫此言听在众人耳中，却是非常的刺耳，这明显得狂妄无比，对所有同阶修为之人的藐视。
<br/>　　来者不惧？
<br/>　　居然这么嚣张，不将所有同阶武者放在眼里，真是不自量力！
<br/>　　一个名声不显的人，不知道得到了什么奇遇，有了一点实力之后，居然就膨胀到了这等地步！
<br/>　　“我来杀你！”暴喝声响起，一名面宽体胖的巫族青年从看台上飞下，落在了战台之上。
<br/>　　这名青年穿着一声紫色长衫，身上气息极为浑厚，浩瀚如海，一看就不是普通的武帝境三重武者。
<br/>　　“巫尘，你的卑劣行径，让我感到不齿，所以我决定亲手解决你！”
<br/>　　紫衣青年面上带着绝强的自信，继续道：“你我同为武帝境三重武者，我也不算欺负你！”
<br/>　　紫衣青年一上场，顿时引起了一片议论之声。
<br/>　　“居然是巫粱宇！巫梁宇都忍不住出手了！”
<br/>　　“哈哈！巫梁宇战力极强，一般的武帝境四重巅峰武者，根本不是其对手，这下巫尘无法嚣张了！”
<br/>　　“梁宇大哥，狠狠的虐死这个卑劣的家伙！”
<br/>　　哗然声四起，呼喊声震天，众人对于紫衣青年极为自信。
<br/>　　苏莫闻听巫梁宇之言，轻笑一声，道：“你哪来的自信？”
<br/>　　“自信来源于实力！”巫梁宇傲然一笑，身为黑耀城有名的天才，其傲气尽显无疑。
<br/>　　“很好！希望你等会还能这么自信！”苏莫淡淡的说道。
<br/>　　这时，战台上的防御气罩升腾而起，两人均是不再说话，目光互相凝视，强大的战意冲天而起。
<br/>　　巫梁宇的手中出现了一柄硕大的紫金铜锤，足有水缸般大小，给人以强大的力感。
<br/>　　唰！
<br/>　　下一息，巫梁宇动了，身形一闪，瞬息十里，以闪电般的速度来到了苏莫的身前。
<br/>　　“碎！”
<br/>　　一声厉喝，紫金铜锤一抡而起，划过一道优美的弧线，向苏莫当头砸下。
<br/>　　这一锤力量浩瀚无比，犹如一座万丈巨山狠狠砸下，一锤之下，空间立刻四分五裂。
<br/>　　这一锤若是砸中苏莫，立刻就会将他砸成肉泥。
<br/>　　轰隆隆！
<br/>　　巨大的紫金铜锤，裹挟浩大的狂风，在苏莫眼中急速放大。
<br/>　　苏莫丝毫无惧，一剑逆斩而上，直接迎了上去。
<br/>　　剑光耀眼，魔气汹涌，瞬间便和紫金铜锤碰撞在了一起。
<br/>　　铛！
<br/>　　一声震耳的金铁交鸣之声响起，刺痛人的耳膜。
<br/>　　紫金铜锤上裹挟的力量，和长剑中澎湃的魔力硬撼，顿时如同核弹爆炸。
<br/>　　轰隆隆！
<br/>　　劲气瞬间炸裂，化为猛烈的风浪，横扫八方，将笼罩在战台之上的战法气罩都冲击的不断晃动。
<br/>　　嗖！嗖！
<br/>　　人影闪烁，苏莫和巫梁宇同时后退千丈。
<br/>　　“实力确实不错！”巫梁宇全身玄力沸腾，眸中精光四溢，与强大的同阶武者一战，已经彻底激起他心中澎湃的战意。
<br/>　　不过，刚才一击不过是稍作试探而已，真正的战斗还未开始呢！
<br/>　　“巫尘，你有资格让我动用全力，接下来，你必败无疑！”巫梁宇冷声大喝。
<br/>　　“你的实力若像你的口气这么大，我已经败了！”苏莫淡漠的说道，对方的实力虽强，但还不被他放在眼里。
<br/>　　他大量的战魂和武魂一旦动用，同阶之中能是他对手的人，屈指可数。
<br/>　　“现在，我便让你见识我们之间的差距！”巫梁宇也不动怒，结果很快便能见分晓。
<br/>　　唰！
<br/>　　话音一落，巫梁宇身形冲天而起，居高临下，再次一锤砸下。
<br/>　　“霸王锤！”
<br/>　　一声厉喝，紫金铜锤之上顿时幻化出一个巨大的锤影，足有一座宫殿般大小，向苏莫镇杀而下。
<br/>　　澎湃的气息如山崩海啸，猛烈的波动如长江大河，巨大的锤影蕴含无可匹敌的威势。
<br/>　　“有点实力！”苏莫心中暗赞，此人的实力比巫烈和巫玉儿强了不少，武帝境三重的修为，能有如此实力，在黑耀城之内的确堪称天才了。
<br/>　　不过，这点实力还不足以抗衡他。
<br/>　　苏莫已经没有了试探的心思，准备以雷霆之势，击败对方。
<br/>(本章完)
