　　“真枯长老，不能退出啊！”苏莫急忙对真枯半圣说道。

<br/>　　要不要退出会武，他自有考量，不可能听一个来观战的普通弟子之言。

<br/>　　不过，阴泽长老的话还没说完，就被一道刺耳的破空声打断了。

<br/>　　阴泽长老见此，神情一怔，顿时停下了口中的话语。
<br/>　　武斗场所有观战的人，见此全部都是神情一怔，真枯半圣都要退出会武了，这个时候居然还有道宗弟子上场。
<br/>　　而一众道宗的人，望着飞落到战台上的人，一时间均是不明所以，因为此人正是苏莫。
<br/>　　苏莫蹬上战台了？
<br/>　　这是要干嘛？
<br/>　　战台上，苏莫长身而立，在阴泽长老话还未说完之际，便飞上了战台。
<br/>　　他不得不这么做，不能再等下去了，不然等真枯半圣带着所以道宗弟子离开了，他自己还如何留下来？
<br/>　　所以，他立刻便上场了。
<br/>　　战台上，苏莫回头望向真枯半圣，朗声道：“真枯长老，会武才开始没有多久，胜负还是未知之数，怎么能就这样退出呢？”
<br/>　　听闻苏莫之言，所有观战之人都是面带怪异之色，这位道宗的弟子是要干嘛？居然当众驳斥真枯半圣的决定。
<br/>　　而此刻，一众道宗之人的脸色，都不禁难看了起来，苏莫这是在故意捣乱吗？还嫌道宗不够丢人？
<br/>　　真枯半圣的脸色黑了下来，被门下弟子当众反驳决定，让他大感脸上无光，而且还是一名跟来观战的无关弟子。
<br/>　　此刻，他心中有一股无形之火在燃烧。
<br/>　　本来今日道宗颜面尽失，他心中就非常的郁闷，现在又被苏莫这么一搞，他即便修养极高，也不禁要怒了。
<br/>　　若不是知道苏莫和真贤圣长老有关系，是真贤圣长老特意交代带着一起来太阴族的人，他现在就要将苏莫当场镇压了。
<br/>　　“苏莫，你干什么？赶快回来？”
<br/>　　冯光脸色一变，急忙对苏莫大喊了起来，同时他心中几乎气的吐血，这个苏莫是愣头青吗？你说你一个观战之人，怎么跑到战台上了？还反驳真枯半圣的决定！
<br/>　　“冯长老，我为什么要回去？现在我要挑战！”苏莫沉声说道。
<br/>　　“啥？”
<br/>　　冯光闻听苏莫之言，顿时面色一呆，差点惊掉了下巴，仿佛是听到了什么不可思议的事情一般。
<br/>　　要挑战？
<br/>　　你这是开玩笑吗？
<br/>　　你才武帝境八重的修为，也不是什么有名的天才，在这种局面下居然扬言要挑战？
<br/>　　冯光一个趔趄，差点跌倒在地。
<br/>　　一众道宗的弟子、长老，均是微微一愣，这个苏莫是脑子有问题吗？区区武帝境八重的修为，这个时候居然要挑战？
<br/>　　他们一直认为，苏莫只是前来观战之人，从来没想过苏莫要挑战，此刻均是有些回不过神来。
<br/>　　“哈哈哈！！”
<br/>　　就在此刻，一阵畅快的大笑声突然响了起来，传遍了整个武斗场。
<br/>　　而发出笑声之人，正是刚刚返回看台上的炎火。
<br/>　　“道宗这是真的无人了吗？居然连武帝境八重的武者，都要上台挑战了？”炎火满脸的戏虐之色。
<br/>　　炎火话音一落，整个武斗场亦是一片哗然之声。
<br/>　　“这个道宗的弟子，是在搞笑吧？”
<br/>　　“道宗三大顶尖天才都败了，他一个武帝境八重的武者，居然还想要挑战？”
<br/>　　“估计这个家伙是看道宗一败涂地，丢了大脸，心中有些气不过，想要上场一搏吧！”
<br/>　　“真是可笑啊！”
<br/>　　议论声、调笑声此起彼伏，很多人都被苏莫逗笑了。
<br/>　　这种场面的确是很可笑，道宗顶尖天才都败了，而一名修为低的普通弟子，还在扬言胜负为分，要继续挑战，岂不是可笑至极。

<br/>　　“苏莫，闹够了吧！我们回宗！”真枯半圣极力压下心中的的怒火，铁青着脸说道。

<br/>　　也是，他只有武帝境八重的修为，在明虚星河又没有丝毫的名气，众人以为他在胡闹，也是理所应当。
