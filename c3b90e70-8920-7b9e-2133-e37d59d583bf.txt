　　第692章
<br/>　　程大雷不得不承认，每次与崔白玉目光对视，他都会下意识心颤。说这姑娘有倾城之貌，其实也并不夸张。估计没多少男子能在这目光下撑住场子。
<br/>　　至于程大雷自己……其实也够呛。
<br/>　　“好姑娘，我有话说与你听。你我本是敌人，我怎样处置你都不过分。但对一个姑娘，本当家是慈悲的性子，毕竟要温柔些。”
<br/>　　“慈悲……”崔白玉忍不住重复这两个字，毫不遮掩自己的鄙视。
<br/>　　“是的，慈悲。”程大雷缓缓点头：“日后你便会发现，其实我是个好人。”
<br/>　　说着，程大雷直起身来，目光落在小鹿三人身上。三人一动不敢动，宛若被猛兽盯上的猎物。
<br/>　　“刚才你肯为她们出手，也不枉她们跟你一场。忠诚是个好东西，本当家也是很欣赏。”程大雷顿了顿道：“但意图对我不利，总不能当做什么都没有发生。本当家要从她们之中选一个服侍，你来做主，从中选一个，本当家便放过这件事。”
<br/>　　三人战战兢兢，虽然早有主忧臣辱，主辱臣死的觉悟，但这种事对女儿家来说，毕竟是天大的耻辱。
<br/>　　崔白玉颤巍巍站起身，忽然对程大雷笑了笑。便是这一笑，笑得程大雷心惊胆战。这似乎是崔白玉第一次对程大雷露出笑容，真令程大雷有种冒犯神女的感觉。
<br/>　　“程当家，你不就是想要我的身子么，何必如此惺惺作态。我愿服侍程当家，只求程大雷放过她们三个。”
<br/>　　程大雷有那么一刻的失神，轻轻拍了拍手。
<br/>　　“好，很好。现在你方才值得我刮目相看。”程大雷挥挥手：“难得你肯为她们出头，罢了，罢了，退下吧。”
<br/>　　崔白玉四人也没想到程大雷会放过她们，这才是真正令人刮目相看。崔白玉临走前，特意看了程大雷一眼。直到现在，她对程大雷仍没有一个大概的估计。
<br/>　　四人回到房中，方才察觉出了一身冷汗。刚才当真恐怖，四人稍微错一错，就没法保证完璧之身。
<br/>　　小鹿三人跪倒在崔白玉面前，道：“属下死罪，不该鲁莽行动。”
<br/>　　崔白玉摆摆手，令三人起身。现在实在不是分主仆之礼的时候，而崔白玉这主人也实在没比一个仆人强到何处。
<br/>　　她现在心里寻思的，还是如何同白空集三人中的任意一人取得联系。只要说动三人中的任意一个，自己就有机会反败为胜。事成之后，若能将程大雷擒住，必定将今日之辱千倍百倍的还回来。
<br/>　　想到此处，崔白玉的眼神炙热起来。
<br/>　　但如何将消息传出去呢……
<br/>　　这是个为难的事，现在她们四人在城主府中，能享受部分自由。但并没有离开城主府的自由，更不用说离开凉州城。
<br/>　　夜渐渐深了，崔白玉仍然想着这件事，难以入睡，身边小鹿三人已经睡熟。
<br/>　　正这时候，忽然响起轻轻的敲门声。崔白玉心中一惊，失声问：“谁？”
<br/>　　小鹿三女也翻身坐起，露出警惕模样。原来这三人也没有睡熟，时刻保持着警惕。
<br/>　　“郡主，是我？”
<br/>　　崔白玉一楞，将窗户推开半扇。一道黑影翻窗进来。屈膝便拜：“属下来迟，郡主受苦了。”
<br/>　　来人不是歌舒离又是那个。崔白玉大喜过望，没料到歌舒离会突然出现。
<br/>　　“你怎么来得？”
<br/>　　凉州城破之时，城内乱糟糟一片。也没有人注意受伤的歌舒离，他为了躲避搜捕，便躲进一口枯井中。等了这么多日子，风声小一些后，才敢冒头。
<br/>　　歌舒离说明了自己的经历，然后道：“这城主府的防卫松得紧，属下已经查探过数日，今夜一路过来，也没碰上什么人。”
<br/>　　看歌舒离此刻的模样，的确狼狈不堪。浑身污泥，身上散发着恶臭，如今是夏季，在潮湿闷热的枯井中躲了数日，也的确是委屈他了。
<br/>　　“郡主，那恶贼没有欺负你吧？”歌舒离揣着一把短刀：“属下这就护送郡主离开，逃出这魔窟。”
<br/>　　崔白玉摇了摇头，道：“你带上我们，怕自己出城都困难，逃出这城主府容易，怕离开凉州城难。”
<br/>　　“那该如何是好。”歌舒离叹了一声：“属下不忍郡主在这里受难。”
<br/>　　崔白玉心中一动，道：“你来得正好，我恰恰有件事要你去做。你连夜潜出城去，到最近的金城，以我的名义，说动他们起名攻打凉州城。”
<br/>　　“是。”歌舒离答应一声，眼睛看着崔白玉舍不得移开目光，道：“属下只是担忧郡主，在这里受了苦。”
<br/>　　歌舒离冒死来救，也不单单是为了一腔忠心。他负责崔白玉的安全工作，面对这样的女子，早已情根深处。此刻恰逢崔白玉遭此大难，他难免就有其他心思。
<br/>　　崔白玉很厌恶这样的目光，如果是之前，歌舒离敢这样盯着他，便是砍头的死罪。但今时不同往日，现在正是用歌舒离的时候，她也不好发作。
<br/>　　“歌舒将军速速去吧，安全要紧，以后日子还长。”
<br/>　　歌舒离顿时精神起来，双手抱拳：“郡主放心，属下即便死了，也要救出郡主。”
<br/>　　说着，翻窗离开，消失在夜幕中。
<br/>　　等他离去，崔白玉脸上的笑容才渐渐消失。与男人虚与委蛇实在不是她擅长的，也是她所不屑的。
<br/>　　但形势比人强，沦落到这种地步，自己不得不利用自己的美貌收拢人心。
<br/>　　只盼着，歌舒离能完成任务，引得大军过来，将凉州城一举拿下。如此，也不枉自己卧薪尝胆，忍辱负重。
<br/>　　这件事无论能否成功，毕竟是个希望，让生存在水深火热中的崔白玉多了点盼头。到了次日天明，崔白玉脸上也多了几抹红润，显得精神几分。
<br/>　　到了中午，程大雷差人将四人唤过去。
<br/>　　程大雷令人捧出一个食盒，一边翻书一边道：“喏，手下刚送来的稀罕玩意，我都不舍得享用，先给你们四人尝尝。”
<br/>　　崔白玉一头雾水，双手将食盒打开，只是一眼，整个人便吓得亡魂皆冒。
<br/>　　食盒内是歌舒离好大一颗人头。
<br/>(本章完)
