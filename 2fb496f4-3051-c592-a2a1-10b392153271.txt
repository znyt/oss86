题，所以让陈德林去了镇上，将自己的图纸和房屋建筑结构都做了备案，然后通过以后才让陈明盖的房子。<br/>　　<br/>　　    农村里盖房，一般是宅基地审批下来后，也就不管不顾的盖了，其实还是需要到村委进行备案的。<br/>　　<br/>　　    但是这一步，在农村没有那户能够遵守执行的。就算是要拆迁的村子，还有很多村民偷偷盖房子，何况是现在呢？本来陈德林还嫌陈默骚情，盖个房子还要去镇上备案，不过架不住陈默的央求，所以也就去办理了。<br/>　　<br/>　　    “你特么的说什么呢？我们联合执法是你胡日鬼的么？说你有问题，就是因为受到投诉后才过来的，你以为我们很清闲么？”领头男子身后一个人年轻人叫喊道。<br/>　　<br/>　　    “你们联合执法想做什么就做什么，跟我没有关系！但是你要给我工地贴封条，难道不能解释一下违反了那个规定有这么执法的么？”陈默沉声问道。<br/>　　<br/>　　    何老那帮人，也是皱着眉头看着，他们没有插言，就像看看基层一些人员的工作方式和工作态度，但是现在他们说看到的是什么呢？蛮横、无理、狂躁、无法无天！<br/>　　<br/>　　    “既然你有要求，那么我就给你说说！”领头的人拦住了身后的年轻人，对着陈默说道：“你知不知道，现在村庄都是集中规划集中住宅？要是另外盖房子的话，那么必须审批？”<br/>　　<br/>　　    领头的男子非常清楚，现在农村盖房有几个人会去审批资格？都是一帮子法盲，私自盖房的多的去了，所以他也就当陈默也是如此。<br/>　　<br/>　　    虽然听说这个叫陈默的是个大学生，可是这又能如何呢，还不是要听自己的！今天领导找自己来给这个叫陈默的找事情，他也是积极响应，好不容易碰见了，当然要好好表现一下自己。<br/>　　<br/>　　    “农村自家盖房子也要审批？”<br/>　　<br/>　　    “当然！要不然我们联合执法工作组成立是为了什么？就是为了管理现在农村乱搭乱建的状态，见一个拆一个！处理一个罚一个！”<br/>　　<br/>　　    说完，很有气势的指着陈默的别墅，说道：“给我封了！”<br/>　　<br/>　　    “等等！”陈默还是阻止了，然后问道：“这位领导，贵姓？”<br/>　　<br/>　　    “王！你也不用求情或者找藉口，你要是在阻止的话，就是在阻挠执法，知道么？”<br/>　　<br/>　　    姓王的领导一副官腔，很是正气的说道。<br/>　　<br/>　　    “那个，你过来的时候没有调查过么？我的房子是在镇建筑管理局备过案的！你们是靠什么来执法？”陈默问道。<br/>　　<br/>　　    姓王的领导听到陈默这么说，顿时一阵尴尬，这是他么的怎么回事？不是领导说这样找茬没有问题么？怎么这个叫陈默的却在建筑管理局背过案？到底怎么回事？<br/>　　<br/>　　    现在的农民盖个房子，难道都这么开明，服从政府领导，执行决策了？这还是法盲的代表么？<br/>　　<br/>　　    上个大学，就能这么厉害？<br/>　　<br/>　　    如果感觉还不错，请推荐和收藏一下，谢谢大家哈！<br/>　　<br/>　　    7真人小姐姐在线服务，帮你找书陪你聊天，请微/信/搜/索热度网文或rdww444等你来撩~<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
