　　紫霄神王心中惊愕不已，剑山之巅不仅被人登上了，这登上之人居然还是秦陨！
<br/>　　不过很快，紫霄神王的心中，惊愕被巨大的惊喜而取代。
<br/>　　秦陨若是真的能解开终极剑道的秘密，掌控终极剑道，这简直就是为他准备的。
<br/>　　毕竟，他手中有秦陨在乎的女人，那个已经算是死亡的女子，这是他要挟秦陨的筹码。
<br/>　　紫霄神王的眼眸之中，惊喜的光芒一闪而过，这是他的机会，上天给予的机会。
<br/>　　本来，他从秦陨身上抢来那个女人，只是为了胁迫绫裳，一番以后绫裳返回。
<br/>　　没想到，显得歪打正着，成为了他的巨大筹码。
<br/>　　“紫霄，这秦陨是你紫霄剑宫的弟子？”终极神王问道，紫霄的模样，显然是认识秦陨。
<br/>　　“殿主，这秦陨是绫裳剑宫的弟子，我曾经见过！”紫霄神王摇头说道。
<br/>　　“哦？”终极神王讶然，原来是绫裳剑宫的弟子，如此不凡之人，居然没有任何的名气，实在奇怪。
<br/>　　“绫裳！”
<br/>　　随即，终极剑王看向绫裳闭关的塔楼，轻喝一声，滚滚声浪震荡四野。
<br/>　　嗖！
<br/>　　随后，便见绫裳从绫裳剑宫之中飞了出来，飞到了高空，停留在了终极神王的身前。
<br/>　　“秦陨是否你门下弟子？”终极神王沉声问道。
<br/>　　“是！”绫裳没有否认，这突然出现的变故，她已经知晓了与苏莫有关，适才她也听到了紫霄等人在谈话。
<br/>　　终极神王闻言，略一沉吟，道：“绫裳，你与紫霄一起，去一趟万道剑冢，务必将秦陨带回来！”
<br/>　　“是，殿主！”绫裳面色平静，微微抱拳，她能猜到，殿主必然会有所安排。
<br/>　　毕竟，终极剑道对于终极神王殿而言，实在太过重要了。
<br/>　　“绫裳，事不宜迟，我们走吧！”紫霄神王催促道，殿主的安排正和他意，他必须立刻将秦陨控制住。
<br/>　　他可是心中清楚，秦陨并不是一般的终极神王殿弟子，很可能逃离终极剑域。
<br/>　　虽然他手中有筹码，但是，为防万无一失，必须立刻控制住秦陨。
<br/>　　“嗯！”绫裳点头，随即，便可紫霄一起，离开了神剑山。
<br/>　　绫裳并不知晓芊寻月在紫霄的手中，也不知晓紫霄心中的打算，苏莫到底如何，还需要见到之后再说。
<br/>　　绫裳和紫霄走后，终极神王立刻安抚弟子，很快，神剑山上便恢复了平静。
<br/>　　……
<br/>　　苏莫在剑山之巅消失，眼前千回百转，被传送到了一处虚无之地。
<br/>　　这里一片光晕弥漫，充满了各种各样的气息，数之不尽。
<br/>　　而苏莫的目光，瞬间就锁定了前方，因为在他的前方，有着一个巨大的球体。
<br/>　　这个球体，直径足有数十丈，颇为巨大，悬浮在虚空之中。
<br/>　　“这是……！”
<br/>　　苏莫瞪大了眼眸，心中惊疑不定，这就是最后一关吗？是什么考验？
<br/>　　苏莫小心翼翼，谨慎的向前方飞去，缓缓的接近了那巨大的球体。
<br/>　　这巨大的球体，其内有无数的光芒在游走，仔细一看，居然是无数的剑气。

<br/>　　火焰剑道，雷霆剑道，狂风剑道，阴阳剑道等等等等，甚至，在这些剑道之中，苏莫还看到了他自己的剑道。
<br/>　　不错，就是他的吞噬剑道，以及融入了不败战气意志的吞噬剑道。
<br/>　　这剑道的气息，和他的剑道一模一样，没有任何的区别。
<br/>　　“果然！果然是这样？”
<br/>　　苏莫心中震动，他之前猜测的果然不错，万道剑冢之中，果然在收集剑道，他的剑道，也被收集了。
<br/>　　他心中明白了，所谓的终极剑道，就是靠收集别人的剑道，才聚齐了几千种剑道，形成最强的终极剑道。
<br/>　　苏莫清点了一番，这巨大球体之内的剑道，足足有两千九百五十余种，接近三千。
<br/>　　随后，苏莫围绕着巨大球体，转了一圈，有些不知所措。
<br/>　　这里，并没有任何的考验，只有这蕴含近三千种剑道意志的球体。
<br/>　　踌躇了良久，苏莫也不知道该如何是好，他有一种感觉，这球体就是施展终极剑道的基础，但是，该如何掌控呢！
<br/>　　吞噬？
<br/>　　参悟？
<br/>　　还是其它方法！
<br/>　　苏莫想不明白，他精神意念探出，牢牢的笼罩住整个球体，检查这球体的状况。
<br/>　　在苏莫的检查之下，这球体并无丝毫的危险性，虽然其内有很多的剑道意志，但是，并不具攻击力。

<br/>　　球体进入体内，立刻便与苏莫的神魂开始了融合，速度奇快无比。

<br/>　　苏莫心中大惊，随着球体融入神魂，很多的信息涌入了他的脑海之中。
<br/>　　“剑源……终极神剑……！”苏莫口中自喃，一瞬间，他便明白了一切，对于融入神魂的球体，心中也没有了任何的排斥。
