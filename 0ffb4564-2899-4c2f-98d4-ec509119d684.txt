<br/>　　“苏莫大哥，我这座行宫可不简单，你知道是用什么建造的吗？”
<br/>　　小八面带笑容，如同是一个小大人一般，背负着双手，面带得意之色。

<br/>　　苏莫闻言，打量起了眼前的宫殿，发现这宫殿是由一种不知名的岩石建成，这岩石呈墨色，其上还有一道道雷纹。
<br/>　　仔细感受，苏莫在这种岩石之中，居然还感受到了滂湃的雷霆之力。
<br/>　　“雷霆之力？”苏莫望向小八，目露询问之色。
<br/>　　“嘻嘻！”
<br/>　　小八闻言一笑，道：“这是蛮荒山脉特有的雷纹石，非常珍贵，师父说了这雷纹石有助于我提升修为！”
<br/>　　“你修炼的是雷霆之力？”苏莫闻言眼眸一亮，惊讶的问道。
<br/>　　若是小八雷属性天赋了得的话，以后倒是可以继承雷霆宫的传承。

<br/>　　还有水月宫、长青宫、金煌宫、厚土宫、雷霆宫没有传人。
<br/>　　其中长青宫是为宏青璇准备的，厚土宫苏莫暂将战乱作为候选人，而剩下的雷霆宫、水月宫和金煌宫三大宫，他还没有看重任何适合的人选。
<br/>　　若是小八雷属性天赋了得，倒不失为雷霆宫的不二人选。
<br/>　　只不过，小八是妖族，不知道适不适合雷霆宫的传承！
<br/>　　“当然，我们八翼飞龙蛇一族，天生便有超控雷霆的本领，我自然修炼雷霆之力！”
<br/>　　小八点了点头，随即拉着苏莫来到了一间练功房之中，笑道：“苏莫大哥，这就是我平常修炼的地方，不错吧？”
<br/>　　“嗯！是不错！”苏莫扫了一眼练功房，微微点头，嘴角却是忍不住的抽了抽。
<br/>　　因为在这间练功房中，散落着不少兽骨，这些兽骨明显被啃食剩下的。
<br/>　　是被什么人啃食，不用想苏莫也知道，必是身前这个小萝.莉。
<br/>　　苏莫不由的心中暴汗，这小八虽然看着模样可爱，但终究是一只妖兽啊！
<br/>　　小八带着苏莫，在她的宫殿中逛了逛，少倾之后，一道淡漠的声音，突然在宫殿中响起。
<br/>　　“小丫头，过来！”
<br/>　　小八听到这道声音，顿时吓得脖子一缩，显然比较惧怕此人。
<br/>　　“师父叫我了，我们过去吧！”小八不敢耽搁，急忙走出自己的宫殿，向另外一座宫殿而去。
<br/>　　苏莫便也抬脚跟了过去，他知道是那名黑衣中年人喊他们了。
<br/>　　出了小八的宫殿，两人走进了属于黑衣中年人的宫殿之中。
<br/>　　宫殿中，光头大汉并不在，只有黑衣中年一人，正盘膝坐在蒲团上。
<br/>　　“小子，你可以离开了！”
<br/>　　黑衣中年的人的目光，落在苏莫的身上，淡淡的说道。
<br/>　　呃~~
<br/>　　苏莫闻言一怔，他还想和对方套套近乎，没想到对方居然直接下逐客令了！
<br/>　　“走吧！看在小丫头的份上，本帝不杀你！”黑衣中年人挥了挥手，一脸淡漠的表情。
<br/>　　苏莫暗叹一声，既然对方不欢迎他，他也不会死皮赖脸的留在这里。
<br/>　　这时，小八脸上露出气恼的表情，皱眉向黑衣中年人问道：“师父，苏莫大哥才刚来，你怎么就让他走呢？”
<br/>　　“小丫头，你和他已经没有关系了，他早已不是你的主人！”
<br/>　　黑衣中年人摇了摇头，对苏莫道：“再不走，你就永远不用走了！”
<br/>　　黑衣中年人的声音淡漠无比，其中已经蕴含了一丝杀机，若不是清楚小八和苏莫的关系，他早已杀了苏莫。
<br/>　　“好吧！告辞了！”
<br/>　　苏莫脸色稍稍有些难看，点了点头，便转身向外走去。
<br/>　　“这……！”
<br/>　　小八见此，气恼的跺了跺脚，但她师父的话，她也不敢阻拦。
<br/>　　随即，小八急忙追了出去：“苏莫大哥，我送你！”
<br/>　　而后，苏莫便和小八一起，向圣山外走去。
<br/>　　“小八，你有其它名字吗？”
<br/>　　苏莫向小八问道，‘小八’这个名字还是他当初取的，因为对方是八翼飞龙蛇，所以他便取了这么一个简单的名字。
<br/>　　但现在，这个名字显然不合适了。
<br/>　　“没有！师父他们都叫我小丫头！”小八摇了摇头道。

<br/>　　苏莫闻言苦笑，看来对方也知道‘小八’这个名字不太好。

<br/>　　苏莫点了点头，沉思了片刻，道：“想好了，翼晓晓，这个名字怎么样？”

<br/>　　小八闻言眼眸一亮，心中顿时大喜，道：“好，我以后就叫翼晓晓了！”
<br/>　　小八显得非常的兴奋，她师父就像个木头，整天板着张脸，都不知道给她起个名字，还是苏莫大哥想的周到。

<br/>　　所谓君子报仇十年不晚，但是苏莫，却是一刻也不想等了。
<br/>　　这一次回中洲，他就要报仇！
