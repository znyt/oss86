陆寒霆来了。
<br/>
<br/>        今天陆寒霆穿了一身手工版的黑色西装，昂贵的衣料被熨烫的没有丝毫褶皱，衬的他高大挺拔，额头的刘海全部往上推，露出了他那张英俊如天工雕琢的俊脸，腕上一块名贵的钢表，他举手投足都弥漫着上流成功人士的优雅精致和考究，简直让人移不开目光。
<br/>
<br/>        现在他迈开长腿踏着稳健的步调走来，那剪裁如刀锋的黑色西裤都迈出了令人炫目的磁场，三个月没见的陆寒霆高高在上的让人仰望，成熟内敛的气场里又透着一股清冷的禁欲感来，直接让现场这些名媛们看的面红耳赤。
<br/>
<br/>        最开心的就是厉嫣然了，她一直在等陆寒霆，现在他来了，她当即开心的迎了上去，“寒霆哥哥，你来啦？”
<br/>
<br/>        陆寒霆停下脚步，他深邃的狭眸落在了厉嫣然漂亮夺目的小脸上，嗓音低沉磁性的开腔道，“我答应你会来的，嫣然，生日快乐。”
<br/>
<br/>        “寒霆哥哥，谢谢你，你能过来就是我今天收到的最好的礼物！”厉嫣然仰慕爱恋的望着他。
<br/>
<br/>        厉嫣然自己就出身高门，是真正的高门贵女，她什么样的男孩子没有见过，但就是对陆寒霆一见钟情。
<br/>
<br/>        陆寒霆跟帝都城的这些豪门子弟们都不同，他是真正的天之骄子，商界贵胄，光这一身高冷而禁欲的气息就能让人心动不已。
<br/>
<br/>        帝都城的名媛们都是怕他的，厉嫣然也有点小怕，但是这些名媛们聚在一起，谈论最多的还是陆寒霆，大家都在赌最后是哪家的名媛能将陆寒霆这个难搞的男神给拿下。
<br/>
<br/>        这时大家都聚了过来，起哄道，
<br/>
<br/>        陆少，今天是嫣然的生日，你的生日礼物呢？
<br/>
<br/>        不如这样吧，陆少拿一个吻当礼物吧，亲嫣然一下。
<br/>
<br/>        亲一下！亲一下！
<br/>
<br/>        大家将两个人围在中间，都起哄“亲一下”，厉嫣然迅速心跳加速，小脸红红的看向男人，“寒霆哥哥…”
<br/>
<br/>        她欲说还休的样子，很期待陆寒霆亲她。
<br/>
<br/>        陆寒霆身高腿长的伫立着，没有动，此时他站在璀璨的灯光下，金黄的光束镀了他一身的光芒，这上等的精致皮囊不知道迷了多少人的眼。
<br/>
<br/>        厉嫣然上前，踮起脚尖就凑到了陆寒霆的俊颊上，快速的亲了一口。
<br/>
<br/>        陆寒霆没有亲她，她就主动去亲。
<br/>
<br/>        退下来的时候众人“哇”了一声，
<br/>
<br/>        嫣然，你好主动哦。
<br/>
<br/>        今天嫣然过生日，嫣然说了算，要不，再亲一口，这一次就要亲…嘴巴了哦。
<br/>
<br/>        亲嘴巴！亲嘴巴！
<br/>
<br/>        厉嫣然是帝都第一美人，追求她的男孩子如过江之鲫，数不胜数，她从来就不会多看一眼，能让这位首富千金这么主动的男人，只有陆寒霆一个。
<br/>
<br/>        亲嘴巴！
<br/>
<br/>        厉嫣然贝齿咬了一下红唇，水汪汪的眸子害羞又紧张的看着陆寒霆，陆寒霆也看着她，没有责备她的意思，眼神温和而纵容。
<br/>
<br/>        厉嫣然又上前，踮起脚尖往他的薄唇上亲。
