　　这突如其来的变化，也让楚风眠的心神，微微平静了下来。
<br/>　　本来楚风眠大闹北邙宫的时候，心中已经是有着鱼死网破的心思了。
<br/>　　因为有着云羽天宫作为底牌，楚风眠也不会怕什么，所以他在斩杀了那些夜家老祖时候，便是打算利用云羽天宫，全身而退，随后离开北邙学院。
<br/>　　等到他实力足够的一日，在重新归来。
<br/>　　可是楚风眠做梦也没有想到，这北邙宫，居然就是剑皇殿，夜皇打算利用北邙宫来对付楚风眠，反倒是带给了楚风眠这样一番机缘。
<br/>　　如此一来，楚风眠倒是还要谢谢夜皇了，若是没有夜皇，也许楚风眠还无法发现，这北邙宫正是剑皇殿，倒时候错过了这等机缘。
<br/>　　行事，在瞬息之间便是逆转过来。
<br/>　　楚风眠凌空而立，脸色轻松无比。
<br/>　　玄北圣者，邓灵副院长，还有那些北邙学院的老祖，一个个都是脸色有些呆滞，目光闪烁，似乎都在思考，一时间都无法想明白，到底是如何发生了这一切。
<br/>　　而相比之下，夜皇等人的脸色，才是极为阴沉，难堪到了极点。
<br/>　　尤其是夜皇的心中，更是恼怒无比，他若是不镇压楚风眠，而是选择直接轰杀楚风眠的话，也就不需要动用北邙宫的力量，不至于如今引得北邙宫，居然是站在了楚风眠这一边。
<br/>　　尤其是他们四人联手之下，居然是被这一道剑芒，轻而易举的给逼退了。
<br/>　　这四尊创始者，如今在刚刚的剑锋之下，都已经是重伤，一个个连连后退，眼神中都是充满了忌惮，生怕是这剑芒，再一次出现。
<br/>　　倒时候他们四人可就没有那么好运了，很有可能都会被轰杀。
<br/>　　武皇，风皇等人，现在一个个都是有些说不出话来，眼神中尽是恐惧，他们身为北邙学院的创始者，天命之境的超级强者，就算是在整个九域不能说是横着走，也算是赫赫有名的人物了。
<br/>　　从未想过，居然会距离死亡那么近，现在没有人还敢再度出手。
<br/>　　“怎么回事？北邙宫怎么发生了异变？”
<br/>　　突然之间，一道身影，从外面突然飞了进来，这是一名中年男子，看起来三十岁出头的年纪，一身长袍，衣着极为尊贵，有着一种高高在上，久居上位的气质。
<br/>　　这中年男子，正是北邙学院的院长，平日里神龙见首不见尾，神秘无比，楚风眠来到北邙学院，都是第一次见到此人。
<br/>　　“院长！”
<br/>　　“院长！”
<br/>　　在场无数老祖，都是恭敬的行礼道，北邙学院的院长，地位甚至是还要高过那些创始者一头，因为这院长的实力，甚至都不在创始者之下。
<br/>　　甚至是现在隐约一看，这北邙学院院长的实力，比起夜皇等人，还要强上几分。
<br/>　　“北邙宫，居然认主了？怎么回事？”
<br/>　　这北邙学院的院长，一来到这北邙宫中，便是心神一动，灵识扫视了周围的一切，脸色突然露出不可知悉的神情来。
<br/>　　整个北邙宫，如今都是发生了翻天覆地的变化，最令他震惊的是，北邙宫居然是认主了。
<br/>　　当初自从北邙圣君死后，北邙宫便是一直坐落于这里，甚至是在当初，北邙圣君都没有能够让这北邙宫认主。
<br/>　　随后的万年时间之中，北邙学院的无数强者，也有不少人都在打这北邙宫的主意，想要让其认主，却都一无所获。
<br/>　　可以说整个北邙学院建立的万年一来，北邙宫都从未认主过，现在却是突然认主了，这让他如何不惊讶。
<br/>　　“玄北，怎么回事？”
<br/>　　北邙学院的院长看向玄北圣者，突然开口道。
<br/>　　他虽然很少在这北邙学院之中出现，但是对于其中发生的种种事情，他也是曾经知晓的，现在这北邙宫认主的人是楚风眠，他第一个便是询问玄北圣者。
<br/>　　毕竟楚风眠，可是玄北圣者的弟子。
<br/>　　“这件事，我也不知。”
<br/>　　玄北圣者平静的回答道。
<br/>　　对于楚风眠身上的种种神秘，玄北圣者也不知道，他也没有去询问的必要。
<br/>　　每一尊武者，都有着各自的奇遇，有着自己的底牌，他就算是身为楚风眠的师尊，也不需要去一一询问。
<br/>　　“北邙宫认主了，这可是大事，我们北邙学院万年一来，都没有发生过的事情！”
<br/>　　夜皇在一旁，目光闪烁，突然对着北邙学院院长开口道。
<br/>　　“院长，此人曾经在众圣堂中，得到过北邙圣君的传承，也许是在那个时候，他得知了这北邙宫的认主方法。”
<br/>　　“这楚风眠，实在是太过危险，也太过狂妄了，目中无人，这样的人决不能留在我们北邙学院中，不如我们现在将此人擒下，将这北邙宫认主的方法，逼问出来！”
<br/>　　夜皇的心中还有些不甘心。
<br/>　　楚风眠之前当着他的面，杀了那么多的夜家老祖，今日要是让楚风眠全身而退的话，他会懊悔终身。
<br/>　　可是现在楚风眠有着北邙宫的保护，他在对楚风眠出手，只是自取灭亡，所以他只能将希望，寄托在这北邙学院的院长身上。
<br/>　　北邙学院的院长，是一尊极为神秘的人物，他的实力要超出大部分的创始者，甚至是隐约之间，有传闻，他便是北邙学院真正的第一强者。
<br/>　　想要对付楚风眠，只有他出手才行。
<br/>　　“夜皇，你真的想死？”
<br/>　　听到夜皇的话，楚风眠目光闪烁，眼神杀意凝聚到了极点。
<br/>　　这夜皇，他必须要找个一个机会斩杀，不然以此人对楚风眠的怨恨，将来必然是后患无穷。
<br/>　　只是如今，在这神秘的院长面前，楚风眠也不好直接出手。
<br/>　　“夜皇，你先稍安勿躁，楚风眠终究还是我们北邙学院的弟子，这北邙宫既然选择认他为主，也是我们北邙学院的一件好事。”
<br/>　　北邙学院院长，沉吟了一会，突然开口道。
<br/>　　“楚风眠，你可知道这北邙宫，为何认主与你？”
<br/>　　“只要你说出来，这北邙宫依然还是你的，最多将这秘密一直传承下去，毕竟这是我们北邙学院一直以来都不知道的一个秘密，你若是说出来，我保证给你足够的好处，甚至是让你成为这北邙学院，下一任的院长。”
