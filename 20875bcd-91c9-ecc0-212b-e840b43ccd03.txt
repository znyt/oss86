　　“水月，你那弟子带走了独孤玉凤等人，可是出于你的授意!”谈了一个晚上，将基本的东西了解的差不多之后，袁守诚终于问出事情的关键。
<br/>　　水月倒也没含糊，想都没想便点了点头：“不错，师兄，正是出于我的授意！”
<br/>　　“为什么？为什么你要做的这么绝呢？寻雪与我们的关系天下尽人皆知，你，你这样做岂不是等于告诉所有人，这件事情是我们在背后搞鬼。”秦英老道佝偻着身子，几乎是一字一顿的问道。
<br/>　　这老道辈份奇高，水月纵然与袁守诚同辈，但却不敢在这老道面前无礼，见他开口相询微微坐正了身体：“前辈，非是晚辈胆大妄为，实在是不得以而为之。”
<br/>　　“为何？”秦英问道。
<br/>　　水月并没有直接回答秦英的问题，反而沉声问道：“前辈可知七星玉佩？”
<br/>　　“略有耳闻！”秦英点点头说道。
<br/>　　“七星玉佩眼下已经齐聚皇宫，再加上玄雪身上的副佩，八佩俱全！”水月脸上露出一副悲天悯人之情：“这是天下将要大乱之兆啊！”
<br/>　　“糊涂！道听途说之言如何能够当真。”秦英干枯的手在椅子扶手上重重一拍。
<br/>　　面对老道士的训斥，水月抗声说道：“为何不能当真，想当年这几块玉佩随诸葛武候现世，那一场持续数百年的杀戮难道不是真实写照么？”
<br/>　　“就算如此，你也不能设计害死长乐公主，你可知道你已经惹下滔天大祸，道门一脉将要面对的是被血洗的后果！”秦英老道士见水月依旧不知会改，气的一个劲的拍打椅子扶手，如果不是身体条件不允许，估计早就跳起来打人了。
<br/>　　“怎么可能会有人知道！”
<br/>　　“为什么不会有人知道，举头三尺有神明，你真的以为李承乾就什么都不知道？你真以为他就是个傻子？”
<br/>　　“……”水月觉得秦英这个老道士实在是胆子太小，有些不耐烦，只是碍于辈份不好再多说什么。
<br/>　　可就在这个时候，玄都观的大门猛的在外面被人轰开，一大队身着黑色作战服的禁军冲了进来，进门之后二话不说见人就抓。
<br/>　　“飞凤……军！”这可真是说曹操曹操就到，秦英老道士苍老人脸上露出一丝错愕，似乎没想到自己的话竟然如此灵验。
<br/>　　水月见秦英已经抖成一团，心中微微有些不屑，瞥了他一眼，站起身直接走了出去：“住手，你们是什么人，为何来玄都观撒野？”
<br/>　　“大唐飞凤军奉旨拿人，查封玄都观！”身着黑服作战服的人群中走出一人，冷冰冰的看着水月：“你又是什么人？！”
<br/>　　“贫道水月！你们……”
<br/>　　“水月？”黑衣大兵皱了皱眉头，似乎努力在想些什么，片刻之后恍然问道：“长白山水月？”
<br/>　　“不错，正是贫道，尔等既然知道贫道的名声，还不……”水月在长白山一带颇有些名声，当年高句丽占据了辽东的时候就算是泉盖苏文见了她都要给几分面子，养成了她孤傲的性子。
<br/>　　所以在她看来，大唐长安似乎也应该如此，只要报上名号就可以震慑住那些大兵。
<br/>　　但是她却并没有发现，其实在她报出自己名字的时候，不管是秦英还是袁守诚全都是面色一变，而她面前的那个本应退下的大头兵更是狰狞一笑：“好好好，得来全不费功夫，来人，将此人拿下！”
<br/>　　“是！”
<br/>　　‘哗啦’一声，随着一声响亮的回答，数支步枪已经上膛，四、五个黑衣大兵将手里的枪对准了水月。
<br/>　　“你，你们干什么，竟然对本座无礼！”水月完全没有想到会有这样的局面出现，脸色瞬间一白。
<br/>　　就在一群大头兵想要上来抓人的时候，袁守诚见机不对，连忙从里面出来，与那些大兵的头头打了个照面：“来将军，来将军且慢，可否容老道讲几句话？”
<br/>　　不过袁守诚同样失算了，李承乾对外下的命令可不仅仅是针对玄都观和水月，就连他袁守诚也同样在必抓之人的行列之中。
<br/>　　那刚刚下完命令的来将军对他的话根本就是置若罔闻，冷冷看了他一眼说道：“道长有什么话一会儿到大理寺再说吧，来人，观中所有人等，一个也不要放过，全都锁拿大理寺！”
<br/>　　“哗……”这下子，整个玄都观都乱了套，一群完全不知道发生了什么就被虎狼一般的飞凤军三拳两脚打翻在地，用绳子一穿绑到一起。
<br/>　　袁守诚、袁天罡、秦英等人也免不了被抓的命运，一个两个在步枪的逼迫下不得不束手被擒。
<br/>　　只有水月还想要挣扎一下，结果被那所谓的来将军掏出手枪打中了胳膊之后，也放弃了无谓的抵抗。
<br/>　　……
<br/>　　一大清早，玄都观被封的事情几乎瞬间传便全城，联系到之前竞技场的那场事故，长安城的百姓瞬间就炸了，无数人都想不明白，这些道士是怎么跟那件事情扯上关系的。
<br/>　　不过皇命就是皇命，容不得任何质疑，况且李承乾在这次的事件之前，在百姓中的威望和名声一直都很不错，所以长安城的百姓虽然不解，但却并没有怎么闹腾，都在等着官方给出一个明确的说法。
<br/>　　而就在长安百姓等着消息的时候，一个更劲爆消息传了出来——隐王一脉犯上做乱，欲引拜占庭人入关！
<br/>　　消息虽然没有经过官方确认，但是长安各处府衙集体缄默，菜市口数百颗血淋淋的人头已经足以说明一切。
<br/>　　这下子长安百姓再也不淡定了！
<br/>　　刚刚平静了没有几天，好不容易大唐不打仗了，结果又跑出一些不甘寂寞的人来想要搅局，这几乎让长安百姓忍无可忍。
<br/>　　一些情绪激动的更是跑到驿馆外面向属于拜占庭人的院子里丢石头，丢大便。
<br/>　　还有愤青则是跑到皇城外面请命，希望皇帝陛下可以派兵征讨不臣，将拜占庭的全部消灭。
<br/>　　仅仅几天时间下来，拜占庭人的名声一下子就臭了，没有任何一个拜占庭人敢离开驿馆半步，否则将会被被人打死在街上的可怕后果。
