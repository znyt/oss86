　　南苑的交易坊市，在一处平坦空旷的空地之上。
<br/>　　这交易坊市，是神武学府弟子自行聚集而成，千百年来，都是在此地。
<br/>　　坊市人员极少，只有数百名南苑弟子，在此地摆摊售卖宝物。
<br/>　　神武神武学府弟子本就不多，外府四苑总共也就十几万人而已，而且均是天赋出众之辈，只有身上无用物品过多，才会过来处理一次。
<br/>　　苏莫在坊市中逛了一圈，也没有发现售卖兽魂的人，售卖武魂和战魂之人，那就更没有了。
<br/>　　这让苏莫非常的郁闷，没有兽魂、武魂和战魂，他的吞噬武魂就无法晋升等级，那就不能进化战魂。
<br/>　　“看样子只能去杀了！”苏莫心中暗道，没有兽魂和武魂，他只能靠自己出去击杀了。
<br/>　　只不过，这个效率有点低，不知道能否来的急！
<br/>　　但是，即便希望不大，他也要拼一次，既然答应了冷江的约战，他就不会退缩。
<br/>　　战，就要胜之！
<br/>　　随后，苏莫也没有浪费时间，立刻离开了神武学府，向途山镇的方向飞去。
<br/>　　他当初来神武学府之时，路上可是发现了不少山高林密的山脉，其内凶兽众多，正是他猎杀凶兽的地方。
<br/>　　值得一提的是，这凶兽和妖兽不同，妖兽强大到一定的等级之后，都纷纷化为人形，成为了一支支妖族。
<br/>　　妖族也是这宇宙中，最为庞大的群体，据闻仅是古灵星河，就有十几万个妖族群体，有的甚至比那些人族超级势力还要强大。
<br/>　　而这凶兽，可以说是未开化的妖兽，就算实力再强也不会化形，灵智极低，凶残暴戾。
<br/>　　这种凶兽，是无数武者猎杀的猎物，同时一旦遇到人类，这些凶兽也会疯狂扑杀。
<br/>　　苏莫离开了神武学府所在的陨落星辰，一路急行，两个时辰之后，便离开了陨落星辰所在的荒原。
<br/>　　而后，他又飞行了几个时辰之后，便来到了一处浓密的山脉中。
<br/>　　这山脉不大，方圆只有几万里，苏莫伫立在山脉上空，灵识扫荡而出，覆盖方圆数千里。
<br/>　　他发现这山脉中，相当于武帝境的八级凶兽极少，大多都是七级和六级凶兽，以及更低级别的凶兽。
<br/>　　苏莫毫不犹豫，立刻开始了吞噬，直接催动吞噬武魂。
<br/>　　强大的吞噬之力，顿时席卷八方，笼罩方圆三千余里。
<br/>　　吼！吼！吼！
<br/>　　霎时之间，山脉中响起了无数的兽吼之声，惊天动地，震耳欲聋。
<br/>　　随即，山脉中爆炸不断，凶兽的惨叫声、哀嚎声，此起彼伏。
<br/>　　大量的凶兽惨死在吞噬之力下，兽魂、血液、精气等等，全部向苏莫汇聚而来。
<br/>　　苏莫只吞噬兽魂，其他的全部都舍弃了，因为他的修为已经到了瓶颈，吞噬也没有任何用处。
<br/>　　吼！
<br/>　　就在此刻，一只庞大的狮型凶兽，从远处山脉中冲了出来，急速向远处飞逃。
<br/>　　这是一只堪比帝级武者的八级凶兽，它虽然灵智不高，但面对苏莫的吞噬之力，也是感觉到了致命的危险，疯狂逃窜。
<br/>　　死！
<br/>　　苏莫手指一点，一柄本命灵剑便****而出，洞穿了虚空，以无与伦比的速度，瞬间击中了这头凶兽的脑袋。
<br/>　　嘭！
<br/>　　脑袋被洞穿，这头狮型凶兽立刻惨死，巨大的兽魂遭到吞噬之力的影响，根本逃无可逃，立刻被吸纳了过来。
<br/>　　苏莫疯狂的吞噬兽魂，他的身形在山脉中飞过，所过之处必定是腥风血雨，所有凶兽，任何生灵，无一幸免。
<br/>　　他吞噬的速度极快，因为这山脉不大，又没有太厉害的凶兽，紧紧是小半个时辰，整个山脉中便再无活物。
<br/>　　“继续！”
<br/>　　苏莫离开了这处山脉，又向下一个目标而去。
<br/>　　他也不挑那些大型山脉，免得遇到实力太强的凶兽，自己不敌，再遇到危险。
<br/>　　只选择那些中小型的山脉，疯狂的吞噬。
<br/>　　他的吞噬武魂，因为有了吞噬奥义之助，威力更强，范围更广，简直就是屠杀弱小的大杀器。
<br/>　　那种方圆万里的小山脉，其内的凶兽，他片刻功夫就能吞噬的一干二净。
<br/>　　真正的血雨腥风，苏莫的身影辗转在一座座山脉中，仅仅是十天的功夫，他已经吞噬了不下上亿的兽魂。
<br/>　　这让身在苏莫吞噬武魂之内的阎魔鬼圣，即震惊又羡慕！
<br/>　　看到无数的兽魂不停的被吞噬，他心中的震惊，简直无以复加。
<br/>　　知道苏莫的武魂能吞噬兽魂是一回事，亲眼见到如此疯狂的吞噬，则又是一回事。
<br/>　　而且，感受到那些被吞噬的兽魂，哺进吞噬武魂中的魂力，他充满了渴望。
<br/>　　“苏莫，汝虚弱不已，这些魂力……能不能分给吾一些？”阎魔鬼圣向苏莫说道。
<br/>　　此时，苏莫正站在一处山巅之中，吞噬方圆三千里之内的凶兽，闻言微微摇头，道：“待我武魂升到天级八阶，可以分你一些！”
<br/>　　苏莫现在不可能给对方好处，不然的话，对方实力恢复，他就控制不住对方了。
<br/>　　“好吧！吾等你！”阎魔鬼圣叹道，随即他便不再多言。
<br/>　　时间缓缓流逝，一日又一日。
<br/>　　眨眼间又过去了三日。
<br/>　　这一日，身处在一座山谷中的苏莫，在吞噬了一头八级一重凶兽之后，吞噬武魂顿时一阵巨颤，其上出现了第八道金色光环。
<br/>　　“终于天级八阶了！”苏莫顿时大喜，十三天的时间，让武魂提升等级，这个速度可是不慢了。
<br/>　　要知道，他在苍穹世界之时，没有修炼资源，也曾大量吞噬妖兽，但都没能让武魂提升。
<br/>　　当然，苍穹世界的妖兽，等级实在太低。
<br/>　　他这十几日吞噬的凶兽，虽然八级凶兽不多，但七级凶兽可以数是百万计。
<br/>　　“还有十七天，应该能提升到天级九阶吧？”
<br/>　　苏莫低声自喃，他也没有多少把握，毕竟越往后越难以提升。
<br/>　　而且，他的吞噬奥义，只是接近一阶的地步，还没有真正达到一阶，还需要时间参悟。
<br/>　　时间，真的是非常紧迫啊！
<br/>　　“真……真的升级了？”阎魔鬼圣的声音响起，声音中充满了震撼。
<br/>　　不算沉睡的十万时间，他也活了几万年，这却是第一次见到能升级的武魂，这让他感觉不可思议。
<br/>　　这打破了修炼界武魂的常理！
