　　第637章
<br/>　　程大雷拥着大氅坐在帅案后，听着赵子龙汇报那一夜战斗的情形。
<br/>　　“这么说来，你顺了一批兵器回来？”程大雷问。
<br/>　　“他们打的一锅乱粥，宋伯康急着追杀百里胜，他们也顾不得我，我看不拿白不拿，索性就顺了一些。也没有多少，最多只能武装一千人。”赵子龙说着，又问道：“是不是不该拿？”
<br/>　　“呃，没有，你做得很好。”程大雷道：“不抢东西怎么是山贼呢，不过，百里胜究竟怎么突围的，凭他的兵力未必做得到这点？”
<br/>　　“百里胜手下有一员大将，本事十分了得，如果没有他，百里胜也无法成功突围。我与他交过手，不是他的对手。”
<br/>　　咦，程大雷是大吃一惊，能令赵子龙说出这番话，可是极为不容易。
<br/>　　“对方什么来历，百里胜手下怎么有如此大将？”
<br/>　　“他骑一匹九花马，用一杆金纂提炉枪，来历不知道，名字么……”赵子龙道：“对了，我听见有人唤他卢俊义。”
<br/>　　“呀，是他。”程大雷吃了一惊。
<br/>　　赵子龙眼前一亮：“大当家果然见多识广，竟还认得此人。”
<br/>　　“唉，也不过有个耳闻而已，却是没照过面。”
<br/>　　程大雷随口道了一句，却是没想到卢俊义竟投靠了百里胜，如此倒也令事情多了许多变数。赵子龙虽强，但境界还和卢俊义差了一级呢，输在他手中也是不委屈。
<br/>　　“好了，这一场你辛苦了，回城后好好休息。”
<br/>　　与卢俊义交手之后，赵子龙方才明白自己与真正高手的差距。如此，倒是隐隐有所领悟，正需要一些时间打磨功夫，以图精进。
<br/>　　程大雷仍然留守朔方城，窝在帅案后思索。卢俊义已经出现了，照道理来说，那杨国忠也该来了，为何迟迟还未现身。
<br/>　　对于他的到来，程大雷倒是不如何期待，他真正在意的还是哪位祸国妖孽。不知自己有没有机会见到……
<br/>　　“大当家。”阿喜从外面过来，道：“您让我查的事情，查到点东西了。”
<br/>　　“喔，快过来讲讲。”
<br/>　　程大雷将阿喜也从蛤蟆城调了过来，让他的喜字队撒出人手查，看朔方城方圆有没有方便动手的势力，派一支队伍过去，抢一批粮食以补充朔方城的消耗。
<br/>　　查访这么多日子，总算是查出几个目标。经过一轮筛选后，还留下的目标有三个。
<br/>　　阿喜将地图摊开，在地图上标注三个位置：“他们都在荆州，最远的是石门城，距离我们大概有三百里，驻兵三万。最近的是金沙镇，只有一百里。稍远些的是黑风寨，大概有三千人，黑风寨很富裕，抢下它，足够朔方城消耗些日子。”
<br/>　　程大雷在这三个位置选择，最后确定了黑风寨：“就黑风寨了，他在河边，我们顺流之下，来去也快些。”
<br/>　　“我立刻就派人去踩盘子。”阿喜道：“大当家，这次行动就让喜字队做吧，速战速决喜字队可以拿得下的。”
<br/>　　经过一段时间发展，喜字队的人数已经超过一千人，都被派到各地打探情报。如果算上各地被收买的探子，人数要更多。
<br/>　　程大雷白他一眼：“你是情报工作者，做好自己的本职工作就好了。”
<br/>　　阿喜略显遗憾：“那大当家准备由谁负责这次行动？”
<br/>　　程大雷想了想，最后确定由单雄信带队，毕竟是贼王，专业的事情自然要由专业的人来做。
<br/>　　“为何要让别人负责，我明明可以自己去做么。”程大雷道：“你回蛤蟆城传个信儿，让单雄信点一千兵马，前来协助我。”
<br/>　　“是。”
<br/>　　阿喜答应一声，退了下去，心里想着：原来大当家也不安分呐。
<br/>　　阿喜离开后，程大雷窝在城主府，研究黑风寨的信息。
<br/>　　黑风寨距离朔方城百里有余，顺着大河往下走，不足半日就可以到达黑风寨。那黑风寨的老大，有个外号叫做混江龙，据说水性惊人，在此地颇有恶名。
<br/>　　啧，程大雷叹了一声，自己来到凉州已经有些年头，都是江湖同道，这混江龙怎么不想着过来打个招呼。
<br/>　　你瞧瞧，连拜码头的道理都不懂，自己完全有必要告诉你什么是规矩呐。
<br/>　　到了夜里，单雄信已经到了朔方城，这是他第一次行动，完全是展现自己能力的时候，所以整个人表现得很兴奋。
<br/>　　“大当家，咱们什么时候行动？”单雄信已经有些迫不及待：“不如咱们立刻出发，天亮就能回来了。”
<br/>　　“着什么急，船明天才能到，咱们明天黄昏出发，夜里动手。”
<br/>　　到得次日，从蛤蟆岛运来的船已经到了，共十艘大船，士兵全副武装，向黑风寨进军。
<br/>　　……
<br/>　　黄昏的水面上，一艘大船被截在滩涂，十几个凶神恶煞的大汉蹭蹭窜上船，望着抱头鼠窜的船夫哈哈大笑。
<br/>　　“我们是混江龙龙老大的人，不想死的都给我蹲下。”
<br/>　　咔嚓一声，为首的大汉将一名船夫的脑袋砍成两半，血喷了他一脸，他咧开大嘴哇哈哈笑着。
<br/>　　一名面白无须的年轻人手持青锋剑，被堵在船舱中。
<br/>　　“哥哥，哥哥，我们会死么？”
<br/>　　在他身后立着一个小女孩，年纪不过十一二岁，生得粉雕玉砌可爱。
<br/>　　“不怕，有兄长在，你死不了。”
<br/>　　年轻人将小女孩抱在怀里，用剑堵着门不让水贼杀进来。可他并无多少武艺，再打下去必定吃亏。
<br/>　　据说这混江龙性情十分凶恶，小妹年纪还小，若落在他手中，定然会遭了毒手。
<br/>　　年轻人咬咬牙，将小女孩的眼泪擦去，口中道：“好小妹，莫怕，我不会让你出事的。”
<br/>　　他抱起小女孩，推开船舱窗户，一头跃入水中。噗通噗通声响，不断有尸体落入水中，也无人注意到他。
<br/>　　几个沉浮，他用力拍水，一个浪头打过来，他便晕了过去，被水流卷着不知冲向何地。
<br/>　　而哪怕昏迷后，他仍用一只手紧紧抱着小女孩，不肯松开一丝。
<br/>　　几名水贼抢了大船，立在船头发出阵阵狂笑。
<br/>(本章完)
