　　第465章
<br/>　　“徐军师，不知你可能推测出程大雷逃向了何处？”一日，陆承问徐神机。
<br/>　　徐神机自从落到相府手中后，待遇一直不错，整天一帮年轻子弟围着他聊天，徐神机偶尔吹吹牛，也唬得他们一愣一愣的。
<br/>　　今日在茶余饭后，陆承终于将这个问题抛了出来。毕竟，各路诸侯都派出了缉拿程大雷的队伍，但现在还没摸到程大雷的下落。
<br/>　　徐神机顿了顿，知道考验终于来了。他当然不想在崔相手中安度晚年，一看那个留山羊胡的男人就不是个好东西啊。这几天，徐神机一直想着该如何脱身，暂时还没有想到办法，以自己的体力，跑一百步就喘，怎么可能从这军营中逃走。
<br/>　　可今日陆承这个问题问出来，徐神机也不能不回答。但他也不想暴露程大雷的位置。
<br/>　　当然，他现在也不知道程大雷在何处，可他觉得，以自己的足智多谋，神机妙算，稍微掐算掐算也就十拿九稳了。
<br/>　　“徐军师，徐军师……”陆承连唤了两声。
<br/>　　“啊！”徐神机这才回过神来，摆摆手道，罢罢罢自己随便指个地方忽悠他们得了。
<br/>　　“着什么急，不就是问那程大雷的位置么。”程大雷随手一指：“他一定往东北方向去了。”
<br/>　　“东北方向？”陆承奇怪道：“徐军师怎么知道的？”
<br/>　　“哈，你问这个做什么，我说了你能听懂么，说了东北方向，就是东北方向。”徐神机已经懒得编瞎话。
<br/>　　陆承在徐神机处得到消息后，悄悄离开，立刻将这消息禀报给崔相。
<br/>　　“东北方向？”崔相皱起眉头：“这不太可能吧，往东北就离凉州越来越远了，他真是这么说的。”
<br/>　　“千真万确，而且语气十分肯定。”陆承道：“相爷，你说他会不会是唬我们？”
<br/>　　“带人去东北方向看看不就知道了，哼，若他敢戏耍我们，直接宰杀了喂狗。”崔相冷冷道。
<br/>　　立刻点齐一千兵马，由陆承和一员武将带队，去东北方寻找程大雷下落。路上，陆承一直在想，徐神机的话的确靠不靠谱，任谁来想，程大雷去往西北方都是不可理解的。但也的确因为这一点，更多兵力布置在并州边境。
<br/>　　难道当真被徐神机说中了……当然，这只是陆承自己的想法，军中所有人都不太重视这件事，否则，也不会只派出区区五百人。
<br/>　　而带队的是周开新，军中一无足轻重的武将。
<br/>　　“陆军师莫要着急，我们慢慢去，快快回，怎么可能嘛，他不往西走往北走。”周开新道。
<br/>　　陆承点点头，倒是没有说什么。只是忽然觉得心绪不宁，似乎有什么不好的事情就要发生。
<br/>　　他极目远眺，望向前方：“那是什么？”
<br/>　　就在不远处，一队人马正在跋涉。当陆承发现对方的时候，对方也发现了陆承。
<br/>　　这支人马，正是程大雷率领的三百义字军，程大雷也没想到在这里遭遇帝国军。
<br/>　　“大当家，我们怎么办？”白平问。
<br/>　　“还能怎么办，有怨报怨，有仇报仇。”程大雷也没打算可以顺利抵达青牛山：“与我杀。”
<br/>　　“是！”
<br/>　　白平一挥手中大刀：“随我杀！”
<br/>　　三百如狼似虎的义字军，携着满腹仇恨冲了过去，这是在程大雷面前的第一战，他们也想好好表现。
<br/>　　程大雷面向身边的单雄信，道：“单二爷，看你的了。”
<br/>　　“大当家尽管放心。”
<br/>　　“还有一件事，你看看他们军中是否有一个叫薛丁山的人？”
<br/>　　“他是大当家朋友？”单雄信知道程大雷的交际人脉很广。
<br/>　　“不是，碰见他，给我宰了他。”程大雷恶狠狠道。
<br/>　　“包在某家身上。”
<br/>　　单雄信快马奔出，手中的金顶枣阳槊举起，虽然后发，但是先至，过去就将一人的脑袋斩了。
<br/>　　义字军是满腔仇恨，想着报仇雪恨，但帝国军这边只想着摸鱼出一场公差，谁也没想到会真遭遇敌人。周开新自己先慌了，将为兵之胆，周开新尚且如此，何况手下的兵丁。
<br/>　　这几乎是一场单方面的杀戮，三百骑兵斩杀五百步兵，几乎没遇到任何抵抗。
<br/>　　程大雷远远看着这一战，并未出战，想要借着这一战看看义字军的能力。结果，自然谈不上满意，勇则勇矣，但配合能力组织能力实在太差，占有如此大的优势，竟还跑了几个漏网之鱼。
<br/>　　没办法，以后慢慢调教吧，当然，是在活着离开这里的前提下。
<br/>　　战斗结束，白平带着搜刮战利品，更换破损的武器，从尸体身上搜寻干粮，还有他们留下的战马。
<br/>　　而程大雷的心情实在谈不上开心，已经暴露行踪，接下来就会有更多人赶过来，困难也许才刚刚开始。
<br/>　　陆承和周开新带着几十名残兵狼狈逃回大营，层层报上去，一直报到崔相面前。
<br/>　　“发现程大雷了？”崔相睁大眼睛问。
<br/>　　陆承满脸鲜血：“找到了，他的确往东北方去。”
<br/>　　“啧。”
<br/>　　崔相叹口气，当真是被其猜中了。他挥挥手，道：“快去将徐军师请来。”
<br/>　　于此同时，徐神机正在提心吊胆中，他胡乱指了个方向，估计快露馅了。等他们一无所获回来，怎么可能饶过自己。如果逃的话，可能也没机会逃，罢罢罢，到时候就多骂程大雷几句，看能不能活下来。
<br/>　　“徐军师，崔相有请。”这是外面有人报来。
<br/>　　徐神机昂起头，感觉伸脖子一刀缩脖子也是一刀，他大步走出，挥挥手道：“头前带路。”
<br/>　　抱着视死如归的心情走入崔相的大帐，徐神机立而不跪，反正事情已经这样啦，有什么冲我来吧。
<br/>　　“徐军师，你果然来啦。”崔相立刻起身。
<br/>　　“当然来啦，来没来你没看见么。”徐神机已经打算破罐子破摔。
<br/>　　崔相有些糊涂，究竟出了什么事，至于对自己这么不客气么。
<br/>　　“徐军师莫要生气，我之前还怀疑过你，在这里老夫就向你赔罪啦。”
<br/>　　满头血污的陆承抬起头：“徐军师当真神机妙算，在下心服口服。”
<br/>(本章完)
