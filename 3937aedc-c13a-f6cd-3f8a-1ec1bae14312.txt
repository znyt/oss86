　　第一名，苏莫！
<br/>　　这五个大字极其的醒目，字体比其他名字更大，位于最上方，无比的耀眼。
<br/>　　整个拍卖会场，一片死寂，所有的人均是瞪大了眼眸，死死的盯着真圣榜上的‘苏莫’二字。
<br/>　　所有的人都是面色呆滞，一脸的懵逼，完全回不过神来。
<br/>　　这是什么情况？
<br/>　　苏莫是谁？
<br/>　　哪族的天才？
<br/>　　众人都是摸不着头脑，因为完全没听说过。
<br/>　　苏莫也是傻了，一脸的茫然之色，怎么会是自己？怎么可能是自己？
<br/>　　他如今才武帝境五重的境界，在古灵星河名声不显，也就在神武学府名气较大而已，怎么会排在第一名？
<br/>　　难道是同名之人？
<br/>　　这倒是很有可能，他的名字很普通，有同名之人也在情理之中。
<br/>　　苏莫不相信此‘苏莫’是他，这完全不符合常理，根本就说不通。
<br/>　　虽然他的天赋超级强大，甚至在别人眼中已然逆天，但还不至于在整个古灵星河的武帝境天才之中，排在第一位吧！
<br/>　　整个拍卖会场，在寂静了几个呼吸的时间之后，终于彻底的沸腾了起来，呼喝声四起。
<br/>　　“这苏莫是谁啊？”
<br/>　　“玄净尊者，你们搞错了吧？是不是写错名字了？”
<br/>　　“肯定是错了，什么苏莫？那族的天才？听都没听说过！”
<br/>　　“姜神夜不上榜，这完全没有道理！”
<br/>　　“哈哈！居然犯了如此低级的错误！”
<br/>　　“……”
<br/>　　整个拍卖场一片哗然，惊疑声、怒喝声、嘲笑声不绝于耳，基本上所有人都认为是搞错了！
<br/>　　因为这个苏莫，基本上没人听说过，完全就是一个无名之人，这明显的是个错误。
<br/>　　当然，也有例外之人。
<br/>　　谢彬几乎是目瞪口呆，随即缓缓的转头，看向身边早就改变了容貌的苏莫。
<br/>　　“这……这不太可能吧？”
<br/>　　谢彬心中狂震，他知道苏莫天赋妖孽，比之学府的那些圣子有过之而无不及，直追学府神子，但也不可能排在第一位吧！
<br/>　　不远处的尤天韩，双眸微眯了下来，他亦是缓缓转头，向苏莫看了过来，眸中露出惊疑之色。
<br/>　　虽然苏莫改变了容貌，但他在苏莫改变容貌之前，就发现了苏莫，现自然认识。
<br/>　　尤天韩心中惊疑不定，真是这个苏莫？这不可能吧！
<br/>　　他也是不太相信，苏莫何德何能，能排在第一名？将所有天才都踩在脚下。
<br/>　　而另一边的巫人王，双眸之中爆射出凶厉的光芒，眼眸死死的盯着真圣榜上‘苏莫’二字，双拳捏的咯咯爆响。
<br/>　　是他！
<br/>　　上次他就是败在了对方的手中！
<br/>　　此人混进巫族，和他争夺少司之位，还杀了大量的族人，罪该万死！
<br/>　　好在，对方逃走之后，大祭司已经重新任命他为少司，并且全权处理苏莫之事。
<br/>　　这一次，他就是来追查苏莫的行踪，因为族中已经查到，苏莫就在古周星。
<br/>　　这一次，他带来了数百族中高手，已经严密的监视住了古周星所有的星空传送阵。
<br/>　　苏莫，插翅难逃！
<br/>　　玄净尊者身形伫立在拍卖台上，面带淡笑的环视着整个拍卖会场，这个结果他早有预料。
<br/>　　事实上，他在刚得知这个结果的时候，也是疑惑不已，根本就不相信。
<br/>　　但是，最终他还是被事实改变了想法。
<br/>　　此真圣榜，乃是两位武圣强者亲自查探，并且仔细斟酌之后，才得出的结果。
<br/>　　或许个别的排名有些偏差，但第一名之人，和榜单上总体的拍卖，绝对不会错。
<br/>　　“玄净尊者，你老快说话啊！是不是弄错了？”
<br/>　　“肯定弄错了，姜神夜不是第一名，我第一个不认可！”
<br/>　　“就是，姜神夜不是第一名也就罢了，居然都没有上榜，开始什么玩笑？”
<br/>　　“……”
<br/>　　整个拍卖会场，依旧喧嚣震天，很多人在催促玄净尊者，想要得到对方的肯定回答。
<br/>　　“众位请安静！”玄净尊者双手抬了起来，不断的下压，示意所有人安静。
<br/>　　片刻之后，拍卖会场终于缓缓的安静了下来。
<br/>　　玄净尊者见此，深深的吸了口气，随即朗声开口道：“众位，这个排名并没有错，第一名的确叫苏莫……！”
<br/>　　然而，玄净尊者话未说完，便被狂涛骇浪般的呼喝声打断了。
<br/>　　“什么？没有错？”
<br/>　　“这怎么可能？”
<br/>　　“这苏莫是什么玩意？何许人也？”
<br/>　　“我去，老子从来都没听说过，也能排在第一名？”
<br/>　　“简直是开逆天玩笑，你是在耍我们吧？”
<br/>　　“第一名不改的话，我第一个不支付一百块下品灵晶！”
<br/>　　“……”
<br/>　　玄净尊者的一句话，直接是将不少人激怒了，众人群情激愤，呼喝声震天。
<br/>　　什么真圣榜？排的什么玩意？
<br/>　　所有人都没听闻过的人，居然能排在第一名，这是当他们是白痴吗？
<br/>　　有的人不禁暗想，这个什么苏莫，不会就是拍卖会背后几个商会的人吧，这是在故意造势呢！
<br/>　　只是，这样做未免太儿戏了，没有足够的天赋和实力，就算排在了第一名，也只是个笑话而已！
<br/>　　“众位能不能听老夫把话说完？”
<br/>　　玄净尊者脸色微沉，朗声高喝，其话音中蕴含了玄力，如同声波一般，在所有人的脑海中炸开。
<br/>　　霎时之间，众人脑海中如同雷霆轰鸣，被震得双耳嗡嗡作响，这次安静了下来。
<br/>　　玄净尊者见此，心中组织了一番语言，朗声道：“老夫知晓众位对于苏莫之名，均是闻所未闻，所以不认同这个排名；事实上在半月之前，老夫和你们一样，也是不闻其名；现在，老夫就为你们介绍一番，这苏莫是何许人也！”
<br/>　　闻听玄净尊者的话，所有人都打起了精神，屏气凝神，静待对方介绍。
<br/>　　苏莫深深的吸了口气，接下来，他就能知道，这第一名的‘苏莫’，到底是不是他。
<br/>　　不过，听了玄净尊者之前的话语，他已经有了预感，很可能真的是他。
<br/>　　“苏莫，乃是神武学府的弟子……！”
<br/>　　玄净尊者朗声开始介绍，但只是这第一句话，就让坐席上的苏莫脸色一苦。
<br/>　　尼玛，真的是自己！
<br/>　　完了！这次是真的完了！
<br/>　　他成了真圣榜第一，绝对名动整个古灵星河，将成为众矢之的！
<br/>　　这一次，怕是逃都无路可逃了！
