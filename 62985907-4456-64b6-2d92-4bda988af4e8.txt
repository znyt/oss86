        &lt;h3&gt;第三卷第二百七十章  再次相聚&lt;/h3&gt;
<br/>
<br/>        陈子昂听完江俞轩的话，背对着江俞轩站了一会，然后大步的离开。
<br/>
<br/>        她上车，启动、一气呵成，汽车绝尘而去，江俞轩追到楼下的时候，只看到红色跑车的影子。
<br/>
<br/>        江俞轩懊恼的拍了自己的右脸：“江俞轩，江俞轩，你怎么这么窝囊啊。你说你爱她能死吗？”
<br/>
<br/>        “那个傲娇的臭女人已经爱无能了，你难道要和她一样一辈子都躲躲闪闪的？这下子死定了，怎么办啊？连等都不等我了。你看，她今天都拍你脸了啊，啊啊啊，我该怎么办啊？”想归想，晚上的聚会还得参加。
<br/>
<br/>        江俞轩满怀心事的开着车朝他们聚会的酒店驶去，晚上还得面临张倩楠，他一定是脑子秀逗了，张倩楠现在很冷静，冷静得让他都感觉不认识她了，尤其张倩楠当着陈天天骂他的话，让他感觉张倩楠似乎是变了一个人。
<br/>
<br/>        当江俞轩赶到酒店的时候，大家都在，张函正在和寒伯安聊天，俩个人感觉相谈甚欢，而陈子昂和安男、张倩楠三个人也是笑语嫣然。
<br/>
<br/>        郝景文看见江俞轩来了，冲江俞轩招招手：“三哥，这里。”
<br/>
<br/>        江俞轩走过坐在郝景文的身边：“单身汪。”
<br/>
<br/>        郝景文看了一下江俞轩：“嘶，我单身汪？你以为你能撒狗粮？你看看你家老婆连一个眼神都不给你。说，你们是不是出什么问题了？”
<br/>
<br/>        江俞轩：“有这么明显？”
<br/>
<br/>        “真出问题了？”郝景文惊奇的问。
<br/>
<br/>        “应该是，我和张倩楠总感觉中间隔着一层东西，我们的相处方式很奇怪，文文，你和陈思宇在一起的时候有没有想抱抱她？”
<br/>
<br/>        郝景文思考了一下：“我和陈思宇在一起，心里都是甜的，我很想一直牵着她的手，牵着她的手我就不舍得松开，至于别的，在没有结婚之前我是尊重她的。”
<br/>
<br/>        “可是，我和张倩楠在一起连拉手的机会都会很少，偶尔的两次拥抱我都记得清清楚楚，我们似乎内心都波澜不惊。”
<br/>
<br/>        “我也没有经验，要不你问问大哥吧，大哥和安男姐姐已经结婚了，他们有经验。”
<br/>
<br/>        “算了，不问了，反正现在就这样子，我觉得也很不错，走一步看一步了。今天就缺你的思宇了哦，要不然全了，对了，文文，你觉得张函和陈子昂还有机会吗？”
<br/>
<br/>        “张函这么多年不是一直在努力，可是结果呢？他们还不是路归路，桥归桥，就是平行的两条线，既没有再次擦出火花，也没有反目成仇，嘿，佩服陈子昂能如此的谈笑风生，淡定自若，再怎么说也是前男友还是孩子的爸爸。”
<br/>
<br/>        “所以，大家都看不透她，连我都看不透她。”
<br/>
<br/>        “可能是生活所迫吧，她的坚强都是伪装起来的，越是坚强的女人内心越脆弱，只不过是一次次的磨炼让她变成了现在样子，哪个女人不喜欢有人宠着她爱着她？”郝景文突然叹了口气，他一定要对陈思宇好一点，不让陈思宇受一点点伤害。
<br/>
<br/>        “我不和你说了，我要过去和她们说说话。”江俞轩听着郝景文的话站起来就朝陈子昂他们走过去。
<br/>
<br/>        “俞轩，你来了？坐吧。”安男和江俞轩打着招呼。
<br/>
<br/>        “嫂子、子昂、倩楠、”江俞轩一个个的喊着她们的名字。
<br/>
<br/>        “我们基本差不多时间出发了，你就迟到了半小时。”陈子昂笑着说。
<br/>
<br/>        “老实交代路上干什么去了？”张倩楠也笑着问江俞轩。
<br/>
<br/>        “你们这是要三司会审的节奏啊。”安男接话。
<br/>
<br/>        “西城堵车厉害，你又不是不知道，我下楼的时候你连人影都不见了，心真够狠了，我就是开了一句玩笑，你就生气了？”江俞轩盯着陈子昂看，对于安男和张倩楠的话他并没有回答。
<br/>
<br/>        “哇，江俞轩，你开了一句什么玩笑，竟然惹我们海纳百川胸怀宽广的子昂生气了？”张倩楠看着江俞轩夸张的笑着。
<br/>
<br/>        “别听他的，我怕我迟到了，就开车先走了，下午开会，讨论了一些公司的发展规划，我和他有分歧，说了几句，他就认为我生气了。”陈子昂对张倩楠解释着。
<br/>
<br/>        “你们俩个人会有分歧，真的是让人不可思议。是什么样的分歧啊。”张倩楠好奇的打破砂锅问到底。
<br/>
<br/>        “我说......”
<br/>
<br/>        “他说他要去甸城！”陈子昂接过江俞轩的话。
<br/>
<br/>        “你也要去甸城？你去干什么？”张倩楠惊奇的问。
<br/>
<br/>        “做生意啊，哪里有生意就往哪里跑啊。”江俞轩淡淡的说。
<br/>
<br/>        “子昂，你看，大家都追随你的步伐，你到哪里，大家都到哪里。”张倩楠叹了口气。
<br/>
<br/>        “你们可以不用追随我的步伐啊，话又说回来，没有你们我的生活也许是平平静静，安安宁宁的呢。”陈子昂有点生气，你们强行插入我的生活，到现在还来说我的不是？
<br/>
<br/>        “子昂，舅舅后天做手术吗？”安男见陈子昂的语气不善，便转变了话题。
<br/>
<br/>        陈子昂深深的吸了口气，她希望他们都从她的眼前消失，那样她就没有纠结，没有烦恼，没有苦闷，她就会带着她的俩个孩子好好生活，没有希望也没有失望。
<br/>
<br/>        “是啊，我这心里很难受了，最近的事情压得让人透不过气来。”陈子昂眉眼低垂，一丝淡淡的感伤附上眼眸，周身笼罩在一股悲凉和哀伤的气息里。
<br/>
<br/>        这样的陈子昂是大家不曾见到了，安男伸出手，轻轻的拉着陈子昂的手握在手中。
<br/>
<br/>        张倩楠静静的看着陈子昂，这么多年，她一直追随着张函的脚步，了解陈子昂的点点滴滴，在她的心中对陈子昂有爱有恨有嫉妒有不甘还有一种说不出来的怜悯，可是如此悲伤的陈子昂是她不曾见到过的，她的心，似乎是被针扎了一下。
<br/>
<br/>        江俞轩看着陈子昂，心揪起来了，当初豪言壮语的要陪着她一起走上人生的顶端，陪着陪着差一点就散了，他不想再丢弃他们之间的美好，他想要一直延续下去。
<br/>
<br/>        他需要时间，给他一点时间，让他重新开始。
<br/>
<br/>        “子昂，一切都会好起来的！”安男柔声的安慰陈子昂。
<br/>
<br/>        “是啊，子昂，你还有我们呢，我们会一直陪在你身边。”张倩楠也开口了。
<br/>
<br/>        “谢谢大家。”陈子昂抬起来头，眼里亮晶晶的浮着一层水雾。
<br/>
<br/>        不管如何，人生之旅还得继续，家里还有老人和孩子！她陈子昂是俩个孩子的天地。
