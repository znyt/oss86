　　高大青年吕罡，大步而行，穿过人群，走到了一名宗务殿长老的身前。
<br/>　　这名长老是一位矮瘦老者，面上黝黑，还留着一缕山羊胡。
<br/>　　“吕罡，的确有适合你的任务！”郑长老微笑着说道。
<br/>　　“哦？什么任务？”吕罡眼眸一亮，大声问道。
<br/>　　“在灵宁星，出现了一支魔道小势力，这个势力嗜杀成性，无恶不作，需要有人前去铲除这支势力！”郑长老说道，言罢，他便递给吕罡一枚任务玉简。
<br/>　　吕罡接过玉简，仔细查看了一番，顿时大笑了起来：“哈哈，这种弹丸小势力，我挥手间便能铲除！”
<br/>　　“嗯，以你的实力完成这个任务不成问题！”
<br/>　　郑长老微微点头，继续道：“你速速完成这个任务，还能来得及去太阴族参加会武！”
<br/>　　“不急不急，会武还有一个多月，时间充裕的很！”吕罡大笑着说道。
<br/>　　“去太阴族参加会武？”一旁的苏莫，听到两人的谈话，顿时眼眸一亮，他现在对太阴族可是非常的敏感。
<br/>　　因为他的目的就是去太阴族，去见青璇。
<br/>　　“吕罡师兄，希望这次会武，你能大放异彩！”一名认识吕罡的弟子，笑着说道。
<br/>　　“是啊，挫败其它六大势力的天才，然后得神女青睐！”
<br/>　　“最好是能让太阴族看重，然后留在太阴族！”
<br/>　　其他一些弟子纷纷开口，面上均是带着尊敬之色，吕罡的实力在道宗的十万核心弟子之中，可是顶尖的存在，的确是有这个可能。
<br/>　　“神女？”
<br/>　　吕罡闻言轻笑一声，不屑道：“我若是能得到太阴族的看重，那是最好不过了，至于神女青睐，还是拉倒吧？”
<br/>　　“怎么？难道吕师兄不喜欢神女？”
<br/>　　“是啊！神女美貌无双，纯阴之体，难道吕师兄还看不上？”
<br/>　　“天才爱美女，吕师兄难道不近女.色？”
<br/>　　其他弟子闻言，纷纷诧异的问道，要知道现在神女云悠悠可是无数年轻武者的梦中情人，身份高贵、美貌无数，天赋绝佳，这等绝世女子，谁不心动？
<br/>　　“呵呵！神女虽好，但我吕罡不屑捡他人破鞋！”
<br/>　　吕罡轻笑一声，摇了摇头道：“神女是纯阴之体不假，但正因为是纯阴之体，谁能保证还是完璧呢？”
<br/>　　众人闻言惊讶，吕罡这是什么意思，难道神女云悠悠已经被人破了处.子之身，不在是完璧？
<br/>　　听吕罡这么一说，众人均是觉得很有可能，听说神女是其它星河之人，身为纯阴之体的体质，还真有可能早已被人破了身子。
<br/>　　“吕罡，注意你的言语！”郑老脸色一沉，口中顿时厉喝一声，这个吕罡真是口无遮拦，这种话居然也敢说。
<br/>　　这种话要是传到太阴族，那还得了，怕是吕罡会有杀身之祸。
<br/>　　幸好这是在宗务殿内，要是在外界，肯定会惹下麻烦。
<br/>　　“呵呵，郑长老息怒，我也只是随口一说而已！”吕罡轻笑一声，却是毫不在意。
<br/>　　不远处，苏莫的脸色阴沉了下来，眸中杀机一闪，这个吕罡居然公然破坏青璇的名誉，真是找死！
<br/>　　唰！
<br/>　　苏莫一步踏出，直接来到了吕罡的面前，目光冷视着对方。
<br/>　　吕罡见此，神情微怔，打量了苏莫一眼，疑惑的问道：“你有何事？”
<br/>　　“我来告诉你一句话！”苏莫冷冷的说道。
<br/>　　“什么话？”吕罡问道。
<br/>　　“管好你的嘴，免得祸从口出！”苏莫眸光直视对方的双眼，声音非常的冷漠。
<br/>　　吕罡闻言，顿时眼眸微眯了下来，冷笑道：“呵呵，你一个小小的武帝境八重的武者，居然还敢命令我？”
<br/>　　“不是命令，只是给你一个忠告！”苏莫淡淡的说道。
<br/>　　众人见此，均是惊讶不已，怎么突然就跳出一个人，来向吕罡发难。
<br/>　　众人打量着苏莫，均是感觉面生的很，暗暗猜测他肯定是神女的爱慕者。
<br/>　　“忠告？”
<br/>　　吕罡闻言不屑，面带戏虐之色，笑道：“小子，你这么维护神女，是不是也有觊觎之心？“
<br/>　　“是有怎样？不是又怎样？”苏莫淡淡的说道。
<br/>　　“我也给你一个忠告！”
<br/>　　吕罡冷笑一声，面带嘲弄之色的说道：“就你这样癞蛤蟆想吃天鹅肉的垃圾，神女就是要找一百个双修道侣，也不会选你，你还是死了这条心吧！”
<br/>　　吕罡目光直视苏莫，面带嘲弄，心有不屑，在他看来，苏莫只是一个贪恋神女身份容貌，和双修体质的小丑而已。
<br/>　　“你在找死吗？”苏莫闻言，眸中杀机大起，眸中顿时爆射出犀利的寒光。
<br/>　　这要是在外面，他早就出手了，绝对一剑劈死这个家伙。
<br/>　　“就凭你，还想与我动手？”吕罡的脸色亦是寒冷了下来，体质玄力涌动。
<br/>　　两人互相冷视，身上气势均是缓缓升腾而起，仿佛大战一触即发。
<br/>　　众人愕然不已，这个突然出来的家伙，居然如此的大胆，武帝境八重的修为，居然就敢与吕罡叫板！
<br/>　　“好了！”
<br/>　　就在此时，郑长老一声厉喝，身上冲起一股庞大的威压，如同万丈大山，狠狠的压在了苏莫和吕罡的身上。
<br/>　　呃~~
<br/>　　苏莫和吕罡两人，感受到这股庞大的威压，顿时闷哼一声，各自后退数步，身上的气势立刻溃散一空。
<br/>　　“你叫什么名字？”郑长老望向苏莫，冷冷的问道。
<br/>　　“苏莫！”苏莫道。
<br/>　　“宗门之内严禁私斗，违者重罚，你不知道吗？”郑长老呵斥道。
<br/>　　苏莫闻言眉头一皱，这郑长老明显偏袒吕罡，居然只怪罪他一人，而不追究吕罡胡言乱语之罪。
<br/>　　道宗既然是太阴族下辖的实力，吕罡言语侮辱神女，难道不是罪过？
<br/>　　“长老，我来领取新晋核心弟子的奖励！”苏莫心中暗叹一声，不想和这位长老多耍嘴皮子。
<br/>　　“你是新晋的核心弟子？”郑长老闻言诧异，吕罡和其他弟子也是略显诧异。
<br/>　　因为新晋核心弟子，应该是武帝境一重的修为，苏莫如此修为的话，若是新晋的核心弟子，那就是刚加入道宗的人。
<br/>　　不少人暗暗惊讶，这苏莫刚加入道宗，就敢和吕罡交恶，真是胆大。
<br/>　　也有少量的武者，认出了苏莫就是前段时间，一直盘坐在山门处的人，顿时恍然，的确是刚加入宗门的人。
