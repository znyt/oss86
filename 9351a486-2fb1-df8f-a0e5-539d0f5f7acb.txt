　　“嗯！你真的要与巫森一战？”东海龙王点头问道，他两日前便已经知晓了此事。
<br/>　　“不错！”苏莫点头。
<br/>　　“你有几分把握？”东海龙王沉声问道，他没有劝告苏莫放弃这个打算，而是先询问苏莫的把握。
<br/>　　“没有把握！”苏莫摇了摇头。
<br/>　　“没有把握？”东海龙王闻言皱眉，没有把握你还敢与巫森一战，这不是自寻死路吗？

<br/>　　所以，他才会询问东海龙王，因为唯有伪龙珠，才能最快的提升他的肉身实力。

<br/>　　一枚在龙腾的身上，一枚正在龙宫中。
<br/>　　这两枚伪龙珠，都是留给龙腾之用，对龙腾的将来，有无法替代的作用。
<br/>　　现在苏莫前来索要，顿时让东海龙王有些迟疑了起来。
<br/>　　伪龙珠太贵重了，他们已经送给了苏莫一枚，只剩下最后两枚了。
<br/>　　“宫中还有一枚，既然你要，本王便给你！”沉默了良久之后，东海龙王郑重的说道。
<br/>　　他虽然不舍得龙珠，但为了能让苏莫有把握对抗巫森，他不得不这么做。
<br/>　　因为一旦苏莫败亡，他们东海蛟龙一族，也会遭受灭顶之灾。
<br/>　　孰轻孰重，东海龙王还是能看清的。
<br/>　　而且，苏莫对龙腾也是有大恩，让龙腾成为金煌宫传人，得到了金煌宫的大帝传承。
<br/>　　这恩情，比龙珠还要贵重的多。
<br/>　　而后，东海龙王立刻便命人取来了伪龙珠，并且他亲自将龙珠交到了苏莫的手上。
<br/>　　“这枚龙珠，是我蛟龙一族最强大的一枚。”东海龙王沉声道。
<br/>　　苏莫手掌托着伪龙珠，打量了一番这枚龙珠，不由的心中暗喜。
<br/>　　这枚龙珠足有两个成人脑袋这么大，不仅比他上次那枚龙珠更大，其内蕴含的精气也更为磅礴浩瀚。
<br/>　　有了这枚龙珠，他的肉身实力，起码还能提升几个层次。
<br/>　　如此的话，他与巫森一战，便能有一些把握了。
<br/>　　“龙王前辈，大恩不言谢，我，不会让你失望的！”收起伪龙珠，苏莫向对方抱拳。
<br/>　　“嗯！”
<br/>　　东海龙王点头，叮嘱道：“切记，一旦不敌巫森，保命要紧，以你的天赋，要不了多久便能翻盘！”
<br/>　　“我明白！”
<br/>　　苏莫沉声道，随即他又和东海龙王聊了片刻，便告辞离开了东海龙宫。
<br/>　　距离约战之日，已经不足五日，他必须要在五日之内，赶到帝玄山脉，时间非常的紧迫。

<br/>　　虚空之中，苏莫盘膝坐在本命灵剑之上，一边催动灵剑赶路，一边准备提升实力。
<br/>　　他时间有限，只能利用赶路的时间来提升肉身的实力了。
<br/>　　取出伪龙珠，苏莫催动吞噬武魂，开始吸收伪龙珠内的精气。
<br/>　　伪龙珠中精气浩瀚如海，蕴含龙之血脉力量，疯狂的流进苏莫的体内。

<br/>　　庞大的龙象虚影仰天咆哮，震天动地，一缕缕金色的气力，在他全身血肉中游走。
<br/>　　苏莫的肉身实力，再次进入了一个持续暴涨的过程之中。
<br/>　　苏莫一路向帝玄山脉急行，龙象虚影的异象和咆哮声，震动了无数人。

<br/>　　武皇境九重！
<br/>　　仅仅大半日的时间，那枚伪龙珠中的精气便被苏莫吸纳的一干二净。
