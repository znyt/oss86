　　既然是青铜树组织的行动，那作为首领的‘枭’自然是不会放任不管的。
<br/>　　虽然在原著中监管这次行动，并且救走金木的，是芳村功善，但因为店长已经提前死亡，故此芳村艾特有很大的几率出现在这里。
<br/>　　而事实，和他的计划也在逐渐重合，当他即将掳走金木的时候，芳村艾特出现了。
<br/>　　吞噬了店长的尸体之后，刘哔现在的实力，已经超越一般的SSS级喰种了，即使是面对芳村艾特，也没有太大的压力。
<br/>　　“将他给我。”和店长的声音不同，独眼‘枭’声音非常尖锐，带着一丝童音，却让人升不起任何好感。
<br/>　　紧了紧手中的拳套，刘哔背后的翅膀黑光大盛，一股强大的力量感自身体上散发出来。
<br/>　　与此同时，他的身影也在不停拔高，本来一米八几的个子，短短数秒的时间内竟然扩长到了两米多高，本来略显单薄的身躯，也变的壮硕起来。
<br/>　　更令人惊恐的是，他后肩上突然凸起的十余个肉包，有些像原本属于店长的那些赫包，却又有些许不同。
<br/>　　这场变化只花费了几个呼吸的时间，芳村艾特自然不会任凭他进入状态，在她冲到刘哔面前的时候，却被撞在了一层灰白色的半透明罩子上。
<br/>　　似乎防御罩还带有腐蚀的属性，仅是瞬间，艾特的双爪就被腐蚀掉了一层肉皮。
<br/>　　这是从上一个位面吞噬来的鬼气，虽然质量不高，但全部挥发出来，也能暂时阻挡艾特的攻击了。
<br/>　　在激活死亡之翼力量的同时，刘哔也进入了赫者化状态，随着身体内所有赫子舒展开，这一刻他的战力达到了巅峰。
<br/>　　此时刘哔的身体，已经彻底脱离人类的范畴了，他身上涌出的数十根血色赫子，盘旋交错在他的后背，组成了一个十分怪异的图案。
<br/>　　像是一尊血红色的龙头，在黑色翅膀的衬托下，正大张着嘴巴，无声的咆哮着，让人感觉无比狰狞。
<br/>　　感受到体内极为强大的力量感，刘哔握拳，挥向了正向他冲来的芳村艾特，力量之强，在空气中留下一连串的音爆声。
<br/>　　“砰！”
<br/>　　一声闷响，艾特根本没有反抗的余地，被一拳击中在面门，极速的向后倒飞出去。
<br/>　　灰白色的面具在半空中落下，露出了一张不修边幅的俏脸，如果有热爱文学的人在这，肯定能认出这张脸庞。
<br/>　　一个被无数人爱戴敬仰的著名青年女性作家，高槻泉。
<br/>　　说起来，这个状态下，更多人还是喜欢叫她独眼‘枭’，或者是芳村艾特。
<br/>　　早就知道枭的真实身份，刘哔根本没有意外，一击得手后，他继续飞身追击，在半空中不断对芳村艾特进攻击。
<br/>　　或者说是殴打。
<br/>　　这根本不是一个级别的战斗，速度，力量……在任何数据上，芳村艾特都被现在的他所碾压。
<br/>　　不仅仅是因为吞噬掉了店长，事实上只是吞噬的话，是不能获得目标全部能力的，但除此之外，他还有其他的各种增幅。
<br/>　　死亡之翼，装备，血脉，以及他原本的能量，种种因素结合起来，方才成就了现在的样子。
<br/>　　……
<br/>　　半空中，艾特口中不断喷出鲜血，面对这样强大的敌人，她根本没有还手的余地。
<br/>　　即使是身体有很强大的再生能力，在这样的攻击下，她也无法坚持太久。
<br/>　　这个家伙，到底是从哪冒出来的啊，居然能强大到这种地步！
<br/>　　心中有些怨恨，艾特将所有赫子都挡在身前，苦苦支撑着，在这一刻，她感受到了死亡的威胁。
<br/>　　但面对速度远超过自己的敌人，她想要逃跑，几乎是一件不可能发生的事情。
<br/>　　打打不过，跑跑不了，场中的情况已经是十分危急。
<br/>　　“别杀我，我臣服。”在巨大的压力之下，艾特也快要坚持不住了，无奈，她也只能求饶，以期能获得一丝生机。
<br/>　　听到艾特的话语，刘哔动作微微一顿，看到了希望，艾特脸上的表情也欣喜了起来。
<br/>　　“砰！”
<br/>　　还不等她来得及在说什么，刘哔一拳轰向了她的头颅，千钧一发之际，艾特稍微往旁边侧了侧头，躲开了要害，却还是被轰掉了小半边脸庞。
<br/>　　脸上的肉芽一阵蠕动，脸部遭到破坏，艾特现在却说不出任何话语了。
<br/>　　没有任何留情，刘哔继续一拳轰向艾特的头颅，攻击头部，这是他长久以来养成的习惯。
<br/>　　他自然听到了艾特想要臣服的话语，但他可不想收下一颗说不定哪天就会爆炸的炸弹。
<br/>　　像她这种喰种，早就已经是自身的巅峰实力了，臣服或者不臣服，对刘哔来说区别已经不大了。
<br/>　　反倒是将艾特吞噬掉，对他来说更实用一些。
<br/>　　艾特脸上的肉芽疯狂蠕动着，剩下的半张脸，露出惊慌的神色，面对刘哔势大力沉的一击，慌忙进行闪躲。
<br/>　　可刘哔对她早有算计，在她左腿微动的时候，金属地面上突然出现了十数根骷髅爪，将艾特的脚腕死死抓住。
<br/>　　突如其来的变故，让她来不及做出其他反应，瞬间，刘哔的攻击落在她额头上。
<br/>　　“扑哧。”
<br/>　　仿若无物般的穿过额头，刘哔带着拳套的手掌，夹杂着一股红白相间的液体从艾特后脑穿了出来，随后，艾特缓缓停止了挣扎。
<br/>　　“叮~恭喜宿主击杀SSS级喰种芳村艾特，获得积分一万点。”
<br/>　　系统提示的声音从脑海中响起，刘哔确定了敌人已死，松了口气，将手掌抽了出来，随手甩掉了上面的液体。
<br/>　　这场战斗，对他来说也没有看上去那么容易，高强度的进攻，对他体内能量的损耗，也是极为严重。
<br/>　　但最终的结果总还是让人感觉满意的，面对着艾特的尸体，刘哔并没有着急吞噬，而是将她收入了空间戒指中，往来路的方向赶去。
<br/>　　他已经感应到，身体内的力量已经开始消退，死亡之翼和赫者化都是有时间限制的，他要趁着这段时间，抓紧从这里出去。
<br/>　　谁知道他的这番作为，会不会引来一些强大的对手，这里四处都是监控，刚才所有的战斗过程，估计也都被录制下来了。
