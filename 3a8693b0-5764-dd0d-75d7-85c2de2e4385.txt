<br/>　　林峰继续打开第四个空投箱。

<br/>　　“我去，似乎比之前见到的又要多一些了。”
<br/>　　陈勾看了眼明月湖附近的怪物更多了。
<br/>　　“的确多了许多，看来这个地方还是个好地方啊，这么多怪物聚集在这里。”

<br/>　　林峰说着就掏出了四个汽油弹。
<br/>　　这是他之前开D级空投箱开到的，之前和尸傀门老头大战时用了一个，还剩下四个。
<br/>　　给力陈勾和李广一人两个。
<br/>　　“我去，汽油弹？”
<br/>　　“这玩意牛逼啊，林峰你那里搞来的。”
<br/>　　林峰随口编了个理由：“自己无聊做的，你以为我这几天在屋内干嘛，就是研究这些玩意。”

<br/>　　“这只是用来当诱饵的，看到那边那群变异野猪没有，这群家伙身上毛很多，很容易燃烧，你们到时候就一个汽油弹丢他们那边，一个丢那边的丧尸群。然后剩下的我来想办法。”

<br/>　　“得嘞。”

<br/>　　直接遥控着军用无人机飞了过去。

<br/>　　咻咻咻。

<br/>　　足足四发导弹落在了变异野猪群之中。
