　　整个龙族都被惊动了，无论是龙帝，还是一众长老，亦或是大量的龙族族人。
<br/>　　四面八方，不断的有大量的龙族之人到来，即便祖龙塔所在的广场不小，也很快就人满为患。
<br/>　　龙洌、龙双双、龙烨等龙族天才，尽数到来。
<br/>　　有人闯到祖龙塔第九层，这是极为惊人的事情，这种天赋和实力，冠绝龙族，唯舞独尊。
<br/>　　不过，在场的所有龙族之人，心中震惊的同时，亦是感觉有些悲哀。
<br/>　　这么多年了，好不容易出现一个，能进入祖龙塔第九层之人，但还不是龙族本族人，而是人族。
<br/>　　无论是炎空还是屠霄，均是人族之人。
<br/>　　虽说炎空和屠霄都是龙族的外族族人，效命与龙族，但是与真正的龙族族人相比，还是有很大差别的。
<br/>　　非我族类，齐心必异。
<br/>　　外族族人终究是外族，不是真正的龙族。
<br/>　　“奇怪，龙腾去了哪里？”龙城环视整个广场，没有见到龙腾的身影，微微有些诧异。
<br/>　　这屠霄和炎空两人之中，有一人闯到了祖龙塔第九层，如此惊天大事，龙腾居然没有出现。
<br/>　　不过，龙城也没有多想，毕竟，龙腾若是在闭关，不知道此地的情况，也在情理之中。
<br/>　　……
<br/>　　祖龙塔，第九层。
<br/>　　苏莫手握终极神剑，目光灼灼的望着眼前怪异的吞虚神龙，他并没有多少压力。
<br/>　　因为，他还有很多的底牌，比如，自己的吞噬剑道，再比如，终极剑道。
<br/>　　所以，战胜眼前的吞虚神龙，他有绝对的自信。
<br/>　　但是，对于第十层，他就没有多大的把握了。
<br/>　　昂！
<br/>　　龙吟声震天，千丈的吞虚神龙，快速的临近，一身白色的毛发，如同一根根银色的纸条。
<br/>　　此龙极为威武，如同天生的帝王，高高在上，藐视苍生。
<br/>　　眼见吞虚神龙，临近了苏莫，突然张开了血盆大口。
<br/>　　其大口之中，仿佛是宇宙黑洞，又仿佛是九幽地狱，可怕无比。
<br/>　　霎时之间，一股恐怖的力量，作用在虚空之中，这股力量让得苏莫和他周身方圆万里，都被彻底的凝固住了。
<br/>　　而后，这被凝固的方圆万里，仿佛是被压缩成了一个小型的世界，直接向吞虚神龙的口中而去。
<br/>　　仿佛是，这吞虚神龙，将要一口将苏莫，以及方圆万里的虚空全部吞食。
<br/>　　传闻，吞虚神龙吞天食地，一口吞食一颗星辰，都如同吃饭喝水一般简单。
<br/>　　而如龙帝那般强大的存在，一口吞食宇宙中的一个星河都不算夸张，无比的恐怖。
<br/>　　“好强大！”
<br/>　　苏莫感觉到周身的恐怖力量，眸中精光四溢，周围方圆万里的虚空，在飞速的压缩和凝实。
<br/>　　这股力量之强大，如同是亿万座神山降临，狠狠的压在了苏莫的身上。
<br/>　　不过，这反而激起了苏莫心中的战意，他意念一动，全身的混沌源力、两种万化神剑元神、不败战气元神，血脉之力，全部催动到了极致。
<br/>　　磅礴浩大的气势，冲他的身上冲天而起，锋锐的剑光爆射苍穹，直接就将周身凝固的空间震碎了。
<br/>　　“混沌剑轮！”
<br/>　　苏莫大喝一声，一剑暴斩而出，直接施展了吞噬剑道的第二招，混沌剑轮。
<br/>　　这一刻，他几乎是施展出了得到终极剑道之前，最巅峰的战斗力。
<br/>　　虽然他没有催动吞噬元神，增加混沌剑轮的威力，但是，因为他手中用的终极神剑，所以，他这一剑的威力，丝毫不差。
<br/>　　轰隆隆！
<br/>　　霎时之间，一道巨大的剑***斩而出，直击吞虚神龙。
<br/>　　这剑轮如同一个方圆十里的巨大轮盘，散发出毁天灭地的气势，恐怖的剑威震荡诸天，强大的吞噬威能浩荡八方。
<br/>　　巨大的混沌剑轮，分开了整个世界，裹挟无敌的气势，瞬间就临近了吞虚神龙。
<br/>　　而吞虚神龙，面对如此强大的一击，居然没有丝毫的退避，再次张开了血盆大口，想要将混沌剑轮都一口吞下。
<br/>　　不过，这是不可能的事情，毫无意外，吞虚神龙直接被混沌剑轮斩爆，彻底灰飞烟灭。
<br/>　　“没有神智是硬伤！”
<br/>　　苏莫见此，微微的摇了摇头，此龙虽然强大，但是并没有多少神智，终极只是阵法之力虚构而成。
<br/>　　若是有同阶武者的神智，此龙不至于败的这么快，最起码不会硬吞混沌剑轮。
<br/>　　嗡～～
<br/>　　前方空间微微颤动，白色的传送光门再次出现了。
<br/>　　苏莫深深的吸了口气，最后一层了，能不能得到祖龙精血，在此一举了。

<br/>　　他知道苏莫闯过第九层，有很大的可能，但是没想到这么快。

<br/>　　毕竟，到目前为止，他都很是轻松，连终极剑道还没有动用呢！
<br/>　　言罢，苏莫大步踏出，身形一闪，便进入了空间光门之中。
