　　苏莫的二十四星宿剑阵，修炼的时间极奇短暂，只是演练了几个时辰。
<br/>　　但虽然还只是演练了几个时辰，此阵的威力远不能发挥道最大，但此阵的威力依旧非常的恐怖。
<br/>　　这就是本命剑客的恐怖，传闻中远古时代，越两个大境界杀敌的本命剑客，并不是虚言。
<br/>　　那种逆天剑客，武王境便能斩杀武帝。
<br/>　　当然，那种本命剑客，苏莫目前还差的很远。
<br/>　　最主要的就是他没有那么多本命灵剑，自然也无法学习那种超级剑阵。
<br/>　　眼见着四柄擎天神剑，向地狱三头狼袭杀而去，巫森脸色凝重了起来。
<br/>　　苏莫的实力出乎了他的预料，让他也感受到了一丝压力。
<br/>　　“地煞之气！”
<br/>　　巫森大喝一声，地狱三头狼顿时狂吼一声，三只头颅上的血盆大口齐齐张开。
<br/>　　轰隆隆！！
<br/>　　三股墨黑色的气流，从三张大口中喷出，形成了一道黑色的大河。
<br/>　　黑色长河威势极其骇人，蕴含浓烈到极致的地煞之气，如同滚滚魔焰，足以毁灭一切。
<br/>　　地煞之气是一种特殊的力量，十亿魔剑蕴含的魔煞之气相似，只不过并没有那种魔气，有的仅仅是大地煞气。
<br/>　　地狱三头狼，乃是巫森的战魂，完全受巫森的操控。
<br/>　　霎时之间，黑色的地煞之气长河，便与四柄擎天巨剑相击在了一起。
<br/>　　轰隆隆！！
<br/>　　巨响连连，四柄擎天巨剑，裹挟玄妙莫测的星宿之力，威力强大绝伦，但地煞之气同样强大无比。
<br/>　　地煞长河遭到剑气的攻击，急速溃散，支离破碎。
<br/>　　四柄擎天神剑，破碎层层地煞之气，洞穿地煞长河，急速向地狱三头狼袭杀而去。
<br/>　　不过，随着破灭大量的地煞之气，四柄擎天神剑的威势，也是有所减弱。
<br/>　　嗤嗤哧~~~
<br/>　　很快，四柄擎天神剑洞穿层层地煞之气，接近了地狱三头狼，而后直接爆射进了两张狼口之中。
<br/>　　轰！轰！
<br/>　　顿时，两声爆响，擎天神剑从狼口中一穿而过。
<br/>　　嗷！嗷！嗷！
<br/>　　地狱三头狼仰天狂吼，那两只被擎天神剑洞穿的脑袋，立刻变得虚幻了起来，而后萎靡不震。
<br/>　　噗！
<br/>　　远处，一直在操控地狱三头狼的巫森，顿时脸色一白，一口鲜血狂喷而出。
<br/>　　地狱三头狼和他心神相连，遭到重创之后，他立刻便跟着受伤了。
<br/>　　“怎么可能？”巫森脸色阴沉如水，本以为能轻易解决苏莫，没想到却自己却受伤了。
<br/>　　唰！
<br/>　　巫森立刻急速暴退，而地狱三头狼亦是化为一道黑色闪电，急速返回。
<br/>　　瞬息之间，地狱三头狼便返回了巫森身边悬浮在了巫森的身后。
<br/>　　咻咻咻！！
<br/>　　远处剑光爆闪，四柄擎天神剑一击之后，立刻又分散开来，化为了二十四柄本命灵剑，飞舞在苏莫的身周。
<br/>　　两人这一次交手，巫森完败，让得无数观战之人纷纷震惊，个个面色呆滞。
<br/>　　作为武帝强者的巫森，居然受伤了！而且看样子伤势还不轻。
<br/>　　“哈哈哈！！”
<br/>　　一声狂笑声骤然响起，只见司空炎仰天狂笑，随即大声道：“怎么样？我说的不错吧！苏莫就是变.态，不能以常理来衡量！你们的眼力也太差了！”
<br/>　　司空炎显得意气风发，之前其他人都担心苏莫会陨落在巫森的手中，只有他不担心。
<br/>　　现在，果然如此，他的眼力根本不是其他人可比，苏莫果然没有让他失望。
<br/>　　夕儿、闻人天都、彩芸等人，见巫森被苏莫击伤，本来正在震惊。
<br/>　　听闻司空炎的话，几人顿时无语。
<br/>　　你的眼力强？
<br/>　　是你没心没肺吧！
<br/>　　“哈哈，你们放心吧！苏莫必然能斩将此人灭杀成渣！”司空炎继续说道。
<br/>　　司空炎此话一出，顿时感受到无边的寒意，从脚底下直冲天灵盖。
<br/>　　不由的，司空炎顿时打了个冷颤，随即转头向另一个方向望去。
<br/>　　便见到那数千名黄泉魔宗的强者，全部都在冷视着他，无数凌厉的杀机，几乎凝为实质。
<br/>　　司空炎顿时一阵心虚，吞咽了一口口水，急忙向海明江身边靠了靠。
<br/>　　这五千名黄泉魔宗的强者，可都是武皇，再加上二魔老的准帝强者，根本不是司空炎能对抗的。
<br/>　　好在因为有帝释和海明江两位准帝在，一众黄泉魔宗的强者并没有出手。
<br/>　　因为他们都知道，巫森和苏莫的战局才是关键，谁胜了谁便能称霸天下。
<br/>　　而此时，在黄泉魔宗的阵营之中，脸色最为难看的人，不是黄泉魔宗之人，而是殷棕。
<br/>　　殷棕的脸色几乎扭曲了，本以为这一次苏莫必死无疑了，但现在的情况，完全不是那么回事！
<br/>　　上次在天海城，苏莫的实力也就和他相当而已，这才两个月的时间，居然就能击伤武帝强者。
<br/>　　这……这简直是开玩笑啊！
<br/>　　所以，殷棕几乎是狠的牙痒痒，今日若是杀不死苏莫，那他这一辈子，基本上都很难报仇了！
<br/>　　“可恶……可恶！天下间怎么可能有这么强天赋的人！”殷棕咬牙切齿，低沉的声音从喉咙中发出。
<br/>　　就在此时，前方的战况又起变化了！
<br/>　　“苏莫，今日若是不杀你，它日.你真的要无敌了！”
<br/>　　巫森大声吼道，他打定主意，今日必杀苏莫，无论付出多大的代价。
<br/>　　不然的话，以苏莫的成长速度，日后他只要被虐杀的份了！
<br/>　　“还有什么本事，都使出来吧！”苏莫面无表情，心中同样打定主意，无论如何今日也要斩杀此人。
<br/>　　但他并没有小看对方，他知道作为武帝强者，显然不可能只有这点本事。
<br/>　　“让你见识见识，什么才是真正的武帝！”巫森面上杀机爆闪，随即立刻大喝一声：“人魂合一！”
<br/>　　刹那之间，随着巫森话音一落，他身后的地狱三头狼战魂缓缓缩小，然后身形下沉，立刻和他的身躯融合在了一起。
<br/>　　随即，惊人的一幕出现了，只见巫森的身体，以肉眼可见的速度膨胀了起来。
<br/>　　一根根漆黑如墨，形如钢针一般的毛发，从他的身上急速长出。
<br/>　　锋利如尖刀般的指甲，同样从其十指手指上快速长出，寒光森森。
<br/>　　嘭！嘭！嘭！！
<br/>　　闷响连连，巫森的身躯眨眼之间便膨胀了一大圈，将他身上的衣袍都撑的爆裂开来。
