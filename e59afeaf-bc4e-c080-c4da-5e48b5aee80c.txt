　　万博现在的心情很不好，之前他一直大声嘲笑楚涵不行，可现在他却连战场都进不去，楚涵的一个领域将所有丧尸都包了进去，撤开之时包括大丧尸在内，都瘫倒在地上。
<br/>　　虽然那只大丧尸还没彻底死亡，但被密密麻麻的小型月牙不断围攻中，应该也离死不远了！
<br/>　　万博万万没有想到楚涵真的能打赢这只十阶大丧尸！
<br/>　　楚涵带着血色丧尸满场跑，可是战场就这么大，加之两个人的速度都极快，一不小心就将这战场绕了好几圈。
<br/>　　这期间其他人都无法插手，别说上去帮忙，能别添乱就不错了。
<br/>　　基地大门更是紧闭，生怕这血色丧尸跑到基地之中。
<br/>　　这种新型的丧尸没有人见过，对其一无所知，但光是看到那骇人的样子和恐怖的速度，就知道很不简单。
<br/>　　万博一行人也傻眼，没想到忽然冒出来这么个东西，尤其是每当楚涵带着丧尸跑近之际，他们都能感受到血色丧尸身上散发出来的强烈气场。
<br/>　　这种丧尸哪怕是十阶的万博，都绝对不想与之动手。
<br/>　　万博也决定带着四人离开这里，此处不宜久留，再拖下去他不仅一点好处捞不到，说不定还会有危险。
<br/>　　于是万博看向了基地战役指挥官：“把我的钱结了。”
<br/>　　基地指挥官一脸莫名，反问道：“现在十阶丧尸已死，你根本没动手，结什么钱？”
<br/>　　不少人也内心大骂万博的不要脸，这种时候竟然还想着结账。
<br/>　　万博阴笑着，指着远处的血色丧尸道：“这头丧尸的钱。”
<br/>　　周军词看不下去，开口道：“我们敬你是十阶高手，可你也不能乱说话，这只血色丧尸你连手都没动，却问我们要钱？”
<br/>　　万博道：“先给钱，我再上，鉴于刚刚你们的诚信问题，我现在对你们没有当场把钱结清的信心。”
<br/>　　“不需要你出手了！”陈正奇气愤道：“我们有楚涵就够了，不需要你装好人强行插一脚。”
<br/>　　从头到尾都是楚涵在战斗，万博就只动了嘴，还有脸要钱？
<br/>　　再说这诚信问题，到底是谁没诚信？
<br/>　　万博却是冷笑，继续道：“你们不会以为靠楚涵一个人，就能对付这只血色丧尸吧？他为什么撤开领域你们心里没数？更何况他要是打得过，用得着还在继续用这种拖延战术？现在的情况是，你们不得不求我动手，明白了吗？”
<br/>　　一番话说出口，也让很多人都思索起来，再看楚涵的情况，似乎也已经筋疲力竭。
<br/>　　胡承泽想了会儿，问道：“你要多少钱？”
<br/>　　现在不是讲面子和自尊的时候，最重要的是将这头血色丧尸解决！
<br/>　　万博嘴角的笑意浮现：“这样才对，我也不乱叫价，把之前谈好的钱一次性付清。”
<br/>　　基地战役指挥抿了抿嘴，道：“那个价格是你和你的团队单独出手的价钱，但现在只需要你们辅助而已，还要全价？”
<br/>　　万博冷笑：“你们要是继续拖延时间的话，我也不会介意，反正基地的死活不关我事。”
<br/>　　战役指挥和周军词商量了一会儿，而后回绝道：“我们不需要了，谢谢你来一趟。”
<br/>　　没想到最终会是这种答复，万博当即沉下脸：“你们的意思就是不给钱？”
<br/>　　周军词皱眉：“不需要你战斗了，不是不给钱。”
<br/>　　“还不是一样！”万博大怒，忽然就一掌而出，将离他最近的陈正奇拍飞：“给钱！不给钱就别怪我不客气！”
<br/>　　陈正奇本来伤就没好，一晚上连续被两大强者打，这让他内心不住的骂娘，他是拉仇恨还是怎样，怎么都朝他下手？
<br/>　　胡承泽连忙接住陈正奇，一看，好嘛，骨头又断了！
<br/>　　虽说对于五阶新人类来说断了骨头及时救助就能很快恢复，但一晚上连续断两次，接下来得卧床静养才行。
<br/>　　看到万博忽然动手，战役指挥也冷了脸：“你这不是在做生意，是在明抢！”
<br/>　　万博这时候也懒得装，随手抓·住一把武器，‘轰’的一声就朝着基地大门掷去。
<br/>　　“我就是抢又怎样！不给钱是吧，好！我就把你们的门砸了，让那头丧尸进去屠了这个基地！”
<br/>　　疯狂的举动，引得所有人都震惊不已，他们全部人都努力的对抗尸潮，在最后一关的时候，竟然还有人在战斗没结束之际，就开始作乱？
<br/>　　基地大门在那一掷之下，也当场被砸出了一个大窟窿，紧接着万博又是冲过去，十阶的力量一拳拳的轰在基地城墙上。
<br/>　　“住手！”基地指挥大喊。
<br/>　　万博坐视不理的继续砸，他的四名同伴更是一起上，直接让一处高墙彻底倒塌。
<br/>　　这个城墙由诸多新人类的生命守护，哪怕是尸潮战的时候，也没让大丧尸冲进去过，却在战斗结束之后，被人类高手破坏？
<br/>　　万博到底是人类还是丧尸的同伙？！
<br/>　　愤怒在很多人的心中滋长，但面对十阶的新人类，却没有一个人能够阻止的了。
<br/>　　这一切楚涵都看在眼里，有那么几分杀意突显。
<br/>　　此时在十阶的大丧尸那里，时空月影斩终于完成了任务，那一柄柄小月牙都快磨没了，才深入大丧尸的脑袋内部，将其大脑结构一举破坏！
<br/>　　狂奔中的楚涵，感受到一阵巨大数字的积分跳动，当下便立即查看！
<br/>　　好家伙，刚刚的那头十阶超级丧尸，竟然一次性就给了八十个积分！
<br/>　　高于楚涵的等阶，还是超级丧尸，更是尸潮战中的老大，体内还寄生了一只特异的血色丧尸。
<br/>　　这让大丧尸的价值一下子远超正常的十阶，也让楚涵摸索出了积分的一丝规律。
<br/>　　越稀有，情况越特殊，积分就越多！
<br/>　　果然丧尸要杀，就要杀最厉害的！
<br/>　　但八十积分距离十阶还差得远，哪怕加上之前杀掉的最后几只丧尸积分，楚涵现在的积分也不够升阶。
<br/>　　不升阶，想要战胜这只寄生的血色丧尸就不可能！
<br/>　　似乎是察觉到大丧尸的死亡，让这只血色丧尸仰头痛苦的嚎叫起来，声音凄厉与正常的丧尸不太一样。
<br/>　　听到这个声音，楚涵当场就瞳孔一缩……
<br/>　　有一种情况他一直没想到，正常丧尸体内怎么可能会有寄生物？
<br/>　　除非！那大丧尸一开始就怀·孕，变成丧尸后由于某些原因，体内胎儿未腐化，而是继续生长至今。
<br/>　　丧尸是不能生育的，这一点与动物丧尸不一样，但血色丧尸这种情况，却相当于二代丧尸！
