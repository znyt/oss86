　　第593章
<br/>　　野原火在草原上露面的时间并不长，却已经搅出足够的风雨。短短时间内，他手下已有五六千精壮儿郎，已是任谁都不能忽视的一股势力。
<br/>　　但野原火的野心从来不止于此，他的目标从一开始就是坚定的，之后再无半点动摇。
<br/>　　不过，想要继续扩张自己的势力，目前有两大困难。一就是兵器，刀剑在戎族永远是奢侈品，以前都是从帝国那里抢。不过，目前野原火还不希望和帝国产生摩擦。第二就是士兵素质，他手下的队伍，识字率几乎是零，就连野原火本身……其实也认不了几个字。
<br/>　　如果只是满足目前的小打小闹，认不认字其实也没关系。但如果有更大的贪图，目前自然远远不够。
<br/>　　所以他才将主意打到齐如海身上，相比之下，兵器倒是小事。野原火现在缺的是可以出谋划策的人才。有些事，他连可以商量的人也没有。每天被一群蠢货包围，野原火大概能体会每天头疼的感觉了。
<br/>　　现在，他的目标已是整个北蛮部。在北方草原，北蛮部是最大的部落。原先它的统治者是北蛮王，但过去这么多年，北蛮王已经寿终正寝，如今继任的是九王子金问道。
<br/>　　他是新的北蛮王。
<br/>　　双方已有过摩擦，但真正的决战尚未展开。只等着这批兵器到了，全副武装的火之部落，就要向北蛮部展开总攻。
<br/>　　夜里，野原火独自望着北方，在那独眼中，恍惚有火光燃烧。
<br/>　　“这么多年过去了，我的乌力罕……你还记得我么？”
<br/>　　……
<br/>　　这个时候，程大雷和小白狼已经进入草原，随之，程大雷就发现一件极为尴尬的事情。
<br/>　　程大雷本来的打算，是找个戎族打听一下燕不归的事情。这倒也不是太困难的事情，毕竟燕不归那样的高手，无论在任何地方都是显眼的存在。
<br/>　　但程大雷疏忽了一件事，这里是戎族势力范围，他和小白狼两个帝国人，委实是太过显眼的存在。想要混入戎族中心地带谈何容易。当然，戎族地广人稀，想不被发现还是蛮容易的，可程大雷这次过来是找人复仇，并不是过来观光旅游的。
<br/>　　小白狼明显也感知到这一点，道：“你别灰心，功夫不怕苦心人，只要你想找到他，总是能找到的。”
<br/>　　程大雷垂头丧气的点点头，其实也没多大把握，要在茫茫如海的草原上寻到一个人谈何容易。况且，他现在还不能确认，燕不归是不是已经回到草原。
<br/>　　为今之计，只有走一步看一步，顺便也查探一下戎族最近的动向。
<br/>　　正这时候，耳边突然响起唿律律的马鸣声，一队人从远处快马奔来，将程大雷和小白狼围住。
<br/>　　“呔！”为首的大汉吼道：“这是爷爷的地盘，不想死的，把马和财物放下，敢说个不字，老子送你们见阎王。”
<br/>　　程大雷和小白狼对视一眼，表情都有几分无奈。程大雷眼睛忽然一亮，他倒是忘了，草原上除了戎族和来往的商队，却还有第三种职业。
<br/>　　那便是马匪。
<br/>　　马匪几乎可以算是山贼海盗的同行，自己想要向戎族打探消息不太现实。可马匪四处钻来钻去，未必不知道草原上的消息。
<br/>　　当真是送上门来的礼物，刚才还苦闷无策的程大雷表情缓和了许多。
<br/>　　他看向为首的马匪，道：“报个名吧，怎么称呼？”
<br/>　　这马匪外号草上飞，本来名字却是无人知道。提着脑袋干这种缺德的事情，他们大多不愿用爹娘给的名字。
<br/>　　程大雷如此镇定，倒是令草上飞刮目相看。他眯起眼睛道：“看来也是道上的兄弟，那就别废话了，把值钱的东西放下，今天老子好心，放你们两个一条生路。”
<br/>　　程大雷活动着手腕，冲对方招招手：“别废话了，你们一起上吧，打完了我还有事情问你们。”
<br/>　　草上飞皱起眉头，冷喝道：“小子，当真是给脸不要脸，弟兄们，给我动手。”
<br/>　　小白狼在一旁看着，对方不过五六个人，这种场合也不需要她动手。只见程大雷骑在黑牛背上，只是几个冲撞便将草上飞一伙收拾了。
<br/>　　草上飞被打翻在地，他刚挣扎着想要站起来，程大雷就从牛背上跳下来，一脚将他踹翻在地。
<br/>　　如此重复几次，草上飞才意识到这次自己踢了铁板。他双手作揖，求饶道：“好汉饶命，我上有八十老母，下有黄口幼子，您饶我有眼不识泰山。”
<br/>　　程大雷摆摆手，制止他继续说下去：“走吧，带我去你的老巢坐坐吧？”
<br/>　　“呃……”草上飞愣住，试探着问道：“敢问好汉尊姓大名？”
<br/>　　程大雷在道上还是有些名气的，哪怕这里草原。当他报出名号时，草上飞顿时吓得一跌，暗骂自己肯定是早上出门没看黄历，竟然招惹到这等煞星。
<br/>　　他将程大雷引到自己山寨，不过是山坡里几栋茅草屋，他手下也不过五六十好人。
<br/>　　过了一会，他提着个箩筐进来，立在程大雷面前，先一左一右给了自己两个大耳光子。
<br/>　　“爷，今天是我有眼无珠，不该招惹您。但我们这里小门小户，实在拿不出什么值钱的东西。山寨的家当都在这里了，求您大恩大德，放弟兄们一条生路。”
<br/>　　程大雷瞥了他一眼，表情有些无语，只见箩筐里，摆着些金银酒器什么的。
<br/>　　“好啦，好啦。”程大雷摆摆手：“谁说我是来劫你们的，我只是顺便路过这里，问你一些事情。”
<br/>　　草原上的麻烦，大部分都是帝国人。许多从前都是军中的老兵，吃了败仗，也回不了家，就做了马匪。
<br/>　　听到程大雷的话，草上飞仍然是将信将疑，贼不走空，他当真会放过自己么。不过话说回来，自己这点家当，当真没什么值得他动手的。
<br/>　　“程当家打算问什么？”
<br/>　　程大雷想了想：“戎族最近有什么新鲜事没有？”
<br/>　　“倒也没什么事，无非就是打来打去。哎，这里不太平呐，不如我们干脆跟着程当家混得了。”
<br/>(本章完)
