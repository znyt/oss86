　　“进还是不进？”罗杰直接问道，他已经受不了在这种恐惧里继续呆着。
<br/>　　白忧定了定神，看了眼在场还剩下的四十名神秘家族成员，所有人的目光都带着期许，他们想要尽快出谷，哪怕迷宫也总比这里耗着强。
<br/>　　“进。”白忧下了决定，带着人便朝着一条道路冲了进去。
<br/>　　四十人抱团走在一起，不敢再分散，一百多人的队伍还没见到敌人就一下子只剩下四十人还在继续，这样的状况在千年内的神秘家族都史无前例，也是一向都高高在上的白家，损失最大的一次。
<br/>　　罗家相对成员数量多一些，哪怕在与高家爆发过一战之后，这回罗杰带出来的人依旧比白家多，只是再如何成员数量多，也绝对经不起这么忽然一下子的大量失散。
<br/>　　更何况那些走散的人之后会怎样，白忧和罗杰心里都很清楚，那就是凶多吉少！
<br/>　　阴谷出口迷宫道路不宽不窄，被大范围的植被墙挡住的一条条道路很干净，没有丝毫危险的样子，也根本看不出这处地方曾在千年前有过什么凶险。
<br/>　　但四十人还是走的小心翼翼，不断摸索着往前走，白忧带队走在最前方，并非因为他忽然的性情大变，而是对于这种磁场变化的迷宫图，他是在场唯一有研究的人。
<br/>　　那座私人岛屿的底下基地，就是建立在这种充满磁场变化的土地之下！
<br/>　　白忧走走停停，口中时不时的默念着什么，双眼也在计算步数，走的万分小心，一群人就跟在他的后面一声不吭，进度很慢也很有可能会就这么被困在这迷宫出不去，但跟着白忧走却是他们唯一的希望。
<br/>　　只是在走了这么长时间之后，之前先一步逃入这迷宫的那些人，他们却一个都没见到。
<br/>　　凭空消失？
<br/>　　罗杰自进来后就一直在想这个问题，他总觉得事情有些诡异的过分。
<br/>　　迷宫之外，那个被秒杀的白家人身上火焰终于消失，骨骼都被烧的不完整，碎裂成一块块看不出原型的东西。
<br/>　　此时的在这堆漆黑的焦炭物旁，一个人缓步走近，他的穿着很普通，状态却很不普通。
<br/>　　因为他的手中，正燃烧着一团火焰！
<br/>　　来者正是火族的少主炎良，与水族的鱼柔女在门族中的地位相等，也是这阴谷内一阴一阳调和的两种极端存在。
<br/>　　“少主，杀阵已启动。”一名火族人在其身后开口。
<br/>　　炎良点头，看着眼前的迷宫入口沉默不语。
<br/>　　身后之后扫了眼地上的灰烬，道：“其他在阴谷内逃散的人怎么说？”
<br/>　　“不必管。”炎良吩咐道：“既然进来了，就只有一条出口，想出去只能经过这里。”
<br/>　　那名火族人有些犹豫，问道：“可眼下的阴谷内阴阳不再失衡，少主您的火焰也正处于最巅峰时期，说明那个水族也在此处，并且就住在阴谷的入口。”
<br/>　　“所以呢？”炎良不以为意：“这些大家都心知肚明，水火两族虽世代为敌，但本身就相辅相成，这就是制衡。”
<br/>　　那名火族人继续道：“我胆心这一代的水族人，会不会帮助那两家族的人从入口出去？要知道如果是水族圣女，完全有这个能力。”
<br/>　　“背靠海岸，水族圣女有着得天独厚的优势。”炎良依旧神色平淡：“但我可不认为水族会管这等闲事。”
<br/>　　那名火族人没有再继续询问，只是在思考着炎良的话。
<br/>　　“这个杀阵困不住他们的。”忽的，炎良又再次出声。
<br/>　　火族人一惊，之后才反应过来炎良说的是这个出口迷宫·内的白家人和罗家人。
<br/>　　“毕竟只有阴阳在谷。”火族人苦笑道：“如果是当年……刚才他们就全死了。”
<br/>　　“虽然困不住，但也不会让他们轻易走出去。”炎良不再关注迷宫，转身朝着浓郁迷雾的地方走去，在这阴谷的禁制完全解除之前，他同样无法在阴谷内自由行动。
<br/>　　——————
<br/>　　狼牙基地，这是步纱来到这一禁地的第七天，这一周内她都待在木屋内没有出来，白允儿也就这么站在木屋外寸步不离，尚九娣离开过两次前去安罗城处理公务，陆玟丞等一干楚涵的心腹则是时不时的前来查看情况，带发现整整一周都没变化之后，便也不再将重心放在这里。
<br/>　　基地内最后的一点兵力，则是在一开始的集体涌来之后没多久，便如潮水般退去，守在周围不被人察觉到的地带。
<br/>　　这一天一切如常，尚九娣也照旧前来给白允儿和步纱送食物和水。
<br/>　　只是在她前脚刚靠近木屋的下一秒——
<br/>　　刷！
<br/>　　忽的一柄锋利刀片，从白允儿的手中飞出，擦着尚九娣的耳畔而过，切断了她一截鬓发！
<br/>　　突然的出手让尚九娣心脏都差点跳停，她愣愣的站在原地，无法形容刚才刀片划过的一瞬间，那与死亡极限接近的感觉，当时她只需要稍微偏离一点点，恐怕就已经当场死亡。
<br/>　　而后尚九娣便听到了‘噗’的一声轻响，一股温热喷洒到她的脖颈后方，带着黏稠感。
<br/>　　她很清楚，那是血。
<br/>　　此时站在尚九娣对面五米开外的白允儿，眼神依旧冰冷，刚才的出手快速至极，但也没让她的神色有丝毫变化。
<br/>　　尚九娣在这一刻思维很混乱，她的第一反应便是有人来暗杀，而非白允儿杀掉了狼牙内部的谁，哪怕在正常情况下看来，能瞒过那些防守人员潜入这里，还悄然无息靠近尚九娣身后，这根本就不可能。
<br/>　　下一秒，尚九娣猛然转身，朝着远处一声大喊：“敌袭！拉响最高警戒！”
<br/>　　等她喊完这句话后，才看清楚了身后的情况，那是一个浑身都穿着黑色衣服的人，喉间没入了一块刀片，已经是死亡状态。
<br/>　　这个人尚九娣并不认识，但却一眼就看出了某种诡异感。
<br/>　　怎么混进来的？
<br/>　　被白允儿秒杀的人，怎么混进来的？
<br/>　　不容她有时间思考，刚才的一声大喊已经让所有听到的人警戒起来，好几个身影在周围的树梢间穿梭，那一双双洞悉分毫的眼睛在确认了地上死去的人并非狼牙成员之后，便又以更快的速度离去。
<br/>　　前后不出三秒，狼牙基地最高级别的哨响，便彻响在整个基地！
