　　陆千刃，是北邙学院的老祖宗。
<br/>　　而楚风眠，充其量也最多只是一名核心弟子。
<br/>　　老祖训斥弟子，本是天经地义的事情，让弟子跪下，便是就要跪下。
<br/>　　不过这什么老祖，弟子，这种身份，从来没有被楚风眠放在眼中过。
<br/>　　“陆千刃，你找死？”
<br/>　　这话一出，在场无数老祖的脸色都是一变，他们训斥过无数弟子，还无人敢反抗。
<br/>　　“不好，楚风眠，你赶紧向陆千刃老祖道歉忏悔，让他宽恕你的鲁莽！”
<br/>　　邓灵副院长听到这几个字，急忙的向着楚风眠开口道。
<br/>　　不管是陆家，还是夜家，都是对楚风眠恨之入骨，一直都想要找个机会，杀了楚风眠。
<br/>　　现在楚风眠说出这话，正好是给了他们一个完美的借口，可以直接轰杀楚风眠。
<br/>　　“道歉！可笑，侮辱老祖宗，按照门规，罪可当诛！小子，去死吧！”
<br/>　　陆千刃脸色狰狞到了极点，他的头发根根林立，满是怒火，如同一头发狂的狮子一般。
<br/>　　他的掌心灵力，都是逐渐凝聚，化为了一座九层宝塔，直接冲着楚风眠，轰击过来，要将楚风眠直接轰杀。
<br/>　　“是吗？陆千刃，你以为你还能杀的了我？本来我这次回来，便是就要杀了你，很好，既然你敢出手，那我现在就杀了你！”
<br/>　　楚风眠看着陆千刃，眼神之中尽是杀意。
<br/>　　区区一个碎命之境的老祖，也想要杀了楚风眠？
<br/>　　“给我破！”
<br/>　　楚风眠凌空猛然一掌，轰击出去，他的掌风都是化为了一条长龙，轰鸣之中，冲了过去。
<br/>　　这九层宝塔，被当场打碎，一条长龙，直接冲着陆千刃轰击过去。
<br/>　　“陆千刃，你就这样的废物，也敢自称老祖？我就拿你的鲜血，去洗刷我的仇恨！”
<br/>　　楚风眠冷喝一声。
<br/>　　这九层宝塔，在楚风眠的一掌之下，全部破碎，似乎是陆千刃凝聚的力量，都是在这瞬间，被全部打散。
<br/>　　大部分的力量，全部轰击到了陆千刃的胸口上。
<br/>　　“噗！”
<br/>　　陆千刃的胸口，都被直接打碎，他的身躯，足足是飞出去了数十里远，在逐渐的停了下来，身上的大半精血，在这一掌之下，全部毁灭，灵力虚弱无比。
<br/>　　“什么？”
<br/>　　“这怎么可能？”
<br/>　　在北邙宫中的不少老祖，看到这个结果，都是一脸震惊。
<br/>　　他们也没有任何一人料到了这一幕。
<br/>　　陆千刃，可是碎命之境的老祖，而且可并非是刚刚突破碎命之境的，已经是有着一段时间，有着积累。
<br/>　　就算是在传闻中，楚风眠有着足以击败碎命之境的实力，但是在众人看来，楚风眠也最多只能击败一些刚刚突破，境界不稳的碎命之境。
<br/>　　谁也万万想不到，在这一掌之下，陆千刃便是败的如此之惨。
<br/>　　这样一尊老祖，便是当场重伤，濒临死亡。
<br/>　　周围的无数老祖中，都无人想到了这一点，所以才无人阻止。
<br/>　　等到他们反应过来的时候，陆千刃已经是身受重伤了。
<br/>　　“好强的力量！”
<br/>　　站在楚风眠身边的邓灵副院长，脸色才是吃惊到了极点。
<br/>　　在楚风眠的身边，他足以完全感觉到刚刚楚风眠动用的力量，没有动用任何的武技，所动用的只是纯粹的力量。
<br/>　　根据他的了解，楚风眠最为强大的，应该是楚风眠的剑术才对，楚风眠的剑道无双，已经是九域众人皆知的事实。
<br/>　　可现在，楚风眠居然是单单动用力量，没有动用剑术，甚至是就连武技都没有动用，就将陆千刃给轻易击败。
<br/>　　楚风眠的实力，比起他所想象的，还要恐怖的多。
<br/>　　邓灵副院长也突然明白想明白了过来，楚风眠这一次来到北邙宫，可不是负荆请罪来的，而是要大闹一顿了。
<br/>　　“畜生，你居然敢打伤老祖！还不速速跪下！交出你的一切奇遇，自废修为！还可以留你一具全尸！”
<br/>　　夜牧看着楚风眠，尖锐的声音大声喊道。
<br/>　　“诸位老祖，你们看到了吧，这楚风眠，桀骜不驯，绝对是我北邙学院的一大祸害，今日不杀了此人，将来必然会颠覆我们北邙学院，我们北邙学院，都会毁在这小子的手中！”
<br/>　　“不错，诸位老祖，这小子目中无人，打伤老祖，大闹北邙宫，已经是罪无可赦，我现在就擒下他，来让各位老祖审判！”
<br/>　　那夜家的副院长，夜众星，也是大声的喊道。
<br/>　　一群夜家老祖，还有那些跟夜家关系不错的老祖，都在一个个大声的喊道。
<br/>　　“对，擒下此子！此子的一身实力，太过诡异了，他肯定是七大宗门的奸细，我们必须要擒下此人来！弄清楚一切！”
<br/>　　“不错，还有他所得到的那些奇遇，都是属于我们北邙学院的，我有一种抽魂之术，可以折磨他的灵魂，倒时候他一定会说出来的！”
<br/>　　“还有，此人的师尊是玄北！玄北肯收此人为弟子，肯定也是跟七大宗门脱不开关系！”
<br/>　　“杀！这种人，不能够留在我们北邙学院之中，今日必须要杀了他！”
<br/>　　群情激昂，一道道目光，看向楚风眠，尽是杀意。
<br/>　　“楚风眠，你还不跪下！还想反抗？那好，我就先废了你的修为，抽出了你的一身血脉来，看你还能如何反抗！”
<br/>　　夜牧看着楚风眠，尖锐的声音大吼起来，一拍王座，猛然从王座上站了起来。
<br/>　　在他的背后，一把长刀，突然出鞘，刀锋闪烁，突然这一刀，便是打破空间，直接向着楚风眠袭来。
<br/>　　这一刀的突然出现，简直是在无声无息之间，在偷袭这楚风眠的背后。
<br/>　　等到夜牧老祖出手的一刻，这刀锋，已经是逼近了楚风眠的背后，令楚风眠甚至是就连任何一点的反应时间都没有。
<br/>　　显然是阴险无比，不等楚风眠任何反应，就要杀了楚风眠。
<br/>　　“死吧！小子，你杀我夜家天才，今日就以你的鲜血，来祭奠！”
<br/>　　夜牧老祖脸色狰狞，大声吼道。
<br/>　　“战龙宝车！”
<br/>　　就在这一道刀锋，逼近楚风眠的时刻，突然之间，就从楚风眠的座下，战龙宝车，突然出现。
<br/>　　其中那宝车之上的罗伞，突然打开，散发出七彩霞光，这一道刀锋，碰到这七彩霞光，顿时全部破碎，烟消云散。
