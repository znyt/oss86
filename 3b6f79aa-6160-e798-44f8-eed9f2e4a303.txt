        想到这里，明司寒浑身不自在地抖了一下，立即刷卡进门。
<br/>
<br/>        这是一间偌大的总统套房，三室一厅。
<br/>
<br/>        连星儿不知道自己该住哪间房，只好将行李先放到客厅里。
<br/>
<br/>        一想到待会还能与洛羽单独见面，连星儿便兴奋不已地掏出手机，和妈咪童筱颖打起了视频电话。
<br/>
<br/>        “妈咪，猜猜我在哪里？”视频电话接通后，连星儿便迫不及待地问道。
<br/>
<br/>        童筱颖打量了一番连星儿的背景环境，笑道：“你是不是住在自家的酒店里呢？”
<br/>
<br/>        傅夜沉曾在云城收购了云城五星级酒店，只提供给来影视城拍戏的明星住。
<br/>
<br/>        不过，近几年，云城又新建了好几家五星级的酒店，所以，云城酒店已经不是明星们的唯一选择。
<br/>
<br/>        “我住的是明丽酒店啦！而且，还是洛羽住的总统套房噢！”连星儿笑嘻嘻地说。
<br/>
<br/>        “你改当‘私生饭’了吗？”童筱颖顿时诧异起来。
<br/>
<br/>        “妈咪，你女儿是那种人吗？”连星儿不禁嘟嘴道。
<br/>
<br/>        童筱颖忍俊不禁起来：“那你倒是跟我说说，你是怎么住到洛羽的总统套房里去呢？难道，你暴露了自己的身份？”
<br/>
<br/>        “这次，我可没靠爹地噢！我是靠我自己办到的！”连星儿得意对童筱颖比了个剪刀手。
<br/>
<br/>        童筱颖笑道：“好吧！既然你不想说，我也不问了。不过，你打算什么时候回来？你不在家，家里变得太安静了，我不习惯。”
<br/>
<br/>        “妈咪，难道我在家的时候，能让家里变得很热闹？”连星儿不以为然地反问。
<br/>
<br/>        童筱颖点了点头：“那当然，至少我能时不时地听到有人喊我妈咪。”
<br/>
<br/>        “那我每天也时不时地给妈咪打电话好了。”连星儿开心地说道。
<br/>
<br/>        童筱颖幸福满满地应了声：“嗯。”
<br/>
<br/>        “妈咪，我要出去吃午饭了噢！就不跟你聊了。爹地在家要是欺负你，你告诉我，我回来再收拾他。”连星儿接着说道。
<br/>
<br/>        童筱颖笑不拢嘴地回答：“嗯，好！在外吃好点，也要好好照顾自己保护自己。”
<br/>
<br/>        “我会的！妈咪，拜拜！”
<br/>
<br/>        “拜拜——”
<br/>
<br/>        “来，啵一个！”连星儿随即给妈咪送了一个飞吻，才将视频电话给挂断。
<br/>
<br/>        酒店里中午有自助餐提供，连星儿也不打算去外边吃了，索性直接在酒店里解决午餐。
<br/>
<br/>        她端着盘子，在把所有美食一一扫视了一遍后，才开始重新挑选自己想吃的东西。
<br/>
<br/>        挑完自己想吃的食物后，连星儿端着盘子找座位的时候，无意间看到了一个让她很不想再见的男生。
<br/>
<br/>        哎——
<br/>
<br/>        真是冤家路窄。
<br/>
<br/>        连星儿扯动着嘴角，坐在了离明司寒很远的角落里用餐。
<br/>
<br/>        明司寒吃完后，起身去放盘子的时候，经过走道时，一眼便在人群里看到了坐在那边角落里的连星儿。
<br/>
<br/>        突然间，他莫名其妙地红了耳根，再度怀疑，自己是不是看花眼了。
<br/>
<br/>        怎么走哪儿，都能遇到她？
<br/>
<br/>        明司寒放了盘子，下意识地加快了步伐，离开了酒店的自助餐厅。
<br/>
<br/>        他在外面买了酒和下酒菜回到了自己的总统套房，没过多久，明希哲便来了。

<br/>        明丽五星级酒店，是他们明家旗下的产业。
<br/>
<br/>        明希哲一进门，直接抱着酒瓶往沙发上躺，丢了偶像的包袱，也就跟平常人没什么两样，完全没有一个偶像明星该有的样子。
<br/>
<br/>        明司寒则还是拿起两个酒杯，给自己和明希哲倒了两杯酒：“有什么苦水，就赶紧吐吧！”
<br/>
<br/>        “你帮我查查，那个洛羽，跟傅氏集团的Boss傅夜沉是什么关系？”明希哲气呼呼地说道。

<br/>        所以，到时候电影上映，一定会全球热卖。

<br/>        明希哲觉得，这部电影对他来说，若是真火了，就是他是从国内知名度升华到国际知名度的机会。
<br/>
<br/>        但是，他的经纪人，好不容易替他争取到的男主角之位，却被那位很没眼光的傅大Boss一句话，给撤掉了，又重新回到了洛羽身上。

<br/>        “查谁都容易，唯独查傅夜沉难。”明司寒淡定道。
