　　辗转良久，刘哔方才沉沉睡去。
<br/>　　第二天醒来时，太阳已经升起，刘哔穿戴好了衣服，动作轻缓的走出了屋门。
<br/>　　屋外，九叔早就已经起来，正坐在一个木椅，在大厅的门口观察着什么。
<br/>　　刘哔揉了揉有些酸痛的双眼，昨天一整夜的噩梦，再加上初临一个陌生环境的警惕感，直接导致了他现在的精神十分萎靡。
<br/>　　磨磨蹭蹭的下了楼，刘哔晃悠到九叔身边，搬了一个小木凳坐下，顺着九叔的目光向院中看去。
<br/>　　是他昨夜回房间的时候看见的那些花朵，此时在阳光下，泛着极为鲜艳的红色，猛地看去，美奂绝伦。
<br/>　　“这是彼岸花。”九叔转头看了刘哔一眼，随后又将自己的目光转向那片红色的花朵：“很多人都很害怕它，说什么这是黄泉路上的花，真是愚昧无知啊。”
<br/>　　刘哔微微颔首，看着九叔淡黄色的皮肤，在阳光照射下逐渐变的通红，他向上卷起舌头，牙龈的地方又有些瘙痒了。
<br/>　　“可是呢，我种植了这么久，都没有真的出现什么意外，我就是想要让所有的人知道，这只是一种很普通的鲜花。”九叔的眼睛没有焦距，像是在自言自语，但刘哔知道，九叔是对他说的。
<br/>　　像是在回应一般，远处的花朵轻轻颤动了几下，纤细的枝干微微摇曳。
<br/>　　“没有黄泉。”九叔语气坚定，停顿了一下，补充了一句：“也没有地狱！。”
<br/>　　一脸漠然的听九叔讲完，刘哔心中没有多大的反应，他对这些花花草草的东西并不了解，甚至之前还认为是这个世界特有的产物。
<br/>　　彼岸花啊，他当然知道，在刘哔原本的世界里，这种花也十分出名。
<br/>　　思考了一会，刘哔感觉身体一阵滚烫，不由得用手遮挡住了双眼，抬头看了看毒辣的太阳，默默的将小板凳向后拉扯了一下，离开了太阳的照射范围。
<br/>　　事实上他挺佩服九叔的，老人家晒太阳是一件很常见的事情，但晒这么毒辣的太阳他倒是还没怎么见过。
<br/>　　紧接着，刘哔就彻底收回了心中先前的想法，随着他的动作，九叔似乎也察觉到了什么，伸手将自己的木椅也往后拉扯了一段距离。
<br/>　　“这都快入秋了，天气还这么热，刚才想事情想的出神，还没有注意到……”九叔嘀咕了一声，重新做回了椅子。
<br/>　　两人待在大厅门口，一时间谁都没有主动开口，刘哔在想怎么让九叔传授自己道术的事情，九叔在想昨夜有意传下衣钵却被拒绝的事情。
<br/>　　气氛变的有些尴尬，片刻的功夫，九叔张了张嘴，想要说些什么，却被文才突如其来的声音打断，只得作罢。
<br/>　　“师父，师弟，早饭做好了，来吃饭喽！”一道憨实的声音响起，九叔叹了口气，站起了身来。
<br/>　　看来是他和这个徒弟无缘啊，罢了，衣钵这件事情，怎么好意思主动开口呢，反正时间还长，指不定这臭小子就自己想通了呢。
<br/>　　“走吧，去吃饭了。”看了一眼刘哔，两人整理了一下衣装，一前一后走出了大堂。
<br/>　　早饭很简单，糯米粥，配一些米饼，还另做了几碟咸菜，很久没吃过热乎乎的食物，刘哔心中微微有些兴奋。
<br/>　　这种饭菜，放到他原本的世界中，是多少大佬都奢求不来的食物啊！
<br/>　　但是怕露出自己嘴角的獠牙，刘哔只能强忍住进食的欲望，小心翼翼的拿起一块米饼，放在嘴边咬了一口。
<br/>　　瞬间，刘哔整个人都有些不好了，一股意想不到的恶心感自口中传进了脑海，连带着他的胃部都开始痉挛，宛若拧成了一股麻花一般。
<br/>　　“刘，你怎么了？身体不舒服啊？”九叔停下了筷子，注意到刘哔有些不太对劲，面色疑惑的开口。
<br/>　　摆了摆手，刘哔示意自己没事，但胃部依旧在不断翻搅着，一阵呕吐的感觉，不断冲击着喉咙。
<br/>　　“是不是昨天夜里着凉了？多喝点糯米粥，驱驱身体里的寒气。”九叔接下来的话语，让刘哔心中一惊，开始手足无措了起来。
<br/>　　他现在可是正八经的丧尸啊，虽然严格来说，和僵尸并不属于同一个物种，但是谁知道会不会对糯米产生什么不好的反映。
<br/>　　所幸九叔只是随口一说，也没有在这个话题上继续纠缠，他只是觉得刘哔有些不太对劲，却也没发现刘哔身体上的异样。
<br/>　　“对啊，喝点粥，清晨就是要喝粥才对身体好。”正在一旁狼吞虎咽的文才，用余光瞥了一眼刘哔，随口附和道。
<br/>　　虽然文才语气颇为关心，可刘哔却有种冲动，想要将这个家伙的嘴给缝住！
<br/>　　现在的情况，有些骑虎难下了，刘哔脸色有些难堪，慢腾腾的端起桌上的糯米粥，放在嘴边轻轻吹了一口。
<br/>　　他当然能找到其他办法搪塞过去，但是以后和九叔这些人生活在一起，也免不了接触这些东西，如果每次都找借口搪塞，总会让他们起疑心的。
<br/>　　再说，他是经过病毒改造的特殊丧尸，而僵尸是死时一口气咽不下去的怪物，这两者从本质上就根本不同，应该不会有同样的弱点吧。
<br/>　　怀着上刑场的态度，刘哔咬了咬牙，狠狠喝了一口，因为动作太过于激烈，刚长出不久的獠牙碰撞在瓷器上，发出一声细微的声响。
<br/>　　滚烫的糯米粥入口，如同一块烙铁一般，顺着食道逐渐落入了胃部，像方才一般的强烈恶心感又涌了上来，刘哔双手捧着粥碗，微微颤抖着。
<br/>　　此时刘哔心中的想法，简直像是打翻了五味瓶一样复杂，看着眼前白花花的米粥，他却是怎么都喝不下去。
<br/>　　他从来没有吃过如此难吃的食物，甚至，对于他来说，与其说这是食物，倒不如说这是一碗搀着剧毒一般的毒水一般，让人根本无法下咽。
<br/>　　“你，到底怎么回事？”九叔皱着双眉，看着刘哔颤抖的双手，继续开口：“要不要送你去看郎中？”
<br/>　　自己这个徒弟现在的状态，明显有些不太正常。
<br/>　　“没事，就是想起来昨天那些僵尸，有些倒胃口。”刘哔缓缓将粥碗从自己脸上拿下，嘴角轻轻一扯，露出了一个有些难看的笑容。
<br/>　　他不知道自己的身体究竟出了什么问题，但系统却死活都不肯出来给他一个解释，无论如何，这顿饭他是不会有胃口了。
