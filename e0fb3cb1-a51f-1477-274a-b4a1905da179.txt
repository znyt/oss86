        “去请安装公司的人，给大厦从北上路变压器那里再新接两条主备用供电进来。”傅夜沉接着命令道，“以后，不要再出现这种一路主备电源突然都停电导致整个公司无法工作的情况。”
<br/>
<br/>        落以柔再次应道：“是！”
<br/>
<br/>        就在刚刚电力局那边确实没有提前通知，就断了整栋大厦的电，造成公司里大家都无法继续工作。
<br/>
<br/>        童筱颖听着脚步声越来越近，连忙将耳朵上的两个助听器摘了下来，放进了口袋里。
<br/>
<br/>        当她站起身来，微笑着转过身去，刚巧迎面遇上了正走下楼来的傅夜沉，以及跟在他身旁的落以柔。
<br/>
<br/>        落以柔一眼就看到了童筱颖，身心一怔的同时，担心傅夜沉会不会对童筱颖又做出什么出人意料的反应来，结果，她的担心是多余的。
<br/>
<br/>        傅夜沉的目光只是淡漠地从童筱颖的身上扫过，直接落在了她的身后。
<br/>
<br/>        然后，他就像完全不认识她童筱颖似的，与她擦肩而过，直接优雅从容地走下楼去。
<br/>
<br/>        落以柔在经过童筱颖身边的时候，得意地瞥了她一眼，紧接着加快了步伐跟上了傅夜沉的脚步。
<br/>
<br/>        这是他第二次视她为空气了吧！
<br/>
<br/>        记得上次，还是在李忆莎的婚礼宴会厅门外。
<br/>
<br/>        俗话说，事不过三，她还是要有点自知之明吧！
<br/>
<br/>        她跟他真的结束了，彻底结束了，她还在对他奢望什么啊？
<br/>
<br/>        是破镜重圆？还是久别重逢？又或者是再续前缘？
<br/>
<br/>        认清自己的位置后，童筱颖不再做那种为了见他，一口气爬上二十多层楼梯的傻事了。
<br/>
<br/>        自我封闭了一段时间后，她也独立完成了体育馆的外形设计，到了交标日期，再次去傅氏集团大厦时，她不再像之前那样有所期待。
<br/>
<br/>        因为她知道，期望越大，失望就越大。
<br/>
<br/>        设计部部长孙莉莉从五十张设计图里，让公司里的员工一起众投，精选了二十五张出来后，命令自己的助理送去总裁办给傅夜沉过目。
<br/>
<br/>        设计图交接的任务，是落以柔接手。
<br/>
<br/>        在设计部部长的助理离开后，落以柔偷偷地揭开了密封线内的封条，找到童筱颖的那张设计图后，将童筱颖的设计图给抽了出来。
<br/>
<br/>        傅夜沉过目后，从二十四张设计图中选了十五张出来，然后让落以柔反馈给设计部部长孙莉莉进行挂网，由网民投票，票数最高的作为体育馆的最终外形设计。
<br/>
<br/>        童筱颖在网上看到那十五张设计图里没有自己的设计图后，心里失落不已。
<br/>
<br/>        而这个时候，傅夜沉忽然想起了什么，于是起身去了设计部。
<br/>
<br/>        傅大总裁几乎从来不来设计部，这次亲自来访，让集中办公室里的男男女女一个个窃窃私语起来，特别是女同胞们，扯着脖子也要一睹这顶级上司的俊容。
<br/>
<br/>        “这次的体育馆外形设计图里面有没有童筱颖的？”走进设计部部长办公室，傅夜沉未打招呼，直接开门见山地问。
<br/>
<br/>        孙莉莉立即从办公桌前起身，颔首道：“有！入围了前二十五名，只可惜没入围您精选的十五名。”
<br/>
<br/>        “她设计的是哪张？调出来给我看看。”傅夜沉接着说道。
<br/>
<br/>        孙莉莉点了下头，拿起一旁的平板电脑，将童筱颖的那张设计图的电子版给调了出来。
<br/>
<br/>        傅夜沉看完后，不容置喙地命令道：“把童筱颖的这张也挂网上去。”
<br/>
<br/>        “好的！”孙莉莉会心一笑，立即安排助理，在票选网上的末尾加上了童筱颖的那张设计图。
<br/>
<br/>        “你刚刚说，你提交上来的是多少张入选设计图？”傅夜沉又问道。
<br/>
<br/>        孙莉莉怔了一下，才回答道：“是二十五张。”顿了顿后，她又弱弱地问，“傅总，是有什么问题吗？”
<br/>
<br/>        傅夜沉并未回答孙莉莉的话，而是直接离开了部长办公室。
<br/>
<br/>        回到总裁办的时候，落以柔正在给他倒红茶。
<br/>
<br/>        落以柔看到傅夜沉进来后，每次都是笑脸迎上，并在没有外人的情况下，亲昵地唤他一声：“大哥。”
<br/>
<br/>        傅夜沉松了松领口上的领带，在老板椅上落座，端起红茶抿了一口后，一边拿起桌上的文件，一边淡淡地说道：“你上次没被调走，是筱颖替你求的请。”
<br/>
<br/>        “……”落以柔不明白傅夜沉为什么会突然开口提起这件事情，顿了顿后，才讷讷地开口，垂下眼帘，笑着说，“是这样啊！那大哥替我谢谢夫人。”
<br/>
<br/>        “从今以后，不管是公私场合，请对我用上司的称呼。此外，我是筱颖一个人的大哥。”
<br/>
<br/>        蓦地，落以柔明眸微抬，怔望着面前一脸冷严的傅夜沉。
