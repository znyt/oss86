　　虚云殿，顾名思义也是一座宫殿，此殿和赤龙营将士所住的石殿略有不同。
<br/>　　此殿同样是一座石殿，但建造的更加的高大，坐落在赤龙营的最后方。
<br/>　　殿门敞开，门内由一片虚幻的水幕，苏莫见此心中恍然，看来这个所谓的虚云殿，是一片阵法构建的空间。
<br/>　　唰唰唰！
<br/>　　祝英、赖羽和苏莫三人，径直穿过了水幕，走进了虚云殿之中。
<br/>　　眼前场景变换，苏莫抬眼看去，自己来到了一片虚空之中，这片虚空极大，方圆不知几许。
<br/>　　虚空中除了有数十位赤龙营的将士之外，别无它物。
<br/>　　见到苏莫和赖羽、祝英三人到来，所有人都望了过来，这些人均是听闻了苏莫要挑战赖羽，前来观战之人。
<br/>　　“参见中队长！”众人纷纷向祝英行礼。
<br/>　　“诸位将士免礼！”祝英随意的回来挥手，随即看向苏莫和赖羽。
<br/>　　“上官昊，来吧！”赖羽大喝一声，随即身形一闪，如同一道赤色的闪电，瞬息间飞到了数千里之外站定。
<br/>　　苏莫虚空踏步，跟着对方走了过去，来到了距对方百余里之处。
<br/>　　所有都凝神观看，这上官昊以武帝境八重的修为，敢挑战赖羽，想必有极大的依仗，这一战估计很有看头。
<br/>　　“到底是不是苏莫，我一看便知！”恒飞小眼睛微眯了下来，心中暗想。
<br/>　　“上官昊，我赖羽做二十九队的小队长一年有余，期间挑战我的人有三位，每一人修为都比你高，但无一成功，你认为你能有希望！”赖羽冷笑着说道。
<br/>　　“懒的跟你废话！”苏莫面色淡漠，心念一动，便催动了梦魔战魂，一股淡薄的紫色雾气升腾而起，瞬间就将赖羽笼罩在了其内。
<br/>　　下一刻，赖羽的面色顿时凝固了，仿佛石化了一般。
<br/>　　“怎么回事？”
<br/>　　“这是什么？”
<br/>　　“是战魂吗？还是幻术？”
<br/>　　众人见此顿时惊讶，但是，紧紧是一个呼吸的时间，众人立刻瞪大了眼眸。
<br/>　　因为赖羽的身躯，仿佛是一具尸体一般，笔直的倒了下去，向虚空下方跌落而去。
<br/>　　这是什么情况？
<br/>　　这就完事了？
<br/>　　赖羽败了？
<br/>　　所有人都是一脸懵逼，一时间个个目瞪口呆，被这种意想不到的情况惊呆了。
<br/>　　赖羽能成为小队长，便可见其实力不弱，最起码在同阶武者之中，也算是难得的高手。
<br/>　　但是，其在上官昊的手上，居然连出手的机会都没有，就败了，而且还败的如此不明不白！
<br/>　　恒飞脸皮一抽，他还想看看这上官昊到底是不是苏莫呢！结果两人都没有出手，战斗就已经结束了！
<br/>　　“中队长，我胜了！”苏莫看向祝英，朗声说道，对于瞬间击败赖羽，他没有丝毫的意外。
<br/>　　随着修为的提升，他的各种能力也都在提升，梦魔战魂对付一个武帝境九重的武者，还是丝毫没有问题的。
<br/>　　当然，这也只限于正常的武帝境九重武者，若是姜神夜那种武帝境九重武者，就另当别论了。
<br/>　　“这……！”
<br/>　　祝英回过神来，一时间竟然不知道该说什么，少倾之后，他深吸了一口气，阴沉着脸道：“苏莫，你战胜了赖羽，从今天起便是二十九小队的小队长。”
<br/>　　“多谢中队长！”
<br/>　　苏莫向祝英微微抱拳，得到了小队长的职务，他就可以自由离开营地了，随即，他继续道：“中队长，若是没有什么事，那我就先回去了。”
<br/>　　“嗯！”祝英摆了摆了手。
<br/>　　随后，苏莫便自顾自的通过出口，离开了虚云殿。
<br/>　　苏莫走后，祝英的脸色彻底铁青了下来，他扫了一眼那些一脸懵逼的士兵，冷喝道：“还傻楞着干什么，还不把赖羽带上来！”
<br/>　　众人闻言，顿时吓的一抖，而后纷纷向下方飞去，将半死不活的赖羽抬了上来。
<br/>　　祝英仔细的检查了一番赖羽，脸色越发的难看了起来，此时的赖羽虽然没有外伤，但两眼翻白，好像精神受到了重大的创伤。
<br/>　　这种创伤最难恢复，没有几个月的时间，根本不可能复原。
<br/>　　……
<br/>　　苏莫离开虚云殿之后，便径直离开了赤龙营。
<br/>　　什么小队长职务？
<br/>　　他根本丝毫不在乎，现在他只想炼制分身，而后离开古灵星河。
<br/>　　出了赤龙营之后，苏莫便脱下了身上的赤色铠甲，不然穿着这一身铠甲，走到外面太扎眼了。
<br/>　　“该去哪里寻找肉身呢？”走在圣皇城的街道上，苏莫有些犯难了，难道随便找个人来杀。
<br/>　　但这不太符合他做人的原则，再说了，修为太低的人，他也看不上。
<br/>　　苏莫心中沉思，盘算着有什么仇人，适合炼化成分身，但他目前除了巫族之外，貌似也没有什么仇人了。
<br/>　　“尤天韩倒是合适！”苏莫心中暗道，尤天韩无论是修为还是天赋，都非常适合做他的分身，最主要的是此人也算是他的仇人。
<br/>　　只不过，对方身在古周星神武学府。
<br/>　　“就去古周星吧！”片刻之后，苏莫决定再回古周星。
<br/>　　就算不选择尤天韩做分身，他也要去古周星。
<br/>　　他临走之前要回苍穹神宫一趟，毕竟这一次走，是离开古灵星河，去往另外一个星河，不知道要多久才能回来，而他并不会将苍穹神宫的人带走。
<br/>　　所以他要回去一趟，交代一些事情，再给苍穹神宫留下一具分身，一是守护苍穹神宫，二是为苍穹神宫收集资源。
<br/>　　还有就是古天意，苏莫不傻，古天意救出聂乾和韩盈之后，不会将两人放了，肯定会带回皇祖圣朝。
<br/>　　因为对方知道，可以用这两人，牵制住他。
<br/>　　不然，他随时可以脱离皇祖圣朝。
<br/>　　古天意很明显非常看重他的天赋，绝对不会允许他脱离皇祖圣朝，会将他束缚在皇祖圣朝，为其效力。
<br/>　　所以，苏莫准备再炼制一具分身，到时候留在皇祖圣朝，安抚住古天意，而他的本体，就能没有任何的顾虑，去明虚星河寻找青璇了。
<br/>　　一切的一切，苏莫很快便有了详细的计划，而后他立刻行动了起来，前往传送之地，向古周星而去。
