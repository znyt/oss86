　　“若是成功了，时空至尊就能超脱纪元？”苏莫惊讶的问道，因为，据他所知，只要能离开荒界，去到其它界，貌似就能超脱纪元。
<br/>　　因为，纪元的终结，便是整个荒界的毁灭，荒界之内的任何生灵，无论是普通的凡人，还是无上的至强者，统统要随着荒界的毁灭而毁灭。
<br/>　　所以，只要离开了荒界，到了其它大世界，那么基本上就不会死了。
<br/>　　“不，若是成功了，所有人都能超脱纪元，因为但凡实力不错的人，都能通过时空之城离开。”终极神王摇了摇头道。
<br/>　　“哦！”苏莫闻言，顿时震惊，这真的是一个伟大的壮举，可惜时空至尊失败了。
<br/>　　“前辈，这时空之城，神王之下的人，根本就走不通吧?”苏莫面带疑惑的问道，不说空门之中，就是时门之中的三十三重时光神殿，想要走到头，都是完全不可能的事情，最起码神王之下的人，是完全不可能走到头的。
<br/>　　“时空至尊，万古人杰，拥有时空元神，将时间法则和空间法则，都修炼到了极致，他当初，炼制了时空之城，并配合上他的无上法则之力，联通无尽的虚空。”
<br/>　　终极神王仰望长空，声音悠远，并未给苏莫解惑，继续道：“你若是走到第三十三重时光神殿，其实，早已进入了无尽的虚空深处，第三十三重时光神殿距离神域大地的路程，以你的速度，需要飞行几百万年。”
<br/>　　“啊！这么远？”苏莫闻言咂舌，第三十三重时光神殿，距离神域大地的路程，居然如此的遥远，以他的实力都要飞行几百万年，这未免太夸张了吧。
<br/>　　“事不可为，时空至尊耗尽了生命本源，也没能打通到其它大世界的通道，最后黯然陨落，而留下的时空之城，算是给予后来者方便了，神域的强者可以通过时空之城离开，去找寻其它大世界。”
<br/>　　终极神王并未隐瞒苏莫，这在神域的顶级强者圈子里，也不算是什么秘密，少倾，他继续道：“或许是不想让后来者送死，亦或者时空之城秉承了时空至尊的意志，高级神王之下的人，根本到不了第三十三重神殿。”
<br/>　　“原来如此！”苏莫闻言，心中恍然，这时空至尊也是一个伟大的人物，他虽然失败了，但却是给后人铺下了一半的路。
<br/>　　“那其他至强者呢?比如混沌神帝？洪荒祖龙？”苏莫再次的问道，对于此事，他自然极为关心，毕竟，听说荒界的纪元毁灭已经不远了。
<br/>　　“老一辈的太古至强者，大多都是早已离去，进入了无尽的虚空深处，在找寻离开的路，在很久很久岁月以前，他们还偶尔会回来，但是，现如今，他们都已经太久没有回来了，至于是多久，本座都记不清了！”
<br/>　　终极神王的面上，露出落寞之色，道：“或许他们已经到达了其它大世界，或者他们已经陨落，亦或者他们彻底的迷失在了无尽的虚空之中，找不到回来的路了。”
<br/>　　苏莫闻言，沉默不语，这些太古至强者，哪一位不是盖世的巨擘，哪一位不是镇压一个时代的存在！
<br/>　　但是，他们依旧要为命运挣扎，即便他们已经在荒界达到巅峰，依旧很难挣脱命运的束缚。
<br/>　　“除了那些老一辈的太古至强者，神域之中，后来还陆续出现了十几位至强者，有人是在时空至尊之前离开的，有人是在时空至尊陨落之后成就至强者，然后通过时空之城离开的，这些人现如今，都是下落不明，生死不知！”终极神王继续说道。
<br/>　　“下落不明，生死不知！”苏莫低声自喃，具体情况如何，他很想知道，但是，估计终极神王也不清楚。
<br/>　　“前辈，您离开过吗？”苏莫沉声问道。
<br/>　　“我怎么可能会离开？这是找死！”终极神王摇了摇头，不到至强者境界，出去等于是送死。
<br/>　　无尽的虚空深处，一片虚无，没有灵气，没有方向，没有资源，什么都没有，连一众至强者努力了这么久，都找不到路，别说是他了。
<br/>　　随即，终极神王转头凝视着苏莫，郑重的道：“若是我得到元神之后，能突破到至强者之境，我肯定会离开。”
<br/>　　“你会如愿以偿的！”苏莫微微一笑，随即便不再多问，继续向玉虚宗而去。
<br/>　　关于这件事，现在他不必过多的了解，待他打败帝一魂和拓跋青云之后，再行了解也不迟。
<br/>　　“如此最好！”
<br/>　　终极神王点了点头，随即大步跟上，若是苏莫敢食言，他一定将苏莫碎尸万段。
<br/>　　很快，两人便来到了玉虚宗的山门之前，终极神王防止被人认出来，和苏莫一样改变了容貌和气息。
<br/>　　他倒是不担心危险，而是被人认出来的话，会有损他的威名。
<br/>　　“来者何人？”
<br/>　　玉虚宗的山门之前，伫立着四名守山弟子，为首的一人是一位虎背熊腰的壮硕青年，修为高达四品低级。
<br/>　　见到苏莫和终极神王来到山门前，他立刻上前一步，拦在了苏莫和终极神王的神前。
<br/>　　“在下求见玉虚子前辈？”苏莫向壮硕青年抱拳，郑重的说道。
<br/>　　“求见宗主？”壮硕青年闻言，仔细的打量了一番苏莫和终极神王，在他的感知之下，苏莫的修为是四品高级，一览无余，而终极神王的修为，仿佛是用秘法遮挡了，无法窥探。
<br/>　　“你是何人，有何事求见宗主？”壮硕青年声音洪亮的问道，作为守山之人，他自然不能随便放入进入山门。
<br/>　　“在下有要事和玉虚宗前辈相谈，你通传一声便可！”苏莫笑着说道。
<br/>　　“报上名讳，以及有何要事？”壮硕青年并没有立刻通报，而是追根问底。
<br/>　　苏莫见此，不由得眉头一皱，这人居然如此追问，可真是恪尽职守。
<br/>　　“不必和他废话！”终极神王冷漠的说道，随即大袖一挥，一股无形的力量卷出，壮硕青年和另外几名守山弟子，顿时身躯僵硬，仿佛是石化了一般。
<br/>　　“这……！”
<br/>　　苏莫见此，有些无语，不过，既然已经这样了，他也不会再多言，随即便大步的走进了玉虚宗。
<br/>　　“嗯？有些不对！”
<br/>　　刚刚走进山门，苏莫便眉头一皱，因为他感觉到了明显的不对劲。
<br/>　　这偌大的宗门，山门之处居然如此的清静，充满了诡异的感觉。
<br/>　　要知道，这个规模的宗门势力，虽然远不如终极神王殿或者人王宫，但最少也有十几万的弟子，山门之处应该人来人往，热闹非凡才对。
<br/>　　而现在，这山门之处，居然非常的清静，放言望去，看不到半个人影。
