　　“隐藏的好深！”
<br/>　　苏莫眉头紧皱，随即再次开口问道：“你们黄泉魔宗，为何要如此隐藏实力？”
<br/>　　“这我就不清楚了！”阴霆摇了摇头。
<br/>　　噗嗤！
<br/>　　阴霆话音刚落，一道寒光陡然闪过，他的一条手臂便立刻被斩了下来。
<br/>　　啊！
<br/>　　阴霆口中顿时发出一声惨叫，剧烈的疼痛让他脸色惨白，脸上渗出一颗颗豆大的汗珠。
<br/>　　“说！”苏莫面色冷漠，不下点狠手，对方根本不会如实招来。
<br/>　　“你……！”
<br/>　　阴霆心中怒急，恨不得立刻和苏莫拼命，但他还是勉强压制住了心中的怒火，因为他清楚自己没有半分胜算。
<br/>　　只有顺着苏莫，方有活命的机会。
<br/>　　“我真的不知道！”阴霆咬着牙，再次摇头。
<br/>　　“看来不吃点苦头，你根本不会老实了！”苏莫冷笑一声，缓缓抬起了手中的十亿魔剑，一副要下狠手的模样。
<br/>　　“这……！”阴霆见此，顿时心神剧颤，吓的面无人色。
<br/>　　“我说！我什么都说！”眼见苏莫要动手，阴霆顿时疾呼了起来，不敢再有丝毫的隐瞒。
<br/>　　“这才对嘛！说说你们黄泉魔宗都有什么目的？”苏莫停止了出手，面露微笑道。
<br/>　　“我们黄泉圣宗是准备一统苍穹世界，所以一直在隐藏实力！”阴霆沉声道。
<br/>　　“一统苍穹世界？”
<br/>　　苏莫闻言眼眸微眯，黄泉魔宗的胃口倒是不小，还想一统苍穹世界！
<br/>　　不过黄泉魔宗虽强，但想一统苍穹世界还是没有什么可能！
<br/>　　“那你们黄泉魔宗准备什么时候行动？”苏莫再次问道。
<br/>　　“已经有所行动了，目前已经有两个大势力愿意合作，支持我们！只等时机成熟，便会动手！”
<br/>　　阴霆沉声道，他倒是知无不言，对于他来说，现在保住自身的性命才是最重要的。
<br/>　　“哪两个势力？”苏莫皱眉问道，若是有其它超级势力帮助黄泉魔宗，一统苍穹世界还真的有可能！
<br/>　　他若是知道是那两个势力，必定先灭了这两个势力。
<br/>　　“这个我真的不知道，以我在圣宗的地位，根本不可能知晓的太清楚！”阴霆摇了摇头，面带苦涩。
<br/>　　“真的不知道？”
<br/>　　苏莫眯着双眼，眸中寒光闪动，道：“若是你敢隐瞒，你会后悔来到这个世上！”
<br/>　　“这个我真的不知道，你就是杀了我也没有用！”阴霆大声说道，他的确不知道，这些隐秘凭他的地位，根本不可能知道。
<br/>　　他的师尊，阴罗殿主也不会告诉他。
<br/>　　苏莫见此，心中沉吟，或许对方真的不清楚。
<br/>　　“那你可知道巫山抓的那名女子在何处？”苏莫又问起了宏青璇，上次从皇甫擎的口中，他没有逼问出结果，希望能从阴霆口中得到消息。
<br/>　　“不清楚，从未见过，也未听过什么消息！”阴霆摇了摇头。
<br/>　　苏莫闻言心中暗叹，宏青璇到底怎么样了？这是他目前最为担心的事情。
<br/>　　“好了！你可以死了！”
<br/>　　再次叹了口气，苏莫不准备在多问了，猛然出手，十亿魔剑如同死神之刃，瞬间斩在了阴霆的脖颈上。
<br/>　　噗嗤！
<br/>　　剑光闪过，鲜血喷涌，冲天而起，阴霆的脑袋亦是高高抛起。
<br/>　　“你不是说会放我？”
<br/>　　脑袋被斩下，阴霆还未彻底死亡，口中传出了不甘的声音，因为之前苏莫说过斟酌考虑。
<br/>　　“我已经斟酌考虑了，本想将你灭杀成渣，现在只斩下你的脑袋。”苏莫淡漠的说道。
<br/>　　“卑鄙！”阴霆的口中突然怒喝一声，随即让脑袋跌落而下，便彻底没有了动静。
<br/>　　至于阴霆的身躯，血液尽数流进十亿魔剑之中，苏莫更是直接挖出了对方体内的神丹。
<br/>　　这一段时间，苏莫诛灭了十几个黄泉魔宗的据点，武王境的神丹，他已经得到数千枚，武皇境的神丹也得到了几枚。
<br/>　　不过，这些对于他来说还远远不够。
<br/>　　他身上的修炼资源已经耗尽，必须多多收集，越多越好。
<br/>　　少倾，苏莫身形一闪，便离开了此地，继续灭杀黄泉魔宗的据点。
<br/>　　半个月之后，他又诛灭了三个黄泉魔宗的据点，来到了中洲的南部地域。
<br/>　　此地，已经距帝玄宫很近了。
<br/>　　略一沉吟，苏莫便向帝玄宫飞去，已经很久没有看到夕儿了，他准备去看看对方。
<br/>　　而且，他诛灭了这么多黄泉魔宗的据点，若是在杀下去，估计那二魔老又要来了。
<br/>　　他现在实力不足，还不宜正面对抗。
<br/>　　两日之后，苏莫便来到了帝玄宫。
<br/>　　帝玄宫还是往常的模样，但是当苏莫身形飞进山门之时，整个帝玄宫外宫为之震动。
<br/>　　“苏莫回来了！”
<br/>　　“苏莫居然回来了！”
<br/>　　“听说苏莫已经有武皇境九重的实力了！”
<br/>　　“苏莫应该是来提亲的吧？”
<br/>　　无数帝玄宫外宫之人惊呼出声，现今的苏莫，无论走到哪里，都足够吸引眼球。
<br/>　　这不仅是因为苏莫的实力，更因为他现在苍穹神宫宫主的身份。
<br/>　　苏莫进入帝玄宫之后，便直接进入了内宫之中，而后向夕儿的居所飞去。
<br/>　　夕儿居住在一座奢华的宫殿中，这宫殿仿佛用白玉建造，其上覆盖着金色的琉璃瓦，霎是美丽。
<br/>　　咯吱！
<br/>　　苏莫身形刚刚落在宫殿之前，宫殿的大门便应声打开了，一袭蓝色长裙的夕儿款款的走了出来。
<br/>　　“夕儿！”苏莫轻唤了一声。
<br/>　　“苏莫哥哥，你终于回来了！”夕儿俏脸上带着如水般柔和的笑容。
<br/>　　“嗯！”苏莫重重的点了点头，面上亦是露出了温柔的笑意。
<br/>　　“苏莫哥哥，苍穹神宫出世，我还没有恭喜你呢！”夕儿笑道。
<br/>　　“没有什么值得恭喜的，苍穹神宫也是应运而生！”
<br/>　　苏莫摇了摇头，笑道：“我们出去走走吧！”
<br/>　　“好！”夕儿乖巧的点了点头，随即两人一起，便在内宫中走动了起来，不多时两人便走到了山巅之上。
<br/>　　山巅之上，云雾缭绕，雾气翻滚，苏莫和夕儿相对而立，互相述说着这一段时间的经历。
<br/>　　其实，主要是苏莫在说，夕儿静静的聆听。
<br/>　　当然，遭到黄泉魔宗之人追杀这种事情，苏莫自然不会说，免得让夕儿担心。
