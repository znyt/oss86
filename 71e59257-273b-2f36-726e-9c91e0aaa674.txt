　　此刻，虽然会武没有进行到最后，但第一名已经是板上钉丁的事情了。
<br/>　　道宗之人个个面带喜色，另外六大势力的人，则是脸色非常难看，这一次会武，风头全部被道宗抢去了。
<br/>　　战台之上，苏莫身上仿佛有无形的光芒，整个人都成为了这方天地的中心，耀眼无比，吸引着所有人的目光。
<br/>　　苏莫转头，遥遥看向北方看台上的云悠悠，发现对方也在望着他，立刻向对方微微一笑，然后便飘然返回了看台，显得潇洒无比。
<br/>　　云悠悠见此，也是嫣然一笑，她倒是对这个苏莫产生了一丝好奇，这真谜一般的人物，名不见经传，但天赋实力，却是秒杀所有妖孽。
<br/>　　“悠悠，这个苏莫肯定对你有意思！”灵萱见到苏莫的举动，轻声说道。
<br/>　　“灵萱姐，你怎么知道？”云悠悠好奇的问道。
<br/>　　“这还用问吗？悠悠你的情商真是太低了！”
<br/>　　灵萱翻了翻白眼，继续道：“你没看他一直在看你吗？明显就是有意思！”
<br/>　　“或许吧！”云悠悠抿了抿红/.唇，不置可否的说道，感情之事对于她来说，是不会考虑的事情。
<br/>　　另一边，苏莫回到看台之后，所有道宗的弟子和长老，全部都是肃然起敬，就连真枯半圣都是如此。
<br/>　　“苏莫，这一次你得到会武的第一，为我道宗争光，壮我道宗的神威，回到道宗之后，我定然请示宗内，给予你打的奖励。”真枯半圣满脸笑意的说道。
<br/>　　真枯半圣非常高兴，会武第一非同小可，这不单单是苏莫的荣誉，也是道宗的荣誉，能让道宗在太阴族和六大势力之面前抬起头颅。
<br/>　　以往，道宗弟子连连败北，宗门地位都受到了很大的影响，现在真是一朝崛起，以后谁还敢轻视道宗？
<br/>　　“多谢真枯长老！”苏莫向对方微微抱拳，既然有奖励，他当然不会拒绝。
<br/>　　“待你回到宗门之后，定然会给你一个惊喜！”真枯半圣眸中笑意连连。
<br/>　　“惊喜？”苏莫闻言一怔，不过他也没有太放在心上，以他目前的情况，已经没有什么奖励能让他惊喜了。
<br/>　　“那我就等着真枯长老的惊喜了！”苏莫微笑着说道。
<br/>　　接下来的时间中，苏莫和真枯长老闲聊，冯光也不时的插上一两句。
<br/>　　而第四轮的会武，在片刻之后，便又继续开始了，一场连着一场，仿若永无止境一般。
<br/>　　而苏莫由于连得四分，接下来的几轮挑战他均是没有上场，因为他的分数远远超越其他人。
<br/>　　直到第八轮之时，苏莫才再次开始挑战，随便选择了一名对手，而对方直接认输了。
<br/>　　第九轮也是如此，对手再次认输。
<br/>　　第十轮同样如此，没有人愿意和苏莫交手。
<br/>　　十轮之后，整个会武彻底结束了，名次也是非常的明朗。
<br/>　　苏莫，毫无意外的得到了第一名。
<br/>　　而第二名则是卓剑豪，第三名灵枫，第四名扬天成，第五名炎澜天，第六名幽狼，第七名雷云皓。
<br/>　　剩下的第八名、第九名和第十名，则分别是闪灵族、黑炎族和雷云禽族的三名天才。
<br/>　　卓剑豪和灵枫等六人，虽然败给了苏莫，但可不会败给其他人。
<br/>　　六人到得最后，也是互相较量了一番，才得出最终的排名。
<br/>　　此刻，阴泽长老站了起来，目光环视全场，朗声道：“这一届的七大势力会武，圆满举行，接下来一个月的时间，你们十人可以免费在九重浮屠塔修炼，一个月后，前往外族的族务府领取其它的奖励！”
<br/>　　“是！”苏莫等十人纷纷应声。
<br/>　　随即，阴泽长老手臂一挥，十枚黑色玉牌飞射而出，飞向苏莫等十人。
<br/>　　“这是九重浮屠塔的通行令，可让你们在塔内免费修炼！”阴泽长老说道。
<br/>　　“谢阴泽长老！”十人接住玉佩，纷纷抱拳道谢。
<br/>　　“好，现在会武结束了，大家各自返回！”
<br/>　　阴泽长老再次朗喝一声，便身形一闪，向出口处飞去。
<br/>　　其他太阴内族之人，亦是纷纷跟随，云悠悠也是跟着阴泽长老，向外飞去。
<br/>　　飞行之中，云悠悠发现苏莫一直在望着她，让她心中有一丝慌乱，但她并未理会，径直的离开了武斗场所在的空间。
<br/>　　苏莫见此暗叹一声，想要接近青璇，或者是让其恢复记忆，怕是还有很远的路要走。
<br/>　　“呵呵！苏莫，我们也走吧！”阴泽长老心情大好，招呼苏莫一声，便也带领着道宗众人，离开了武斗场所在的空间。
<br/>　　随着会武的结束，和所有观战之人的离开，会武的结果也在太阴城中传播了开来。
<br/>　　庞大无比的太阴城，顿时一片哗然，议论纷纷，苏莫之名彻底的响彻在了太阴城。
<br/>　　“听说这次会武道宗出现了一个妖孽！”
<br/>　　“是啊！据说仅仅是武帝境八重的修为，战力却极度逆天，横扫六大势力的妖孽天才，强势夺得了会武的第一！”
<br/>　　“太强了，不知这个叫苏莫的人，是道宗在哪里招揽的妖孽？”
<br/>　　此种议论之声，响彻在太阴城的大街小巷，无处不在。
<br/>　　而苏莫离开武斗场之后，便和道宗众人分开了，他要去九重浮屠塔修炼，而道宗之人不可能等他，便先行离开了太阴城，返回道宗。
<br/>　　临分开之前，真枯半圣再三叮嘱了苏莫，让他在太阴族小心谨慎，不得闹出什么乱子。
<br/>　　苏莫独自而行，没过多久便再次来到了九重浮屠塔，直接走了进去。
<br/>　　向负责管理九重浮屠塔太阴族强者出示了黑色玉牌，苏莫畅通无阻，一路来到了第九层。
<br/>　　此时，浮屠塔的第九层空间内，炎澜天、扬天成等九名天才，已然早他一步到来，均是在盘膝修炼。
<br/>　　苏莫默默走到一边，便也盘膝而坐，闭上眼眸开始了修炼。
<br/>　　不远处，扬天成微微睁开了眼眸，阴冷的目光望着苏莫，眸中寒光闪烁。
<br/>　　扬天成心中，对于苏莫恨之入骨，后者不仅狠狠的折磨了他一顿，还让他在神女面前丢近了脸面，这个仇不可不报。
<br/>　　只不过，他的实力不如苏莫，想要报仇，还要从长计议。
