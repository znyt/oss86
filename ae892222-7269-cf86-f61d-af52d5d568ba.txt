　　碧清冷哼一声，“那就让你见识一下这大阵真正的威力！”
<br/>　　真正的威力？那是什么？
<br/>　　叶少阳倒不怕碧清亲自跳出来杀了他们。之前的交锋，让叶少阳认清了碧清的实力，硬干的话，的确是有一打五团灭他们的可能，不过如果他们死命逃走的话，她绝对没法把人全留下。
<br/>　　她想留下所有人，并且垂涎他们身上的罡气，想要借助五行杀阵将他们杀死，然后吸取他们的罡气，转化成自身的修为……
<br/>　　如果她现在厌倦了这么做，在五行杀阵已经困住他们的情况下，自己动手，将他们一个个解决掉，应该容易的多……难道她是想亲自上阵了？
<br/>　　叶少阳正在为自己这个想法紧张，突然，身边的“五角星”一阵收缩，重现变回成五道灵光，悬在空中，然后龙气所化的白雾更加浓烈，将众人裹在中间，什么也看不见，只有这五点灵光，像灯光一样悬在半空，忽明忽暗……
<br/>　　“怎么办？”有人问。
<br/>　　妙心劝说先等等看，不要慌张。
<br/>　　大概过了有十几秒钟，雾气散去一些，露出了藏在白雾里的东西，是一栋虚幻的楼宇……叶少阳四下观察，不知道怎么样形容，他们这些人站在一个巨大的建筑物的回廊上，四周都是墙，前方有八道门悬在半空中，灵光环绕，缓缓地旋转着。
<br/>　　“这是什么？”卢晓清吸着气问道。
<br/>　　妙心观察片刻，说道：“应该是某种阵法造成的幻象……”
<br/>　　“蜃景……这应该是蜃景。”梧桐喃喃说道。
<br/>　　大伙听了这话，心里都是一紧。
<br/>　　蜃景，就是类似海市蜃楼的一种，只不过是法术界对于某种能够幻变出虚景的一种阵法的说法，能够产生蜃景的阵法，都是极为玄妙而灵力极强的阵法，所虚化出来的景物，不管多么宏大复杂，身在阵法之中，想要出去，都要依据阵法的规律，寻找出路，想要硬闯肯定是不行的。
<br/>　　每一个阵法产生的蜃景，想要堪破，都有不二法门，凶险到了极点，基本上可以说是九死一生。
<br/>　　大伙静静地打量着八道门，发现这些门一模一样，只是门上有一些灵光组成的三条横线，却是有长有短，大伙对这个再熟悉不过了，这是八卦的八种卦象，有三条实线，代表“乾”，为天，三条虚线，代表“坤”，为地，其余还有各种排列方式，分别代表坎、兑、震、离、艮、巽几种卦象。
<br/>　　八道门，分别有一个对应的卦象，随着不断旋转，门上的卦象也在不断变幻。
<br/>　　大伙傻傻地看着眼前这一幕，不知道是什么情况。
<br/>　　“这八门遁甲，有死无生，我要看着你们一个个怎么死！”碧清的声音再度传来，带着一丝得意。
<br/>　　大伙面面相觑。
<br/>　　说话的工夫，周围那八道门不断旋转，离他们越来越近，一旦八门合围，阵法即成，就算他们有再强的本事，也是一个死。
<br/>　　大伙都很清楚这一点，看着八门迫近，心中也是不由紧张起来。
<br/>　　“怎么办？”陈晓宇叫起来。但是没有人回答。眼下这局面，真的有种山穷水尽的感觉。
<br/>　　“八门之中，有生有死，现在唯一的办法，就是找出生门的所在，钻进去，就可以破阵……”
<br/>　　叶少阳话刚落音，陈晓宇就叫起来：“别扯淡了，这里不止你一个人懂，八门遁甲，****相乘，生门早就被封锁了，怎么可能会存在！”
<br/>　　叶少阳不想理他，但是妙心也问了类似的问题。
<br/>　　叶少阳看着众人，缓缓说道：“你说，她既然法力强过我们，为什么不直接灭了我们，反而搞出这些名堂？”
<br/>　　大伙互相看了看，梧桐说道：“因为想要借助阵法，杀死我们之后，炼化我们体内的罡气？”
<br/>　　“那现在她已经用阵法困住我们了，为什么还不动手，按说从之前我们破阵开始，直到现在，她都有机会偷袭我们，前后夹攻之下，我们必死无疑，为什么她不这么做？”
<br/>　　大伙沉默了一会，毛小方说道：“她想看着我们一个个被阵法杀死，看个热闹？”
<br/>　　叶少阳摇摇头，“只要我们不死，变数就是有的。她不会放心的。所以你这想法不对。”
<br/>　　“那是为什么？”
<br/>　　没等叶少阳开口，梧桐抢先说道：“她分不出精力动手！”
<br/>　　叶少阳赞许地看了她一眼，点头说道：“她自己也在这阵法之中，主持着阵法，所以没有工夫对我们动手，只能借助阵法的力量杀死我们。”
<br/>　　“那又怎么样！”陈晓宇叫起来。“她在不在阵法里，跟我们有什么关系！”
<br/>　　“所以说，你是人头猪脑。”
<br/>　　“你——”陈晓宇刚要还嘴，妙心叫道：“你别插嘴！”
<br/>　　随即转头望着叶少阳，说道：“我明白你的意思了，这阵法之中，必然有一处是安全的，不然的话，就算是她自己，也不可能在阵法中存活下来。”
<br/>　　叶少阳点头说道：“所以不要听她忽悠，这八门之中，必然有一道门是通往生门的，我是说至少，只要进入生门，我们就能破阵而出。”
<br/>　　不管阵法再强大，只要能出去，那阵法就等于废了。这道理当然谁都懂。
<br/>　　大伙沉吟起来，一边望着不断迫近的八道玄门。
<br/>　　“这八门遁甲，毫无规律可言，似乎无解……我找不到生门所在，难道我们要碰运气吗？”妙心喃喃道。
<br/>　　地师一脉，虽然与道佛二宗不同，但风水术中，也是以五行八卦为基础的，妙心对于这方面的理解，可以说比卢晓清等人还要强不少，连她都摸不到头绪，其余人自然更是没办法了。
<br/>　　叶少阳受托下巴，观察着八道旋转的玄门，门上不断变幻的八卦符文，是它唯一的规律。
<br/>　　叶少阳先牢记每扇门移动的方位，和上面的卦象，将卦象一一记在心里……
<br/>　　“你的确很聪明……”碧清的声音再度响起，“这八门遁甲，六十四种卦象变化之中，的确有一道隐藏的生门，但其中经略玄妙，却不是人力所能推演得出，除非，你不是人，我等着看！”
