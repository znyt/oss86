　　昆吾老祖一剑生世界，九天剑界之中，将元始紧紧包围，整个剑界化为攻击，全部攻向元始。
<br/>　　试想一下，一个世界的力量全部用来攻击一个人，这是何等的威力，更何况这个世界还是由攻伐第一的剑道所组成的。
<br/>　　这与元始的灭世印很像，都是用一个世界来攻击一人。但又不同于灭世印，灭世印是用一个世界的毁灭终结来结束这一切，而九天剑界是用世界内的剑道来攻击敌人。
<br/>　　“咦？！一剑生世界！有意思！那也来尝尝本座这一招。”元始有些惊异这昆吾老祖能够做到一剑生世界的地步，显然这是剑道走到了极致，不过，他也并没有慌张，一剑生世界是很强，但还没有强到可以无惧一切。
<br/>　　“四象印！”
<br/>　　元始手捏印法，四象神印，黑白赤青四色神光闪现，分别化为四道神兽身影，窜出神印。
<br/>　　朱雀、白虎、玄武、青龙！
<br/>　　四象神兽，分别飞向九天剑界的四个方向，朱雀飞舞，飞向南方，神焰冲天，焚天裂地，火光耀天。白虎奔越，纵向西方，金戈杀伐，口吐金气，虎啸震世，面对漫天剑气，貌不犹豫，一口吞下，虎威割裂天地。
<br/>　　玄武来到南方，抬头向天，呈擎天之势，脚镇四海，破涛翻滚，大浪滔天，毁天灭地，成灭世之象。青龙腾起，飞向东方，瞬间涨得巨大，龙威凌天，口吐龙息，飞龙在天，破天碎地。
<br/>　　四象神印演化四大神兽，大道加持，万道轰鸣，天地剧烈动荡，地水火风涌动，九天剑界摇摇欲坠，灭世之威，震动九天。
<br/>　　“御剑——破界！”
<br/>　　只听一声暴喝，昆吾老祖，剑锋所指，剑势瞬变，剑光凛凛，原本就摇摇欲坠的九天剑界顿时破碎成空，灭世之威，化为巨剑亘古横空，强烈的毁灭之道，剑道之中，毁灭剑气肆掠散发，将周围山谷毁灭一空。
<br/>　　剑光比九天之上的星辰还要耀眼，星辰摇曳，风起云涌，日月失色，乾坤破灭，煌煌之威，化为毁天灭地的风暴，卷向元始。
<br/>　　这一剑，似乎可以横断苍穹，毁天灭地，仿佛没有什么可以阻挡这一剑！
<br/>　　“灭世印！”
<br/>　　元始无悲无喜，又是一印轰出。
<br/>　　四象神兽再次聚集在一起，化为一个巨大的磨盘，将毁灭风暴尽数笼罩。
<br/>　　磨盘搅动，将毁灭风暴吞噬，一切具皆绞碎，巨大的磨盘流出瀑布般的混沌破灭之气，转瞬间，昆吾老祖的剑刃毁灭风暴就具皆被灭世磨盘磨灭，而且，磨盘在吞噬完毁灭风暴后，并未停下来，而是化作一道流光，飞向昆吾老祖，要将他毁灭。
<br/>　　“可恶！”
<br/>　　昆吾老祖暗骂一声，不得不提起十二分的精神来对待，面对向他飞来的灭世磨盘，他感受到了强烈的危机感，一个不小心，他就会被毁灭。
<br/>　　昆吾老祖丝毫不敢放松，感受到元始的强大之处，他心里不对不承认，这是一个劲敌，一点也不比他弱。但，他也并没有太过担心，因为他手中还有一件灵宝，一件他至今还未参悟透的灵宝。
<br/>　　一枚灰色的石印被他祭出，石印中是黑白两道光芒。这两道黑白之光就如同两个突然炸开的宇宙一般，狂暴的能量与死亡的气息轰向灭世磨盘。
<br/>　　这才是他的杀手锏，这是他未转劫之前，身为混沌魔神，在混沌之中的一处混沌风暴中捡到的灵宝生死石印。内含生死两大大道法则，生死轮转，宇宙幻灭。
<br/>　　这生死石印的黑白光芒并没有直接斩向灭世磨盘，而是形成一个巨大的黑白混沌世界，一面生，一面死，将灭世磨盘与元始一同牢牢锁在其中。
<br/>　　一道恐怖的浩瀚宇宙气息从生死石印的黑白世界暴炸，这爆炸的目标正是被锁在其中的元始与灭世磨盘。
<br/>　　感受到其中浓浓的死亡危机，元始知道，自己不是不如昆吾老祖，而是被他用生死石印暗算了。现在一个不小心，哪怕他不陨落，也会暂时失去战斗能力。在此地失去战斗能力，其实和陨落并没有太大区别。
<br/>　　元始冷静下来，想到了过去百万年中始终不能成功的一道印法，但此时已经顾不得那么多了。他抬手接连使出各种印法。
<br/>　　五行印，阴阳印，四象印，元心印，不周印，开天印，灭世印，诛神印……
<br/>　　各印按某种玄妙的规则相加，万回三，三回二，二回一，一归道，合而为一，种种大道，尽皆归一，万道相融，万法相合，形成最为初始的一印。
<br/>　　元始知道，自己成功了。
<br/>　　“道一印！”
<br/>　　万道归一，形成最初的道！
<br/>　　周围突然安静下来了，空间仿佛停滞，时间流速也在减缓。
<br/>　　各种大道形成的道一印，成为一个点，就仿佛天雷勾动地火，又仿佛各种大道产生了某种特殊的反应。在汇聚在一点的刹那，元始隐隐约约看到了一片虚无的世界……在那虚无的世界之中，有着一个缓缓滚动旋转的巨大球体……
<br/>　　无尽虚空，不管是时间，还是空间。
<br/>　　在这旋转的巨大球体上，这一切都在汇聚，万物汇聚于一点。
<br/>　　这一点，是一切的起点，也是一切的终点。
<br/>　　这一点，就是道。
<br/>　　元始各种大道形成的大印交织在一起，也汇聚了模糊的“一点”，而后贯穿一切！
<br/>　　……
<br/>　　昆吾老祖还在疯狂挥动生死石印笼罩元始，正持剑碾压过去，面对元始那疯狂击来的一点，他大笑着，无比自信眼前此人肯定挡不住。不是因为别的，就因为他对生死石印的威力无比自信。到现在为止，他还没有遇到一个能够在生死石印的攻击下活下来的人，不管是谁，都挡不住生死石印的威力。
<br/>　　忽然——
<br/>　　轰出道一印的元始，竟然整个人突然消失了！
<br/>　　凭空消失的无影无踪！
<br/>　　昆吾老祖愣住了，不见了？怎么会不见了，这怎么可能？他简直不敢置信。
<br/>　　就在昆吾老祖愣住的一瞬间，元始又凭空出现在昆吾老祖正前方不足一米之处！那道一印形成的“一点”，更是出现在前方距昆吾老祖额头仅仅一寸之外。
<br/>　　“怎么可能？”昆吾老祖惊骇，全身战栗。
<br/>　　他怎么可能在生死石印的笼罩之下忽然消失？
<br/>　　又穿过生死石印的封锁，穿过一段空间，突然出现在自己面前？这难道是空间法则？这简直难以置信！
<br/>　　只是那一点距离自己如此之近，自己想挡都挡不住啊！
<br/>　　“噗嗤！”
