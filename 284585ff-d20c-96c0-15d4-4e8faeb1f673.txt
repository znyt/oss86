　　第二天叶帝他们去比赛的时候，到了现场依旧是会听到有人在那里议论着他，就是连他的那些粉丝都认为他的这首歌曲创作的毫无意义。
<br/>　　不过这边叶帝却是没有理会这些，依旧是开心的和自家的几名队友在那里交谈着，而这一幕被那些直播间里的观众看到，在有心人的带动下，倒是成了叶帝膨胀了表现。
<br/>　　“膨胀了，特么的出了这么破的一首歌居然不道个歉，现在还在那里有说有笑的。”
<br/>　　“楼上的你sb吧，大魔王出的歌曲你特么爱听不听，不听就滚好么，道什么歉？谁都有瓶颈的时候，我相信大魔王肯定能成功的。”
<br/>　　“凭什么不道歉啊，这是欺骗我们的感情啊，他这首歌受到了这么多抨击，就算是不道歉也应该出面解释一下吧。”
<br/>　　“别的不说，单单就说昨天大魔王跳的那个舞还是很惊艳的，我很多学舞蹈的朋友现在都崇拜死大魔王了，都说这是他自创的。”
<br/>　　......
<br/>　　直播间里的观众疯狂的议论着，只不过这一切叶帝却是看不到了，不过相信就算是他看到了，估计也不会在意这些的，是金子总是会发光的，叶帝心中始终坚信着这首歌曲肯定会大火的。
<br/>　　今天是他们和RG的比赛，RG在国内的实力倒是也非常的强劲，能够排在第三梯队的前列，只不过是他们这几支战队表现的太亮眼了，所以才一直没有打响名头。
<br/>　　时间到达3点15的时候，他们的比赛才正式开始，之所以开的有些晚，完全是因为上一场的比赛耽误了一些时间。
<br/>　　“不知道今天大魔王会拿出什么英雄，自从大魔王直播拿出冥火之拥那件装备后，现在这件装备基本上已经成为ap刺客中单的首选装备了。”米粒大声的开口说道。
<br/>　　“没错，因为这件装备的出现，原本非ban必选的机械先驱和沙漠皇帝这两个英雄最近的选取率也是有些下降，相反狐狸这种基本上没上过场的英雄，倒是又流行了起来，大魔王真的是时时刻刻都在引领着游戏的节奏啊。”林雅楠跟着开口说道。
<br/>　　“对了，你最近听到一个消息没，据说咱们夏季赛的决赛开幕式请到了国际巨星叶芷澜到现场表演。”米粒有些兴奋的说道。
<br/>　　“怎么可能不知道，要知道那位可还是咱们这位大魔王的亲妹妹呢，短短一年的时间从一个在校学生闯到国际巨星，这个真的是太厉害了，她和她的哥哥崛起的都太过迅速了。”林雅楠笑着说道。
<br/>　　而直播间里的众人听到这个消息的时候，原本还在那里带着叶帝节奏的弹幕瞬间为之一遍，全部刷起了666.
<br/>　　要知道自从叶芷澜的名声闯入到国际社会后，她最近这半年的时间基本上都是在国外开演唱会，而且名气一天比一天的火爆，加上她那甜美的外表，现在她已经是被评为全球影响力最大的一个巨星了。
<br/>　　这些人骤然听到叶芷澜要回到国内，而且还担任夏季赛开幕式的嘉宾，自然是兴奋的不得了，而这也是米粒和林雅楠两人想要的结果。
<br/>　　对于直播间里的弹幕他们是能够看到的，两人自然是不想让叶帝承担那么大的压力，所以才悄然的引领了一波节奏。
<br/>　　不过如果他们知道叶帝非常自信，甚至不在意这点事情的话，不知道他们会作何感想。
<br/>　　场上双方轮流ban人，RG战队这边ban的都是叶帝他们这边非常擅长的英雄，而叶帝他们这边倒是ban掉了几个版本比较热门的英雄，并没有针对对方ban人。
<br/>　　这倒不是说叶帝他们就瞧不起对方了，如果真要是针对ban，那么恐怕RG这样的战队就更不是他们的对手了，如今给他们留一手，也是叶帝几人在给对方机会，希望他们能够在游戏中成长，在游戏中进步。
<br/>　　“嚯！大魔王居然拿出了卡萨丁，这个英雄虽然qw两个技能在被砍了一刀之后，已经很少有人在赛场上使用了，当然了这个英雄之所以被砍，也是源于当初大魔王在春季赛上那一场震惊四座的单人偷家，没想到今天大魔王再次拿出了这个英雄。”看到叶帝选择了卡萨丁，解说席上的米粒眼睛一亮，顿时大声的喊了起来。
<br/>　　“这一场可是有意思了，大魔王这边选择了卡萨丁，而对面则是选择了小鱼人这样的中单，在卡萨丁这个英雄被削弱之后，如果单论对线能力的话，其实这个英雄是次于小鱼人的，现在就看双方选手的个人能力了，看的出来RG这边的中单是想要单杀大魔王的。”看到RG中单的选人，林雅楠跟着开口说道。
<br/>　　而随着比赛的开始，大家也是再次将目光放到了游戏之中，虽然有很多人在黑叶帝昨天创作的那首歌曲，但是对于叶帝的游戏水平，却是没人能够黑的起来的。
<br/>　　如果他们赶在英雄联盟这个领域黑叶帝的话，那估计那人纯粹是在找骂，可以毫不客气的说，如果真的有人在媒体上黑叶帝的游戏水平，起码有八层的职业选手会站出来为叶帝说话。
<br/>　　而且这还是世界范围内的，叶帝对英雄联盟做出的贡献，这一点火神都是承认的，正是在叶帝一个个套路和英雄的出现后，才有了火神那一刀接着一刀的砍人。
<br/>　　很快双方阵容便是确定了下来，最终白队这边选择了：上单鳄鱼、打野盲僧、中路卡萨丁、下路则是女警和扇子妈。
<br/>　　而RG战队这边则是选择了：上单巨魔、打野男枪、中路小鱼人、下路则是ez和锤石这两个英雄。
<br/>　　度过了等待时间，在现场无数观众的欢呼声中，荣耀白队的这一场比赛终于是正式开始。
<br/>　　只不过让这些人没有想到的是，这一场游戏才刚刚开始，就出现了非常让人震惊的一幕，在毫无征兆的情况下，叶帝居然送出了自己的第一个人头，而且这个人头还是被对方的小鱼人给拿了下来。
