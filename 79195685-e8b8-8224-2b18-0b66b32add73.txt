        &lt;h3&gt;第三章第四十五章  三姐到西城&lt;/h3&gt;
<br/>
<br/>        三姐陈子澜到了西城，三姐出差顺路去看三姐夫，也联系了陈子霖，想一起吃顿饭。
<br/>
<br/>        三姐并不知道陈子昂回到了西城。
<br/>
<br/>        潘青松调都西城有两年了，陈子澜一直在原单位混着，不是她不想上进，她现在也算是一个领导了。
<br/>
<br/>        之前四妹有给她说，托关系找人将她也调到西城，作为陈子澜来说，在那里工作都一样，她已经习惯了她的这个工作岗位，要不是上面的领导强行提拔她，她宁愿一辈子做她的教导员。
<br/>
<br/>        还有泉城离家里近，可以照顾到父母亲，父母亲年龄都大了，他们都走了，有点事情身边没个人不行，除了值班，她隔一个礼拜就要回去看看父亲和母亲，陈子澜乐此不彼的跑着单位和家庭之间。
<br/>
<br/>        她就像是一个全能王，将单位和家庭都照顾得很好。
<br/>
<br/>        看着被她训练出来的年轻人，一个比一个能干，她充满了自豪。每天和年轻人一起松松筋骨，打打闹闹，她感觉她一直是十八岁。
<br/>
<br/>        她也算是他们业界的一杆旗帜，身手了得，一个女人都做教导员这么长时间，不得不说是一个奇迹，他们的团队在洛市那是响当当的，几个县的比武大会，他们年年都是第一，这十几年来无人超越，他们单位和她家里的奖状都糊了一面墙。
<br/>
<br/>        低调的三姐从不居功自傲，她一如既往地的和同事们一起执勤，泉城的治安在他们的管辖范围内一年比一年好。
<br/>
<br/>        这次单位出差，刚好逢周末，顺道出来看看潘青松，和弟弟也见一面，他们兄妹七八个，现在都是各有各家，大家忙着赚钱升职，一年都难得见上几面。
<br/>
<br/>        陈子霖正在开车，接到了姐姐的电话
<br/>
<br/>        “子昂，你接一个电话。”
<br/>
<br/>        “三姐。”
<br/>
<br/>        “是子昂吗？”
<br/>
<br/>        “是我呢，三姐，你在哪里？”
<br/>
<br/>        “子昂，你回西城了？”
<br/>
<br/>        “是的，我现在在回西城的路上。”
<br/>
<br/>        “我也在西城呢，子昂下午我们一起吃饭吧，我叫上你姐夫，我们都好久没有见面了。”三姐喜悦的声音传过来。
<br/>
<br/>        “好，那我们下午见，我们回去大概还得2个小时。”
<br/>
<br/>        “行，我看看你五姐有没有时间，我也叫上。”
<br/>
<br/>        “五姐这几天好像也在西城呢。”陈子昂记得五姐这几天在西城开分店。
<br/>
<br/>        “那就太好了，子昂，我挂了啊，我给你五姐打电话。”
<br/>
<br/>        陈子澜挂了电话，给潘青松打电话，她这会事情已经办完了，坐在警/车上，这几天的西城比泉城要热很多，陈子澜将窗户全开，她舍不得开空调，最近的油价一直在上涨。
<br/>
<br/>        虽说是单位上会报销油费，但是他们单位的经费不多，他们的工资也不是很高，干着一份危险的工作，拿着低工资。再说，他们经常执勤，天气热、凉对他们来说已经习以为常了。
<br/>
<br/>        “青松，你下班了吗？”
<br/>
<br/>        “快了，老婆，你在哪里？”
<br/>
<br/>        “我今天出差，刚好到西城，刚刚给子霖打电话，子霖说子昂回来了，你下午有没有时间，我们一起吃顿饭？”
<br/>
<br/>        “老婆大人来了，我怎么会没有时间呢，老婆，你每次来出差都不提前见我，我这颗小心脏很疼，求安慰。”潘青松在电话那头嗲着声音说。
<br/>
<br/>        陈子澜噗嗤笑出了声，潘青松永远都是一副调皮的模样，长不大。
<br/>
<br/>        “你太忙，再说，我这不是想给你一个惊喜嘛。”
<br/>
<br/>        “哪我俩先惊喜一会，再见弟弟妹妹们？”
<br/>
<br/>        陈子澜耳根一热，潘青松放个屁，不用闻，她都知道是香的还是臭的。
<br/>
<br/>        夫妻这么多年，他们还和结婚的时候一般恩爱，别人说的什么三年之痒、七年之痛，在他们这里统统都不存在。
<br/>
<br/>        “老婆、老婆！”潘青松继续腻歪着。
<br/>
<br/>        “今天不是周五吗？我这周在西城过。”平时都是潘青松来来回回的跑着，坐火车得5个小时。
<br/>
<br/>        “嗯嘛，老婆，i  love  you!”
<br/>
<br/>        “青松，我给子悦联系一下，你看看二哥？”
<br/>
<br/>        “好，我来联系二哥。”
<br/>
<br/>        陈子澜挂了电话，心里满满的都是幸福与甜密。
<br/>
<br/>        潘青松是西城人，父母亲在泉城上班，当时是在他们单位实习，她也是到单位不久，实习结束后，原本潘青松要回西城的，可是潘青松因为她的原因留在他们单位，这一留就是好多年，他们是志趣相投，俩个人就工作和家庭上的问题都一致，潘青松回了西城，她留在泉城，别人都说两地分居不好，相反，他们俩个人因为不在一起，反而更加珍惜彼此在一起的时间。
<br/>
<br/>        陈子澜收回了思绪，给五妹陈子悦打电话，陈子悦这几年也是发展得太快了，分店开得愈来愈多，原来从一个小县城已经开到省城了。
<br/>
<br/>        “子悦，你在西城？”
<br/>
<br/>        “三姐呀，我在西城呢。”陈子悦那边有些嘈杂。
<br/>
<br/>        “子悦，下午有时间吗？子昂回来了，我们下午一起吃顿饭。”陈子澜很希望大家能在一起坐坐。
<br/>
<br/>        “子昂回来了？她都不和我联系。”陈子悦似乎有些失落。
<br/>
<br/>        “也没有和我联系，我给子霖打电话，他们俩个在一起。”其实他们家几个小的年龄相差不大，一个就比一个差了三四岁，反而是他们几个大的，和几个小的年龄相差很大，大姐的儿子都和子昂一样大。
<br/>
<br/>        也不知道爸爸妈妈怎么想的，生下他们四个大的，中间隔了好几年，后来就掉葫芦一样连着几年又生了四个小的。孩子少，家里当然能顾得过来，孩子一多，家里就捉襟见肘了，尤其是二哥娶二嫂的时候，基本将家里的东西都给了二哥，家里一下子就穷了。
<br/>
<br/>        二哥结婚的时候，她和老四都在上大学，几个小的也都在上学，父母亲供着他们读书已经是很吃力了，到了几个小的，家里有时候真正是穷得揭不开锅。尤其是子昂上学的时候，家里又增添人口。
<br/>
<br/>        她也刚刚结婚不久，家里条件不好，没有照顾到几个弟弟和妹妹，尤其是六弟，到现在都不见人。
<br/>
<br/>        陈子悦听说子昂回来了，当然是要见一见了，她此时正在和闫欢一起看着超市开业的盛况。
<br/>
<br/>        “三姐，那下午见，我来找个地方吧。”
<br/>
<br/>        “不用，我一会让你姐夫联系一个吃饭的地方，我给你地址，你们过来就是。”
<br/>
<br/>        “好的，三姐，那辛苦你了啊，我这会还有点忙，等会见。”
<br/>
<br/>        陈子澜打完电话，松了一口气，她现在开车去潘青松单位，这会事情已经办完了，她过去等着他下班。
