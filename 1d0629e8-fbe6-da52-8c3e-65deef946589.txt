        童筱颖画完后，拿起纸张举到了冷逸峰的眼前，高兴道：“我都不知道，我还有这样的特技哦！”
<br/>
<br/>        “你最擅长的就是画画。”冷逸峰看着童筱颖，看出了神，就像是在对他的那个女孩说，“你还说，你这一辈子都不会画我。因为，一旦画上了，就会从成为回忆。”
<br/>
<br/>        “我可没对你说过这样的话！”童筱颖冷不丁地反驳。
<br/>
<br/>        冷逸峰不禁黯然神伤地垂下了眼帘。
<br/>
<br/>        另一边，季雅珊跟着傅夜沉一起离开咖啡厅后，迫不及待地追上傅夜沉的脚步，询问道：“傅大哥，那个女孩真的是筱颖吗？她真的没有死吗？”
<br/>
<br/>        “我希望你日后离筱颖远一点，她不需要你这样的朋友。”傅夜沉正眼都不看季雅珊一下，冷言冷语道。
<br/>
<br/>        季雅珊心知肚明，这半年的时间里，傅夜沉恨她入骨。
<br/>
<br/>        可是，当时她有得选吗？
<br/>
<br/>        她压根就没得选。
<br/>
<br/>        一个是自己的救命恩人，一个是自己的深爱着人。
<br/>
<br/>        两个人一起去救筱颖的话，相当于去送死。
<br/>
<br/>        如果牺牲一个人，能救两个人的话，她为什么不牺牲筱颖？哪怕筱颖是她这一辈子最真挚的朋友。
<br/>
<br/>        季雅珊从来都不认为自己的选择是错的，如果还有第二次，她选择的结果都会是一样。
<br/>
<br/>        “你放心，我会把筱颖永远记在心里，不会去打扰她。”季雅珊给了傅夜沉一个答复。
<br/>
<br/>        咖啡厅里，童筱颖有些难以置信地看着十爷冷逸峰。
<br/>
<br/>        他居然把她的所有喜好和特长说得准确无误。
<br/>
<br/>        而同时，冷逸峰对童筱颖的身份起了疑心。
<br/>
<br/>        他为婴宁立了墓碑，却不用婴宁的真名，是因为婴宁的尸首被神秘人带走了。
<br/>
<br/>        想到这里，冷逸峰忽然问道：“你后背，左边的蝴蝶骨上是不是有一颗痣？”
<br/>
<br/>        “有吗？我不知道诶！”童筱颖不知情地摇了摇头。
<br/>
<br/>        “介不介意让我看看？”冷逸峰微微蹙起了眉头。
<br/>
<br/>        童筱颖二话不说地拒绝道：“我的身子只有大哥才可以看。如果你想知道的话，你可以去问大哥。”
<br/>
<br/>        “你爱他？”冷逸峰突然间心痛了一下。
<br/>
<br/>        就好像曾经的那段时间，他逼问婴宁一样：“你是不是还爱着他？”
<br/>
<br/>        他不是婴宁的初恋，但当他爱上婴宁的时候，婴宁当时的心里确实还装着另外一个男人。
<br/>
<br/>        “对，我爱他。”童筱颖坦然地微笑道，顿了顿后，又补充了一句，“跟你一起来的那个女孩，她也爱你。”
<br/>
<br/>        “我知道。”冷逸峰苦笑。
<br/>
<br/>        “我觉得吧！我真的不是你口中的那个女孩。”童筱颖忽然一脸严肃地再三强调。
<br/>
<br/>        冷逸峰回过神来，应了声：“我们去找他们。”
<br/>
<br/>        “好！”童筱颖顿时眉开眼笑了起来。
<br/>
<br/>        他们两离开咖啡厅走出去跟傅夜沉和季雅珊汇合的时候，冷逸峰问出了他心里的那个疑惑。
<br/>
<br/>        傅夜沉也给了他一个肯定的答案：“筱颖左边的蝴蝶骨上确实有一颗痣。”
<br/>
<br/>        他知道，连童筱颖她自己都不知道的事情。
<br/>
<br/>        也对，当初婴宁也不知道自己后背左边的蝴蝶骨上有颗痣，还是他冷逸峰告诉她的。
<br/>
<br/>        这意味着什么？
<br/>
<br/>        “你放心，我不会动，也不会抢你的女人。我爱婴宁，但是我还没有失去理智。”冷逸峰心里坦荡地对傅夜沉允诺。
<br/>
<br/>        “那我带筱颖先回家了。”傅夜沉接着说道。
<br/>
<br/>        冷逸峰点了点头，忽然间想起了什么事情，便问道：“五帝门现在在临海城已经把势力统一了，你要不要考虑沿着海岸线往滨江城这边发展？”
<br/>
<br/>        “五帝门的事情你做主即可，我还是那句话。五帝门，是我们用来保护我们想要保护的人的力量。”傅夜沉淡定道。
<br/>
<br/>        冷逸峰朝傅夜沉走进了一些，附到傅夜沉的耳边，细语低沉地说：“我介意你，再查查这个筱颖的来历。她所有喜好和特长，以及言行举止，都和婴宁一模一样，包括后背左边蝴蝶骨上的那颗痣。”
<br/>
<br/>        “如果，我跟你说，我不打算去追究这件事情呢？”傅夜沉微微蹙起了眉头。
<br/>
<br/>        连华生也提过要深查筱颖的事情，如果要查，很费时间。
<br/>
<br/>        但其实，是他不想再去面对那些查出来的意想不到的结果。
<br/>
<br/>        因为，现在，他只是觉得，结果对他和筱颖来说已经不重要。
<br/>
<br/>        “算我求你，帮我这个忙。”冷逸峰语重心长地说。
<br/>
<br/>        实际上，是他想查婴宁的事情。
<br/>
<br/>        但是他的能力有限。
<br/>
<br/>        而傅夜沉却不一样。
<br/>
<br/>        他有这个钱财和这个时间去查这件事情。
