　　异种王就站在一群拥护他的人群中，他身披厚重的斗篷，将外貌遮挡的严严实实，浑身散发着不可亵渎的高冷气息，让那些对他好奇或崇拜的猎者们，再如何激动也不敢靠太近。
<br/>　　场面略有些混乱的持续了一段时间，江凌轩更是一刻不停的紧盯着，以防出现意外。
<br/>　　只是王之暴君并没有做出任何出格举动，只是逛了一圈后，就自顾自的来到高级猎者的休息室，而后关上了门。
<br/>　　江凌轩止步在门外，望着这间屋子陷入沉思。
<br/>　　“总管？”旁边的一名工作人员快步走来，小声道：“要不要开启秘密摄像头？”
<br/>　　猎者联盟总部的任何一个角落，都安装了最全面的监控设备，也是这两年猎者联盟的发家相当猛，赚的钱多到数不清，才能让总部如此挥霍。
<br/>　　这一点一开始还让楚涵颇为嫉妒和不满，但考虑到猎者联盟的特殊性，以及其内某些猎者的身份危险程度，也就任由猎王折腾。
<br/>　　而当下这一并没有公开的设备，终于派上了用场。
<br/>　　“最高权限，开启全面监控。”江凌轩下令。
<br/>　　“是！”
<br/>　　江凌轩没有离开原地，而是依旧守在这处休息室外，这里只有一道门，王之暴君要离开必然会经过他。
<br/>　　而最高权限的全面监控开启，则是需要一定的时间，这段时间里江凌轩会寸步不离的守在这里。
<br/>　　时间一分一秒的过去，这间休息室没有任何动静传来，这让江凌轩有些焦虑，他无法从门外得知内部的任何情况，因为在核心地带的所有房间，都有隔音效果。
<br/>　　“可别出什么意外……”江凌轩内心紧张的祈祷。
<br/>　　他虽然不知道这个王之暴君的具体身份，但根据猎王透露的信息，加之此人显露出来的战力，并不难锁定。
<br/>　　无非就是异种！
<br/>　　一个异种，竟然混进了人类阵营，甚至还是猎者联盟中的高级猎者，拥有一大批崇拜者，这无论如何都让人接受不了。
<br/>　　尤其是江凌轩很清楚，这个王之暴君很有可能，就是异种王！
<br/>　　正当江凌轩分析思考中，离开很长一段时间的一名工作人员忽然快步跑来，面带焦急。
<br/>　　“总管！”他气喘吁吁的来到江凌轩身旁，小声汇报道：“果然有问题，监控内显示这间休息室没有人！”
<br/>　　“什么？！”江凌轩大惊，随后二话不说猛然用力的一脚踹在了门上。
<br/>　　嘭！
<br/>　　一声巨响，爆发在这处安静的走廊上。
<br/>　　休息室的门应声而倒，随着腾起的灰尘散去，屋内的场景呈现而出。
<br/>　　正如监控内所显示的那般，王之暴君所在的这间屋子，早已空无一人，不仅如此，上方的天花板还被砸出了一个洞！
<br/>　　王之暴君从这里离开了！
<br/>　　工作人员被江凌轩的这一脚惊到，站在原地目瞪口呆的望着这一幕，要知道这核心地带的所有休息室，那防御力都相当惊人，却挡不住江凌轩的一脚？
<br/>　　这战力，也太恐怖了吧！
<br/>　　如此厉害的家伙，竟然待在总部常年不出去，甘心当一名幕后总管？
<br/>　　就在工作人员的呆愣中，江凌轩已经从屋内走出，他黑着脸快速下令：“封锁总部！调动最高权限的全面监控，开启所有摄像头，一旦发现王之暴君的位置立即汇报！”
<br/>　　“是！”工作人员连忙回神照办。
<br/>　　江凌轩也立即前往监控室，在满墙壁的大屏幕中，目光不断移动的寻找王之暴君的身影，同时他心中更是腾起了无数疑问，与紧张焦虑的心情一起。
<br/>　　王之暴君的目的究竟是什么？为何要避开他们的视线？这里还有什么他不知道的被遗漏？猎王又为何在明知可能有诈的情况下依然离开？
<br/>　　诡异的情况，让江凌轩万分紧张，甚至有些不知道如何应对。
<br/>　　但他很清楚，他不能坐以待毙，这个猎者联盟里不仅有他，还有他的亲弟弟江凌天！
<br/>　　江凌轩之所以放弃很多优势，心甘情愿的在这里当一个管事，原因就是江凌天不知道中了什么邪，非要在这里当个猎者，还每次都选一些很危险的任务。
<br/>　　思绪运转只是一瞬间，一眨眼内江凌轩就已经扫过了很多处地方的监控，可是完全找不到王之暴君的所在，同时监控上的画面，各处地方都一切正常。
<br/>　　到底在哪？
<br/>　　还有哪里漏掉了？
<br/>　　其他人也紧急的分工合作，每个人都盯着固定的几个地方反复检查，可当他们细无巨细，将所有可以查看的地带都查了一个遍后，依然不见王之暴君的踪影。
<br/>　　“总管！没有！”一人汇报着，肯定道：“整个总部里里外外，包括中心处的那个冷潭都360度无死角的查看过，还是没有。”
<br/>　　“这怎么可能呢？”另一人满心疑惑：“难道凭空消失？”
<br/>　　“整个总部都没有，他能藏在哪？”
<br/>　　江凌轩也不解，但还是谨慎道：“可能知道有监控，躲在了死角，例如天花板与下层地面之间的空洞，但不管他怎么躲都还是要出来，既然带着目的而来，就不可能永远躲下去，继续盯着！”
<br/>　　“是。”
<br/>　　江凌轩没有在发现天花板上的大洞后就立即追上去，也是这个原因，利用这些通风口道路找人更难，不如打开监控，王之暴君的目的不可能在那些毫无用处的地方。
<br/>　　只是在接下来的时间里，时钟的指针转了一圈又一圈，江凌轩额间的冷汗已经密密麻麻，监控上的画面依旧没有任何反常，王之暴君就如同人间蒸发一般，再也没有出来过。
<br/>　　“查一下，有无伤亡。”江凌轩用颤抖的声音，下达命令。
<br/>　　这时候忽的，一名工作人员猛然想起了什么，瞪大双目开口道：“还有一个地方没有查过！”
<br/>　　“哪里？！”江凌轩瞬间扭头，犀利发问：“怎么不早说！”
<br/>　　“我知道了！”另一名工作人员也想到了什么，随后又有些迟疑的开口道：“可是那个地方，没有监控，并且属于禁区。”
<br/>　　一提到禁区，江凌轩也反应了过来，猛地心脏漏跳了半拍后瞬间冲出监控室，大声的命令更是立即吼出：“集合高战力猎者！目的地为长老室！”
<br/>　　当下在禁区内唯一有人的地方，就是何培源所在的长老室！
