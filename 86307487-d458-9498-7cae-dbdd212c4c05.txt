　　布克尔特带着佟千雪进入他在山林中安排的安全屋中，打开的安全屋地下室的门，押着佟千雪进入到地下室中。
<br/>　　“你还是不要负隅顽抗了，你是跑不掉的！”佟千雪挣扎了一下，但是布克尔特锁在她手腕上的塑料扎带收得很紧，根本没有任何可以挣扎的余地。
<br/>　　“跑不掉？”
<br/>　　布克尔特冷哼出声，说道：“虽然他们跟比洛多合作有些出乎我的意料，但是我也有我的底牌，最后到底是谁跑不掉，不到最后一刻又有谁知道呢？告诉你，这个地下室的入口已经用十公分厚的脱氧钢板封死，就算是用塑胶炸药进行定向爆破，短时间内也不可能炸开它。”
<br/>　　布克尔特打开了安全附近的监控探头并且把地下室入口锁死，监控探头把山林中的情况清晰的反馈到地下室那一排挂在墙壁上的液晶屏幕上，布克尔特拿出卫星电话，拨通了一个号码。
<br/>　　这是美国加勒比海基地群司令部的电话，是西伦科特交给布克尔特的，让布克尔特在走投无路的时候寻求帮助。当然西伦科特如此帮助布克尔特也是有他自己的目的，打造“影子部队”需要大量的钱，仅仅是他通过自己手中的权利弄到的那些钱，根本不足以维持“影子部队”的消耗，所以他需要布克尔特贩毒集团在金钱上对自己的支持。
<br/>　　加勒比海基地群赫克托跟西伦科特私交很好，当年赫克托惹上了一个大麻烦，如果不是西伦科特的帮助，赫克托早就被关进了联邦监狱中，根本不可能有他现在加勒比基地群司令官的身份。
<br/>　　电话很快被接通。
<br/>　　“赫克托将军？”布克尔特拿着卫星电话在地下室中走动着，说道：“不知道你还记不记得西伦科特？”
<br/>　　坐在椅子中，正在享用着纯正美式咖啡的赫克托手一抖，杯子中的咖啡泼了出来，落在他的手上，令他倒吸一口冷气，赫克托把咖啡杯放在桌子上，从椅子上站起身来，对电话那头的布克尔特问道：“你是谁！？”
<br/>　　“我叫布克尔特，是西伦科特同父异母的弟弟。”布克尔特看着出现在墙壁上液晶屏幕中的战术中队队员，说道：“我哥哥让我在有需要的时候打这个电话。”
<br/>　　“西伦科特？”赫克托冷笑道：“他自己都已经完蛋了，难道还指望我做些什么吗？”
<br/>　　“看来赫克托将军是忘了当年我哥哥帮你的忙了吗？”布克尔特对赫克托的态度并不感到意外。
<br/>　　人走茶凉，说到底，西伦科特已经死了，知道赫克托当年那件事情的人已经不在了，他自然是不会再想着还西伦科特这个人情。
<br/>　　“事情已经过去了十几年！十几年！”赫克托有些恼怒的说道：“我在这十几年中也帮西伦科特做了很多事情！明白吗？就算他帮过我，我也还清了！”
<br/>　　赫克托到底有没有帮西伦科特做过事情，只有他自己清楚，布克尔特手中没有证据，就算赫克托不愿意承认，布克尔特也拿他毫无办法，不过布克尔特从最开始就没打算用西伦科特这个不知道是不是真的管用的人情让赫克托来帮自己逃出这个国家。
<br/>　　“赫克托将军，你也不用这么激动。”
<br/>　　布克尔特看着液晶屏幕中正朝着安全屋靠拢过来的战术中队，说道：“我给你这个电话，只是想要跟你谈一笔交易。我手中有一支P病毒，一支改进过的P病毒，我想这个东西你们应该很感兴趣。”
<br/>　　“你想要得到什么？”
<br/>　　赫克托被布克尔特所说的话打动了，如果他能得到布克尔特手中这支P病毒，这意味着什么，赫克托太清楚了，绝对会给自己的仕途带来一个质的飞升，他甚至可以直接入驻五角大楼！
<br/>　　“我现在遇到了一点麻烦，被人困在了一处山林中，我要你派特种部队把我救出去。”布克尔特说道：“只要我能安全的离开，我就把我手中的P病毒交给你。”
<br/>　　“坐标？”赫克托问道。
<br/>　　布克尔特说出了自己所在位置的坐标。
<br/>　　“不可能！”赫克托在自己办公桌的电脑上输入了布克尔特所说的坐标后，说道：“距离太远了，就算我让特种部队现在就出发，短时间内也不可能赶到你所在的位置。”
<br/>　　“那就让CiA来！”布克尔特提高声音说道：“CiA中有不少是前特种部队的队员吧？通知他们，让他们过来带我离开！你可以告诉我在这个国家没有CiA的人，我也会相信你的谎话，但前提是你们不想要我手中的P病毒！”
<br/>　　“留在原地，很快就会有人支援你。”
<br/>　　赫克托说完就挂断了电话。
<br/>　　作为世界四大情报组织的CiA，在全世界很多国家都有自己的安全屋、情报人员和行动人员，正如布克尔特所说，CiA有相当一部分的行动队员是来自于特种部队，他们的战斗力绝对不可小觑。
<br/>　　布克尔特放下电话，对佟千雪说道：“现在我们赌一把，看看是救你的特种部队先打开地下室的门，还是支援我的人先到。”
<br/>　　……
<br/>　　罗昊带着战术中队队员很快就找到了布克尔特的安全屋，就算是有树木的遮挡，可是这么大一栋建筑杵在那儿，想要不被发现几乎是不可能的。
<br/>　　罗昊发现了安装在周围树干上的监控探头，他对着一个探头竖起中指，然后拔出手枪，一枪把那个监控探头给打烂。
<br/>　　安全屋地下室中，一块液晶屏幕中立刻遍布雪花，看不到任何画面。
<br/>　　十几名战术中队将布克尔特的安全屋包围起来，陈应天砸开玻璃窗，甩手扔了一颗闪光震撼弹进去。
<br/>　　哐！
<br/>　　白光闪烁而过的瞬间，罗昊一脚把安全屋的门踹开，握着手枪闪身冲进了安全屋中；可是安全屋中除了地上那颗被烧的焦黑，还冒着缕缕青烟的闪光震撼弹外，并没有看到布克尔特和佟千雪的踪影！
<br/>　　“人不在这儿？”司徒空看着空荡荡的安全屋，眨巴了一下眼睛疑惑道。
