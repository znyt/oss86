　　楚涵的突然出现显然让丧尸头领没想到，可以明显的看到它的动作有了一瞬间的僵硬，神情也刹那间变化万千，那森白的瞳孔里映射·出了一种惊讶的情绪。
<br/>　　但哪怕如此，它的前进还在继续，并且也一下子进入战斗状态。
<br/>　　蒋天庆待在不远处的楼层里，在近距离的观察下，他终于看清了这只丧尸头领的全部外貌，这要比在远处用望远镜观察来的更加真切，也更令人心生惊恐，以至于让这名年轻的中将面色都惨白。
<br/>　　丧尸头领虽皮肤灰白与正常的丧尸一样，但它却有头发，并且整个身形都与人类完全一致，每一处细微的地方都完好无损，更别说那透露着和人类一样有着千变万化的神情。
<br/>　　这是蒋天庆第一次见到如此像人类的丧尸！
<br/>　　此时楚涵这一方更早已是最强的进攻状态，他扬起巨大黑斧，在两者高度刚好到达一致的时候，双手握着黑斧狠狠往前一劈。
<br/>　　“爆炎斩！”
<br/>　　轰然一声巨响，带着强大能量波动的修罗战斧，发出了楚涵最强的一击！
<br/>　　那丧尸头领更不是盖的，当即一个轻微闪避躲开了最致命的一击，在半空中强行调整了姿势，双手利爪顺势伸出就抓向楚涵的脖颈。
<br/>　　双方的一攻一反攻不过发生在刹那间，甚至爆炎斩的爆破还未在半空中炸开之际，那丧尸头领的利爪就已经距离楚涵的皮肤不到一毫米！
<br/>　　丧尸头领明明森白的瞳孔，这时候却流露出了一抹残暴和得逞的快·感，它知道眼前这个人类是它唯一需要警惕的对象，只要能杀死就再无威胁。
<br/>　　楚涵的战斗方式一向不退反进，在无数次交战之中哪怕对方已经突脸到随时能取他性命，楚涵也会在最快的速度内反应过来，而后不顾一切的反攻！
<br/>　　防守，向来不是楚涵的作风。
<br/>　　这一点所有狼牙的成员都知道，甚至人类中的很多敌军也知道，连异种王都知道，近乎不是秘密。
<br/>　　眼下这只丧尸头领，好似也知道这一点似得，将速度提升到极致，更将所有心神都放在了楚涵身上，只要它的速度比楚涵快，就可以在对方反攻之前，先将其杀死！
<br/>　　楚涵没有任何停顿，也没有任何例外的手腕一翻，将修罗战斧的反向调整。
<br/>　　这是他战斗中最寻常的动作，无论是反向的调整还是反攻的角度，他都轻车熟路如同一种本能。
<br/>　　此时两者的对战力已经抵消了双方前进的弹跳力，再加之在半空中的两招已过，让他们的高度急速下落，下一瞬间就会落地。
<br/>　　楚涵反攻的动作被丧尸头领看在眼里，让它欣喜的裂开嘴露出一个笑容，紧接着——
<br/>　　啪！
<br/>　　它忽然一个角度翻转，在即将落地的一霎那，赶在楚涵之前一只脚用力一弹，带动整个身躯好似闪现一般，瞬间出现在了楚涵身后侧。
<br/>　　这时候才刚落地的楚涵，却依然是手握修罗战斧向前斩击的姿态，方向角度完全处于最劣势的状况。
<br/>　　他或许来得及回头，也或许来得及在此瞬间调整修罗战斧的角度，但绝对来不及在这只丧尸头领攻击之前挥出战斧！
<br/>　　这是两个过程，除了反应能力更需要时间。
<br/>　　丧尸头领却已经占尽优势，当下面对楚涵的后侧位，想要将其重伤亦或是一击毙命，都轻而易举！
<br/>　　而这只丧尸头领，也确实这么做了……
<br/>　　锋利的利爪冒着寒光，对准楚涵的后颈就猛地抓去，其一瞬间爆发出来的力量、速度和能量强度，不仅能一下子将楚涵的脖子抓断，更能将颈椎骨震荡成粉末！
<br/>　　待在不远处看到这一幕的蒋天庆，一股寒意顿时蔓延整个身躯，他惊恐的瞪大眼睛，想要叫出声却完全喊不出来，因为到达最极限的惊恐，让他丧失了所有协调能力。
<br/>　　刷！
<br/>　　丧尸头领的进攻强势无比，在空中带出了破空声。
<br/>　　但这么一抓过后，令人意外的是却没有任何插入血肉，亦或是碰到物体的声音……
<br/>　　下一秒什么都没有发生，那丧尸头领竟然扑了一个空！
<br/>　　时间好似在这一刻静止，空荡荡的道路上只有丧尸头领站在原地，有些呆滞的看着前方。
<br/>　　那是一个背影……
<br/>　　楚涵毫不犹豫朝着远处狂奔的背影！
<br/>　　观战的蒋天庆也彻底惊呆，这是他此生第一次见到楚涵在对战中的非反攻行动。
<br/>　　逃跑？防守？撤退？
<br/>　　这一定是个假的楚涵吧？！
<br/>　　蒋天庆有些不知道用什么语言来形容，甚至他都不知道自己此刻是什么心情，完全凌·乱了。
<br/>　　丧尸头领也有些没反应过来，还处在原地想不明白，这与它了解的楚涵根本不一样！
<br/>　　就在这两人都被楚涵的反应搞懵的时候，已经在瞬间跑出二十米远的楚涵脚步一顿，紧接着立即——
<br/>　　回身扬斧而上！
<br/>　　楚涵的转身动作流畅无比，修罗战斧轮旋的弧度完美无缺，整个点爆发的一切都如行云流水般精妙绝伦！
<br/>　　如果不是此刻存在着太过于明显的反常情况，楚涵的这个动作恐怕还会被记录为经典……
<br/>　　蒋天庆大脑一个卡壳，这又是哪一出？！
<br/>　　震惊中他再次怀疑这是不是一个假的楚涵，要跑已经够反常了，结果都跑出二十米了还回什么头反什么攻。
<br/>　　正常情况下如果有反攻的想法而假意逃跑，也应当在跑开五米之内三米最佳，一旦超过这个范围就不存在任何优势，更别说楚涵还是朝着完全直线的反向在跑，回头反攻更是没有任何方向改变。
<br/>　　他最擅长的侧位移动呢？
<br/>　　被堪称经典的无死角巨斧轮旋呢？
<br/>　　Z字形反向升龙上寮呢？
<br/>　　无数楚涵最为擅长又漂亮的战斗动作，一个都没有出现！
<br/>　　这与平常一向精于细节又擅长找优势的楚涵，完全不符！
<br/>　　那丧尸头领显然也和蒋天庆一样陷入了怀疑，眼前楚涵这愚蠢到极致的行动，不是在故意给它反应时间和取胜机会吗？
<br/>　　但谁也没想到就在楚涵刚转身的那一霎那——
<br/>　　嘭！
<br/>　　枪声响起。
