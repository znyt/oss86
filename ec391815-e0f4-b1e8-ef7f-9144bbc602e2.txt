　　洛冥拿出了难得一见的耐心，继续边看边逐步分析：“可是同样，楚涵只亲自训练了第一批狼牙战团的成员，在此之后的第二代第三代，以至于到现在扩军到了接近十万的队伍，从挑选到训练再到上战场，楚涵都没管过，完全交给了其他人处理。”
<br/>　　牧司令想了想，道：“第一代狼牙战团成员负责训练新兵，紧接着一代接一代的传递下去，训练模式越来越完善，狼牙战团越来越强。”
<br/>　　“对。”洛冥说着，又翻到了另一页的资料：“如果说前两件事，楚涵是太忙了或者对那些人信任，但这第三、第四件事呢？”
<br/>　　牧司令凑过去一看，忍不住疑惑：“风花雪月城，安罗城？”
<br/>　　洛冥的语气神态越来越玩味，道：“风花雪月城没有城主，是一处纯商业娱乐地带，其内部的管理都交由警戒队，楚涵同样没管过，只是提供了初步建设和发展方案。”
<br/>　　“而安罗城就更绝了！”洛冥说着，啧啧称奇：“楚涵这货在安罗城正式成立，并且城主上·位的大典那天，甚至都不在狼牙地界！”
<br/>　　牧司令眼神带着微疑：“管理分配给下属来负责，并没有什么问题，楚涵一个人无暇分身兼顾，把尚九娣派到安罗城当城主，基地改建成纯军事基地，这很正常。”
<br/>　　洛冥看着牧司令：“连着五个事件都是如此，并且狼牙内部的情况你也知道，那个军事基地也不是楚涵本人在管理，而是何峰啊！楚涵待在基地的时间根本不多，基地不管，城池不管，战团不管，什么都不管……哦对了还有那些狼牙战团的队伍，杀羽、黑芒、神隐三战队，侦查队、射击队、警戒队、防守队，还有狼牙暗部。”
<br/>　　“所有的一切楚涵似乎都只在开头带了下节奏，然后就撒手不管事，完全的放任姿态。”洛冥的语速忽然快的惊人，滔滔不绝后一顿，拼命摇头：“他这是懒得管，只想无事一身轻啊！”
<br/>　　牧司令愣了愣，有些脑子发懵。
<br/>　　当楚涵第一次进入他视线的时候，他就注意到了这个年轻人，这几年来楚涵的一路走来他都看在眼里，一次又一次的翻盘逆袭，以及无数次的将局面改变，让牧司令坚信楚涵有着异于常人的大局观。
<br/>　　这可不仅仅是战场上的输赢，不单单是看到了楚涵率领狼牙战团无败绩的军事才能，而是此人本身在大局趋势上的可怕敏锐力。
<br/>　　正是这种想法，让牧司令对楚涵尤其关注，而当他和洛冥被楚涵救出，在狼牙基地见到了齿轮般一环扣一环的体系管理，则是让牧司令一下子冒出了让贤的想法。
<br/>　　但当此时此刻洛冥根据楚涵一路发展而来，各大事件的分析结果，却让牧司令忽然察觉到自己可能是在一厢情愿。
<br/>　　显然只有从客观的角度，才能看出楚涵处理每一项事件的习惯，以及似乎可以被定论为套路的发展。
<br/>　　“我忽然想到一件事。”洛冥再次出声，道：“咱们在狼牙基地的时候，因为身份敏感问题，似乎对狼牙当下该由谁做主有过探讨吧？还将这事摆在明面上，好让其他基地的首领都别自乱阵脚。”
<br/>　　牧司令也忽然想起了这么一件事：“不错，那会儿我就说的很清楚，在狼牙一切由楚涵说了算，不用管我等的意见。”
<br/>　　洛冥点头，紧接着指着档案资料上的某一页：“但这件事楚涵那里的态度，你一定不知道。”
<br/>　　牧司令低头望去，顿时瞪大了眼睛：“这真是楚涵说的？这种细节怎么也会被记录下来？”
<br/>　　洛冥将资料翻到最后，拎起来指着最后那页上的奇怪字符，神色诡异：“看到没有，这份档案资料不是普通情报，而是来自神秘家族白家。”
<br/>　　牧司令定了定神，紧接着便是苦笑：“不为人知的细节，也只能是神秘家族才能知道的情报，看来完全是我想太多啊！以楚涵这样的性格，哪怕我硬塞给他，他估计都能当场给我退回来。”
<br/>　　洛冥让牧司令看的那一页资料，正是很久之前当蒋天庆向楚涵请示牧司令和洛冥身在狼牙的特殊情况后，楚涵某次在黑芒那二十人面前暴露的态度。
<br/>　　也是巧合，当时有神秘家族白家的成员在附近，一不小心听了个正着！
<br/>　　楚涵的态度带着异常明显的无所谓风格，好似狼牙谁说了算他压根不关心，只一心想带着黑芒战队疯狂报复。
<br/>　　这哪里是一名合格继承人所该有的态度？
<br/>　　完全的任性和为所欲为，就像脱缰的野马栓都拴不住！
<br/>　　有着方面的才华与能力是一回事，而楚涵自己想不想这么做，喜欢不喜欢却是另一回事。
<br/>　　逍遥自在，不爽就骂，被骂就打，被打就杀……
<br/>　　想让楚涵依照流程依照规矩老老实实的呆着，实在太难！
<br/>　　狼牙一路发展至今到当下这么多的部门、城池、军队，楚涵肯用心在最开始领个头，或许都不过是一个最简单的理由。
<br/>　　他要救父母，却老有人想跟他作对，所以他反抗，而跟他作对的人越来越多，到最后上演到了暗杀和各种陷害，才逼的楚涵不得不壮大自己，才有了狼牙战团，才有了狼牙基地。
<br/>　　牧司令和洛冥双双都被这显然为事实的分析所惊，两人对望了一眼后忍不住笑容古怪，他们在这件事上都各自想法颇多，假设更是拟定过上百种，一切可能都设想到，甚至洛冥还想过楚涵是否会黑化的问题。
<br/>　　但却偏偏没想过楚涵愿不愿意这一最基础的问题！
<br/>　　正当两名老者都忽然感觉到无尽疲惫，也有些惆怅的时候，一名士兵快步走进屋打破沉寂：“段氏基地传来的紧急密报！”
<br/>　　紧急，密报，两个关键词一下子让空气都凝聚了紧张感。
<br/>　　牧司令当即正色起来，洛冥则早已接过了士兵手中的密报打开查看。
<br/>　　牧司令不等洛冥看完就着急出声：“怎么说？是不是南署基地的战事？是不是有关狼牙的？大事件？”
<br/>　　哪知洛冥在看完急报后却神色一呆：“是狼牙的，也是大事件没错……但跟南署基地的大战无关。”
<br/>　　“恩？”牧司令一愣：“眼下跟狼牙有关的大事件除了南署大战，还有什么？”
