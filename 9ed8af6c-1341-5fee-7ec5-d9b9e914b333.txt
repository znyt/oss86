<br/>    陈明看向鬼仙，仔细思考了一会儿，心里顿时有了一个新的主意。
<br/>    
<br/>        作为一个阵法师，要发挥自己的职业优势。
<br/>    
<br/>        俗话说的好啊，每一个提前布阵的阵法师都是耍流氓，像是耍流氓这种事，我们陈仙师自然是轻车熟路了。
<br/>    
<br/>        不过现在最重要的事情，是先确定并不是所有的钥匙都已经落入了鬼仙的手里。
<br/>    
<br/>        陈明打开罗盘：
<br/>    
<br/>        陈明：现在还有几把仙路的钥匙在你们手里？
<br/>    
<br/>        雪山宗宗主：我的钥匙已经被老祖抢走了。
<br/>    
<br/>        紫霞仙子：我爹的钥匙也被抢走了。
<br/>    
<br/>        铁牛谷谷主：我的钥匙也已经被抢走了。
<br/>    
<br/>        剑王阁阁主：你们怎么没有保住钥匙，这可是封印鬼仙的关键，我的钥匙还在.......额......也被抢走了......
<br/>    
<br/>        所以说你们几个到底有什么用？
<br/>    
<br/>        陈明站在高大的青铜大门前，看向远方，“所以说，我现在，根本没有机会离开这里了是吧？”
<br/>    
<br/>        从门缝中传来鬼仙的嚣张地大笑，“哈哈哈哈哈，你已经输了，小老弟！”
<br/>    
<br/>        陈明看向自己地万道天竹，作为一个阵法师，保命的阵法可是随身带着的啊。
<br/>    
<br/>        我，即是阵法！
<br/>    
<br/>        陈明的脸上忽然出现一抹不爽之色，“哎！”
<br/>    
<br/>        鬼仙嚣张道：“怎么了，小老弟，是因为感受到自己的死亡，觉得难受吗？”
<br/>    
<br/>        陈明道：“一般一个人拯救世界的时候，不是应该旁边陪个美女，献上一吻，打开我的小宇宙吗？为毛我现在身旁是你这个丑逼？”
<br/>    
<br/>        鬼仙：......
<br/>    
<br/>        陈明看向旁边的朱雀，然后道：“准备绞杀这里的所有活尸。”
<br/>    
<br/>        朱雀道：“可是鬼仙的这只手臂.....”
<br/>    
<br/>        陈明摆了摆手，然后道：“这只手臂，交给我了。”
<br/>    
<br/>        鬼仙笑了笑，然后道：“慢慢等吧，我会调遣我的所有活尸，前来杀了你。”
<br/>    
<br/>        陈明也是丝毫不惧，“人多有用吗？大阵之下，众生平等？”
<br/>    
<br/>        鬼仙继续道：“至少，我可以慢慢消耗你，让你法力耗尽，我只需要一瞬间的时间，就能打开这扇门。”
<br/>    
<br/>        陈明看向鬼仙，双眼之中露出一抹惊喜的神色，“不得不说，你说的很有道理，开门确实不需要太久，而且，现在只有我一个人，根本没有任何人可以来帮我。”
<br/>    
<br/>        鬼仙道：“你也意识到了你的死亡了吗？绝望，是你死前唯一能够感受到的！”
<br/>    
<br/>        陈明朝着鬼仙的那一条手臂走去，他挥手，破空出鞘，他再次挥手，春秋入手，他将万道天竹插在背后，左手破空，右手春秋。
<br/>    
<br/>        “我这个人，向来不喜欢绝望的感觉！”
<br/>    
<br/>        鬼仙的声音出现了一丝疑惑，“你到底想要做什么？”
<br/>    
<br/>        陈明看向鬼仙，双眼之中跳动着狂暴的神色，“灯光就绪，舞台建好，演员到位，好戏，要开场了！”
<br/>    
<br/>        “杀了你，那么，什么都解决了！”
<br/>    
<br/>        陈明左手掷出破空，飞到门缝之后，而后迅速飞行，然后陈明挥剑，开启。
<br/>    
<br/>        右手之中，春秋朝着鬼仙的左臂挥出，一剑斩下！
<br/>    
<br/>        鬼仙的左臂被一剑斩断，陈明回头，看向朱雀，认真道：“如果我回不来了，告诉我的徒弟们，我爱他们！”
<br/>    
<br/>        陈明踏上一柄国器长剑，然后传送到破空上，他传送过去，断臂的鬼仙愤怒地看向他，而后目光中露出一抹惊喜之色，“阵法师，在你的剑上留下了阵法的阵法师？”
<br/>    
<br/>        “那么，这柄剑，可以传送到大陆之上？”
<br/>    
<br/>        鬼仙已经动了心思，只要他得到这把剑，他就可以直接进入大陆。
<br/>    
<br/>        陈明打了个响指，然后认真道：“你没说完这句话的时候，确实是可以的。”
<br/>    
<br/>        沟通破空的那一柄国器之上，传送阵法崩坏，“但是现在，不行了。真是抱歉。”
<br/>    
<br/>        鬼仙从黑暗之中朝着陈明走来，露出他的面庞，他看起来就像是一个干枯的老头，皮包骨头，双眼之中跳动着魂火。
<br/>    
<br/>        “所以，你是要以凡人之躯，挑战我这个仙人吗？”
<br/>    
<br/>        陈明看向鬼仙，危险指数9900，看来还真是一个可敬的对手。
<br/>    
<br/>        鬼仙看向地面上的断剑，“你的剑已经断了，你是要告诉我，你觉得自己不是我的对手，所以要在钥匙打开这扇门之前给你的弟子争取时间逃走吗？未战先认输，可不是能够获胜的心态，你已经认输了，如何能胜？”
<br/>    
<br/>        陈明笑了笑，然后道：“不，你的思路有问题，正是因为没有后路，我现在只有一条路走，那就是，你死，我活。”
<br/>    
<br/>        鬼仙笑道：“不，你还有另外一条路，你死，我活。”
<br/>    
<br/>        陈明道：“我不喜欢这种可能性。”
<br/>    
<br/>        独臂的鬼仙并没有露出太多的神色，“一千年了，我已经一千年没有动手了，你杀了我两个分身，也是时候给我那两具分身找回一点利息了。”
<br/>    
<br/>        陈明的背后，一千颗星辰逐渐升起，周天星辰大阵，开启。
<br/>    
<br/>        鬼仙抬头，看向头顶的这些星辰，他能够敏锐地感受到这些星辰之中蕴含着大道的力量，而且，那是劫的气息，鬼族，不，应该说所有人都最为害怕的气息。
<br/>    
<br/>        “我真的很好奇，你窃取了天道的部分权柄，为什么还可以好好的活着。”
<br/>    
<br/>        陈明笑了笑，然后道：“可能是因为三千大道，都是我所爱吧，兼爱众生，便是天道。”
<br/>    
<br/>        鬼仙呵呵一笑，“死到临头，油嘴滑舌，你死了，我同样能够知道其中的秘密。”
<br/>    
<br/>        鬼仙的手中，出现一根如同白玉般的骷髅杖，中间由一节节白骨组成，最上面则是一个骷颅头，加上他身披一身黑色的披风，看起来倒是有些阴森。
<br/>    
<br/>        鬼仙将手中的骷髅杖插在地面上，只是插出了一个小洞，地面之上，瞬间遍布寒霜，朝着陈明蔓延而来。
<br/>    
<br/>        陈明倒是也不怎么在意，看了一眼周围，身后百里之外有着一条通道，在通道之外则是迷茫一片，不知道是什么，而身前这百里，则是一个巨大的平台，看得出来，身后这条通道便是通向其他大陆的通道，而这个平台，便是进入赤潮大陆所驻足的地方。
