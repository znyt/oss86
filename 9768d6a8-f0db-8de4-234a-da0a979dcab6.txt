　　凯特是金·富力士的弟子，在金的安排下，小杰和凯特相遇。并在短时间内，和这个潇洒却执着的猎人，结下深厚友谊。
<br/>　　在ngl境内，凯特为了小杰和奇犽，单独阻击尼飞比多，从此一去不复返。等再次找到的时候，已变成一个只知攻击的傀儡。
<br/>　　小杰心怀愤懑，奇犽也心有不甘，两人遂决定，直接参与剿灭嵌合蚁，找尼飞比特报仇。
<br/>　　如果是原剧情，两人自然是有惊无险，虽有波折，却顺利度过这一大事件。。
<br/>　　现在不同，因为苏重的加入，当前形势早就被搅的一塌糊涂。现在参与，无异于找死！
<br/>　　“真哒？！”小杰一脸喜色，奇犽也松了一口气。他不像小杰那样粗神经，从小接受杀手训练，他想的更深，已隐隐发觉嵌合蚁事件的诡异。
<br/>　　“凯特的能力你们见过，他一共有十种武器。其中一种更像一种概念，叫做极度奈死，能以及其诡秘的方式，避死延生！”苏重淡然解释道。
<br/>　　“嚯！”莫老五深吸一口气：“这可真是了不得的能力。”
<br/>　　尼特罗眼里带着好奇：“你难道不心动？”面对这种堪称多一条命的能力，苏重竟然说的毫无情绪波动，淡然至极。
<br/>　　“极度奈死？呵！”苏重不屑一笑，以魔龙七段修炼出来的强悍身体。只要不是一瞬间轰杀成渣，再重的伤势，他也能快速回复。简直就是想死都难！
<br/>　　“老头子，我还年轻的很。倒是你，可别老死喽。”苏重一脸戏谑。
<br/>　　尼特罗也不着恼怒，风轻云淡道：“恐怕要让你失望，如果没什么意外，等你变成老头子的时候，我依然会是个老头子。”
<br/>　　这是自信还能再活五百年？苏重哑然失笑。
<br/>　　猎人世界的念能力很特殊。它是人体的生命力，是人的生物场。一旦掌握念能力，学会收摄生命力的缠，就能够在一定程度上减缓衰老。
<br/>　　揍敌客家里可就有着一个老怪物。马哈的年龄，可一点儿都不比尼特罗小！
<br/>　　压下这些思绪，苏重不会忘记自己来此的目的：“好了老头子，我们该走了。”
<br/>　　“会长，那我们？”诺布有些不满的皱着眉头，苏重只招呼尼特罗，显然不想让他们跟随。
<br/>　　“这次你们就不要去了，如果不想走，就呆在这里等待消息。”出乎众人预料，尼特罗也不赞成诺布等人前往东拓国。老狐狸一般的尼特罗，也感觉到当下形势的诡异。
<br/>　　伸手对准天空，无数丝线，好似活物一样从手心猛冲而出。顷刻间，一个遮天蔽日的三角魔龙横亘在天空。
<br/>　　不理会还在啰嗦的尼特罗，苏重一跃而上，直接站在龙头上。
<br/>　　“你们看到了，我该走了。”尼特罗笑呵呵的摊手，旋即脸色肃然：“听我的，不要试图进入东拓国。这是以猎人协会会长的身份，对你们下达的命令！”
<br/>　　说罢同样跳上魔龙脊背。旋即，魔龙宽阔翅翼煽动，带着飓风，冲天而起。
<br/>　　……
<br/>　　东拓国皇宫上空云层内。
<br/>　　“下面就是东拓国，嵌合蚁目前的老巢。”尼特罗脸色严肃：“整个皇宫都在蚁王护卫尼飞比多的念力感知内，很难获得准确情报。只知道蚁王行踪莫测，不知道在干什么。”
<br/>　　思考半晌，尼特罗揉了揉太阳穴苦笑道：“可能性太多，想要一个万全之策，难啊！”
<br/>　　苏重早就等的不耐烦：“哪里用得着那么麻烦！嵌合蚁就在下面，直接打进去不就完了！”
<br/>　　右手并指撑到，朝着下方皇宫，狠狠一斩。
<br/>　　斩击线·网！
<br/>　　嗡！
<br/>　　晶莹手掌边缘处，陡然亮起一点黑红光芒。旋即光明大方，无数丝线下从光点内钻出。顷刻间织就一张泼天大网，死死笼罩皇宫！
<br/>　　“不要冲动！”尼特罗大惊失色，怎么这么鲁莽！
<br/>　　斩！
<br/>　　一声断喝，苏重根本不管尼特罗。右手往下一压，大网好似天外陨石，轰然落下。
<br/>　　一团团乌云，瞬间变成一块块整齐方块。线网带着凌厉的劲气，瞬间冲破大气层，朝下方皇宫，狠狠砸下！
<br/>　　尼飞比多蹲在石柱上，占据皇宫最高点，念力监控方圆数公里。苏重发动能力，他瞬间就察觉到云层上变化。
<br/>　　看到熟悉至极的黑红丝线网，尼飞比多不仅没有恐惧，反而满是兴奋。
<br/>　　“来的好！”双手抓住石柱顶端，身体整个弯曲成团。大小腿部肌肉猛然膨胀。好似两个金瓜锤，被纤细的膝盖链接在一起。
<br/>　　砰！
<br/>　　整根石柱骤然炸裂成粉，一圈劲力波纹猛然扩散。尼飞比多突兀消失在原地，眨眼冲到半空。死死盯着遮天巨网，尼飞比多双手成爪，掌心相对缓缓扣合。
<br/>　　虎啸！杀！
<br/>　　纤细的手臂吹气似胀大，竟比他的大腿还要粗壮！凶猛的力道灌注双手，轰然推向天空。
<br/>　　轰！吼！
<br/>　　一股直径数十米的恐怖气浪，从尼飞比多双掌间爆发。好似龙卷风一样，带着凄厉虎啸，轰然撞上黑红线网！
<br/>　　线网下坠趋势，立刻减缓，好似生了锈似的，嘎嘎吱吱终于停在半空。
<br/>　　僵持数息，白色气浪肆虐，线网砰的一声炸碎。
<br/>　　尼飞比多双脚连踏。阵阵气爆白烟从脚底喷发，整个人炮弹一样冲上天空！瞬间来到苏重面前。
<br/>　　“哈！抓到你了！”尼飞比特裂开大嘴，眼睛里满是兴奋凶残：“为报断臂之仇，我一刻不停的修炼。猎食数百念能力者，就为增强力量！哈哈，怎么样，怎么样，你的能力不管用了！”
<br/>　　“能力被迫，我看你……你……”突然瞥见苏重右手，尼飞比多笑容猛然僵住。
<br/>　　凭空站在半空，苏重一身黑色衣衫，被冷风吹的猎猎作响。摊开的五指指尖处，一团黑红光芒缓缓浮现。
<br/>　　光芒胀大，五颗圆珠垂在指尖，黑红光芒流转，好似岩浆在蔓延。
<br/>　　“你……你怎么还能发动能力！念能力被迫，不是要元气大伤吗？！”尼飞比多大吃一惊！
<br/>　　“我为什么不能发动能力呢？”好似弹钢琴，苏重五指雀跃按下。
<br/>　　嗡嗡嗡嗡嗡！
<br/>　　五张遮天巨网，瞬间成形，轰然砸下！
<br/>　　一股寒意，直冲大脑，尼飞比多一脸绝望……
