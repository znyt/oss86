　　“大人，我们就这么在这里盯着他们么？”张玄问道。
<br/>　　叶子博冷哼一声，想了想道：“我们要演一出戏。”
<br/>　　张玄眼神闪动起来，嘴角弧度扬起到老高：“好啊。”
<br/>　　两人并没有在这个高原基地安静的呆太久，一来是这个基地内的人本就不多，想长久不被发现不现实，二来是叶子博嫌弃这里的环境恶劣，毫无继续等下去的耐心。
<br/>　　于是在短暂的合计、踩点两天后，叶子博便挑选了一个清晨，穿着一身异常肮脏的衣服出现在了叶家人所在的附近。
<br/>　　叶永阳这天照例带着几个后代出门劳作，勤勤恳恳也就想在这里永远隐居下去，一家人虽然过的辛苦，但却也没有精神上的压力。
<br/>　　只是他们刚一出门，就看到了不远处一个鬼鬼祟祟的身影，那身影一看到他们出来，就立即头也不回的调头离开。
<br/>　　“爷爷？”一名叶家人警惕的询问出声。
<br/>　　“你们几个过去看看。”叶子博的心脏突突的跳，谨慎道：“虽然刻意伪装，也完全不像他平日里的个性，但这个身影我太熟悉，像是叶子博。”
<br/>　　“子博叔？”一名叶家子孙惊讶出声：“他不是在上京当首领么？有着白家当后台，跑来这里干什么？”
<br/>　　“我们在这里猜也没用，还是直接抓来问清楚。”另一人开口，道：“爷爷您就在屋里等着，我们几个难道还打不过他一个人？”
<br/>　　叶永阳本想让大家小心，但一想到此处人迹罕至，叶子博自身战力更是不足，如果有什么手段也是靠着武器，而这处基地却有些特殊，是个武器管理很严格的地方，如果叶子博有武器，就不可能进的来。
<br/>　　而想带着武器进来，就不得不表明身份，少不了大张旗鼓，这样一来他们怎么会不知道？
<br/>　　于是叶永阳便也没有多说，只是点头道：“好，你们别走散了。”
<br/>　　几个叶家最后的年轻人抱团在一起，朝着叶子博消失的方向就追了过去。
<br/>　　叶永阳年岁已高，便直接回了屋内等待。
<br/>　　只是当他刚进屋的一刹那，一股寒芒在背的感觉就油然而生，紧接着——
<br/>　　啪！
<br/>　　房门自动闭合，发出带着一丝阴森感的刺响。
<br/>　　叶永阳猛然回头，当即便看到紧闭的房门旁站着一个人，一个笑容无比阴森的人。
<br/>　　“你是谁？！”叶永阳的声音充满了惊惧感，苍老的面容上更涌上了恐慌。
<br/>　　来者正是张玄，他与叶子博上演了这么一出戏，计划中是叶子博一边避开杨飞杰的注意，一边悄悄将叶家人都引开，最终由张玄悄悄潜入屋内，将叶永阳掳走。
<br/>　　只是当下的张玄并没有第一时间行动，而是依靠在门边上，以一种极为可怕的笑容盯着眼前的老人。
<br/>　　“你是谁！”叶永阳又问了一遍，语气带上了质问。
<br/>　　张玄环顾了一周，然后开口：“你就是叶永阳，叶家最后一代家主，叶永安的后人？”
<br/>　　这一问话出口，叶永阳立即心跳加速，并问出了第三遍一模一样的问题：“你是谁？”
<br/>　　只是这一次，叶永阳的语气比前两次平静很多，也似乎释然了不少，找上门来的，还知道这一终极秘密，显然他已经无处可逃。
<br/>　　“看来是了？”张玄面上绽放了一个灿烂的笑容，手中一柄断刃‘噌噌’的摩擦着他另一只手腕上的铁皮。
<br/>　　随口说道：“叶子博还真是可怜啊，到现在还不知道白家为什么要找你，更不知道叶永阳这个名字，他只是凭着几条毫无用处的线索，就一门心思的想要独吞某个大秘密。”
<br/>　　叶永阳瞳孔一缩：“叶子博果然在这里？”
<br/>　　“当然。”张玄嗤笑出声：“就是他带我来的，搞笑的是他以为是他聘请了我当助手，却不知道白家才是我的真正的雇主，而他只不过是一条……导盲犬？”
<br/>　　说着张玄似乎是被自己的比喻逗乐，毫不掩饰的笑出声：“哈哈！若不是叶子博将你们离开的方向隐瞒了下来，白家何须这么麻烦还要用他这么个废物？导盲犬就是导盲犬，带路就行了，不过他鼻子真灵，还真让我给找到你了！”
<br/>　　“你是白家派来的，你想做什么？”叶永阳已经不想在多问有关叶子博的事，而是直入主题。
<br/>　　张玄耸耸肩：“我只是个做任务的人，对雇主的隐私并不感兴趣，他给钱，我办事，我只负责找到你，然后带你去白忧少爷的面前。”
<br/>　　“白忧。”叶永阳咬牙切齿的吐出这两个字：“他们白家过了这么多年还要——”
<br/>　　“停！”张玄忽然打断叶永阳，道：“别说太多，我可不想小命不保。”
<br/>　　叶永阳眼神发狠，已经到了他这个岁数，在如此情绪起伏的刺激下，身躯已经有些不稳的摇摇晃晃。
<br/>　　“你可别半路死了啊！”张玄看着他这状态，上前一步就将叶永阳抓·住：“干脆在你死之前呆走！”
<br/>　　张玄的速度很快，做事也极为小心，他捂住叶永阳的嘴不让其发声，身形鬼魅的便来到了基地的大门处。
<br/>　　这里有着基地的守卫，进出基地都要有严格的把关，张玄进来时便接受了一系列盘问，想要出去还带着个人，这种温和的方式当然行不通。
<br/>　　于是他在靠近大门的时候，便没有再继续小心隐藏身形，而是加快了速度朝着大门冲去。
<br/>　　守卫很快发现了疾走而来的张玄，也看到了其肩上还扛着个人，当下大喊出声：“怎么回事？！停下！”
<br/>　　张玄却根本对其置之不理，朝着大门猛然冲刺，同时空出的右手更是高举，一拳轰出！
<br/>　　轰！
<br/>　　一声巨响，基地的木质大门一下子便被张玄一拳打烂，到处都是碎屑和木头渣。
<br/>　　其恐怖的八阶新人类爆发力，也在这一瞬间显露在外，同时还有两名守卫的尸体，翻滚着在地上流淌出腥红鲜血。
<br/>　　基地大门是张玄的最后一关，他也根本没想要在这里浪费时间，杀掉守卫破开大门，直接便冲了出去！
