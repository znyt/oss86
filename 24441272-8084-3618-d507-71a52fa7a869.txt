　　第369章
<br/>　　凉州城，军备营。
<br/>　　人们在紧张的忙碌着，火把将黑夜驱散，一车又一车的粮食被从仓库中运出。
<br/>　　这次运粮由宋伯康亲自监督，作为一州之主，他自然是不愿意去给程大雷送粮的，这岂不是养虎为患么。但自己这个凉州牧，只不过是给崔相当看家狗，何况薛问婷如今还在琴川关扣着。
<br/>　　薛问婷虽然是崔相的外孙女，但他的父亲差不多等于入赘到崔家，薛问婷从小在相府长大，这薛问婷和亲孙女也差不多。
<br/>　　因为薛问婷的事，自己已经在崔相面前丢了人，如今还希望把崔弄海从琴川关救出来，以此挽回一些自己在崔相心中的形象。
<br/>　　所以不管程大雷提出的条件有多么苛刻，宋伯康都是会答应的，时间上千万不要出了岔子，以免夜长梦多。对于程大雷的为人，宋伯康实在谈不上有多信任。
<br/>　　“他真的就直接答应了？”宋伯康还是有些放心不下，询问身边的钟伟虎。
<br/>　　“很愉快就答应了啊。”钟伟虎道：“大人，这程大雷见钱眼开得很，根本没有任何犹豫。”
<br/>　　“他不知道崔公子的身份吧？”
<br/>　　“这个应该还不知道，不然也不会这么痛快答应。”钟伟虎道。
<br/>　　“那就好，那就好。”宋伯康说着，却还是觉得心绪难宁，他道：“我却还是觉得放心不下，明天天一亮你就去，一定要将崔公子平安救出来，免得事情迟了又有变化。”
<br/>　　“遵命，若明天我救不回崔公子，就提头来见大人。”
<br/>　　事实上，根据宋伯康估计，崔弄海来凉州，大概就是奔着薛问婷来的。崔弄海是个杀人变态的事实，谁也不会反对，可偏偏这个杀人变态对薛问婷格外有感情。大概是从哪里知道消息，所以特意来凉州救薛问婷。
<br/>　　其实这些年，崔弄海真正去了哪里，谁也不知道。相府也不怎么在意，毕竟以崔弄海的本事，也没人制得住他，只要查查哪里有杀人变态出现，肯定就是崔弄海没跑了。
<br/>　　今日看到这些发生在凉州城的惨案，宋伯康大概就猜到是他，他专门给程大雷传讯，也是觉得程大雷抓不住对方。
<br/>　　可真的没料到，程大雷竟然真的擒住了崔弄海，这让宋伯康有些无语。
<br/>　　程大雷一定是个比变态更变态的存在。
<br/>　　只希望明日能平安将崔弄海捞出来。
<br/>　　第二天天没亮，钟伟虎就出发了，押送粮车的人，都是凉州城中精锐的精锐。同时，这也是个亮肌肉的好机会，也让琴川关看看：
<br/>　　凉州词也是有好汉子的。
<br/>　　大概在辰时赶到的，迎接钟伟虎的是和珅，和珅将钟伟虎请到屋里喝茶。
<br/>　　“麻烦钟大人了，不辞辛苦的跑一趟，唉，实在让我心里不好受啊。”和珅笑道：“钟大人，喝茶，喝茶。”
<br/>　　二人算是老相识了，互相之间不知打过多少太极，可今天钟伟虎实在没这个心情。
<br/>　　“和当家，这个崔弄海呢，能不能让我先见见？”
<br/>　　“咦，钟大人怎么知道他的名字？”
<br/>　　“呃……这个，哈哈，听说的，听说的。”
<br/>　　“呵呵，钟大人果然消息灵通呐，不过不用着急，我们总要点一点数目，一点清楚，立刻交人。”和珅道：“我这可不是信不过钟大人，实在是上面卡得紧。”
<br/>　　钟伟虎心急如焚，却又不得不按下性子等待，他问道：“程当家呢，能不能让我见见？”
<br/>　　“睡着呢，这才什么时间，他那样的人怎么会醒。”和珅叹口气：“受苦受累的，还不是咱们这些手下人。”
<br/>　　二人磨着时间，正这时候，有人进来，道：“和当家，都盘清楚了，一共十万石稻米，分毫不差。”
<br/>　　“走，出去看看。”
<br/>　　和珅大手一挥，踏出屋门，粮车排在琴川关的校场上，骡马排成行。凉州城的人各个严肃，手中举着长戈，保持肃穆的姿态。
<br/>　　琴川关的弟兄也都各个端着一张脸，眼睛恨不得长在头顶，双方目光碰撞，空气中恍惚有火花。
<br/>　　假若一句话说不对，怕双方当场就会杀起来。
<br/>　　和珅用刀扎开一个口袋，从里面流出白花花的稻米，都是上好的粮食。
<br/>　　他脸上露出满意的笑容，道：“弟兄们，这些都是宋大人给的，还不谢谢宋大人。”
<br/>　　“谢宋大人！”
<br/>　　上千人一起吼，势冲斗牛，钟伟虎脸上神色变化。
<br/>　　“和当家，是不是该把人放出来？”
<br/>　　“放？”和珅一楞。
<br/>　　“不，不，是交给我们，让我们处置。”钟伟虎忙道。
<br/>　　“钟大人太着急了，怎么也得把粮食搬进仓库，把这些马呀，车呀腾出来。”
<br/>　　“不必了，不必了，这些全都当送给琴川关了。”
<br/>　　“真的，这可不少呢。”
<br/>　　“都送了，都送了。”钟伟虎忙不迭声道，现在的他，急的心里直想骂娘。
<br/>　　和珅笑笑：“那好吧，把人带上来。”
<br/>　　和珅拍着手，随之一辆马车被人拉上来，相比那宋弄海就在马车中。
<br/>　　“人就在里面，钟大人把人和马一起牵走，钟大人这么大方，我们也不能小气，送给你们啦。”和珅大手一挥，显得十分豪横。
<br/>　　钟伟虎心底冷哼一声，我上百匹骡马，就换你这一头毛发不全的瘦马，你还显得自己蛮吃亏。
<br/>　　当然，这对钟伟虎来说并不重要，他如今只担心崔弄海的情况。
<br/>　　“人，还活着么？”在想将车帘打开时，钟伟虎突然有些忐忑，试探的询问。
<br/>　　“活着，当然活着，活蹦乱跳的。”和珅拍着胸脯道：“不过钟大人还是要小心些，他很凶的，小心咬着你。”
<br/>　　钟伟虎咬咬牙，将车帘打开，向里面张望了一眼。
<br/>　　只是一眼，他整个人完全愣住，脸上轻一阵白一阵，肌肉渐渐绷紧，歇斯底里发出一声大叫。
<br/>　　“程大雷！”
<br/>　　刷地一声，凉州城的卫兵双手握矛，指着前方。蛤蟆寨的弟兄，发出声声冷笑，有些不屑。
<br/>　　“怎么啦？”程大雷慵懒的声音响起，他打着哈欠出现：“我这刚睡醒，就听见有人喊我名字，什么事啊？”
<br/>　　
<br/>(本章完)
