　　徒然，诡异的事情发现了，余子童皮包骨里面，既然有一条修长的样子在皮肤中游曳，显得非常的诡异恐怖。
<br/>　　苏羽看到这一幕，猛然倒吸了一口凉气：“那影子就是蛊蛇？”

<br/>　　捕蛇人凭借着自己的智慧跟手段，与毒蛇生死搏斗，直至最后捕抓。
<br/>　　涂小安记得当初自己拜师何震南，成为一个捕蛇人，他就问过：“师父，什么才是天底下最可怕的毒蛇”

<br/>　　那便叫蛊蛇。
<br/>　　但蛊蛇并不是世界上的什么蛇类品种，而是经过非常特殊且阴邪的手段创造出来的物种，好像只有巫师才能做到。

<br/>　　“你去医院怎么说，说余大师身体内藏着一条蛇？麻烦你们医生开刀将蛇给取出来吗”涂小安揶揄的笑了笑：“况且你刚才也说了，医院的仪器根本查不到这条蛊蛇”
<br/>　　事情要真那么简单，到没有那么可怕了。
<br/>　　苏羽急了：“那该如何是好，难道看着余大师就这样惨死吗，小安兄弟，你一定有办法的对吧”
<br/>　　涂小安奇怪苏羽怎么跟余子童关系变得那么好，工地上的事件，余子童也就是一个打酱油的啊。
<br/>　　莫非他还想用钱买余子童的那种黄金瞳，所以先对他施以恩惠？
<br/>　　“苏哥莫急，余大师要是身上是蛊虫，我还真没什么好办法，可要是蛊蛇，你找我，还真找对人了”
<br/>　　涂小安笑了笑，又给了余子童一个你命真好的表情。
<br/>　　他不管这条蛇是自然的产物，还是认为的制作，只要你是蛇，那么涂小安就有办法。
<br/>　　他可是登基坐殿的蛇王，更有对蛇类的统治能力。
<br/>　　“来，余大师，你把嘴巴张开，我现在要尝试，将你体内吸食你血肉的蛊蛇给召唤出来，无论你多么的难受，你都不可以把嘴巴闭上，懂吗”
<br/>　　“咳咳...”余子童点了点头，涂小安是他最后的救命稻草。
<br/>　　“统治能力，开”
<br/>　　涂小安在心中默念，霎时间，一股皇者之气让他的身上散发出来，旋即笼罩到了余子童的身上。
<br/>　　十几秒后，苏余盯着余子童的身体，惊呼：“那影子好像往上游了”
<br/>　　涂小安看到自己的统治能力对蛊蛇有影响，暗暗的松了一口气。
<br/>　　接下来，他就用统治能力，引导蛊蛇从余子童的喉咙处游出来。

<br/>　　苏羽道：“余大师加油，那条蛊蛇已经在你胸口位置了，现在朝着你的喉咙游去”

<br/>　　他连咬牙这样的动作都不能做，嘴巴不能合，非常的痛苦，冷汗涔涔而下。

<br/>　　就在这时，余子童不知道是不是太痛苦，剧烈的咳嗽呕吐起来，让人感觉他全身骨架都要支离散架了。
