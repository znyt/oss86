　　苏莫几乎要咬碎了满嘴的牙齿，死死的坚持着，不过，他即将坚持不住了。
<br/>　　这魔魂中魔念的强大，超越了他的想像，不断的将他的精神意念撕碎，很快就要灭杀他的神魂了。
<br/>　　“完了！”苏莫的心沉到了谷底，他根本抵挡不住这魔念，要被彻底夺舍了。
<br/>　　就在苏莫准备自爆肉身，和对方同归于尽之时，他手中的十亿魔剑中，同样涌出了一股魔念。
<br/>　　这股魔念乃是十亿魔剑的魔灵，魔灵的魔念冲进了苏莫的识海中，立刻帮助苏莫一同抵挡魔魂的魔念。
<br/>　　苏莫见此，顿时心中大喜，立刻联合魔灵的魔念一同绞杀魔魂的魔念。
<br/>　　不过，即便苏莫和魔灵联手，依旧不是魔魂的对手。
<br/>　　不多时，魔灵的魔念便被绞杀的支离破碎。
<br/>　　啊！
<br/>　　魔灵传出一声惨叫，魔念便开始渐渐消散。
<br/>　　“魔灵！”苏莫大惊，难道魔灵要消亡了吗？
<br/>　　“桀桀！小子，你完成了主人的意志，我死也瞑目了~~！”魔灵大笑一声，便没了声息，彻底的消亡了。
<br/>　　苏莫心中有些悲凉，魔灵死了，他也快要死了！
<br/>　　“嗯？”就在此时，苏莫突然发现，魔魂的魔念居然在快速的减弱。
<br/>　　这让得他顿时大喜，来不及多想，立刻便动用所有的精神意念，绞杀魔魂的魔念。
<br/>　　而他的吞噬武魂，亦是疯狂的催动催动，开始对魔魂展开反击。
<br/>　　呼吸之间，魔魂的魔念便一泻千里，被他的精神力击溃了。
<br/>　　“给我吞！”苏莫立刻疯狂怒吼了起来，想要将魔魂彻底吞噬。
<br/>　　很快，苏莫便如愿以偿了，强大的不可思议的魔魂，被他的吞噬武魂直接吞没。
<br/>　　呼！
<br/>　　苏莫长出了一口气，身躯直接从半空中跌落了下来，他这才发现自己全身都被汗水打湿了。
<br/>　　“终于过去了！”苏莫心中一阵后怕，差一点，就差一点她便彻底死亡。
<br/>　　虽然他不清楚魔魂的魔念，为何突然后继无力，但他只要保住性命便好了。
<br/>　　“嗯？”就在此刻，苏莫面色一变，因为他突然发现，那魔魂被吞噬之后，居然不像其它被吞噬的武魂或战魂那样，和他的吞噬武魂融合在一起。
<br/>　　魔魂依旧在吞噬武魂的漩涡之中挣扎，居然无法和吞噬武魂融合。
<br/>　　“给我吞！”苏莫心中发狠，拼命的催动吞噬武魂，但足足又过去了一刻钟，依旧无法融合。
<br/>　　怎么回事？
<br/>　　苏莫有些懵了，无法融合的话，魔魂不仅不能成为他武魂的养料，还随时有可能逃走。
<br/>　　“可恶啊！天下间居然有如此奇特的武魂！”
<br/>　　这时，魔魂之中传出一声不甘的怒吼，显然这魔魂也是没想到，夺舍一名不到武帝境的人，居然还失败了！
<br/>　　“哼！我会想办法弄死你！”苏莫心中冷哼，他现在虽然无法融合这魔魂，但等到吞噬武魂再次提升等级，或者是等他修为提升之后，定然能够融合。
<br/>　　魔魂不甘的怒吼了一声之后，便不在出声，沉寂在了吞噬武魂之中，也不再挣扎，仿佛彻底消失了一般。
<br/>　　少倾之后，苏莫深吸口气，这一次真是危险，差点就栽了。
<br/>　　以后，他只需注意着吞噬武魂之中的魔魂便可，对方只要想逃，他便立刻催动吞噬武魂。
<br/>　　若是对方不逃更好，待他再次将武魂提升，再试着吞噬融合对方。
<br/>　　看了一眼手中的十亿魔剑，苏莫微微的叹了口气，他能感觉到剑中魔灵已经不在，魔剑的威力也减弱了大半不止。
<br/>　　“聂兄、韩姑娘，没事了！”苏莫抹了把额头上的冷汗，将魔剑收进储物戒后，转头看向通道口处的聂乾和韩盈。
<br/>　　“苏兄，你……你挡住了那魔魂的夺舍？”聂乾满脸的惊讶的问道。
<br/>　　刚才那魔魂的威势，他可是感受的非常真切，最起码都是武尊境巅峰的魔甲族强者的魔魂。
<br/>　　而这等强者的魔魂夺舍，苏莫居然挡住了，这简直就是奇迹！
<br/>　　“当然，不然的话我还会和你说话吗？”苏莫点头道。
<br/>　　“太好了，苏兄总算逃过了此劫！”
<br/>　　韩盈面上露出了喜色，但她心中的震惊，简直无以复加，苏莫不仅实力惊人，面对如此危局都能保住性命，当真是不可思议。
<br/>　　“我们走吧！”苏莫不愿意在此地多待，招呼两人一声，便原路返回。
<br/>　　聂乾和韩盈心中唏嘘，但也立刻跟着苏莫返回。
<br/>　　返回的路畅通无阻，即没有凶灵的围杀，也没有迷宫出现。
<br/>　　当然，就算有迷宫苏莫也不怕，因为廖云剑如何走出迷宫的规律，他早已暗暗记下了。
<br/>　　三人很快便出了地宫，走出了山洞，再次回到了海底的那处山谷中。
<br/>　　三人也没有在海底停留，立刻便离开了海底，飞到了海面之上。
<br/>　　“苏兄，我们还历练吗？”聂乾向苏莫问道，言语客气至极。
<br/>　　“当然，才刚来没多久，难倒就这样回去？”苏莫沉声道。
<br/>　　言罢，他转头看向韩盈，道：“韩姑娘，我们继续猎杀魔甲人，靠你多多费力了！”
<br/>　　韩盈的战魂特殊，对于寻找魔甲人，帮助非常的大。
<br/>　　“没有问题！”韩盈点了点头。
<br/>　　随即，三人简单商量了一番，便在大海之上漫无目的飞行了起来，寻找魔甲人的踪迹。
<br/>　　有韩盈的帮助，即便相隔数十万里，他们也能提前发现魔甲人的踪迹，一旦有强大的魔甲人，他们立刻远远避开。
<br/>　　而实力不强的魔甲人，苏莫则斩尽杀绝，血液和魔魂全部吞噬一空，战利品三人瓜分。
<br/>　　魔甲人虽然很穷，但身上均有少量的魔晶，也是一笔财富资源。
<br/>　　苏莫使用了廖云剑的长剑，这是一柄银色的帝级上品兵器，品质极高，威力比之拥有魔灵的十亿魔剑还要强大一分。
<br/>　　在苍茫的大海之上游荡了五天，三人斩杀了四十多名魔甲人，魔魂全部被苏莫吞噬，但他的吞噬武魂并没有再晋级。
<br/>　　苏莫吞噬一切的情景，让得聂乾和韩盈两人，即震惊又恐惧，他们哪里见过如此的情景！
<br/>　　连续游荡了五天，最后，他们找到了一座弹丸小岛，暂时停留了下来，在岛上恢复元气。
