<br/>　　怪不得它控制不了这块石头，连是什么东西还没弄明白。

<br/>　　满地打滚的涂小安左眼光芒大亮，随后便是一片血海，散发妖艳的光，威势极强，非常的可怕。
<br/>　　而涂小安自己的感知中，他的左眼白茫茫一片，根本看不清东西，像是被一片白雾给阻挡住了视线。
<br/>　　这是还没彻底的融合。
<br/>　　只是彻底的融合了，疏通了，还才与正常的眼睛无异。
<br/>　　“还不给我通”
<br/>　　涂小安右眼一眯，将所有的力量都汇聚在左眼之上，霎时间，这只血海般的左眼迸发出一股恐怖如斯的威慑力。
<br/>　　好像有一种力量要宣泄出来。
<br/>　　旁边的猫精海棠龟缩成一团，浑身忍不住的瑟瑟发抖，现在这个山洞阴寒的要凝固了。
<br/>　　咻...
<br/>　　终于，眸光实质，一股红光从涂小安的左眼中骤射了出来，一闪而逝。
<br/>　　轰...
<br/>　　正前面一块岩壁墙壁出现了一个深坑，碎石簌簌而落，紧接着山洞震动起来。

<br/>　　疏通了，蛇王眼跟涂小安本身的眼睛融为一体。
<br/>　　就在这时...
<br/>　　“叮...恭喜宿主得到蛇王左眼，获得一股巨大的能量值，等级成功提升到了24级”
<br/>　　“叮...恭喜宿主的等级提升到了25级”
<br/>　　“叮...恭喜宿主的等级提升到了26级”
<br/>　　瞬间，涂小安的等级一连提升三级，左眼眸光不断化为实质性的攻击手段，在这个山洞中狂轰乱炸。

<br/>　　蛇精也成了这只眼睛真正的主人，掌控这只眼内蕴含的神通。

<br/>　　在看涂小安，他现在状态的好极了，等级直接提升到二十六级，让他的实力再次得到了一次前所未有的提升。
<br/>　　这是他万万没想到的。
<br/>　　可想而知，这蛇王眼到底有多么的可怕。
<br/>　　这是机遇，只属于涂小安的机遇，他是登基坐殿的蛇王，只有他才能融合跟掌控这只蛇王眼。
