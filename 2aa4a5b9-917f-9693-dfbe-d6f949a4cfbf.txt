　　苏莫三人速度极快，不多时，巫族小镇便出现在了他们的视线之中。
<br/>　　远远望去，这座小镇不大，规模和苍穹神宫所在的途山镇差不多，但居住的巫族之人数量，却远远不如途山镇的武者多。
<br/>　　三人停在了小镇千里之外，苏莫灵识扫荡而出，扫过整个小镇，暗暗的点了点头。
<br/>　　这小镇确实只有七、八万巫族之人，人族更是稀少，不足百人，而最强者是一名武帝境三重的中年人。
<br/>　　“聂兄，韩姑娘，你们两人留在此地，若发现有强者路过，立刻通知我！”苏莫向韩盈和聂乾说道。
<br/>　　“苏兄放心吧，二十万余里之内，任何风吹草动，都逃不过我的监听！”韩盈说道。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即身形一闪，直接向小镇飞去，瞬间便飞临了小镇上空。
<br/>　　“吞噬！”轻喝一声，苏莫立刻释放出了吞噬战魂。
<br/>　　巨大的吞噬旋涡，横亘在高空之上，遮天蔽日，强大的吞噬之力，瞬间席卷而出，笼罩了整座小镇。
<br/>　　嘭嘭嘭！！
<br/>　　刹那之间，闷响声不断，整个小镇的人还没有反应过来，便纷纷爆体而亡，化为滚滚驳杂的气流，向苏莫狂涌而去。
<br/>　　不仅如此，整个小镇之中，无数的院落、阁楼、宫殿等等一切，直接拔地而起，向吞噬旋涡飞了过去。
<br/>　　这一幕极为震撼，整个小镇的一切，都要被吞噬旋涡吸纳。
<br/>　　“啊！怎么回事？”
<br/>　　“你是谁？”
<br/>　　“你想要干什么？”
<br/>　　惊恐的喝声此起彼伏，小镇中武帝境武者足有两百余名，他们没有爆体而亡，但根本控制不住自己的身形，急速向吞噬旋涡飞了过去。
<br/>　　“杀！”小镇中的最强者，那名武帝境三重的巫族中年人，眼见要被吞噬旋涡吞噬，一刀狠狠的斩向苏莫。
<br/>　　犀利的刀气威力不凡，撕碎空间，急速向苏莫斩了过去。
<br/>　　不过，这刀气虽然强大，但根本抵挡不住吞噬的力量，还未斩击到苏莫身前，便陡然转向，飞进了吞噬旋涡之中。
<br/>　　“啊啊啊！！”
<br/>　　毫无意外，包括这名武帝境三重的巫族中年人，所有人都被吞噬了。
<br/>　　而整个过程，耗时仅为一个呼吸的时间。
<br/>　　不远处的聂乾和韩盈已经吓傻了，望着变成了一块平地的小镇，两人脑中掀起惊涛骇浪。
<br/>　　他们虽然知道苏莫战魂的能力，但哪里能想到，这战魂这么恐怖，一股呼吸的时间，便将原本繁华的小镇，变成了不毛之地。
<br/>　　两人对于此情此景，感到全身冰冷，心中直冒凉气。
<br/>　　苏莫身形屹立在高空之上，停止了催动吞噬战魂，周身无尽的驳杂气浪，全部被他震散了。
<br/>　　因为现在显然不是冲击境界的时候，只要有了足够的资源，日后可以安心的冲击境界。
<br/>　　下一刻！
<br/>　　嗖嗖嗖嗖！！！
<br/>　　密密麻麻的人影，伴随着无数的长刀、利剑、大枪、储物戒等乱七八糟的物品，突然从吞噬战魂的旋涡之中飞射而出，纷纷落在了下方的大地上。
<br/>　　眨眼之间，下方便堆积成了一座小山。
<br/>　　这些人影，正是此阵的那些武帝境武者，苏莫吞噬了他们的战魂之后，便将他们的尸体全部抛了出来。
<br/>　　至于那无数的兵器、储物戒等杂物，则属于这全镇武者所有。
<br/>　　当然，有很多低等级的宝物，被吞噬战魂吞噬之后，都直接被炼化成了虚无，这些是苏莫刻意保留下来的。
<br/>　　毕竟，他的目的便是资源，若是全部吞噬的一干二净，那就白忙活了。
<br/>　　他的吞噬武魂进化成战魂之后，吞噬之力变得如臂指使，控制起来非常的方便。
<br/>　　想要如何吞噬就如何吞噬，想要如何炼化就如何炼化，堪称玄妙无比。
<br/>　　随即，苏莫一挥手，将这小山一般的尸体和宝物，全部收进了储物戒中，便转身返回。
<br/>　　“两位，搞定了！”苏莫飞到了聂乾和韩盈的身前。
<br/>　　“苏……苏兄，我们是不是有些太残忍了？”韩盈俏脸有些发白，她本以为苏莫只是单纯的洗劫，哪里知道会是这个情况。
<br/>　　此刻，面对苏莫之时，韩盈已经没有了之前的淡然，眸中隐隐有一丝畏惧。
<br/>　　聂乾不比韩盈好多少，脸色亦是有些苍白，暗暗吞咽了一口口水。
<br/>　　“残忍？”
<br/>　　苏莫闻言摇了摇头，面带阴郁的说道：“你们若是知道，我的家乡有千亿武者、无尽生灵，都死于巫族之手，就不会感觉我残忍了！”
<br/>　　韩盈两人闻言一惊，面带不可思议之色，巫族屠戮了苏莫家乡千亿武者？居然这般残忍？
<br/>　　“不仅如此，我的家乡已经不存在了，被彻底毁灭！”苏莫叹了口气，幽幽的说道。
<br/>　　以前没出苍穹世界之时，他曾有个想法，那就是日后有能力之后，让得苍穹世界摆脱巫族，再次成为自由的世界，可以自由出入外界，但现在都不可能了。
<br/>　　还有苍穹神宫的山门，虽然没有人了，但几位傀儡人还在其内。
<br/>　　虽然金一等人都是傀儡人，但灵智与常人无异，苏莫与它们相处了这么久的时间，也是很有感情了。
<br/>　　苍穹世界一毁，他们虽然隐藏在虚空深处，但估计也全部遭到了毁灭。
<br/>　　“苏兄，既然巫族如此灭绝人性，那我们也不必客气了！”聂乾剑苏莫面带落寞之色，立刻沉声说道。
<br/>　　“苏兄，请节哀！”韩盈说道。
<br/>　　苏莫闻言叹了口气，道：“这一次，算是我给巫族的报复，将来必灭此族！”
<br/>　　言罢，他双掌一翻，两只手掌中均是出现了十几个储物戒。
<br/>　　“这些储物戒你们拿着，想必里面资源宝物都不少！”苏莫将两把储物戒递向了韩盈两人。
<br/>　　“这……苏兄，我们也没有出什么力，要不了这么多！”
<br/>　　“苏兄，你太慷慨了！”
<br/>　　韩盈两人见此，立刻推脱了起来，不太好意思收取这么多的储物戒。
<br/>　　“拿着吧！此地不宜久留，我们快走！”不由分说，苏莫将两把储物戒塞到两人的手中，随即立刻向远处飞去。
<br/>　　韩盈两人只能无奈的收下了储物戒，不过两人心中也很是激动，苏莫如此慷慨，这一次他们说不定真能赚的盆体满钵。
<br/>　　而后，两人立刻跟随苏莫而去。
