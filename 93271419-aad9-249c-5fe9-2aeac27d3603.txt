　　太阴内族，是太阴族的核心之地，其入口就在太阴城之中。
<br/>　　褐袍老者带着苏莫，大步而行，向太阴城的北部方位走去。
<br/>　　“前辈，这次给予我指点的是哪位神长老？”一边行走，苏莫一边沉声问道。
<br/>　　“当然是普阳神长老，我族之中，除了族长之外，也就普阳神长老还偶尔现身！”褐袍老者说道。
<br/>　　“是吗？那族内共有几位神长老？”苏莫好奇的问道，太阴族这么强大，应该虚神境的大能不少吧！
<br/>　　“拥有灵魂玉牌的神长老有十几位，但是真正还在本族的神长老，只有三位了！”褐袍老者叹道，他倒没有隐瞒，这也不是什么秘密，而且以苏莫的天赋，将来肯定被吸收进太阴族。
<br/>　　“三位？”苏莫闻言点了点头，道：“看来虚神境的大能，都去游历宇宙去了。
<br/>　　“不清楚，有的神长老上万年都没有出现了，但灵魂玉牌并没有破碎，说明还活着！“褐袍老者道。
<br/>　　苏莫闻言暗暗惊讶，上万年都没有出现了，这些大能人物也真是潇洒。
<br/>　　“苏莫，你能得到普阳神长老的指点，是你的一次机缘！”褐袍长老告诫道。
<br/>　　“嗯！”苏莫点了点头，心中不禁感慨，能得到虚神境大能的一次指点，就是很大的机缘，那青璇作为虚神境大能的弟子，这机缘是有多大？
<br/>　　这从青璇的修为，便能清晰的看出来，当年离开苍穹世界的时候，青璇的修为的何等之低。
<br/>　　才能短短的几年时间，青璇的修为，居然都赶上了他，这个修炼速度就恐怖了！
<br/>　　苏莫跟着褐袍老者一边闲聊，一边快速而行，通过交谈，他得知这名褐袍老者是太阴族内族执事，名为阴蒙。
<br/>　　不多时，两人便来到了一座宫殿之中。
<br/>　　宫殿防守严密，殿内只有一个空间之门，通向另外一个世界，也就是太阴内族所在的世界。
<br/>　　进入空间之门，穿过虚幻的空间通道，苏莫来到了一处广阔的世界之中。
<br/>　　这个世界一望无尽，浩瀚的大地，汹涌的河流，擎天的巨山，茂盛的丛林，应有尽有。
<br/>　　“好浓郁的灵气！”
<br/>　　感受到浓郁的灵气扑面而来，苏莫深深的吸了一口气，只感觉神清气爽。
<br/>　　“呵呵，我内族中，有数千条顶级的龙脉，灵气当然浓郁！”褐袍老者笑道。
<br/>　　“数千条顶级龙脉？”
<br/>　　苏莫闻言一怔，随即满脸惊讶的问道：“阴蒙执事，这太夸张了吧？哪来的这么多龙脉？”
<br/>　　苏莫惊讶不已，对方说得顶级龙脉，肯定是能诞生灵晶的龙脉，而且可能还不止是下品灵晶。
<br/>　　而此地，居然拥有数千条这种龙脉，这就恐怖了。
<br/>　　“很正常，这些都是族中大能在很多星辰上抓取的，只要虚神境强者出手，龙脉根本逃不了！”阴蒙执事轻笑道。
<br/>　　“原来如此！”苏莫心中恍然，有大能强者出手，自然便能抓取到强大的龙脉。
<br/>　　“走吧！普阳长老住在普阳山上！”
<br/>　　阴蒙执事招呼苏莫一声，便向前方飞去，苏莫急忙跟了上去。
<br/>　　太阴内族非常的大，飞行之中，苏莫看到了一座座城池，这些城池虽然人口不多，也没有外界的太阴城大，但每座城池之中均是有大量的磅礴气息传出，说明每个城池之中都有大量的强者。
<br/>　　整个太阴内族，仿佛就是一个国家，或者说是一块大陆。
<br/>　　苏莫和阴蒙执事的速度很快，飞行了约莫一个时辰之后，来到了一座通天巨峰之前。
<br/>　　此峰高近万丈，通体灵气缭绕，仿佛是一座仙山一般。
<br/>　　山峰之上，建筑极少，只有寥寥几座宫殿。
<br/>　　苏莫跟着阴蒙执事飞到了山脚下，而后沿着山上的小道徒步而行，这是对于那位普阳神长老的尊敬。
<br/>　　若是他们直接飞到山上，就是明显的藐视这山峰的主人，普阳神长老了。
<br/>　　“苏莫，普阳长老性格温和，你也无需有什么紧张？”阴蒙执事说道。
<br/>　　“嗯！”苏莫点了点头。
<br/>　　片刻之后，两人来到了接近山顶的位置，这里零零散散的坐落着几座宫殿。
<br/>　　阴蒙执事带着苏莫，来到了其中最大的一座宫殿之前。
<br/>　　“普阳神长老，这次会武的一名，我已经带来了！”阴蒙执事向殿门紧闭的宫殿抱拳，恭敬的说道。
<br/>　　宫殿内寂静无声，少倾之后，宫殿的大门自动的打开了。
<br/>　　“来后院吧！”一道充满磁性的声音响起。
<br/>　　阴蒙执事闻言点了点头，向苏莫示意，他便躬身告退了。
<br/>　　苏莫见此，深深的吸了口气，便大步的走进了宫殿之中。
<br/>　　宫殿中很是宽敞，其内有一座通道，通向宫殿的后方，苏莫闯过通道，来到了宫殿的后方。
<br/>　　宫殿之后，是一片巨大的院落，院落中只有一座水潭。
<br/>　　水潭不大，方圆约莫百余丈，潭水呈墨黑色，充满了浓烈至极的阴气，这阴气之浓烈，简直世所罕见。
<br/>　　不过，院落之上显然布置了阵法，让得水潭内冲天而起的阴气，不会冲出院落。
<br/>　　苏莫抬眼望去，只见水潭之上，正盘膝坐着一道身影。
<br/>　　这道身影穿着紧身的黑衣，体型匀称，头发很长，一直垂落到水面上。
<br/>　　由于此人背对着苏莫，苏莫也看不清对方的面容，而他也不敢贸然用灵识查探对方。
<br/>　　不过，此人给他的感觉，完全是无法捉摸，对方仿佛是无尽的大海，深不可测，又仿佛是一个普普通通之人，完全没有任何修为。
<br/>　　甚至，此人明明就在眼前，但苏莫感觉对方距他极远，仿佛是亿万里之遥。
<br/>　　“拜见神长老！”苏莫遥遥向对方抱拳。
<br/>　　“你叫什么名字？”黑衣人轻声问道，声音淡漠，不含丝毫感情。
<br/>　　“我名苏莫！”苏莫沉声道。
<br/>　　“嗯！”
<br/>　　黑衣人微微颌首，随即沉默了少倾，开口道：“你可以提出三个修炼中的问题，我会为你解惑！”
<br/>　　苏莫闻言讶然，他之前还在猜测对方会怎么指点他，没想到居然是让他提问题。
<br/>　　可是，该提什么问题呢？
<br/>　　苏莫低头思索了起来，思索自己有什么不懂的问题，对于他来说，虽然不懂的问题很多，但都是一些小问题，不值得此刻提出。
<br/>　　毕竟，他只能提出三个问题，也就是只有三次机会。
