　　“你…”
<br/>　　莫悠闻言，眉头霎时一皱。
<br/>　　“哥，还与这恶贼废话些什么。”莫忧撇撇嘴。
<br/>　　“这恶贼，根本就是在耍你。”
<br/>　　莫悠摇了摇头，“我相信以易兄的威名，不至于耍我。”
<br/>　　“不过。”莫悠直视着萧逸。
<br/>　　“你要舍妹的性命，绝不可能。”
<br/>　　“这样吧…”
<br/>　　锵…一声惊天锵鸣响起。
<br/>　　一把利剑，握于莫悠手中。
<br/>　　“易兄，你我一战。”
<br/>　　“若我胜了，还请易兄卖我个面子，别与舍妹再计较。”
<br/>　　“当然，之前答应易兄的，在下仍旧会做到。”
<br/>　　“赔礼也罢，道歉也好，在下无不应允。”
<br/>　　莫悠很清楚，以易霄今日之声名，被他盯上者，几无人能活命。
<br/>　　邪修尚且低调隐秘，手段诡异，却还是逃不过易霄的追杀。
<br/>　　更何谈他的妹妹？
<br/>　　故此事，他希望能这般解决。
<br/>　　当然，他言语中，只说若他胜了，便如何；却未说半句他败了，便如何。
<br/>　　显然，莫悠自信十足。
<br/>　　别的他不敢说，但年轻一辈天骄妖孽，他未曾怕过谁。
<br/>　　“出手吧。”萧逸言简意赅。
<br/>　　“好，爽快。”莫悠点了点头。
<br/>　　锵…
<br/>　　惊天剑意，霎时从莫悠剑中爆发，随后席卷云霄。
<br/>　　这一剑，仅仅是莫悠仗剑而出，直指萧逸。
<br/>　　这一剑，还未真正劈出，仅仅是剑意酝酿。
<br/>　　嘭…
<br/>　　萧逸身上火焰，竟是瞬间在剑意之下悉数湮灭。
<br/>　　蹬…蹬…蹬…
<br/>　　萧逸一连退了数步。
<br/>　　周遭，受伤倒地的一众学宫长老，脸色一惊，“好惊人的战意。”
<br/>　　“忘忧剑莫悠，竟已有这般实力？”
<br/>　　林星河双眼一眯，“此子的修为，未达我们这个层次。”
<br/>　　“但论剑意锋芒，一身剑道造诣与实力，却已然不容小觑。”
<br/>　　林星河，乃是剑域林家曾经的家主，两千多年前便已成名的绝世剑修。
<br/>　　他的眼光，何等毒辣。
<br/>　　他身怀斩星剑道，一身剑道造诣，何等高深莫测。
<br/>　　连他都面露惊色，赞誉有加，不难想象现今莫悠的剑道水准之强。
<br/>　　嘭…
<br/>　　萧逸身上，火焰再次爆发。
<br/>　　“绝世9994道修为。”
<br/>　　萧逸淡漠地点了点头，一眼便看出了莫悠的修为。
<br/>　　不错，莫悠单单是修为，已达绝世9994道。
<br/>　　论修为，与水凝寒一致。
<br/>　　但，单单刚才那指剑一瞬，那滔天剑意，萧逸就能判断出，莫悠能爆发的实力，绝对远超水凝寒。
<br/>　　“易兄，准备好了吗？”莫悠沉声问着，脸上战意滔天，但同时也脸色凝重。
<br/>　　他虽自信，却不会大意。
<br/>　　易霄的战绩，摆在这。
<br/>　　周遭各学宫长老受伤到底，亦赫然在目。
<br/>　　几乎是之前落到地面之时，莫悠就已然知晓，易霄很强。
<br/>　　如无意外，这将是他遇到过的天骄妖孽中，最强的对手。
<br/>　　“出手吧。”萧逸仍旧淡漠说着。
<br/>　　“好。”莫悠点点头，瞬间出手。
<br/>　　手中忘忧剑，散发惊人黑芒。
<br/>　　黑芒之内，剑影重重。
<br/>　　萧逸凝视着袭来之剑，竟是觉眼眸一花，丝毫分不出何为剑影，何为实剑。
<br/>　　萧逸皱了皱眉，一拳轰出。
<br/>　　拳头，直直轰向剑影。
<br/>　　然而，意料中的轰鸣，并没有出现。
<br/>　　袭来的黑芒剑影，竟是一闪而过。
<br/>　　萧逸轰出的拳头，竟是仿佛轰在了空气中。
<br/>　　此时，莫悠的剑，从萧逸拳下一分，直刺而来。
<br/>　　萧逸反应极快，脚步一退，堪堪躲过。
<br/>　　锋锐的剑尖，堪堪在他脖间划过，脖间紫炎，瞬间溃散。
<br/>　　若他刚才慢一步，未躲开此剑，此刻想必已然重伤。
<br/>　　“厉害，不愧是哥哥。”不远处，莫忧得意大笑。
<br/>　　“什么紫炎易霄，什么双生子，这才一个照面便落败了？”
<br/>　　周遭，各学宫长老，同样面露惊色，“不愧是忘忧剑莫悠。”
<br/>　　“不愧是十数年前便能力压我们十二峰首席的妖孽。”
<br/>　　“百院之争第一名，果真名不虚传。”
<br/>　　林星河满意地点了点头，“这一手幻剑，果真了不起。”
<br/>　　“强如易霄此子，竟也险些抵挡不住…”
<br/>　　然而，林星河的话，还未说完。
<br/>　　萧逸这边。
<br/>　　咔…
<br/>　　萧逸的兽手，竟是瞬间握住了忘忧剑。
<br/>　　忘忧剑上，散发着极致黑芒，锋利到极点，强悍到极点。
<br/>　　可那剑尖、剑身，却是奈何不了兽手分毫。
<br/>　　萧逸的手，仅仅握着。
<br/>　　莫悠下意识抽剑而回，却发现，自己的剑，竟是抽不动丝毫。
<br/>　　“好大的力气。”莫悠眉头一皱。
<br/>　　“硬握我的忘忧剑？厉害。”
<br/>　　“我没兴趣与你浪费时间。”萧逸淡漠地摇了摇头。
<br/>　　咔…
<br/>　　萧逸兽手重重一握。
<br/>　　忘忧剑，瞬间断裂。
<br/>　　“你…”莫悠眉头一皱，嘴角一丝鲜血溢出。
<br/>　　这剑，是他的武魂。
<br/>　　强行被捏断，他自然会受反噬。
<br/>　　不过，能强行捏断他的忘忧剑武魂，绝非等闲实力可以做到。
<br/>　　“卑鄙。”不远处，莫忧冷喝一声。
<br/>　　“易霄，你就只会这等下三滥的手段？”
<br/>　　萧逸未理会，只是直视莫悠，“出你最强一剑吧。”
<br/>　　“如你所愿。”莫悠点了点头。
<br/>　　“圣湮剑。”
<br/>　　莫悠的剑，陡然变了。
<br/>　　剑上，湮灭气息滔天。

<br/>　　“这一剑，怕是我都挡不住。”
<br/>　　“圣湮剑？”萧逸眯了眯眼。
<br/>　　他没记错的话，当年他过了登云道后，副院长与他说，会有大奖励。

<br/>　　那惊人的湮灭气息，同样难以奈何兽手分毫。

<br/>　　“你…”莫悠脸色霎时难看。
<br/>　　咔…嘭…

<br/>　　身影，瞥了眼不远处。
<br/>　　高空中，三道身影，凝视着他。
