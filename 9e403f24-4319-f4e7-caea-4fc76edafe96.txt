　　坐在窗口，一杯烈酒下肚，听着客人们谈天说地，慕清霄剑眉皱了皱。
<br/>　　因为，他居然没听到大厅的客人们，谈论一丝关于兰若寺的事情，反倒是谈论起一些可有可无的事情。
<br/>　　按道理来说，兰若寺这种存在，对于乡镇里的居民来讲，绝对是能谈一下午的事情。
<br/>　　“我们客栈厨子最拿手的招牌菜，卢湖鱼，客观您快尝尝。”
<br/>　　片刻后，店小二端着一锅色香味俱全的鱼肉摆放在桌上。
<br/>　　将酒杯放在桌上，慕清霄拿起筷子，夹起香气扑鼻的鲜嫩鱼肉放入口中，慢慢品尝起来。
<br/>　　“手艺确实不错，赏给你们了。”
<br/>　　赞赏一句厨子的手艺，慕清霄就取出几两银子放在桌上。
<br/>　　“多谢客观，多谢客观。”
<br/>　　拿过白花花的银子，店小二小眼神中充满了激动之色。
<br/>　　这回真是遇见大财主了，将银子揣进兜里，一脸献媚，道：“客观，您有什么需要的，尽管吩咐。”
<br/>　　微微点头，慕清霄道：“那好，我正好有些事情需要了解一下。”
<br/>　　“客观您问，小的一定知无不答。”
<br/>　　“我想知道，关于兰若寺的信息。”
<br/>　　话语刚落，客栈大厅中的议论声瞬间停止，整个环境变得针落可闻，气氛诡异无比。
<br/>　　互相谈论的客人们，纷纷停止口中的话题，转过头来，用复杂且古怪的眼神盯着慕清霄，甚至含有一丝敬畏。
<br/>　　至于店小二，脸上的笑容早已凝固，额头上浮现一层细汗，眼中满是惊恐。
<br/>　　“兰......兰若寺，客观您没看玩笑吧？”
<br/>　　夹起一朵鱼肉放入嘴中，慕清霄自然将客人们的表情尽收眼底，但脸色依旧平淡。
<br/>　　“你觉得，我是在跟你开玩笑吗？”
<br/>　　“不不不，难道您是猎妖师大人吗？”
<br/>　　慕清霄对于店小二口中的猎妖师知之甚少，说的应该就是燕赤霞那一类人吧？
<br/>　　“我并不是你口中的猎妖师，不过也有些自保的手段。”
<br/>　　话语刚落，客厅中的客人们看向慕清霄的眼神也变了，有的不屑，有的好奇，也有的幸灾乐祸。
<br/>　　吃了口鱼肉，慕清霄取出一定金子放在桌上，道：“继续说下去。”
<br/>　　望着桌上的金子，店小二吞咽了口唾沫，强忍着惊恐，道：“客观，您人不错，劝您最好别去兰若寺了，最近那里可是死了不少人。”
<br/>　　“嗯，为什么？”
<br/>　　“小......小的听说，兰若寺里面，有山野精怪，还有......还有专门吃人的女鬼，近一个月里，已经死了不下四五人，死相都非常的凄惨。”
<br/>　　说着说着，店小二本人腿肚子都在打颤。
<br/>　　难怪无人谈及兰若寺，显然，无论是那个时代的人，对于山野精怪，孤魂野鬼这类东西，都充满了惧怕之意。
<br/>　　在他们眼中，神仙是救世的菩萨，而山野精怪，孤魂野鬼则是害人，吃人的可怕存在。
<br/>　　店小二话语刚落，慕清霄思绪万千，兰若寺是千年树妖的地盘，里面确实有山野精怪，孤魂野鬼。
<br/>　　“金子是你的了。”
<br/>　　酒足饭饱后，慕清霄就跟在店小二身后，来到二楼的上等客房，见店小二一副欲言又止的模样，不在意的笑了笑。
<br/>　　无论山野精怪，或者是孤魂野鬼，对他来说，都是一些蝼蚁罢了。
<br/>　　酒店客房干净整洁，外面是一竹栏阳台，晚上能清楚的看见天上的星星与月亮，相对于这个时代，算是非常不错。
<br/>　　站在阳台上，慕清霄向西侧瞥了一眼，在视线的尽头，雾霾笼罩，与周围环境显得格格不入，诡异至极。
<br/>　　听店小二说，整个郭北镇，唯独西侧的黑林常年被雾霾笼罩，他们已经见怪不怪了。
<br/>　　黑林，正是兰若寺所在地点。
<br/>　　随着时间的推移，夕阳西下，霞光满天。
<br/>　　黑林附近，周围空间轻微波动一下，慕清霄的身影就出现在上空。
<br/>　　此时，映入眼中的是一片光线暗淡无比的森林，脚下的是一条残缺破旧的青石小道，四周杂草丛生，皆是荒凉一片。
<br/>　　转身回头看了一眼，此处距离郭北镇有些距离，难怪居民们敢在镇里居住。
<br/>　　望了眼荒凉的丛林，慕清霄沿着破旧的青石小道而上，道路弯弯扭扭，不知通向何处。
<br/>　　夜色沉沉，不知何时，太阳已经下山，整个黑林中都黑黝黝一片，不时刮起阴风，深处则有狼嚎声响起。
<br/>　　若是普通人，恐怕早已吓得面无人色，更别说向树林深处前进。
<br/>　　陡然，一道白影飘过，慕清霄随意瞥了一眼就收回目光，不快不慢的向内部走去。
<br/>　　约莫十分钟左右，慕清霄方才停下脚步，映入眼帘的，是一座破败荒凉的古朴寺庙。
<br/>　　不知经历多少风吹雨打，寺庙残破不堪，门口的匾额上结满了蜘蛛网，但其上龙飞凤舞的三个字，却清晰可见。
<br/>　　“兰若寺！”
<br/>　　兰若寺，是围绕着聂小倩，燕赤霞与宁采臣发生的一些列故事。
<br/>　　这里有山野精怪，也有鬼怪，对于凡人来说，是有进无出之地，至于生为普通人的宁采臣为何没事，那只能说主角光环太强了。
<br/>　　而且，大胡子燕赤霞也起到了一定的作用。
<br/>　　根据剧情中的信息，燕赤霞能御使飞剑，斩杀鬼怪，绝对是筑基修士才有的能力。
<br/>　　天色昏暗，古老的寺庙阴风阵阵，站在倒塌的石碑旁，慕清霄思索片刻，便向其中走去。
<br/>　　寺庙前的石阶残缺，坑坑洼洼，杂草丛生，布满灰尘，不时有毒虫爬过。
<br/>　　兰若寺占地面积不小，以前应该是一所香火不错的寺庙，只是不知为何，现在却残破成这般摸样。
<br/>　　此时，兰若寺中环境清幽寂静，若非这阴森森的气氛，倒不失为一处修炼的好地方。
<br/>　　走进东边的僧舍，将窗户打开，随手一卷，其中的杂木碎屑，满地灰尘都冲出窗外，一时间变得干净许多。
<br/>　　随意找了个木板垫在地上，慕清霄就地盘坐而下，闭目养神，运行心法，开始炼化体内的焚寂。
<br/>　　不多时，僧舍大门打开，一名背负长剑的中年男子走了进来，当看到慕清霄的时候，脚步一顿，眼中满是怪异之色。
