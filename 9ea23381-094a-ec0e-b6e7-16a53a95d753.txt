　　天冥星巫族一脉，五年一次的少司选拔，已经近在眼前了。
<br/>　　这一段时间，天冥圣城和九大卫城，纷纷热闹了起来，很多闭关的巫族天才纷纷出关，很多在外历练的巫族天才，也纷纷返回。
<br/>　　每一次少司选拔，都是天冥星一脉的盛世，亦是巫族年轻一代的盛世。
<br/>　　虽然每次只选拔一名少司，但亿万巫族年轻天才，依旧前仆后继。
<br/>　　只因为成为少司，便能成为天冥星巫族一脉真正的核心，便能彻底的改变命运，修习巫族的禁忌巫术。
<br/>　　而这一次的少司选拔，也出现了不少声望极高的天才，在整个天冥星巫族之内，呼声非常的高，是很多巫族族人看好的对象。
<br/>　　甚至，有好事之人，根据自己的主观判断，以及在族中的声望、名气等等，排出了一个少司候选榜。
<br/>　　这个榜单，排出了一百名族中天赋最强大的天才，根据潜力从第一名排到一百名。
<br/>　　这一百个人，无一不是人中龙凤，天之骄子。
<br/>　　虽然这个榜单只是好事之人闲暇之作，但却是切实排出了天冥星巫族一脉最强的天才。
<br/>　　所有人都知道，这一次的少司选拔，胜出的人必定在这一百人之中。
<br/>　　而巫尘之名，也进入了这少司候选榜之中，荣登第七十八名。
<br/>　　这个名次看似不高，但相比于如同恒河之沙般多的巫族天才而言，已经非常的高了。
<br/>　　这也是巫尘上次连败十几名巫族天才，又杀死了巫魔而造就的名气，不然绝对不可能上榜。
<br/>　　而巧合的是，巫天御居然也名列榜中，甚至排名被苏莫还要高，荣登第五十三名。
<br/>　　……
<br/>　　这一日，苏莫离开了地灵塔，因为少司选拔的日期快要到了，他自然要提前了解一些情况。
<br/>　　当然，他是以巫尘之身离开，吞噬战魂再次进入了巫尘的体内，主导了巫尘的肉身。
<br/>　　至于分身的两种战魂，和他心意相通，自然不会排斥苏莫。
<br/>　　黑耀城中变得有些萧条了起来，因为很多人都去了天冥圣城，那里是少司选拔的地点。
<br/>　　苏莫在黑耀城中，稍一打听，便知道了这次少司选拔的具体情况。
<br/>　　此次少司选拔，将在三日后进行，地点是天冥圣城之中的圣巫广场。
<br/>　　至于具体如何选拔，有什么项目，每一次都不尽相同，只有到时候再看情况了。
<br/>　　粗略的了解了一番，苏莫便回到了他的住处，还有三日时间，也不用着急。
<br/>　　所谓的少司候选榜，他也听闻了，那些都是他强大的劲敌，他只有超越所有人，才能脱颖而出。
<br/>　　这一次的少司选拔，呼声最高的三人，是苏莫重点关注的对象。
<br/>　　这三人分别是排名第一的巫人王，排名第二的巫玛，排名第三的巫幽兰。
<br/>　　这三人无论是实力还是各方面天赋，都是冠绝整个天冥星，每一位都是特殊体质，战魂极度强大。
<br/>　　那排名第一的巫人王，不仅是特殊体质，还拥有强大的双生战魂，天赋逆天到极致。
<br/>　　对于这种天赋的人，苏莫不敢小觑，唯有打起十二分精神。
<br/>　　时间过得很快，三日时间转瞬即过。
<br/>　　一大清早，天刚蒙蒙亮，朝阳还未升起，苏莫便离开了住处，向天冥圣城而去。
<br/>　　黑耀城是天冥圣城的卫城，所以距离天冥圣城非常的近，一路无话，没过多久的时间，苏莫就来到了天冥圣城，直奔圣巫广场。
<br/>　　天冥圣城非常的巨大，堪比天月国之巨，而圣巫广场，同样辽阔无比，方圆两百余里，通体用一种青石的岩石铺就而成。
<br/>　　苏莫到来之时，整个圣巫广场上已经人山人海，喧嚣震天，聚集了不下千万人。
<br/>　　一眼望去，全是黑压压的人头，延绵无际。
<br/>　　“看，那就是巫尘，刚崛起的顶尖天才！”
<br/>　　“听说他杀了巫魔，战力极度强大！”
<br/>　　“荣登了少司候选榜，虽然只是第七十八名，但也是热门人选！”
<br/>　　“此人修炼魔功，听说非常恐怖，在黑耀城拥有同阶无敌的实力！”
<br/>　　苏莫的到来，顿时吸引了不少人的关注，议论声四起，天冥圣城和其它八座卫城的人不认识巫尘，但黑耀城的人却大多都认识他。
<br/>　　毕竟，上次武斗场的战斗，几乎传遍了整个黑耀城。
<br/>　　苏莫打量了一番广场，便在广场的边缘处静立，等待选拔赛开始。
<br/>　　目前，巫族高层还未到来，选拔赛自然也不会开始。
<br/>　　对于这次少司选拔，他还是有很大信心的，毕竟，巫族对于少司的人选，从来都是只看重天赋、潜力和战力，修为还是其次。
<br/>　　当然，巫族的少司选拔，只在武帝境的后辈之中选取，只有武帝境的修为，才有资格参加。
<br/>　　时间缓缓流逝，圣巫广场上的人越来越多，巫天御也已经来到了广场上，被一群人簇拥着，显得身份非凡。
<br/>　　巫天御的目光，透过人群，在广场上扫视，很快便看到了广场边缘的苏莫。
<br/>　　随即，他抬起脚步向苏莫走了过来。
<br/>　　“巫尘，好久不见！”来到苏莫的身前，巫天御向苏莫微微一笑。
<br/>　　“武帝境三重！”苏莫打量了对方一眼，顿时眸光一凝，对方的修为居然提升的如此之快！
<br/>　　“不错，现在你我是同等境界，希望这次少司选拔，我们有一战的机会！”巫天御点了点头，眸中露出强烈的战意。
<br/>　　巫尘在黑耀城同阶无敌？
<br/>　　巫天御可不服，这次少司选拔，他若是能和巫尘一战，定然当着所有人的面，堂堂正正的击败巫尘。
<br/>　　若是没有一战的机会，待选拔结束之后，他也会向巫尘挑战。
<br/>　　“那时候你会后悔！”苏莫面无表情的说道，他也不急着杀对方，只要成为了巫族的少司，他能随意捏死对方。
<br/>　　“后悔？”
<br/>　　巫天御闻言轻笑一声，讥讽道：“虽然我不知道你得了什么奇遇，能短短时间内突然崛起，但你也不过是靠着强大的魔功而已，在我眼中不值一提！”
<br/>　　巫天御没有将巫尘当作劲敌，毕竟，巫尘的战力虽强，但主要是来自于修炼的魔功强大。
<br/>　　随着日后时间的推移，功法的优势会逐渐的减弱。
<br/>　　“你在我眼中亦是如此！”
<br/>　　苏莫不屑一笑，淡淡的道：“你号称天才，拥有境界优势的情况下，却连苍穹世界的一个土著都打不过，我都替你感到害臊！”
<br/>　　“你找死！”巫天御闻听此言，顿时脸色一沉，眸中杀机爆射。
